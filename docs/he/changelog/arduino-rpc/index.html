יומן שינויים של ספריית OpenMV Arduino RPC¶

ספריית OpenMV Arduino RPC מאפשרת ל-Arduino לשלוט ב-OpenMV Cam — ולהזרים ממנה נתונים — באמצעות CAN, I²C, SPI, UART (טורי אסינכרוני) ו-USB. היא יציבה ועמידה בפני תקלות, ומתאוששת אוטומטית מחיבורים רופפים, השחתת חבילות ופסקי זמן. הנה מה שהשתנה בכל גרסה — החדשה ביותר תחילה.
