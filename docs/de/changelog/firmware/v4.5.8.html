v4.5.8¶

v4.5.8 ist ein kleines Release: uint16-Modellausgabe-Unterstützung und eine MicroSpeech-Korrektur. Es bringt keine Breaking-API-Änderungen — v4.5.7-Skripte laufen unverändert.

Auf dieser Seite

  • Höhepunkte

  • Neue Funktionen

  • Fehlerbehebungen

  • Breaking-API-Änderungen

Höhepunkte¶

  • uint16-Modellausgaben — ml.Model passt jetzt auch uint16-ndarrays ab.

  • Keine Breaking-API-Änderungen — neu flashen und Ihre v4.5.7-Skripte laufen unverändert.

Neue Funktionen¶

  • ml.Model — uint16-Unterstützung hinzugefügt, sodass Modell-Ausgabetensoren, die uint16 verwenden, korrekt abgeglichen/behandelt werden.

Fehlerbehebungen¶

  • Behebung von ml.apps MicroSpeech.listen(), dem das return fehlte, sodass das erkannte Ergebnis jetzt tatsächlich an den Aufrufer zurückgegeben wird.

Breaking-API-Änderungen¶

v4.5.8 bringt keine Breaking-Python-API-Änderungen. Skripte, die auf v4.5.7 liefen, laufen unverändert auf v4.5.8 — flashen Sie einfach die Firmware neu.
