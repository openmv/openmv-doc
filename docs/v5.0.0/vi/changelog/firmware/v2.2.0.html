v2.2.0¶

v2.2.0 giới thiệu hỗ trợ AprilTags (image.find_apriltags()), kích hoạt CAN2 trên OpenMV 2 / 3, thêm các module MicroPython bổ sung, khôi phục các hàm gọi lại giả lập Pixy, và căn chỉnh xử lý Ctrl-D/C/A trong REPL theo các bo mạch MicroPython chuẩn. Bản này không có thay đổi API gây lỗi — các tập lệnh v2.1.0 chạy không thay đổi.

Điểm nổi bật¶

  • AprilTags — phát hiện marker fiducial image.find_apriltags().

  • CAN2 — được kích hoạt trên OpenMV 2 và OpenMV 3.

  • Thêm module MicroPython được kích hoạt.

  • REPL — Ctrl-D / Ctrl-C / Ctrl-A nay hoạt động như các bo mạch MicroPython chuẩn.

  • Không có thay đổi API gây lỗi — nạp lại firmware và các tập lệnh v2.1.0 chạy không thay đổi.

Tính năng mới¶

  • AprilTags — thêm tính năng phát hiện marker fiducial AprilTag (image.find_apriltags()), cùng các bản sửa lỗi.

  • CAN2 — kích hoạt CAN2 trên OpenMV 2 / 3 (với khởi tạo/hủy khởi tạo CAN0 trong main).

  • Kích hoạt các module MicroPython bổ sung và khôi phục các hàm gọi lại giả lập Pixy.

Các thay đổi và cải tiến khác¶

  • REPL nay xử lý Ctrl-D / Ctrl-C / Ctrl-A giống các bo mạch MicroPython chuẩn; cập nhật các tập lệnh ví dụ phát hiện khuôn mặt và theo dõi khuôn mặt.

Sửa lỗi¶

  • Sửa lỗi giải mã mã QR, bước cascade Haar, và đèn LED bootloader.

Hỗ trợ phần cứng và bo mạch¶

  • OpenMV 2 / 3 — CAN2 được kích hoạt; thêm cờ lập trình SWD.

Thay đổi API gây lỗi¶

v2.2.0 không có thay đổi Python API gây lỗi. Các tập lệnh chạy trên v2.1.0 sẽ chạy không thay đổi trên v2.2.0 — chỉ cần nạp lại firmware.
