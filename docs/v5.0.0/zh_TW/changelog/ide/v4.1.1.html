v4.1.1¶

v4.1.1 著重於範例選單：範例項目現在會經過篩選以符合所連接的開發板與 sensor，多餘的資料夾會被收合，並在未連接任何裝置時顯示清楚的佔位提示。此版本也在 framebuffer 檢視器中加入即時對焦度量值、更新隨附的範例，並隨附 4.5.1 版開發板 firmware。本版本以 Qt Creator 10.0.1 為基礎建置；本次發行未進行 Qt Creator rebase。範例篩選會改變範例選單的預設行為——請參閱「破壞性變更」。

重點摘要¶

  • 範例篩選只會顯示與您所連接的開發板與 sensor 相關的範例，由 examples/index.csv 設定檔驅動，並可透過 Tools 選單的開關停用。

  • 對焦度量值現在會即時顯示於 framebuffer 檢視器與終端機狀態列中，協助您調整鏡頭對焦。

  • 更簡潔的範例選單：多餘的單一子項開發板資料夾會被收合，並以佔位提示取代空白選單。

  • 已從序列終端機移除軟重開機行，因此主控台不再於每次執行時被雜訊干擾。

  • 隨附的開發板 firmware 在所有支援的 OpenMV 與 Arduino 開發板上更新為 4.5.1。

新功能¶

  • 範例篩選。 Examples 選單現在會篩選項目，只顯示與所連接開發板與 sensor 相關的範例。最初的開發板/sensor 篩選由 filters.txt 設定檔驅動（ef7190282），後來改為使用 examples/index.csv，搭配各開發板與各 sensor 的正規表示式以及一個扁平化欄位（795cc20e9），並支援不分大小寫的開發板資料夾與 sensor 類型比對（3e976cb9a）。新增的 Tools > Filter examples by board and sensor type 開關（預設勾選，狀態會保留）可停用篩選（fd3defe00）。

  • 對焦度量值。 framebuffer 檢視器與終端機狀態列現在會顯示由 JPEG 壓縮 frame 計算而得的即時 Focus 數值，協助您調整鏡頭對焦（5834e3a41）。

其他變更與改進¶

  • 連接後，Examples 選單現在會收合多餘的單一子項開發板資料夾，並顯示清楚的「No examples found for your board」或「Connect to your board first」佔位提示，而非空白選單（795cc20e9）。

  • MicroPython 的 MPY: soft reboot 行現在會從序列終端機輸出中移除，因此不再於每次執行時干擾主控台（b3cf63299）。

  • 隨附範例已重新整理為以開發板命名的資料夾（例如 50-Board-Control），更新了 MIT 授權標頭，且 Image Reader/Writer 範例已簡化為使用原始的 image.ImageIO 視訊串流。記憶體內與磁碟上的「陰影移除」frame 差異範例已移除（1358eeb84、7b32f2814、9d89f1c62）。

錯誤修正¶

  • 強制指定特定 bootloader firmware 檔案時，現在會正確使用所提供的 firmware 路徑，而非總是回復為開發板的預設 firmware.bin（a362c1c6b）。

  • 修正 Windows 上 i.MX (RT1060) 裝置探索因過於嚴格的 RawInputDeviceList 計數檢查而可能失敗的問題（49571962d）。

  • 透過將累加向量加寬為 64 位元，修正大型或明亮 image 造成的 histogram 溢位問題（5aaf923a3）。

  • 修正範例資料夾的排序，使未編號的範例項目順序一致（3cb994433）。

平台與工具支援¶

  • Qt Creator 基礎版本： 10.0.1。

  • 隨附開發板 firmware： 在所有支援的 OpenMV 與 Arduino 開發板上由 4.5.0 更新為 4.5.1——包括 OPENMV2/3/4/4P、OPENMV PT (Pure Thermal)、OPENMV RT1060、ARDUINO GIGA、NICLA VISION、PORTENTA H7、NANO RP2040 CONNECT 與 NANO 33 BLE SENSE——並包含 UVC 與 bootloader 映像（f6a5b12ba）。

  • 新指令碼產生現在會為 HM01B0、HM0360、MT9V0X2 與 MT9V0X4 替換為 grayscale 像素格式（先前的 MT9V034 項目已由 MT9V0X2/MT9V0X4 取代）（ef7190282）。

破壞性變更¶

啟用篩選時，Examples 選單現在需要連接開發板，而這是新的預設行為。在您連接之前，選單會顯示「Connect to your board first to see examples」，連接後則只會顯示符合所連接開發板與 sensor 的範例。可透過新增的 Tools > Filter examples by board and sensor type 開關停用此功能。
