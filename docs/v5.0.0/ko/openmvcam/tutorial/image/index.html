5. 이미지 처리¶

지금까지의 모든 페이지는 픽셀 데이터를 얻을 만큼만 다뤘습니다: 프레임을 캡처하고, 센서를 구성하고, 바이트를 카메라에서 RAM으로 옮기는 것입니다. 이미지 처리 는 방향을 뒤집습니다 – 픽셀이 입력이고, 목표는 애플리케이션이 그로부터 필요로 하는 무엇이든입니다.

image 모듈은 그 도구 모음입니다. 모든 연산은 하나 또는 두 개의 Image 객체를 받아, 변환된 Image – 필터링되거나, 임계값 처리되거나, 왜곡되거나, 다르게 합성된 픽셀 – 또는 결과 객체의 목록 중 하나를 반환합니다: 블롭, 선, 코드, 매치, 통계입니다. 이 두 가지 종류는 조합됩니다. 전형적인 파이프라인은 캡처된 프레임을 정리하고, 그로부터 결과 객체 집합을 추출하고, 그것을 측정값으로 바꾸고, 표시를 위해 주석을 다시 프레임에 그립니다.
