tfp410 — DVI/HDMI-controller¶

De tfp410-module biedt een stuurprogramma voor de TFP410 DVI/HDMI-serializer die wordt gebruikt om een extern DVI/HDMI-display aan te sturen via een 24-bits parallelle LCD-bus.

klasse TFP410 – DVI/HDMI-controller¶

class tfp410.TFP410(*, i2c_addr: int = 0x3F)¶

Initialiseert de TFP410 DVI/HDMI-controllerchip.

i2c_addr is het I2C-adres van de TFP410.

isconnected() → bool¶

Geeft True terug als er een extern display is aangesloten, anders False.

hotplug_callback(callback: Callable[[bool], None] | None) → None¶

Registreert een callback-functie die wordt aangeroepen telkens wanneer de verbindingsstatus van een extern display verandert. De nieuwe verbindingsstatus (bool) wordt als enige argument aan callback doorgegeven.

Geef None door als callback om de hotplug-callback uit te schakelen.

Roep TFP410.isconnected() niet aan terwijl er een callback is geregistreerd.
