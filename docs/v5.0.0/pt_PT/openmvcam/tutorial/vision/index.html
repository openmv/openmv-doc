4. Sensores de Visão¶

Até agora, cada página tratou o OpenMV Cam como um pequeno microcontrolador que, por acaso, tem uma câmara ligada. Os sensores de visão invertem a perspetiva: a câmara é o produto, e todo o resto na placa existe para transferir pixels desta para Python.

O módulo csi é a ponte. Uma única classe (CSI) envolve a longa cadeia de física e silício entre fotões e um buffer. Uma lente foca a cena numa grelha de fotodíodos que convertem fotões em carga; um controlador de exposição / ganho / leitura decide com que brilho cada pixel é reportado; um processador de sinal de imagem (ISP) corrige, efectua debayering, gradua a cor e empacota os pixels num formato escolhido; e um periférico do MCU captura o fluxo resultante para a RAM, pronto para snapshot() devolver como um image.Image.
