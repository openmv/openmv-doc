math — funções matemáticas¶

O módulo math fornece algumas funções matemáticas básicas para trabalhar com números de ponto flutuante.

Nota: Na OpenMV Cam, os números de ponto flutuante têm precisão de 32 bits.

Funções¶

math.acos(x: float) → float¶

Retorna o cosseno inverso de x.

math.acosh(x: float) → float¶

Retorna o cosseno hiperbólico inverso de x.

math.asin(x: float) → float¶

Retorna o seno inverso de x.

math.asinh(x: float) → float¶

Retorna o seno hiperbólico inverso de x.

math.atan(x: float) → float¶

Retorna a tangente inversa de x.

math.atan2(y: float, x: float) → float¶

Retorna o valor principal da tangente inversa de y/x.

math.atanh(x: float) → float¶

Retorna a tangente hiperbólica inversa de x.

math.ceil(x: float) → int¶

Retorna um inteiro, sendo x arredondado em direção ao infinito positivo.

math.copysign(x: float, y: float) → float¶

Retorna x com o sinal de y.

math.cos(x: float) → float¶

Retorna o cosseno de x.

math.cosh(x: float) → float¶

Retorna o cosseno hiperbólico de x.

math.degrees(x: float) → float¶

Retorna os radianos x convertidos para graus.

math.erf(x: float) → float¶

Retorna a função de erro de x.

math.erfc(x: float) → float¶

Retorna a função de erro complementar de x.

math.exp(x: float) → float¶

Retorna a exponencial de x.

math.expm1(x: float) → float¶

Retorna exp(x) - 1.

math.fabs(x: float) → float¶

Retorna o valor absoluto de x.

math.floor(x: float) → int¶

Retorna um inteiro, sendo x arredondado em direção ao infinito negativo.

math.fmod(x: float, y: float) → float¶

Retorna o resto de x/y.

math.frexp(x: float) → Tuple[float, int]¶

Decompõe um número de ponto flutuante em sua mantissa e expoente. O valor retornado é a tupla (m, e) tal que x == m * 2**e exatamente. Se x == 0, então a função retorna (0.0, 0), caso contrário, vale a relação 0.5 <= abs(m) < 1.

math.gamma(x: float) → float¶

Retorna a função gama de x.

math.isfinite(x: float) → bool¶

Retorna True se x for finito.

math.isinf(x: float) → bool¶

Retorna True se x for infinito.

math.isnan(x: float) → bool¶

Retorna True se x não for um número

math.ldexp(x: float, exp: int) → float¶

Retorna x * (2**exp).

math.lgamma(x: float) → float¶

Retorna o logaritmo natural da função gama de x.

math.log(x: float, base: float = e) → float¶

Com um argumento, retorna o logaritmo natural de x.

Com dois argumentos, retorna o logaritmo de x na base fornecida.

math.log10(x: float) → float¶

Retorna o logaritmo de base 10 de x.

math.log2(x: float) → float¶

Retorna o logaritmo de base 2 de x.

math.modf(x: float) → Tuple[float, float]¶

Retorna uma tupla de dois floats, sendo as partes fracionária e inteira de x. Ambos os valores retornados têm o mesmo sinal de x.

math.pow(x: float, y: float) → float¶

Retorna x elevado à potência de y.

math.radians(x: float) → float¶

Retorna os graus x convertidos para radianos.

math.sin(x: float) → float¶

Retorna o seno de x.

math.sinh(x: float) → float¶

Retorna o seno hiperbólico de x.

math.sqrt(x: float) → float¶

Retorna a raiz quadrada de x.

math.tan(x: float) → float¶

Retorna a tangente de x.

math.tanh(x: float) → float¶

Retorna a tangente hiperbólica de x.

math.trunc(x: float) → int¶

Retorna um inteiro, sendo x arredondado em direção a 0.

Constantes¶

math.e: float¶

Base do logaritmo natural.

math.pi: float¶

A razão entre a circunferência de um círculo e seu diâmetro.
