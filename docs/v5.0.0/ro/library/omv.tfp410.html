tfp410 — Controler DVI/HDMI¶

Modulul tfp410 oferă un driver pentru serializatorul DVI/HDMI TFP410 utilizat pentru a acționa un afișaj DVI/HDMI extern printr-o magistrală LCD paralelă pe 24 de biți.

class TFP410 – Controler DVI/HDMI¶

class tfp410.TFP410(*, i2c_addr: int = 0x3F)¶

Inițializează cipul controler DVI/HDMI TFP410.

i2c_addr este adresa I2C a TFP410.

isconnected() → bool¶

Returnează True dacă un afișaj extern este conectat, False în caz contrar.

hotplug_callback(callback: Callable[[bool], None] | None) → None¶

Înregistrează o funcție callback care va fi apelată ori de câte ori se schimbă starea conexiunii unui afișaj extern. Noua stare a conexiunii (bool) este transmisă drept unic argument către callback.

Transmiteți None ca callback pentru a dezactiva funcția de retroapelare la conectare/deconectare.

Cât timp o funcție de retroapelare este înregistrată, nu apelați TFP410.isconnected().
