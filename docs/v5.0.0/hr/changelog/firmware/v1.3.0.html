v1.3.0¶

v1.3.0 je veliko izdanje sa značajkama: uvodi podršku za WiFi štit (upravljački program WINC1500 network.WINC i usocket socket API s alatima za upravljanje ugrađenim programom), novi fir toplinsko-IR modul, postavljače modula sensor za balans bijele / zrcaljenje / okretanje, sensor.skip_frames() te mean / median / mode / midpoint filtre slike. Ne uvodi nikakve promjene API-ja koje narušavaju kompatibilnost — skripte za v1.2.0 rade nepromijenjeno.

Istaknuto¶

  • WiFi štit — WINC1500 network.WINC (scan / connect / ifconfig) i usocket socket API (TCP/UDP, DNS, istek vremena) s alatima za ažuriranje/dump/verziju ugrađenog programa.

  • fir modul — podrška za toplinsko-IR (MLX) senzor.

  • Senzor — set_whitebal() / set_hmirror() / set_vflip() i skip_frames().

  • Filtri — image.mean() / median() / mode() / midpoint().

  • Nema promjena API-ja koje narušavaju kompatibilnost — preflashajte i vaše skripte za v1.2.0 rade nepromijenjeno.

Nove značajke¶

  • WiFi štit (WINC1500) — dodan je upravljački program network.WINC (WiFi scan, connect, ifconfig) i usocket socket API: send() / recv(), bind() / sendto() / recvfrom(), listen() / accept(), settimeout() / blokirajući istek vremena i DNS (s asinkronim povratnim pozivima za socket/DNS), uz alate za ugrađeni program winc.fw_update() / winc.fw_dump() / winc.fw_version() i WiFi primjere skripti (scan, connect, TCP klijent, DNS, NTP, MJPEG streamer).

  • fir — dodan je fir toplinsko-IR modul (init / deinit / read_ta / read_ir / draw_ta / draw_ir, s internim skaliranjem/miješanjem) i primjeri fir.py / fir_lcd.py.

  • Senzor — dodani su sensor.set_whitebal(), sensor.set_hmirror(), sensor.set_vflip() i sensor.skip_frames() (ustaljivanje kamere nakon promjene postavki).

  • Filtri slike — dodani su image.mean(), image.median(), image.mode() i image.midpoint(ksize, bias=).

  • Dodan je prerađeni BLE upravljački program/primjer i alat za generiranje MLX IR tablice.

Ostale promjene i poboljšanja¶

  • Primjeri skripti reorganizirani su u numerirane direktorije tema u Arduino stilu (01-Basics itd.) i dodani su primjeri za save / GIF / MJPEG; u OpenMV IDE dodan je alat za automatski prag mrlje boje „copy color”.

Ispravci grešaka¶

  • Ispravljeni su WINC1500 gethostbyname / DNS redoslijed bajtova IP adrese, verifikacija ugrađenog programa u fw_update() te isporučena ažurirana jedinstvena slika WINC ugrađenog programa.

Hardver i podrška za ploče¶

  • WiFi štit — WINC1500 (firmware 19.4.4).

  • fir — MLX toplinsko-IR senzor.

Promjene API-ja koje narušavaju kompatibilnost¶

v1.3.0 ne uvodi nikakve promjene Python API-ja koje narušavaju kompatibilnost. Skripte koje su radile na v1.2.0 rade nepromijenjeno na v1.3.0 — samo preflashajte ugrađeni program. (WINC1500 WiFi/usocket stog i fir toplinski modul novi su u ovom izdanju; njihov interni API finaliziran je unutar v1.3.0.)
