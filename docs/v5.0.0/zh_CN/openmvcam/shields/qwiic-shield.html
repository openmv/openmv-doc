QWIIC 扩展板¶

QWIIC 扩展板为 OpenMV Cam 提供两个可菊花链连接的 Qwiic（JST-SH 4 针）连接器，它们接到摄像头的 I2C 引脚上，因此 SparkFun Qwiic 生态系统的板卡可以直接插入使用。

完整数据手册、照片以及订购信息请参见 QWIIC 扩展板产品页面。

亮点¶

  • 两个 Qwiic JST-SH 连接器，可菊花链连接 I2C 外设

  • 引出 OpenMV Cam 的 P4（SCL）和 P5（SDA）引脚

  • 兼容 SparkFun Qwiic 生态系统

  • 板载无上拉电阻（请使用软件上拉或外部上拉）

功能

P4

Qwiic SCL——通往两个 Qwiic 连接器的 I²C 时钟

P5

Qwiic SDA——通往两个 Qwiic 连接器的 I²C 数据

3.3V 电源轨

为连接的 Qwiic 设备供电

GND 电源轨

公共接地（也接到两个 Qwiic 连接器）

用法¶

扫描 I²C 总线上已连接的 Qwiic 设备。该扩展板板载未安装上拉电阻——请在软件中启用内部上拉，或依靠所连 Qwiic 板卡内置的上拉:

from machine import Pin, SoftI2C

bus = SoftI2C(scl=Pin("P4", Pin.OPEN_DRAIN, Pin.PULL_UP),
              sda=Pin("P5", Pin.OPEN_DRAIN, Pin.PULL_UP),
              freq=100_000)

print("Qwiic devices:", [hex(addr) for addr in bus.scan()])
