hts221 — czujnik wilgotności/temperatury HTS221¶

Sterownik dla pojemnościowego czujnika wilgotności względnej i temperatury ST HTS221 podłączanego przez I2C. Sterownik odczytuje przy tworzeniu obiektu fabryczne wartości kalibracyjne z pamięci nieulotnej i wykorzystuje je do przeliczania surowych odczytów rejestrów na skalibrowane wartości.

Przykład:

import time
from machine import Pin, I2C
from hts221 import HTS221

bus = I2C(1, scl=Pin("P15"), sda=Pin("P14"))
hts = HTS221(bus)

while True:
    print("rH: %.2f%% T: %.2fC" % (hts.humidity(), hts.temperature()))
    time.sleep_ms(100)

Klasy¶

class hts221.HTS221(bus: machine.I2C, data_rate: int = 1, address: int = 0x5F)¶

Tworzy instancję HTS221.

bus

Skonfigurowana magistrala machine.I2C, do której podłączony jest czujnik.

data_rate

Selektor szybkości danych wyjściowych zapisywany w polu ODR rejestru CTRL_REG1. Dozwolone wartości to 0 (pojedynczy pomiar), 1 (1 Hz), 2 (7 Hz) i 3 (12,5 Hz).

address

7-bitowy adres I2C urządzenia. Domyślnie 0x5F - adres fabryczny używany przez wszystkie układy HTS221.

humidity() → float¶

Odczytuje rejestry wyjściowe wilgotności i zwraca wilgotność względną w procentach (%RH), już skompensowaną z użyciem wartości kalibracyjnych zapisanych w urządzeniu.

temperature() → float¶

Odczytuje rejestry wyjściowe temperatury i zwraca temperaturę w stopniach Celsjusza, już skompensowaną z użyciem wartości kalibracyjnych zapisanych w urządzeniu.
