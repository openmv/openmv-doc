v1.0.3¶

v1.0.3 to pierwsze wydanie pakietu OpenMV Python — biblioteki Python działającej po stronie hosta oraz narzędzia wiersza poleceń openmv służącego do komunikacji z OpenMV Cam z Twojego komputera za pośrednictwem Camera Protocol V2. Jako pierwsze wydanie nie ma niczego, z czego trzeba by migrować; ten wpis podsumowuje, z czym zostało dostarczone.

Najważniejsze zmiany¶

  • CLI ``openmv`` — łączy się z kamerą przez jej port szeregowy USB i strumieniuje jej obraz na żywo na Twój komputer.

  • Uruchamianie skryptów na kamerze — wysyła skrypt .py do kamery i uruchamia go z wiersza poleceń.

  • Biblioteka Camera Protocol V2 — implementacja protokołu w Pythonie do skryptowego sterowania kamerą po stronie hosta.

  • Instalowalny przez pip — pip install openmv.

Nowe funkcje¶

Narzędzie wiersza poleceń openmv:

  • Strumieniowanie obrazu na żywo z podłączonej kamery (--port), z regulowaną skalą podglądu (--scale).

  • Uruchamianie skryptu na kamerze (--script).

  • Pomiar przepustowości protokołu (--bench).

  • Wczytywanie symboli ELF oprogramowania układowego, aby profiler pokazywał nazwy funkcji (--firmware).

  • Tryby cichy (--quiet) i szczegółowego debugowania (--debug).

Zmiany łamiące API¶

Brak. v1.0.3 to pierwsze wydanie — nie ma wcześniejszej wersji, z którą można by złamać zgodność.
