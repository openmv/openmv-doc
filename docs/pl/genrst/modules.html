Parametry wyłącznie pozycyjne¶

Aby zaoszczędzić miejsce w kodzie, wiele funkcji, które akceptują argumenty słów kluczowych w CPython, akceptuje tylko argumenty pozycyjne w MicroPython.

MicroPython oznacza parametry wyłącznie pozycyjne w ten sam sposób co CPython, wstawiając / na końcu parametrów pozycyjnych. Każda funkcja, której sygnatura kończy się na /, przyjmuje wyłącznie argumenty pozycyjne. Więcej szczegółów znajdziesz w PEP 570.

Przykład¶

Na przykład w CPython 3.4 sygnatura konstruktora socket.socket wygląda następująco:

socket.socket(family=AF_INET, type=SOCK_STREAM, proto=0, fileno=None)

Jednak sygnatura udokumentowana w MicroPython to:

socket(af=AF_INET, type=SOCK_STREAM, proto=IPPROTO_TCP, /)

Znak / na końcu parametrów wskazuje, że wszystkie są wyłącznie pozycyjne w MicroPython. Poniższy kod działa w CPython, ale nie w większości portów MicroPython:

import socket
s = socket.socket(type=socket.SOCK_DGRAM)

MicroPython zgłosi wyjątek:

TypeError: function doesn't take keyword arguments

Poniższy kod będzie działał zarówno w CPython, jak i w MicroPython:

import socket
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

array¶

Porównanie między różnymi kodami typów nie jest obsługiwane¶

Przyczyna: Rozmiar kodu

Obejście: Porównuj poszczególne elementy

Przykładowy kod:

import array

array.array("b", [1, 2]) == array.array("i", [1, 2])

CPython output:

MicroPython output:

 
Traceback (most recent call last):
  File "<stdin>", line 10, in <module>
NotImplementedError:

Sprawdzanie przepełnienia nie jest zaimplementowane¶

Przyczyna: MicroPython implementuje niejawne obcinanie w celu zmniejszenia rozmiaru kodu i czasu wykonania

Obejście: Jeśli wymagana jest zgodność z CPython, jawnie zamaskuj wartość

Przykładowy kod:

import array

a = array.array("b", [257])
print(a)

CPython output:

MicroPython output:

 
Traceback (most recent call last):
  File "<stdin>", line 10, in <module>
OverflowError: signed char is greater than maximum

array('b', [1])

Wyszukiwanie liczby całkowitej nie jest zaimplementowane¶

Przykładowy kod:

import array

print(1 in array.array("B", b"12"))

False

  File "<stdin>", line 10, in <module>

Usuwanie elementów tablicy nie jest zaimplementowane¶

Przykładowy kod:

import array

Indeksowanie z krokiem != 1 nie jest jeszcze zaimplementowane¶

Przykładowy kod:

errno¶

MicroPython does not include ENOTSUP as a name for errno 95.¶

Cause: MicroPython does not implement the ENOTSUP canonical alias for EOPNOTSUPP added in CPython 3.2.

Workaround: Use errno.EOPNOTSUPP in place of errno.ENOTSUP.

Przykładowy kod:

import errno

print(f"{errno.errorcode[errno.EOPNOTSUPP]=!s}")

CPython output:

MicroPython output:

 
errno.errorcode[errno.EOPNOTSUPP]=ENOTSUP

errno.errorcode[errno.EOPNOTSUPP]=EOPNOTSUPP

import json

    print("TypeError")

MicroPython output:

 
TypeError

"shouldn't be able to serialise bytes"

os¶

Atrybut environ nie jest zaimplementowany¶

Obejście: Używaj getenv, putenv i unsetenv

Przykładowy kod:

import os

try:
    print(os.environ.get("NEW_VARIABLE"))
    os.environ["NEW_VARIABLE"] = "VALUE"
    print(os.environ["NEW_VARIABLE"])
except AttributeError:
    print("should not get here")
    print(os.getenv("NEW_VARIABLE"))
    os.putenv("NEW_VARIABLE", "VALUE")
    print(os.getenv("NEW_VARIABLE"))

CPython output:

MicroPython output:

 
None
VALUE

should not get here
None
VALUE

os.putenv("NEW_VARIABLE", "VALUE")

random¶

Metoda getrandbits może zwrócić maksymalnie 32 bity naraz.¶

Przyczyna: Wewnętrzny stan PRNG ma tylko 32 bity, więc może zwrócić maksymalnie 32 bity danych naraz.

Obejście: Jeśli potrzebujesz liczby o więcej niż 32 bitach, skorzystaj z modułu random z micropython-lib.

Przykładowy kod:

import random


x = random.getrandbits(64)
print("{}".format(x))

CPython output:

MicroPython output:

 
3426663832757540188

Traceback (most recent call last):
  File "<stdin>", line 11, in <module>
ValueError: bits must be 32 or less

Metoda randint może zwrócić tylko liczbę całkowitą nie większą niż natywny rozmiar słowa.¶

Przyczyna: PRNG jest w stanie generować tylko 32 bity stanu naraz. Wynik jest następnie rzutowany na int o natywnym rozmiarze zamiast na pełny obiekt int.

Obejście: Jeśli potrzebujesz liczb całkowitych większych niż natywny rozmiar słowa, użyj modułu random z micropython-lib.

Przykładowy kod:

import random


x = random.randint(2**128 - 1, 2**128)
print("x={}".format(x))

CPython output:

MicroPython output:

 
x=340282366920938463463374607431768211455

Traceback (most recent call last):
  File "<stdin>", line 11, in <module>
OverflowError: overflow converting long int to machine word

struct¶

Struct pack with too few args, not checked by MicroPython¶

Przykładowy kod:

import struct

try:
    print(struct.pack("bb", 1))
    print("Should not get here")
except:
    print("struct.error")

CPython output:

MicroPython output:

 
struct.error

b'\x01\x00'
Should not get here

Struct pack with too many args, not checked by MicroPython¶

Przykładowy kod:

import struct

try:
    print(struct.pack("bb", 1, 2, 3))
    print("Should not get here")
except:
    print("struct.error")

CPython output:

MicroPython output:

 
struct.error

b'\x01\x02'
Should not get here

Struct pack with whitespace in format, whitespace ignored by CPython, error on MicroPython¶

Przyczyna: MicroPython jest zoptymalizowany pod kątem rozmiaru kodu.

Obejście: Nie używaj spacji w ciągach formatujących.

Przykładowy kod:

import struct

try:
    print(struct.pack("b b", 1, 2))
    print("Should have worked")
except:
    print("struct.error")

CPython output:

MicroPython output:

 
b'\x01\x02'
Should have worked

struct.error

sys¶

Nadpisywanie sys.stdin, sys.stdout i sys.stderr nie jest możliwe¶

Przyczyna: Są przechowywane w pamięci tylko do odczytu.

Przykładowy kod:

import sys

sys.stdin = None
print(sys.stdin)

CPython output:

MicroPython output:

 
None

Traceback (most recent call last):
  File "<stdin>", line 10, in <module>
AttributeError: 'module' object has no attribute 'stdin'
