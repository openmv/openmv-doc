v3.4.3¶

v3.4.3 is een kleine fix-release: deze verhelpt een lange vertraging wanneer JPEG-encoding overflowt en keert netjes terug wanneer HAL_JPEG_Encode wordt afgebroken. Er zijn geen breaking API-wijzigingen — v3.4.2-scripts draaien ongewijzigd.

Hoogtepunten¶

  • JPEG-fix — geen lange vertraging meer wanneer JPEG-encoding overflowt.

  • Geen breaking API-wijzigingen — flash opnieuw en je v3.4.2-scripts draaien ongewijzigd.

Bugfixes¶

  • Een lange vertraging wanneer JPEG-encoding overflowt verholpen en keert nu terug wanneer HAL_JPEG_Encode wordt afgebroken.

Breaking API-wijzigingen¶

v3.4.3 introduceert geen breaking Python API-wijzigingen. Scripts die op v3.4.2 draaiden, draaien ongewijzigd op v3.4.3 — flash gewoon de firmware opnieuw.
