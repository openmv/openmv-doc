v1.0.5
======

v1.0.5 adds a small CLI feature.

Changes
-------

- The ``openmv`` CLI can now read and preview a **custom data channel** with
  ``--channel`` — for example, previewing a named channel such as ``ticks``
  that a script publishes alongside its video.

Breaking API changes
--------------------

None.
