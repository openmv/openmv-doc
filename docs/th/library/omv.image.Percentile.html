class Percentile -- Percentile Object¶

อ็อบเจกต์ percentile คือ attrtuple ที่คืนค่าโดย histogram.get_percentile() รายงานค่าถังที่ตำแหน่ง CDF ที่ผู้ใช้กำหนด (เช่น 0.05 สำหรับเปอร์เซ็นไทล์ที่ 5, 0.5 สำหรับค่ามัธยฐาน, 0.95 สำหรับเปอร์เซ็นไทล์ที่ 95) สำหรับแต่ละช่องของ Histogram ที่อยู่เบื้องล่าง

สำหรับฮิสโตแกรมระดับสีเทา ใช้ value (อีกสามฟิลด์จะเป็น 0) สำหรับฮิสโตแกรม RGB565, l_value / a_value / b_value ให้เปอร์เซ็นไทล์สำหรับช่อง L, A, และ B ของ LAB ตามลำดับ

สามารถเข้าถึงฟิลด์โดยชื่อแอตทริบิวต์ (percentile.value) หรือดัชนี (percentile[0]) อ็อบเจกต์นี้ไม่มีคอนสตรักเตอร์สาธารณะ

class image.percentile¶

กรุณาเรียก histogram.get_percentile() เพื่อสร้างอ็อบเจกต์นี้

value¶

ค่าถังระดับสีเทาที่เปอร์เซ็นไทล์ที่ต้องการ จำนวนเต็ม 0 -- 255 ดัชนี [0]

l_value¶

ค่าถังช่อง L ของ LAB ที่เปอร์เซ็นไทล์ที่ต้องการ จำนวนเต็ม 0 -- 100 ดัชนี [1]

a_value¶

ค่าถังช่อง A ของ LAB ที่เปอร์เซ็นไทล์ที่ต้องการ จำนวนเต็ม -128 -- 127 ดัชนี [2]

b_value¶

ค่าถังช่อง B ของ LAB ที่เปอร์เซ็นไทล์ที่ต้องการ จำนวนเต็ม -128 -- 127 ดัชนี [3]
