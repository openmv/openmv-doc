Changelog de OpenMV¶

Cada producto de OpenMV mantiene su propio historial de versiones. Elige el que te interese: cada entrada describe qué hay de nuevo, qué cambió y (cuando importa) exactamente cómo migrar.

OpenMV Firmware

El software de visión artificial MicroPython que se ejecuta en la OpenMV Cam. Nuevas funciones de cámara, procesamiento de imagen y soporte de placas, más cambios de API incompatibles y pasos de migración para tus scripts.

Ver versiones →

OpenMV IDE

El entorno de desarrollo de escritorio para escribir, ejecutar y depurar scripts en la OpenMV Cam. Nuevas funciones de editor, cámara y herramientas, correcciones de errores y soporte de plataforma.

Ver versiones →

OpenMV Python Package

La biblioteca Python del lado del host y la herramienta de línea de comandos openmv para comunicarse con una OpenMV Cam desde tu computadora a través del Camera Protocol V2 — transmisión, scripting y benchmarking.

Ver versiones →

OpenMV Arduino RPC Library

La biblioteca Arduino para controlar una OpenMV Cam por CAN, I²C, SPI, UART y USB. Nuevo soporte de placas, interfaces de bus y correcciones de fiabilidad.

Ver versiones →
