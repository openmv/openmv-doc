v4.4.9
======

v4.4.9 is a maintenance release that bumps the OpenMV IDE version. It remains
based on Qt Creator 14.0.2 with no rebase, and there are no breaking changes.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Maintenance release.** Version bump to 4.4.9 with no user-facing feature
  changes (`5de4aa347 <https://github.com/openmv/qt-creator/commit/5de4aa347>`__).

Platform and tool support
-------------------------

- **Qt Creator base:** 14.0.2.

Breaking changes
----------------

None. This release introduces no workflow or behavior changes; existing
projects continue to work unchanged.
