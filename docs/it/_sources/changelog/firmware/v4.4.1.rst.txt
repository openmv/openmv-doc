v4.4.1
======

v4.4.1 is a small release: QSPI flash on the OpenMV Cam H7 Pro and a camera
DCMI pin de-init fix. It introduces **no breaking API changes** — v4.4.0
scripts run unchanged.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **OpenMV Cam H7 Pro** — QSPI flash enabled (more on-board storage).
- **No breaking API changes** — reflash and your v4.4.0 scripts run
  unchanged.

Bug fixes
---------

- Fixed STM32 camera DCMI pin de-initialization, improving sensor
  re-init/de-init reliability.

Hardware and board support
--------------------------

- **OpenMV Cam H7 Pro** — QSPI flash enabled.

Breaking API changes
--------------------

v4.4.1 introduces **no breaking Python API changes**. Scripts that ran on
v4.4.0 run unchanged on v4.4.1 — just reflash the firmware.
