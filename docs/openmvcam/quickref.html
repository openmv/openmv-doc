Quick reference for the OpenMV Cam

Each OpenMV camera has its own pinout, set of on‑board peripherals (sensor, IMU, microphone, networking, etc.) and supported drivers. The pages below cover one board at a time — pick yours for the pinout, hardware control, bus mappings, and the list of board‑specific libraries.

If this is your first time working with an OpenMV Cam, start with:
