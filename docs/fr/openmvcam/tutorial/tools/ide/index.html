13.1. OpenMV IDE¶

OpenMV IDE est l’application de bureau pour travailler avec votre OpenMV Cam. Tout y sert une seule boucle : éditer un script MicroPython, l’exécuter sur la caméra, et regarder le tampon d’image et la sortie série revenir en direct pendant son exécution. L’IDE est construit sur le cœur d’éditeur Qt Creator, et toute l’application est façonnée autour de cette boucle – un script ne nécessite aucun fichier de projet, configuration de compilation ni installation de compilateur. Vous écrivez un script et l’exécutez sur une caméra ; c’est tout le modèle.

La fenêtre principale comporte quatre zones de travail. L”éditeur occupe le côté gauche et contient vos scripts ouverts, avec le terminal série en dessous affichant tout ce que le script imprime. La visionneuse du tampon d’image se trouve en haut à droite et montre ce que la caméra a capturé en dernier, et l”histogramme en dessous trace les statistiques de pixels de ce qu’affiche le tampon d’image. Une colonne de boutons de barre d’outils tout à gauche contient les commandes de fichier, d’édition, de connexion et d’exécution, et une barre d’état le long du bord inférieur indique ce que fait la caméra connectée. Vous pouvez réduire n’importe quel volet lorsque le script a besoin de place.

Les raccourcis clavier de ce chapitre sont écrits pour Windows et Linux ; sur macOS, remplacez Ctrl par Cmd.
