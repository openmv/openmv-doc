classe Statistics – objet Statistics¶

L’objet statistics est un attrtuple renvoyé par histogram.get_statistics() (et ses alias histogram.get_stats() / histogram.statistics()) ou directement par Image.get_statistics(). Il résume la distribution des classes de l”Histogram sous-jacent avec huit réductions par canal : moyenne, médiane, mode, écart-type, min, max, quartile inférieur et quartile supérieur.

Pour les données en niveaux de gris, seuls les champs sans préfixe (mean, median, mode, stdev, min, max, lq, uq) portent des valeurs utiles, et ils sont réutilisés comme réductions du canal LAB L pour les données RGB565 – les champs l_* reflètent donc les champs sans préfixe et sont fournis par souci de symétrie. a_* et b_* ne portent des valeurs utiles que pour les histogrammes RGB565.

Les champs sont accessibles par nom d’attribut (statistics.mean) ou par index (statistics[0]). L’objet n’a pas de constructeur public.

class image.statistics¶

Veuillez appeler histogram.get_statistics() ou Image.get_statistics() pour créer cet objet.

Niveaux de gris / LAB L (également exposé en tant que l_* ci-dessous)¶

mean¶

Moyenne en niveaux de gris de l’histogramme. Entier 0 – 255. Index [0].

median¶

Médiane en niveaux de gris de l’histogramme. Entier 0 – 255. Index [1].

mode¶

Mode en niveaux de gris de l’histogramme. Entier 0 – 255. Index [2].

stdev¶

Écart-type en niveaux de gris de l’histogramme. Entier 0 – 255. Index [3].

min¶

Minimum en niveaux de gris de l’histogramme. Entier 0 – 255. Index [4].

max¶

Maximum en niveaux de gris de l’histogramme. Entier 0 – 255. Index [5].

lq¶

Quartile inférieur (25e percentile) en niveaux de gris de l’histogramme. Entier 0 – 255. Index [6].

uq¶

Quartile supérieur (75e percentile) en niveaux de gris de l’histogramme. Entier 0 – 255. Index [7].
