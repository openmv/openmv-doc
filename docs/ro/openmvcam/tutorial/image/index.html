5. Procesarea imaginilor¶

Fiecare pagină de până acum a manipulat date de pixeli doar suficient cât să le obțină: capturarea unui cadru, configurarea unui senzor, mutarea octeților din cameră în RAM. Procesarea imaginilor inversează direcția – pixelii sunt intrarea, iar scopul este orice are nevoie aplicația de la ei.

Modulul image este trusa de unelte. Fiecare operație preia unul sau două obiecte Image și returnează fie un Image transformat – pixeli filtrați, supuși pragului, deformați sau compuși diferit – fie o listă de obiecte de rezultat: blob-uri, linii, coduri, potriviri, statistici. Cele două variante se compun. Un pipeline tipic curăță cadrul capturat, extrage din el un set de obiecte de rezultat, le transformă în măsurători și desenează adnotări înapoi pe cadru pentru afișare.
