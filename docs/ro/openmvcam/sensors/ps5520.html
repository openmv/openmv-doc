Cameră PS5520 5MP HDR¶

PS5520 este un senzor HDR de 5MP cu fluxurile FinePixel și HDR de la PixArt, conceput pentru scene cu gamă dinamică dificilă.

Pentru fișa tehnică completă, fotografii și comenzi, consultați pagina de produs Cameră PS5520 5MP HDR.

Notă

Acceptat doar pe OpenMV N6.

Aspecte principale¶

  • 5MP (2592x1944) la 30 FPS în modul HDR

  • HDR cu o singură captură, gamă dinamică de 85 dB, SNR de 39 dB

Utilizare¶

Pe N6 rata de cadre este setată de codificatorul MJPEG VENC.

Transmiteți un flux video color HDR de 30 FPS la 1280x720 (HD) în RGB565:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.HD)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

Transmiteți un flux video color HDR de 15 FPS la 1920x1080 (FHD) în RGB565:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.FHD)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

Transmiteți un flux video color HDR de 5 FPS la 5MP complet — 2592x1944 (WQXGA2) — în RGB565:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.WQXGA2)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())
