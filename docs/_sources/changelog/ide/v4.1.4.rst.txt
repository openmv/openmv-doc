v4.1.4
======

v4.1.4 is a maintenance release built on Qt Creator 10.0.1, unchanged from the
previous release base. It carries an IDE version bump and a single
simplified-Chinese translation correction, with no functional or behavioral
changes. There are no breaking changes, and existing projects are unaffected.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Maintenance release** — an IDE version bump and a simplified-Chinese
  translation fix on Qt Creator 10.0.1, with no functional changes
  (`a2b5e627d <https://github.com/openmv/qt-creator/commit/a2b5e627d>`__,
  `86d59d69b <https://github.com/openmv/qt-creator/commit/86d59d69b>`__).

Other changes and improvements
------------------------------

- **Chinese translation fix** — corrected the simplified-Chinese rendering of
  the ``Preferences...`` menu entry
  (`86d59d69b <https://github.com/openmv/qt-creator/commit/86d59d69b>`__).

Platform and tool support
-------------------------

- **Qt Creator base:** 10.0.1.

Breaking changes
----------------

None. Existing projects, scripts, and workflows are unaffected.
