v4.0.5
======

v4.0.5 is a maintenance release of OpenMV IDE built on Qt Creator 10.0.1, the
same base as the previous release. It refreshes the bundled example scripts so
new projects start from the latest, up-to-date code. There are no breaking
changes in this release.

.. contents:: On this page
   :local:
   :depth: 1

Highlights
----------

- **Refreshed bundled examples** — the example scripts shipped with the IDE
  were updated across the camera, machine-learning, and other categories so
  new sketches open with current, maintained code
  (`51797ced1 <https://github.com/openmv/qt-creator/commit/51797ced1>`__).

Other changes and improvements
------------------------------

- Bumped the IDE version to v4.0.5
  (`8634634a7 <https://github.com/openmv/qt-creator/commit/8634634a7>`__).

Platform and tool support
-------------------------

- **Qt Creator base:** 10.0.1.

Breaking changes
----------------

None. Existing projects, workflows, and settings continue to work unchanged;
only the bundled example scripts were updated.
