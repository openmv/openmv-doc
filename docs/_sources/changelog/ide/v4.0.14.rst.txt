v4.0.14
=======

v4.0.14 is a maintenance release built on Qt Creator 10.0.1. It carries
internal updates only, with no user-facing feature, behavior, or workflow
changes, and no breaking changes.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Maintenance release** with internal updates only; no user-facing
  changes.

Platform and tool support
-------------------------

- **Qt Creator base:** 10.0.1.

Breaking changes
----------------

None. This release contains no user-visible workflow or behavior changes,
and existing projects continue to work without modification.
