v1.7.1
======

v1.7.1 is a maintenance release built on Qt Creator 4.0.2 that corrects a
version misfire by bumping the OpenMV IDE version and compatibility version
from 1.7.0 to 1.7.1. It shares the same Qt Creator base as the previous
release, so existing user projects are unaffected. There are no user-facing
feature or behavior changes, and no breaking changes.

.. contents:: On this page
   :local:
   :depth: 1

Highlights
----------

- **Version bump** that updates OpenMV IDE to 1.7.1 with no functional
  changes (`3316d25f8 <https://github.com/openmv/qt-creator/commit/3316d25f8>`__).

Platform and tool support
-------------------------

- **Qt Creator base:** 4.0.2.

Breaking changes
----------------

None. This release is fully compatible with existing OpenMV IDE projects and
workflows.
