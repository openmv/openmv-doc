v4.1.5
======

v4.1.5 is a maintenance release that bundles OpenMV firmware v4.5.2 for all
supported boards. It remains based on Qt Creator 10.0.1, with no rebase in this
range, and there are no breaking changes.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Bundled firmware updated to v4.5.2** for every supported board.

Platform and tool support
-------------------------

- **Qt Creator base:** 10.0.1.
- **Bundled OpenMV firmware:** updated from v4.5.1 to v4.5.2 for all supported
  boards, including OpenMV2/3/4/4P/PT, OpenMV RT1060, and the Arduino GIGA,
  Nano RP2040 Connect, Nicla Vision, Portenta H7, and Nano 33 BLE Sense
  (`ea8f7732f <https://github.com/openmv/qt-creator/commit/ea8f7732f>`__).

Breaking changes
----------------

None. This release only updates the bundled firmware and does not change any
IDE workflow or behavior.
