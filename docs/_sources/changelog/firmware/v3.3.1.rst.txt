v3.3.1
======

v3.3.1 is a small release: it adds a **BNO055** library for the IMU shield and
fixes the H7 JPEG encoder boundaries and the CMSIS-NN enum sizes. It
introduces **no breaking API changes** — v3.3.0 scripts run unchanged.

.. contents:: On this page
   :local:
   :depth: 1

Highlights
----------

- **BNO055 library** for the IMU shield.
- **Fixes** — H7 JPEG encoder boundaries and CMSIS-NN enum sizes.
- **No breaking API changes** — reflash and your v3.3.0 scripts run
  unchanged.

New features
------------

- Added a :mod:`bno055` library for the IMU shield.

Bug fixes
---------

- Fixed the H7 JPEG encoder boundaries and the CMSIS-NN enum sizes.

Breaking API changes
--------------------

v3.3.1 introduces **no breaking Python API changes**. Scripts that ran on
v3.3.0 run unchanged on v3.3.1 — just reflash the firmware.
