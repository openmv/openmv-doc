v4.1.4
======

v4.1.4 is a maintenance release that updates the bundled MicroPython
submodule. It introduces **no breaking API changes** — v4.1.3 scripts run
unchanged.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **MicroPython submodule updated.**
- **No breaking API changes** — reflash and your v4.1.3 scripts run
  unchanged.

Other changes and improvements
------------------------------

- The bundled MicroPython submodule was updated (upstream fixes and
  improvements).

Breaking API changes
--------------------

v4.1.4 introduces **no breaking Python API changes**. Scripts that ran on
v4.1.3 run unchanged on v4.1.4 — just reflash the firmware.
