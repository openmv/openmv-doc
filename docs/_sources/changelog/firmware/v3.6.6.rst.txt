v3.6.6
======

v3.6.6 is a tiny release: it adds facial-recognition example scripts and
reverts the "reset sensor state in init0" change from v3.6.5 (restoring the
earlier sensor-init behavior). It introduces **no breaking API changes** —
v3.6.5 scripts run unchanged.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Facial-recognition examples** added.
- Reverted the v3.6.5 "reset sensor state in init0" change.
- **No breaking API changes** — reflash and your v3.6.5 scripts run
  unchanged.

New features
------------

- Added facial-recognition example scripts.

Other changes and improvements
------------------------------

- Reverted "Reset sensor state in init0" (from v3.6.5), restoring the prior
  sensor-initialization behavior.

Breaking API changes
--------------------

v3.6.6 introduces **no breaking Python API changes**. Scripts that ran on
v3.6.5 run unchanged on v3.6.6 — just reflash the firmware.
