v1.0.6
======

v1.0.6 brings several CLI and protocol improvements.

Changes
-------

- Updated the camera protocol implementation.
- Added a ``--raw`` option to the ``openmv`` CLI.
- Fixed a keyword-argument bug in the camera API.
- Fixed flickering when profile data is rendered in the CLI.

Breaking API changes
--------------------

None.
