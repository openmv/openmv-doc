v3.4.3
======

v3.4.3 is a small fix release: it fixes a long delay when JPEG encoding
overflows and returns cleanly when ``HAL_JPEG_Encode`` is aborted. It
introduces **no breaking API changes** — v3.4.2 scripts run unchanged.

.. contents:: On this page
   :local:
   :depth: 1

Highlights
----------

- **JPEG fix** — no more long delay when JPEG encoding overflows.
- **No breaking API changes** — reflash and your v3.4.2 scripts run
  unchanged.

Bug fixes
---------

- Fixed a long delay when JPEG encoding overflows and now returns when
  ``HAL_JPEG_Encode`` is aborted.

Breaking API changes
--------------------

v3.4.3 introduces **no breaking Python API changes**. Scripts that ran on
v3.4.2 run unchanged on v3.4.3 — just reflash the firmware.
