OV5640 FPC Camera Module
========================

The OV5640 FPC Camera Module is a compact 5 MP rolling-shutter colour sensor with autofocus. It connects to any modular OpenMV Cam base board through the board-to-board connector for low-profile builds.

.. image:: ../ov5640-hero.jpg
    :alt: OV5640
    :width: 400px
    :align: center

For full datasheet, photos, and ordering see the
`OV5640 product page <https://openmv.io/products/ov5640-fpc-camera-module>`_.

Highlights
----------

* 5 MP (2592x1944) rolling-shutter sensor
* Output formats: RGB565, Grayscale, JPEG, YUV422, Bayer
* Autofocus, F2.0 aperture, variable focal length
* Up to 120 FPS depending on resolution
* Compatible with all modular OpenMV Cam base boards
