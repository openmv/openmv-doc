Python Overview
===============

The OpenMV Cam runs MicroPython, the same language as desktop Python
with a smaller standard library and a few extra modules for talking
to hardware. Everything written for the camera -- from a two-line
script that blinks the LED to a full machine-vision application --
is Python code.
