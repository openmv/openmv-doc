14.4. Bekerja dengan sertifikat TLS¶

Modul ssl pada OpenMV Cam (dibangun di atas mbedTLS) memungkinkan skrip membuat koneksi jaringan terenkripsi dan terautentikasi. Untuk melakukan apa pun di luar enkripsi biasa, Anda memerlukan sertifikat. Halaman-halaman di bagian ini mencakup apa itu sertifikat, jenis kunci dan format file apa yang diterima kamera, cara membuat sertifikat self-signed untuk pengembangan dan mendapatkan sertifikat bertanda tangan CA untuk produksi, cara memasangnya ke kamera dan memverifikasi server jarak jauh, cara melindungi kunci privat, serta cara kedaluwarsa dan rotasi sertifikat memengaruhi perangkat yang sudah di-deploy.
