v4.8.10¶

يضيف الإصدار v4.8.10 دعماً للوحة OpenMV AE3 ويحدّث سلاسل أدوات الذكاء الاصطناعي المرفقة، مع تحديث مترجم Ethos-U Vela NPU إلى 5.0.0 وأدوات ST Edge AI إلى 4.0.0. يبقى هذا الإصدار مبنياً على Qt Creator 14.0.2، دون إعادة تأسيس، ولا يحتوي على أي تغييرات قاطعة.

أبرز النقاط¶

  • OpenMV AE3 أصبحت لوحة Alif Ensemble E3 مدعومة الآن، بما في ذلك برمجة محمّل الإقلاع/DFU عبر alif_tools وتسريع Ethos-U55 NPU لكل نواة.

  • Ethos-U Vela تم تحديث مترجم نماذج NPU من 4.2.0 إلى 5.0.0، مع إعادة بنائه مقابل واجهة Python 3.13 ABI.

  • ST Edge AI تم تحديث أدوات تحويل النماذج (stedgeai) من 3.0.0 إلى 4.0.0 على جميع المنصات.

  • عودة اتصال أسرع: تتخلى الآن واجهة OpenMV IDE عن اتصال كاميرا فاشل بعد محاولتي إعادة اتصال بدلاً من 10.

ميزات جديدة¶

  • OpenMV AE3 دعم لوحة Alif Ensemble E3 (الاسم الرمزي الداخلي "AE3 classic"). يضيف نوع اللوحة AE3 / OPENMV AE3 SBL2، باسم العرض في OpenMV IDE OpenMV AE3 SBL2 (VID:PID 37C5:16E3)، وبرمجة محمّل الإقلاع/DFU عبر alif_tools، وتخطيط romfs ثنائي النواة (HP/HE)، وتكوينات تسريع Ethos-U55 Vela NPU لكل نواة (نواة الأداء العالي ethos-u55-256، ونواة الكفاءة العالية ethos-u55-128). اللوحة هدف مخفي/مبرمج عبر DFU يتم الوصول إليه عبر تعيينات VID:PID الخاصة بمحمّل الإقلاع بدلاً من كونه هدف اتصال قابل للاختيار مُدرج بشكل عادي (adbed3825).

تغييرات وتحسينات أخرى¶

  • تم تقليل الحد الأقصى لعدد محاولات إعادة اتصال الكاميرا التلقائية من 10 إلى 2، بحيث تتخلى واجهة OpenMV IDE بشكل أسرع عن اتصال فاشل (2ddecec53).

  • تم تحديث مترجم نماذج Ethos-U Vela NPU المرفق من 4.2.0 إلى 5.0.0 على Windows و Linux و macOS arm64، مع إعادة بنائه مقابل واجهة Python 3.13 ABI وإعادة ترجمة الملفات الثنائية regor و mlw_codec وتحديث numpy المرفق إلى 2.4.4 عبر جميع المنصات (be9d29d9d, 04c73f187, 93260af21, a58458cfb, a6e1ac7ed).

  • تم تحديث أدوات تحويل النماذج ST Edge AI (stedgeai) المرفقة من 3.0.0 إلى 4.0.0 على Windows و Linux و macOS arm64 (1a727567c, 8dcfd8472, 27aabc740).

دعم المنصة والأدوات¶

  • قاعدة Qt Creator: 14.0.2.

  • لوحة جديدة: OpenMV AE3 SBL2 (Alif Ensemble E3)، هدف مخفي/مبرمج عبر DFU يُبرمج عبر alif_tools مع تسريع Ethos-U55 NPU ثنائي النواة.

  • Ethos-U Vela مترجم NPU: 4.2.0 -> 5.0.0 (Windows، Linux، macOS arm64)، أُعيد تثبيته مقابل واجهة Python 3.13 ABI.

  • ST Edge AI أدوات (stedgeai): 3.0.0 -> 4.0.0 (Windows، Linux، macOS arm64).

  • numpy المرفق (بيئة Vela): تم تحديثه إلى 2.4.4 على جميع المنصات.
