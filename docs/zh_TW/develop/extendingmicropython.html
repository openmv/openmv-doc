以 C 語言擴充 MicroPython¶

本章說明在 C 語言中實作額外功能的各種選項，但這些程式碼是寫在 MicroPython 主儲存庫之外的。第一種做法適用於建置你自己的客製化韌體，並加入一些可從 Python 存取、針對特定專案的額外模組或函式。第二種做法則用於建置可在執行期載入的模組。

關於建置位於 MicroPython 主儲存庫中之核心模組的更多資訊，請參閱 函式庫一節。
