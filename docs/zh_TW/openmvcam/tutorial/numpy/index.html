6. NumPy¶

Image 類別涵蓋了相機已知如何處理的逐像素工作 -- 閾值處理、色塊尋找、邊緣偵測，以及影像函式庫的其餘部分。numpy 模組則涵蓋其他所有工作：ADC 讀數及其他純數字緩衝區、一次橫跨整個緩衝區執行的數學運算，以及影像函式庫尚未內建的逐像素轉換。

於是輪到 ndarray 登場了，這是一個用來存放一整塊型別相同、緊密排列數字的單一類別。numpy 中的其他一切都是對 ndarray 進行運算的數學函式。將兩個陣列相加、對其中一個求和、對每個元素取正弦 -- 每一項都是單一的函式庫呼叫，一次處理整個緩衝區，比起對等的 Python for 迴圈要快得多。
