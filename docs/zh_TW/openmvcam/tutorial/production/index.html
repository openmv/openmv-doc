14. 量產¶

在工作檯上能運作的指令碼是起點，而非終點。一台出貨在產品中的相機，必須在無人看顧下完成它的工作，運行在指令碼無法控制的網路上，而且不讓拆開外殼的任何人讀取到原始碼。本節的頁面涵蓋了介於展示能跑與裝置出門之間的實務工作。
