class Statistics -- Statistics 物件¶

statistics 物件是由 histogram.get_statistics()（及其別名 histogram.get_stats() / histogram.statistics()）回傳，或直接由 Image.get_statistics() 回傳的 attrtuple。它以每個通道八項歸納值總結底層 Histogram 的分箱分布：平均值、中位數、眾數、標準差、最小值、最大值、下四分位數與上四分位數。

對於灰階資料，只有無前綴的欄位（mean、median、mode、stdev、min、max、lq、uq）帶有有用的值，且對於 RGB565 資料它們會重複用作 LAB L 通道的歸納值 -- 因此 l_* 欄位會鏡射無前綴的欄位，純為對稱性而提供。a_* 與 b_* 只在 RGB565 直方圖時帶有有用的值。

欄位可透過屬性名稱（statistics.mean）或索引（statistics[0]）存取。此物件沒有公開的建構子。

class image.statistics¶

請呼叫 histogram.get_statistics() 或 Image.get_statistics() 來建立此物件。

灰階 / LAB L（亦以下方的 l_* 形式公開）¶

mean¶

直方圖的灰階平均值。整數 0 -- 255。索引 [0]。

median¶

直方圖的灰階中位數。整數 0 -- 255。索引 [1]。

mode¶

直方圖的灰階眾數。整數 0 -- 255。索引 [2]。

stdev¶

直方圖的灰階標準差。整數 0 -- 255。索引 [3]。

min¶

直方圖的灰階最小值。整數 0 -- 255。索引 [4]。

max¶

直方圖的灰階最大值。整數 0 -- 255。索引 [5]。

lq¶

直方圖的灰階下四分位數（第 25 百分位數）。整數 0 -- 255。索引 [6]。

uq¶

直方圖的灰階上四分位數（第 75 百分位數）。整數 0 -- 255。索引 [7]。
