OpenMV IDE 變更日誌¶

每個 OpenMV IDE 版本都帶來新功能與改進。以下是各版本的新增內容——新的編輯器、相機與工具功能、錯誤修正、平台支援，以及任何工作流程變更及其對您的影響。版本依主要版本分組，由新到舊排列。點選某個版本即可展開其各次發行。

OpenMV IDE 建構於 Qt Creator 應用程式框架之上；每次發行皆會註明其所基於的 Qt Creator 基礎版本。此處僅記錄 OpenMV 自身的變更——數以千計與此無關的上游 Qt Creator 提交並不在記錄之列。
