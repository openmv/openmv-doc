OpenMV MicroPython 语言与实现¶

MicroPython 在语言语法方面力求实现 Python 3.4 标准（并选取了后续版本中的部分特性），MicroPython 的大多数特性与 docs.python.org 上的“语言参考”文档所描述的内容相同。

MicroPython 标准库在 相应章节 中描述。MicroPython differences from CPython 章节描述了 MicroPython 与 CPython 之间的差异（这些差异大多涉及标准库和类型，但也包括一些语言层面的特性）。

本章描述 MicroPython 实现的特性与特殊之处，以及使用它们的最佳实践。
