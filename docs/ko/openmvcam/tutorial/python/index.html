2. Python 개요¶

OpenMV Cam은 MicroPython을 실행합니다. MicroPython은 데스크톱 Python과 동일한 언어로, 더 작은 표준 라이브러리와 하드웨어와 통신하기 위한 몇 가지 추가 모듈을 갖추고 있습니다. LED를 깜빡이는 두 줄짜리 스크립트부터 완전한 머신 비전 애플리케이션까지, 카메라를 위해 작성되는 모든 것은 Python 코드입니다.
