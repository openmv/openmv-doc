13.1.5. 환경 설정 및 외관¶

Edit → Preferences(macOS에서는 애플리케이션 메뉴)는 설정 대화 상자를 엽니다. 대부분의 항목은 그대로 두어도 됩니다. 기본값이 이 장의 나머지 부분에서 설명하는 구성이기 때문입니다. 하지만 몇 가지 범주는 알아 둘 가치가 있습니다.

13.1.5.1. 인터페이스 및 테마¶

Environment → Interface 페이지에서 테마를 선택합니다. IDE는 dark(기본값)와 light 테마를 함께 제공하며, 에디터의 구문 색상은 테마를 자동으로 따릅니다. 같은 페이지에 사용자 인터페이스 언어 및 디스플레이 배율 옵션이 있습니다.

일상적인 레이아웃은 대화 상자 외부에 있습니다. 각 하단 창(시리얼 터미널, 검색 결과)에는 상태 표시줄에 토글 버튼이 있습니다. 측면 창은 분할자를 끌어 닫으면 접히고 창 가장자리의 서랍 화살표로 다시 열 수 있습니다. 그리고 Window 메뉴의 Full Screen(Ctrl+Shift+F11)은 데모나 벽걸이 모니터에 전체 화면을 제공합니다.

13.1.5.2. 텍스트 에디터 설정¶

Text Editor 범주는 편집 영역을 제어합니다. Font는 글꼴과 크기를, Display는 행 번호, 공백 시각화, 텍스트 줄 바꿈을, Behavior는 들여쓰기(탭 너비, 공백 대 탭)와 자동 공백 정리를 담당합니다. Ctrl++ / Ctrl+- / Ctrl+0으로, 또는 Ctrl을 누른 채 마우스 휠로 글꼴 크기를 즉석에서 변경할 수도 있습니다.

OpenMV IDE에서 키보드 단축키는 고정되어 있습니다. 각 단축키는 해당 메뉴 항목 옆에 표시되며, 이 장에서는 해당 기능이 등장하는 곳에서 중요한 단축키를 인용합니다.

13.1.5.3. 기타 범주¶

Python 및 Copilot 범주는 에디터의 언어 인텔리전스, 즉 내장된 Python 언어 서버와 선택적인 GitHub Copilot 로그인을 구성합니다. 나머지 범주는 Qt Creator 코어에 속하며 거의 신경 쓸 필요가 없습니다.
