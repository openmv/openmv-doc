Proto Shield¶

Proto Shield는 OpenMV Cam에 10x10 스루홀 프로토타이핑 영역을 제공하며, 카메라에 연결된 3.3 V 및 GND 레일과 함께 사용자 정의 회로를 납땜할 수 있도록 분기된 I/O 핀을 제공합니다.

전체 데이터시트, 사진, 주문 정보는 Proto Shield 제품 페이지 를 참고하세요.

주요 특징¶

  • 10x10 스루홀 프로토타이핑 그리드

  • OpenMV Cam에 연결된 3.3 V 및 GND 전원 레일

  • 분기된 OpenMV Cam I/O 핀

  • 가속도계, 센서, LED 회로 납땜 가능

핀아웃¶

모든 헤더 핀은 프로토타이핑 그리드에 연결되지 않은 채로 분기되어 있어 사용자가 원하는 대로 배선할 수 있습니다. 3.3 V 및 GND 레일은 카메라까지 연결되어 있으므로 납땜된 회로에 직접 전원을 공급할 수 있습니다.

사용법¶

프로토타이핑 그리드에 배선한 것은 무엇이든 적합한 주변장치 API — GPIO, PWM, ADC, I²C, SPI, UART 등 — 를 통해 구동됩니다. 간단한 예로, 분기된 I/O 패드 중 하나에 납땜된 LED를 깜박여 보겠습니다:

from machine import Pin
import time

led = Pin("P0", Pin.OUT)
while True:
    led.on()
    time.sleep_ms(500)
    led.off()
    time.sleep_ms(500)
