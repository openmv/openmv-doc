v3.5.1¶

v3.5.1은 TensorFlow conv2d / sepconv3 연산을 수정하고, TensorFlow와 ulab을 업데이트하며, 이미지 포맷 오류 메시지를 개선합니다. 호환성을 깨는 API 변경은 없으며, v3.5.0 스크립트가 변경 없이 실행됩니다.

이 페이지에서

  • 주요 내용

  • 기타 변경 사항 및 개선

  • 버그 수정

  • 호환성을 깨는 API 변경

주요 내용¶

  • TensorFlow 수정 — conv2d와 sepconv3의 경계 처리를 수정하고, TensorFlow를 업데이트했습니다.

  • 더 명확한 오류 — 더 구체적인 이미지 포맷 오류 메시지.

  • 호환성을 깨는 API 변경 없음 — 재플래시하면 v3.5.0 스크립트가 변경 없이 실행됩니다.

기타 변경 사항 및 개선¶

  • TensorFlow와 ulab을 업데이트했으며, OpenMV 2(공간이 부족한 M4)에서 ulab을 비활성화하고, 구형 보드(특히 OpenMV 3)에서 WiFi 디버그 기능을 비활성화했으며, 사용하지 않는 서브모듈을 제거하고, WiFi 펌웨어 업데이트 스크립트를 업데이트했습니다.

버그 수정¶

머신러닝:

  • TensorFlow 라이브러리의 conv2d와 sepconv3 경계 처리를 수정했습니다.

이미징:

  • mutable_bayer 오류 메시지를 수정하고 이미지 포맷 오류를 더 구체적으로 만들었습니다.

시스템:

  • 디버그 빌드 링커 오류를 수정했습니다.

호환성을 깨는 API 변경¶

v3.5.1에는 호환성을 깨는 Python API 변경이 없습니다. v3.5.0에서 실행되던 스크립트는 v3.5.1에서도 변경 없이 실행됩니다 — 펌웨어만 재플래시하면 됩니다. (참고: ulab은 제한된 플래시로 인해 OpenMV 2에서 사용할 수 없습니다.)
