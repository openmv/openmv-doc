rpc вҖ” rpc лқјмқҙлёҢлҹ¬лҰ¬В¶

OpenMV Camмқҳ rpc лӘЁл“ҲмқҖ OpenMV Camмқ„ лӢӨлҘё л§ҲмқҙнҒ¬лЎңм»ЁнҠёлЎӨлҹ¬лӮҳ м»ҙн“Ён„°м—җ м—°кІ°н•ҳм—¬ OpenMV Camм—җм„ң мӣҗкІ© нҢҢмқҙмҚ¬(лҳҗлҠ” н”„лЎңмӢңм Җ) нҳём¶ңмқ„ мӢӨн–үн•  мҲҳ мһҲкІҢ н•ҙмӨҚлӢҲлӢӨ. rpc лӘЁл“ҲмқҖ к·ё л°ҳлҢҖлҸ„ м§Җмӣҗн•ҳлҜҖлЎң, OpenMV Camмқҙ лӢӨлҘё л§ҲмқҙнҒ¬лЎңм»ЁнҠёлЎӨлҹ¬лӮҳ м»ҙн“Ён„°м—җм„ң мӣҗкІ© н”„лЎңмӢңм Җ(лҳҗлҠ” нҢҢмқҙмҚ¬) нҳём¶ңмқ„ мӢӨн–үн•ҳлҸ„лЎқ н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

лқјмқҙлёҢлҹ¬лҰ¬ мӮ¬мҡ© л°©лІ•В¶

UARTлҘј нҶөн•ҙ н•ҳлӮҳмқҳ мҪңл°ұмқ„ л…ём¶ңн•ҳлҠ” мөңмҶҢн•ңмқҳ slave мҳҲм ңмһ…лӢҲлӢӨ:

import rpc
import csi

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.QVGA)

interface = rpc.rpc_uart_slave(baudrate=115200)

def snapshot(_):
    return csi0.snapshot().compress().bytearray()

interface.register_callback(snapshot)
interface.loop()  # Does not return.

slaveм—җкІҢ JPEG н”„л Ҳмһ„мқ„ мҡ”мІӯн•ҳлҠ” лҢҖмқ‘ master мҳҲм ңмһ…лӢҲлӢӨ:

import rpc

interface = rpc.rpc_uart_master(baudrate=115200)

result = interface.call("snapshot")
if result is None:
    print("communication failed")
elif len(result) == 0:
    print("remote function not registered on the slave")
else:
    # result is a memoryview of the JPEG bytes returned by the slave.
    print("received", len(result), "bytes")

лӢӨлҘё м „мҶЎ л°©мӢқмқ„ мӮ¬мҡ©н•ҳл Өл©ҙ rpc_uart_master / rpc_uart_slave лҘј лҢҖмқ‘н•ҳлҠ” can, i2c лҳҗлҠ” spi мҢҚмңјлЎң көҗмІҙн•ҳм„ёмҡ”.

мқјл°ҳм ҒмңјлЎң м»ЁнҠёлЎӨлҹ¬ мһҘм№ҳк°Җ rpc лқјмқҙлёҢлҹ¬лҰ¬лҘј мӮ¬мҡ©н•ҳл Өл©ҙ rpc лқјмқҙлёҢлҹ¬лҰ¬лЎң мқён„°нҺҳмқҙмҠӨ к°қмІҙлҘј мғқм„ұн•©лӢҲлӢӨ. мҳҲлҘј л“Өл©ҙ:

interface = rpc.rpc_uart_master(baudrate=115200)

мқҙлҠ” rpc slaveмҷҖ нҶөмӢ н•ҳкё° мң„н•ң UART мқён„°нҺҳмқҙмҠӨлҘј мғқм„ұн•©лӢҲлӢӨ.

мқён„°нҺҳмқҙмҠӨк°Җ мғқм„ұлҗҳл©ҙ лӢӨмқҢмқ„ мҲҳн–үн•ҳкё°л§Ң н•ҳл©ҙ лҗ©лӢҲлӢӨ:

memory_view_object_result = interface.call("remote_function_or_method_name", bytes_object_argument)

к·ёлҹ¬л©ҙ rpc лқјмқҙлёҢлҹ¬лҰ¬лҠ” slaveм—җм„ң н•ҙлӢ№ "remote_function_or_method_name" мқ„ мӢӨн–үн•ҳл Өкі  мӢңлҸ„н•©лӢҲлӢӨ. мӣҗкІ© н•ЁмҲҳ лҳҗлҠ” л©”м„ңл“ңлҠ” мөңлҢҖ 2^32-1л°”мқҙнҠё нҒ¬кё°к№Ңм§Җ к°ҖлҠҘн•ң bytes_object_argument лҘј л°ӣмҠөлӢҲлӢӨ. мӣҗкІ© л©”м„ңл“ңмқҳ мӢӨн–үмқҙ лҒқлӮҳл©ҙ м—ӯмӢң мөңлҢҖ 2^32-1л°”мқҙнҠё нҒ¬кё°к№Ңм§Җ к°ҖлҠҘн•ң memory_view_object_result лҘј л°ҳнҷҳн•©лӢҲлӢӨ. мқёмһҗмҷҖ мқ‘лӢө лӘЁл‘җ мқјл°ҳм Ғмқё л°”мқҙнҠё м»Ён…Ңмқҙл„Ҳмқҙкё° л•Ңл¬ём—җ rpc лқјмқҙлёҢлҹ¬лҰ¬лҘј нҶөн•ҙ л¬ҙм—Үмқҙл“  м „лӢ¬н•  мҲҳ мһҲкі  м–ҙл–Ө нҳ•мӢқмқҳ мқ‘лӢөмқҙл“  л°ӣмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. мқёмһҗлҘј м „лӢ¬н•ҳлҠ” к°„лӢЁн•ң л°©лІ•мқҖ struct.pack() мңјлЎң мқёмһҗлҘј мғқм„ұн•ҳкі  л°ҳлҢҖнҺём—җм„ң struct.unpack() мңјлЎң мқёмһҗлҘј л°ӣлҠ” кІғмһ…лӢҲлӢӨ. мқ‘лӢөмқҳ кІҪмҡ°, л°ҳлҢҖнҺём—җм„ң л¬ёмһҗм—ҙ к°қмІҙлӮҳ json л¬ёмһҗм—ҙмқ„ кІ°кіјлЎң ліҙлӮҙл©ҙ masterк°Җ мқҙлҘј н•ҙм„қн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мҳӨлҘҳмҷҖ кҙҖл Ён•ҙм„ң, мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” н•ЁмҲҳлӮҳ л©”м„ңл“ң мқҙлҰ„мқ„ мӢӨн–үн•ҳл Өкі  н•ҳл©ҙ rpc_master.call() л©”м„ңл“ңлҠ” л№Ҳ bytes() к°қмІҙлҘј л°ҳнҷҳн•©лӢҲлӢӨ. rpc лқјмқҙлёҢлҹ¬лҰ¬к°Җ slaveмҷҖмқҳ нҶөмӢ м—җ мӢӨнҢЁн•ҳл©ҙ rpc лқјмқҙлёҢлҹ¬лҰ¬лҠ” Noneмқ„ л°ҳнҷҳн•©лӢҲлӢӨ.

лӢЁмҲңн•Ёмқ„ мң м§Җн•ҳкё° мң„н•ҙ rpc лқјмқҙлёҢлҹ¬лҰ¬лҠ” masterмҷҖ slave мһҘм№ҳ мӮ¬мқҙмқҳ м—°кІ°мқ„ мң м§Җн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. rpc_master.call() л©”м„ңл“ңлҠ” slaveм—җ м—°кІ°мқ„ мӢңлҸ„н•ҳкі , мӣҗкІ© н•ЁмҲҳ лҳҗлҠ” л©”м„ңл“ңмқҳ мӢӨн–үмқ„ мӢңмһ‘н•ҳл©°, кІ°кіјлҘј к°Җм ёмҳӨлҠ” кіјм •мқ„ мәЎмҠҗнҷ”н•©лӢҲлӢӨ.

мқҙм ң slave мӘҪм—җм„ңлҠ” masterмҷҖ нҶөмӢ н•ҳкё° мң„н•ң rpc мқён„°нҺҳмқҙмҠӨлҘј мғқм„ұн•ҙм•ј н•©лӢҲлӢӨ. лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ:

interface = rpc.rpc_uart_slave(baudrate=115200)

мқҙлҠ” rpc masterмҷҖ нҶөмӢ н•ҳкё° мң„н•ң UART мқён„°нҺҳмқҙмҠӨ кі„мёөмқ„ мғқм„ұн•©лӢҲлӢӨ.

slave мқён„°нҺҳмқҙмҠӨлҘј мғқм„ұн•ң нӣ„м—җлҠ” мқён„°нҺҳмқҙмҠӨ к°қмІҙм—җ masterк°Җ нҳём¶ңн•  мҲҳ мһҲлҠ” мҪңл°ұмқ„ л“ұлЎқн•ҙм•ј н•©лӢҲлӢӨ:

def remote_function_or_method_name(memoryview_object_argument):
    <lots of code>
    return bytes_object_result

interface.register_callback(remote_function_or_method_name)

slaveм—җлҠ” мӣҗн•ҳлҠ” л§ҢнҒј л§ҺмқҖ мҪңл°ұмқ„ л“ұлЎқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. л§Ҳм§Җл§үмңјлЎң мҪңл°ұ л“ұлЎқмқ„ л§Ҳм№ҳл©ҙ лӢӨмқҢмқ„ мӢӨн–үн•ҳкё°л§Ң н•ҳл©ҙ лҗ©лӢҲлӢӨ:

interface.loop()

мқҙлҘј slaveм—җм„ң мӢӨн–үн•ҳл©ҙ rpc лқјмқҙлёҢлҹ¬лҰ¬к°Җ мӢңмһ‘лҗҳм–ҙ masterлҘј мҲҳмӢ  лҢҖкё°н•ҳкё° мӢңмһ‘н•©лӢҲлӢӨ. rpc_slave.loop() л©”м„ңл“ңлҠ” л°ҳнҷҳн•ҳм§Җ м•ҠлҠ”лӢӨлҠ” м җм—җ мң мқҳн•ҳм„ёмҡ”.

class rpc вҖ“ rpc кё°л°ҳ нҒҙлһҳмҠӨВ¶
