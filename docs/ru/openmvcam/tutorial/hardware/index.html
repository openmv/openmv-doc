3. Управление оборудованием¶

Каждое значение Python в начальном руководстве жило в памяти: число с именем x, строка, возвращаемая функцией, список, обрабатываемый циклом. Управление оборудованием заменяет работу в памяти физическими эффектами – напряжением на выводе, последовательностью импульсов на проводе, байтом, передаваемым по шине в другую микросхему.

Модуль machine – это мост. Его классы (Pin, ADC, PWM, UART, SPI, I2C, CAN, WDT) каждый сопоставляют часть кремния камеры с Python: настройте экземпляр, вызовите его методы и наблюдайте реальные эффекты на выводах платы.
