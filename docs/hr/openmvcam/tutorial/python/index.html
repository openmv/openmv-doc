2. Pregled Pythona¶

OpenMV Cam pokreće MicroPython, isti jezik kao i Python na stolnom računalu s manjom standardnom bibliotekom i nekoliko dodatnih modula za komunikaciju sa sklopovljem. Sve napisano za kameru – od skripte u dva retka koja treperi LED diodom do potpune aplikacije strojnog vida – jest Python kod.
