struct — pakiranje i raspakiravanje primitivnih tipova podataka¶

Ovaj modul obavlja pretvorbe između Python vrijednosti i struktura u stilu jezika C predstavljenih kao Python bytes objekti, koristeći formatne nizove za opis rasporeda podataka.

Razmaci nisu podržani u formatnim nizovima.

Primjeri¶

Pakiranje i raspakiravanje little-endian vrijednosti. Prefiks < odabire little-endian redoslijed bajtova sa standardnim veličinama i bez poravnanja:

import struct

# Pack an unsigned short (H, 2 bytes) then an unsigned int (I, 4 bytes).
data = struct.pack("<HI", 7, 1000)
# data == b'\x07\x00\xe8\x03\x00\x00'

# Unpack returns a tuple of the values, in order.
struct.unpack("<HI", data)
# (7, 1000)

# calcsize() reports how many bytes the format needs.
struct.calcsize("<HI")
# 6

Pakiranje u postojeći međuspremnik i raspakiravanje iz njega na pomaku bajta:

buf = bytearray(8)

# Write a little-endian signed int (i) at offset 2.
struct.pack_into("<i", buf, 2, -12345)
# buf == bytearray(b'\x00\x00\xc7\xcf\xff\xff\x00\x00')

# Read it back from the same offset.
struct.unpack_from("<i", buf, 2)
# (-12345,)

Funkcije¶

struct.calcsize(fmt: str) → int¶

Vraća broj bajtova potreban za pohranu zadanog fmt.

struct.pack(fmt: str, *values: Any) → bytes¶

Pakira values prema formatnom nizu fmt. Povratna vrijednost je bytes objekt koji kodira vrijednosti.

struct.pack_into(fmt: str, buffer: Any, offset: int, *values: Any) → None¶

Pakira values prema formatnom nizu fmt u buffer počevši od offset. offset može biti negativan kako bi se brojalo od kraja međuspremnika buffer.

struct.unpack(fmt: str, data: bytes) → Tuple¶

Raspakirava iz data prema formatnom nizu fmt. Povratna vrijednost je torka raspakiranih vrijednosti.

struct.unpack_from(fmt: str, data: bytes, offset: int = 0, /) → Tuple¶

Raspakirava iz data počevši od offset prema formatnom nizu fmt. offset može biti negativan kako bi se brojalo od kraja data. Povratna vrijednost je torka raspakiranih vrijednosti.
