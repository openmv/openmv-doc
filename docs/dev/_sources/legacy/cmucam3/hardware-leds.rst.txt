LEDS
====

* **LED 0** -- This pin is shared with the MOSI pin and should be used only when
  CS is disabled while an MMC card is inserted.
* **LED 1** -- This pin is shared with the Servo 2 pin. When using Servo 2, the
  LED will flash.
* **LED 2** -- This pin is shared with the Servo 3 pin. When using Servo 3, the
  LED will flash.
