Lua API
=======

The CamScripter Lua API describes the CMUcam3 functions that you can use in your
Lua programs to control the actions of the CMUcam3.

The original hub page for the Lua API was not preserved in the archive. The
individual API reference pages are:

* :doc:`Lua API: Functions <luaapifunctions>`
* :doc:`Lua API: Constants <luaapiconstants>`
* :doc:`Lua API: pixel <luaapistructpixel>`
* :doc:`Lua API: image <luaapistructimage>`
* :doc:`Lua API: framediff <luaapistructframediff>`
* :doc:`Lua API: tracker <luaapistructtracker>`
