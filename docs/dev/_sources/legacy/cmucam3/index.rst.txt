CMUcam3
=======

.. |c3a| image:: cmucam3-a.jpg
   :alt: CMUcam3
.. |c3b| image:: cmucam3-b.jpg
   :alt: CMUcam3 board

.. container:: legacy-center

   |c3a| |c3b|

The CMUcam3 is an ARM7TDMI based fully programmable embedded
computer vision sensor. The main processor is the NXP LPC2106
connected to an Omnivision CMOS camera sensor module. Custom C
code can be developed for the CMUcam3 using a port of the GNU
toolchain along with a set of open source libraries and example
programs.

For more information visit our :doc:`wiki <wiki>`.
