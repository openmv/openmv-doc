Contributing to cc3
===================

We encourage people to contribute their projects, libraries, bugfixes, and
enhancements.

Bugfixes
--------

The best way to contribute bugfixes are through the ticket system. Patches are
welcome.

Wiki
----

We are investigating having personal pages on the wiki for contributed code and
ideas.

Discussion
----------

The discussion board is open for posting. Consider posting ideas or code!

SVN Access
----------

We are also investigating giving out commit access to parts of the version
control repository. This way, interested people can directly contribute code to
the project.
