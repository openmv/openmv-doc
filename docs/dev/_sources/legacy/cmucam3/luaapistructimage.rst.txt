Lua API: image
==============

This wiki page documented the CamScripter Lua API ``image`` data structure, but
its content was not preserved in the archive.

The ``image`` data structure is created with
:doc:`image_new <luaapifunctions>`, and is used with functions such as
:doc:`pixbuf_read_rows <luaapifunctions>` and
:doc:`framediff_scanline <luaapifunctions>`.
