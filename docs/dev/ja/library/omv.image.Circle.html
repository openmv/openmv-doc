class Circle -- Circle オブジェクト#

circle オブジェクトは Image.find_circles() が返す attrtuple です。各円は、その中心、半径、および画像内のエッジピクセルがその円にどれだけ強く投票したかを表すハフ空間のマグニチュードによって記述されます。

フィールドは属性名（circle.x）またはインデックス（circle[0]）でアクセスできます。このオブジェクトには公開コンストラクタがありません。

class image.circle#

このオブジェクトを作成するには Image.find_circles() を呼び出してください。

x#

中心の x 座標（ピクセル単位）。整数。インデックス [0]。

y#

中心の y 座標（ピクセル単位）。整数。インデックス [1]。

r#

半径（ピクセル単位）。整数。インデックス [2]。

magnitude#

ハフ空間における円のマグニチュード。値が高いほど多くのエッジピクセルがこの円に投票したことを意味します。これは信頼度スコアとして扱ってください。整数。インデックス [3]。
