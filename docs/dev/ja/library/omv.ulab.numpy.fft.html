numpy.fft --- 高速フーリエ変換ルーチン#

numpy.fft サブモジュールは、1次元の高速フーリエ変換ルーチンを提供します。入力配列の長さは 2 のべき乗でなければなりません。そうでない場合は ValueError が送出されます。

変換の実部と虚部は別々の numpy.ndarray オブジェクトに保持され、関数は 2 要素タプル (real, imag) を返します。

関数#

numpy.fft.fft(r: ndarray, c: ndarray | None = None) → tuple[ndarray, ndarray]#

r の1次元離散フーリエ変換を計算します。

パラメータ:

  • r -- 長さが 2 のべき乗である1次元配列。入力信号の実部を保持します。

  • c -- r と同じ長さの省略可能な1次元配列で、入力の虚部を含みます。省略された場合、虚部はゼロとみなされます。

戻り値:

変換の実部と虚部を保持する numpy.ndarray オブジェクトの 2 要素タプル (real, imag)。

例外:

ValueError -- 入力の長さが 2 のべき乗でない場合。

numpy.fft.ifft(r: ndarray, c: ndarray | None = None) → tuple[ndarray, ndarray]#

1次元逆離散フーリエ変換を計算します。

パラメータ:

  • r -- 長さが 2 のべき乗である1次元配列。スペクトルの実部を保持します。

  • c -- r と同じ長さの省略可能な1次元配列で、スペクトルの虚部を含みます。省略された場合、虚部はゼロとみなされます。

戻り値:

逆変換の実部と虚部を保持する numpy.ndarray オブジェクトの 2 要素タプル (real, imag)。結果は N（サンプル数）で正規化されるため、ifft(fft(x)) は元の入力を再現します。

例外:

ValueError -- 入力の長さが 2 のべき乗でない場合。
