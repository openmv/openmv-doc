FLIR Boson アダプタ¶

FLIR Boson アダプタモジュールは、FLIR Boson および Boson+ サーマルコア（別売）を OpenMV Cam に接続します。Lepton シリーズよりも高解像度のサーマルイメージングを行いたい場合に使用します。

完全なデータシート、写真、注文方法については、FLIR Boson アダプタ製品ページ を参照してください。

注釈

OpenMV N6、RT1062、および H7 Plus でサポートされています。

ハイライト¶

  • FLIR Boson および Boson+ サーマルコア（別売）に対応

  • より細かなサーマルディテールを得られる、Lepton シリーズよりも高い解像度

使い方¶

Boson は csi.CSI を通じて駆動します。お使いのモジュールに合わせて framesize を設定してください。320 モジュールの場合は csi.QVGA（320x256）を、640 モジュールの場合は csi.VGA（640x512）を指定します:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.GRAYSCALE)
csi0.framesize(csi.QVGA)  # csi.VGA for the 640 module
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

センサーから直接カラー化されたフレームを取得するには、pixformat を csi.RGB565 に切り替え、csi.CSI.color_palette を image.PALETTE_IRONBOW に設定します。ドライバはパレットを適用した RGB565 フレームを出力するため、snapshot() は直接アイアンボウ色のフレームを返します:

csi0.pixformat(csi.RGB565)
csi0.color_palette(image.PALETTE_IRONBOW)

注釈

現時点では、FLIR Boson は温度測定をサポートしていません。
