4.2. レンズとフォーカス¶

ピンホールは機能しますが暗くなります。レンズはピンホールをより大きな開口に置き換え、入射するすべての光線を像面上の1点へ再び集束させるので、像は明るく、かつシャープになります。ピンホールが強いていたトレードオフが解消されるのです。

4.2.1. 屈折¶

光は、軽い媒質（空気）からより密度の高い媒質（ガラス）に入ると速度が遅くなり、界面でのこの速度変化が光線を曲げます。レンズは、あるシーン点から来るすべての光線が、後壁上の同じ点で再び収束するのにちょうど必要なだけ曲がるように成形されたガラスの一片です。別のシーン点から来る光線は別の点に収束し、以下同様です。像はピンホールの場合とまったく同じく、シーン点を1点ずつ積み重ねて構築されますが、点ごとにはるかに多くの光が集まります。

4.2.2. 薄レンズモデル¶

実際のレンズ設計では、ガラスの形状、複数のエレメント、そしてそれらを透過する光の波長を考慮します。本節の残りの部分で必要となる幾何学は、より単純な理想化、すなわち薄レンズモデルから得られます。これはレンズの実際の厚みを無視し、レンズを光軸上にある垂直な平面として扱い、そこで光線が瞬間的に向きを変えるとみなすものです。

このモデルは1つの出発点となる観察に基づいています。光軸に平行にレンズへ到達する光線は、すべてレンズの後方の同じ点を通るように屈折します。その点が焦点であり、レンズからその点までの距離がレンズの焦点距離で、慣例的に \(f\) と書かれます。「50 mm レンズ」とは焦点距離が 50 mm のレンズのことです。すべてのレンズには2つの焦点があり、両側に等しい距離 \(f\) のところに1つずつあります。1つは像側に、もう1つは物体側に対称に位置します。

この1つの事実から、2つの光線追跡の規則が導かれ、モデルは任意の像点を特定できるようになります。

  • 軸に平行にレンズに入射した光線は、像側の遠いほうの焦点を通るように屈折します。

  • レンズの中心を通る光線は曲がらずにそのまま直進します。中心ではレンズが十分に薄く、光線を曲げるガラスが実質的に存在しないためです。

これらの規則は単一の光線追跡の説明のように見えるかもしれませんが、レンズがすべてのシーン点に対して同時に行っていることを記述しています。見えている各点はあらゆる方向に光を散乱させます。そのうちレンズに入射した光線は、反対側でその点の像として再び収束します。完全な像は、すべて並行して起こる数百万のそうした点ごとの収束の和集合です。

単一のシーン点に拡大すると、その作図が明確になります。そのシーン点から出る2本の光線、すなわち軸に平行な1本（遠い焦点を通るように屈折する）とレンズの中心を通る1本（曲がらない）が、レンズの反対側で再び交差し、それらが交差する場所がその点の像です。

同じ幾何学を代数的に表現したものが薄レンズの方程式です。これは物体距離 \(u\)、像距離 \(v\)、焦点距離 \(f\) を関連付けます。

\[\frac{1}{u} + \frac{1}{v} = \frac{1}{f}\]

3つのうち任意の2つが与えられれば、方程式は3つ目を与えます。

非常に遠いシーン（\(u\) が大きい）では、項 \(1/u\) が無視できるほど小さくなり、\(v\) は \(f\) に近づきます。遠いシーンは焦点に結像するのです。より近いシーンでは \(v\) が \(f\) より大きくなる必要があり、つまりピントを合わせ続けるにはレンズがセンサーからより遠くに位置しなければなりません。これこそが、あらゆるフォーカス機構、すなわちマニュアルの鏡筒、オートフォーカスのモーター、固定焦点のシムが物理的に行っていることです。\(v\) がカメラにシャープに撮像させたいシーンの \(u\) と一致するように、レンズを前後に動かしているのです。

4.2.3. 被写界深度¶

ある物体距離にピントを合わせたレンズは、ちょうどその距離にある点についてのみ完全にシャープな像を結びます。より近いまたはより遠い点はセンサーの手前または後方の位置に結像し、センサー上には小さなぼけの円として到達します。それらのぼけの円が十分に小さくシャープに見える物体距離の範囲が被写界深度（DOF）です。

被写界深度はレンズを絞り込むと大きくなります。より小さな穴は各シーン点からより狭い光束を取り込み、その狭い光束はピントの外れた点に対してより小さなぼけの円を生成します。したがって、絞りを小さくするとDOFは増えますが取り込む光は減り、絞りを大きくすると取り込む光は増えますがDOFは減ります。絞りはレンズが撮影者に渡す2つ目のつまみであり、その前のピンホールとレンズの選択と同じく、シャープさと明るさのトレードオフです。

4.2.4. 絞りとF値¶

レンズの絞りはF値として表され、焦点距離と絞り口径の比です。

\[N = \frac{f}{D}\]

ここで \(D\) は開口の直径です。25 mm 幅の開口を持つ 50 mm レンズは \(N = 2\) となり、f/2 と書かれます。F値が小さいほど開口が広く（光が多く、DOFが浅い）、F値が大きいほど開口が狭く（光が少なく、DOFが深い）なります。絶対的な直径ではなく比が重要なのは、焦点距離にかかわらず、同じ \(f / D\) の比は同じシーンに対して同じ像の明るさを与えるからです。

OpenMV Cam の標準レンズには、汎用的な用途に合わせて選ばれた固定絞りが付いています。F値はレンズのデータシートに記載される仕様の1つです。これらのカメラでは、絞りは日常的には焦点距離ほど重要ではありませんが、データシートを読む上でこの概念は重要です。
