Downloads#

Manual do Usuário#

O Manual do Usuário da CMUcam1 v2.00 para o firmware v1.12 está disponível em formato PDF. Ele contém uma lista de peças, o esquema da placa e a descrição do protocolo de software. Manuais para versões mais antigas do firmware podem ser encontrados aqui: v1.11, v1.1, v1.0.

Se você decidir experimentar alterar as configurações de registro da câmera, também pode se interessar em baixar o Manual da Câmera OV6620 da OmniVision.

Construindo a Placa#

Se você estiver construindo-a você mesmo do zero, precisará baixar este arquivo de código HEX (contendo o firmware mais recente v1.12) para programar o chip SX28. Este é o código que implementa a funcionalidade e o protocolo serial descrito no manual. Você pode obter o downloader e a aplicação SX-Key da Parallax Inc. Versões mais antigas do firmware podem ser encontradas aqui: v1.11, v1.1, v1.0.

Programa de Interface Java#

Temos um programa java de código aberto que você pode baixar para fazer interface com e testar a sua CMUcam1. Este programa de interface permite que você rastreie objetos, analise dados e capture imagens completas da CMUcam1. Aqui estão as Capturas de Tela do Programa Java desta interface.

Este é o projeto java compactado escrito para Linux.

Esta é a versão da GUI que detecta automaticamente o seu SO. Ela é executada em sistemas UNIX que suportam E/S serial como se fosse um arquivo (como o Linux), bem como no Windows 95/98/2000/NT/XP. Recomendamos usar no mínimo um processador Pentium 2 de 400 MHz executando Java 1.2.2 ou superior.

Para ver exemplos de código C para o microprocessador Cerebellum e o Handyboard BotBall se comunicando com a CMUcam1, acesse esta página web.
