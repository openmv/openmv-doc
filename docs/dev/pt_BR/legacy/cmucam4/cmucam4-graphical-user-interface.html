Interface Gráfica do Usuário da CMUcam4#

O que é o CMUcam4GUI#

O CMUcam4GUI é uma ferramenta útil que facilita o trabalho com a CMUcam4. O CMUcam4GUI permite que você tire fotos com a sua CMUcam4 e as visualize no seu computador, e use as fotos para calcular os limites de rastreamento de cor de objetos coloridos na foto que você deseja que a CMUcam4 rastreie. Usando o CMUcam4GUI, você pode determinar rapidamente os limites de rastreamento de cor para um objeto em uma cena de iluminação e alargar os limites de rastreamento de cor para suportar o rastreamento do objeto em cenas de iluminação variável. O CMUcam4GUI também permite que você veja o que a CMUcam4 está rastreando em tempo real sem uma TV, exibindo graficamente o conteúdo dos pacotes de rastreamento, média, histograma e quadro que são enviados pela CMUcam4. Além disso, o CMUcam4GUI permite que você brinque com todas as opções da CMUcam4 em tempo real, como controlar os servos de pan e tilt, opções de rastreamento de cor e controles de câmera de baixo nível. Por fim, o CMUcam4GUI permite que você gerencie o cartão secure digital conectado à CMUcam4 remotamente através da CMUcam4.

Configuração no Windows#

Siga os passos abaixo para configurar o CMUcam4GUI para Windows:

  1. Baixe o arquivo CMUcam4GUI.zip

  2. Descompacte-o e extraia o conteúdo

  3. Execute o arquivo CMUcam4GUI.exe

Configuração no Linux#

Siga os passos abaixo para configurar o CMUcam4GUI para Linux:

  1. Adicione seu nome de usuário ao grupo dialout

sudo adduser username dialout

  1. Agora verifique digitando

groups username

  1. Agora saia da sessão e entre novamente

  2. Baixe o arquivo CMUcam4GUI.zip

  3. Descompacte-o e extraia o conteúdo

  4. Execute o arquivo CMUcam4GUI.sh

Configuração no Macintosh#

Siga os passos abaixo para configurar o CMUcam4GUI para Macintosh:

  1. Instale o Qt 1.2.1 SDK (Desktop 4.8.1) para Mac

  2. Baixe o Código-Fonte do CMUcam4GUI

  3. Abra o arquivo CMUcam4GUI.pro usando o QtCreator e mude do modo debug para o modo release

  4. Abra o arquivo CMUcom4.h e altere o #define CMUCOM4_FAST_BAUD_RATE de 230400 para 115200 (os Macs não conseguem lidar com a taxa de baud de 230400 BPS)

  5. Clique em run

Pergunta: Por que você não tem uma aplicação MAC pré-compilada?

Resposta: Kwabena, o desenvolvedor da CMUcam4, não tem um MAC…

Manual do CMUcam4GUI#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

Comandos da Barra de Menu e da Barra de Ferramentas

Comandos da Barra de Ferramentas

  • Begin/Reset - Conectar à CMUcam4

  • End - Desconectar da CMUcam4

  • Sleep Lightly - Colocar a CMUcam4 em um estado de energia muito baixa

  • Sleep Deeply - Colocar a CMUcam4 em um estado de energia ultrabaixa

  • Send Frame - Capturar uma imagem colorida e exibi-la na janela de visualização

  • Send Bitmap - Capturar uma imagem binária e exibi-la na janela de visualização

    • Esta é uma imagem binária de 80x60 do que quer que a CMUcam4 esteja rastreando - a CMUcam4 deve estar rastreando algo para que isso seja útil

  • Dump Frame - Capturar uma imagem colorida e salvá-la no cartão SD

  • Dump Bitmap - Capturar uma imagem binária e salvá-la no cartão SD

    • Esta é uma imagem binária de 80x60 do que quer que a CMUcam4 esteja rastreando - a CMUcam4 deve estar rastreando algo para que isso seja útil

Comandos da Barra de Menu

  • Save Frame - Salvar a imagem colorida na janela de visualização em um arquivo

  • Save Bitmap - Salvar a imagem binária na janela de visualização em um arquivo

    • Esta é uma imagem binária de 80x60 do que quer que a CMUcam4 esteja rastreando - a CMUcam4 deve estar rastreando algo para que isso seja útil

  • Clear Frame - Apagar a imagem colorida na janela de visualização

  • Clear Bitmap - Apagar a imagem binária na janela de visualização

    • Esta é uma imagem binária de 80x60 do que quer que a CMUcam4 esteja rastreando - a CMUcam4 deve estar rastreando algo para que isso seja útil

  • Open - Abrir uma imagem colorida para usar com o recurso de seleção de rastreamento

  • Exit - Sair da aplicação

Aba Viewing

  • Send Frame Control - Selecionar a resolução horizontal e vertical para o botão de envio de quadro na barra de ferramentas

    • Você não pode mover aquilo que está fotografando enquanto o comando send frame está sendo executado

    • Se a qualidade da imagem parecer ruim, desligue o auto-ganho e o balanço de branco automático e certifique-se de que a CMUcam4 está olhando para um objeto iluminado

    • Use uma resolução menor para capturar a imagem mais rapidamente - como 160x120

  • Dump Frame Control - Selecionar a resolução horizontal e vertical para o botão de despejo de quadro na barra de ferramentas

    • Você não pode mover aquilo que está fotografando enquanto o comando dump frame está sendo executado

    • Se a qualidade da imagem parecer ruim, desligue o auto-ganho e o balanço de branco automático e certifique-se de que a CMUcam4 está olhando para um objeto iluminado

    • Use uma resolução menor para capturar a imagem mais rapidamente - como 160x120

  • Opacity Control - Selecionar a opacidade do quadro enviado e do bitmap enviado - entre 1,00 para 100% e 0,00 para 0%

  • Tracking Control - Selecionar um blob de cor na janela de visualização para rastrear

    • O retângulo de seleção deve conter apenas os pixels dentro do blob de cor que você deseja rastrear

    • Após selecionar o blob de cor, clique no botão track color para calcular os limites de cor para rastrear esse blob de cor

      • O algoritmo funciona calculando a média e o desvio padrão dos canais de cor vermelho, verde e azul dentro do retângulo de seleção

      • O Coeficiente de Nível do Intervalo de Confiança seleciona quanto do desvio padrão adicionar +/- às médias dos canais de cor vermelho, verde e azul - entre 0 para +/- (stdev * 1) até 99 para +/- (stdev * 10)

Aba Tracking

Habilite o modo Line no painel de opções para ver os pixels azuis rastreados.

Aba Options

Aba Disk

  • Disk Information - Obter informações sobre a geometria do disco

  • Disk Space - Obter informações sobre o espaço no disco

  • Disk Control - Desmontar/formatar o disco

    • Nota: Você deve desmontar o disco após usar qualquer comando relacionado ao cartão SD antes de fechar a interface gráfica do usuário

    • Nota: Clicar no botão de formatar disco formatará imediatamente o disco

  • New File - Criar um novo arquivo com algum texto ou anexar algum texto a um arquivo

  • New Folder - Criar uma nova pasta

  • File Explorer - Interagir com os arquivos/pastas no cartão SD a partir do seu computador

    • Clique em refresh para listar o conteúdo do diretório atual

    • Altere os atributos de um arquivo ou pasta clicando nas caixas de seleção de atributos

      • Dê um duplo clique em uma pasta para entrar nela

      • Dê um duplo clique em um arquivo para abri-lo

        • Não tente abrir arquivos maiores que cerca de 10 KB (a menos que você queira esperar para sempre), porque a transferência de arquivos é muito lenta, em torno de 4 KBs

        • Você pode mover arquivos e pastas renomeando-os com um novo caminho, por exemplo, renomear txt.txt para ../txt.txt

Como delimitar cores#

  • Vá até options

  • Ligue o YUV

  • Aguarde 5 segundos

  • Desligue o auto-ganho e o balanço de branco automático

  • Selecione a resolução horizontal de send frame e defina como 160

  • Selecione a resolução vertical de send frame e defina como 120

  • Aponte a CMUcam4 para o objeto que você deseja rastrear, então, sem mover a CMUcam4 ou o objeto, e com ambos parados, clique em send frame

  • Aguarde até que o quadro seja enviado à GUI, não mova o objeto

  • Selecione a área de cor do objeto (e nada fora dessa área)

  • Clique em track selection

  • Clique em okay com 50 como valor padrão

  • Se apenas o objeto tiver azul desenhado sobre ele, clique em track selection novamente e mude o 50 para 60

  • Repita o processo acima até empurrar o valor de track selection o mais alto possível; você pode parar quando estiver em 99. Se você começar a delimitar coisas indesejadas, então você também terminou.

  • Então vá até options… defina o valor mínimo de green como 0 e o valor máximo de green como 255 (isso define Ymin como 0 e Ymax como 255). Se você começar a delimitar coisas indesejadas, então tente limites menos amplos até estar delimitando apenas o objeto que você deseja. Você pode atualizar o que está delimitado clicando no botão sendBitmap.

  • E… então você deve ter suas seis opções de rastreamento… Além disso, você também pode alargar os limites (V/vermelho) e (U/azul) manualmente, se quiser. Isso ajudará a tornar o sistema mais robusto em diferentes condições de iluminação e afins.

Para ver o que a CMUcam4 vê….

  • Ligue o modo de linha (há uma caixa de seleção nas opções)

  • Em seguida, vá até viewing e clique em track color.
