Changelog do Firmware OpenMV¶

Cada lançamento de firmware da OpenMV traz novos recursos e melhorias. Aqui está o que há de novo em cada um — além de quaisquer mudanças incompatíveis na API e exatamente como migrar seu código. Os lançamentos são agrupados por versão principal, do mais recente para o mais antigo. Clique em uma versão para expandir seus lançamentos.
