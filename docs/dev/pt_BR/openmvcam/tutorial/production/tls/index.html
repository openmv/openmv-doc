14.4. Trabalhando com certificados TLS#

O módulo ssl na OpenMV Cam (construído sobre mbedTLS) permite que um script faça conexões de rede criptografadas e autenticadas. Para fazer qualquer coisa além de criptografia simples, você precisa de certificados. As páginas desta seção cobrem o que eles são, quais tipos de chave e formatos de arquivo a câmera aceita, como criar certificados autoassinados para desenvolvimento e obter os assinados por CA para produção, como colocá-los na câmera e verificar um servidor remoto, como proteger a chave privada e como a expiração e a rotação de certificados afetam um dispositivo implantado.
