4. Sensores de Visão¶

Cada página até agora tratou a OpenMV Cam como um pequeno microcontrolador que por acaso tem uma câmera acoplada. Os sensores de visão invertem o enquadramento: a câmera é o produto, e tudo o mais na placa existe para alimentar pixels saindo dela para dentro do Python.

O módulo csi é a ponte. Uma única classe (CSI) encapsula a longa cadeia de física e silício entre os fótons e um buffer. Uma lente foca a cena sobre uma grade de fotodiodos que transformam fótons em carga; um controlador de exposição / ganho / leitura decide com que brilho cada pixel é reportado; um processador de sinal de imagem (ISP) corrige, faz o debayer, ajusta a cor e empacota os pixels em um formato escolhido; e um periférico do MCU captura o fluxo resultante para a RAM, pronto para que snapshot() o retorne como uma image.Image.
