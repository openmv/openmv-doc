2. Visão geral do Python¶

A OpenMV Cam executa o MicroPython, a mesma linguagem do Python para desktop, com uma biblioteca padrão menor e alguns módulos extras para se comunicar com o hardware. Tudo o que é escrito para a câmera – de um script de duas linhas que pisca o LED a uma aplicação completa de visão de máquina – é código Python.
