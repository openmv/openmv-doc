v3.4.1
======

v3.4.1 is a tiny fix release: it fixes the FLIR Lepton clock. It introduces
**no breaking API changes** — v3.4.0 scripts run unchanged.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **FLIR Lepton clock fix.**
- **No breaking API changes** — reflash and your v3.4.0 scripts run
  unchanged.

Bug fixes
---------

- Fixed the FLIR Lepton clock.

Breaking API changes
--------------------

v3.4.1 introduces **no breaking Python API changes**. Scripts that ran on
v3.4.0 run unchanged on v3.4.1 — just reflash the firmware.
