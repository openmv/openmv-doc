OpenMV Python Package Changelog
===============================

The OpenMV Python package is a host-side Python library and the ``openmv``
command-line tool for talking to an OpenMV Cam from your computer over the
Camera Protocol V2 — streaming video, running scripts, and benchmarking
throughput. Here's what changed in each release — newest first.
