2. Python’a Genel Bakış¶

OpenMV Cam, masaüstü Python ile aynı dil olan MicroPython’u çalıştırır; daha küçük bir standart kütüphaneye ve donanımla iletişim kurmak için birkaç ek modüle sahiptir. Kamera için yazılan her şey – LED’i yakıp söndüren iki satırlık bir betikten tam bir makine görüşü uygulamasına kadar – Python kodudur.
