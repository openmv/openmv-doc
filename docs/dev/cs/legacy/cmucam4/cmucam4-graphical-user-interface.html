Grafické uživatelské rozhraní CMUcam4#

Co je CMUcam4GUI#

CMUcam4GUI je užitečný nástroj, který usnadňuje práci s CMUcam4. CMUcam4GUI vám umožňuje pořizovat snímky pomocí vaší CMUcam4 a prohlížet je na počítači a používat snímky k výpočtu mezí sledování barvy barevných objektů na snímku, který chcete, aby CMUcam4 sledovala. Pomocí CMUcam4GUI můžete rychle určit meze sledování barvy pro objekt v jedné světelné scéně a rozšířit meze sledování barvy, aby podporovaly sledování objektu v měnících se světelných scénách. CMUcam4GUI vám také umožňuje vidět, co CMUcam4 sleduje v reálném čase bez televize, tím, že graficky zobrazuje obsah paketů sledování, průměru, histogramu a snímku, které CMUcam4 odesílá. Kromě toho vám CMUcam4GUI umožňuje pohrát si se všemi možnostmi CMUcam4 v reálném čase, jako je ovládání serv otáčení a náklonu, možnosti sledování barvy a nízkoúrovňové ovládání kamery. A nakonec vám CMUcam4GUI umožňuje spravovat kartu secure digital připojenou k CMUcam4 vzdáleně prostřednictvím CMUcam4.

Nastavení Windows#

Pro nastavení CMUcam4GUI pro Windows postupujte podle níže uvedených kroků:

  1. Stáhněte soubor CMUcam4GUI.zip

  2. Rozbalte to a extrahujte obsah

  3. Spusťte soubor CMUcam4GUI.exe

Nastavení na Linuxu#

Pro nastavení CMUcam4GUI pro Linux postupujte podle níže uvedených kroků:

  1. Přidejte své uživatelské jméno do skupiny dialout

sudo adduser username dialout

  1. Nyní ověřte zadáním

groups username

  1. Nyní se odhlaste a poté se znovu přihlaste

  2. Stáhněte soubor CMUcam4GUI.zip

  3. Rozbalte to a extrahujte obsah

  4. Spusťte soubor CMUcam4GUI.sh

Nastavení na Macintoshi#

Pro nastavení CMUcam4GUI pro Macintosh postupujte podle níže uvedených kroků:

  1. Nainstalujte Qt 1.2.1 SDK (Desktop 4.8.1) pro Mac

  2. Stáhněte Zdrojový kód CMUcam4GUI

  3. Otevřete soubor CMUcam4GUI.pro pomocí QtCreator a přepněte z režimu ladění do režimu release

  4. Otevřete soubor CMUcom4.h a změňte #define CMUCOM4_FAST_BAUD_RATE z 230400 na 115200 (Macy nezvládnou přenosovou rychlost 230400 BPS)

  5. Klikněte na run

Odpověď: Kwabena, vývojář CMUcam4, nemá MAC…

Příručka CMUcam4GUI#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

  • Save Bitmap - Uložit binární obraz z okna zobrazení do souboru

  • Clear Bitmap - Vymazat binární obraz v okně zobrazení

  • Open - Otevřít barevný obraz pro použití s funkcí výběru sledování

  • Exit - Ukončit aplikaci

Karta Viewing

  • Send Frame Control - Vybrat vodorovné a svislé rozlišení pro tlačítko send frame na panelu nástrojů

    • Nemůžete pohybovat tím, co fotografujete, dokud běží příkaz send frame

    • Pokud kvalita obrazu vypadá špatně, vypněte automatické zesílení a automatické vyvážení bílé a ujistěte se, že se CMUcam4 dívá na osvětlený objekt

    • Použijte menší rozlišení pro rychlejší zachycení obrázku – například 160x120

  • Dump Frame Control - Vybrat vodorovné a svislé rozlišení pro tlačítko dump frame na panelu nástrojů

    • Nemůžete pohybovat tím, co fotografujete, dokud běží příkaz dump frame

    • Pokud kvalita obrazu vypadá špatně, vypněte automatické zesílení a automatické vyvážení bílé a ujistěte se, že se CMUcam4 dívá na osvětlený objekt

    • Použijte menší rozlišení pro rychlejší zachycení obrázku – například 160x120

  • Opacity Control - Vybrat průhlednost odeslaného snímku a odeslané bitmapy - mezi 1.00 pro 100 % a 0.00 pro 0 %

  • Tracking Control - Vybrat v okně zobrazení barevnou skvrnu ke sledování

    • Výběrový obdélník by měl uzavírat pouze pixely uvnitř barevné skvrny, kterou chcete sledovat

    • Po výběru barevné skvrny klikněte na tlačítko track color pro výpočet barevných mezí ke sledování této barevné skvrny

      • Algoritmus funguje tak, že vypočítá průměr a směrodatnou odchylku červeného, zeleného a modrého barevného kanálu uvnitř výběrového obdélníku

      • Koeficient úrovně intervalu spolehlivosti vybírá, kolik směrodatné odchylky se přidá +/- k průměrům červeného, zeleného a modrého barevného kanálu - mezi 0 pro +/- (stdev * 1) až 99 pro +/- (stdev * 10)

Jak ohraničit barvy#

  • Přejděte do options

  • Zapnutí YUV

  • Počkejte 5 sekund

  • Vypnutí automatického zisku a automatického vyvážení bílé

  • Vyberte vodorovné rozlišení send frame a nastavte jej na 160

  • Vyberte svislé rozlišení send frame a nastavte jej na 120

  • Namiřte CMUcam4 na objekt, který chcete sledovat, poté bez pohybu CMUcam4 nebo objektu, přičemž oba se nesmí pohybovat, klikněte na send frame

  • Počkejte, dokud není snímek odeslán do GUI, nehýbejte objektem

  • Vyberte oblast barvy objektu (a nic mimo tuto oblast)

  • Klikněte na track selection

  • Klikněte na okay s výchozí hodnotou 50

  • Pokud má modrou barvu nakreslenou pouze objekt, klikněte znovu na track selection a nastavte 50 na 60

  • Opakujte výše uvedené, dokud nezvednete hodnotu track selection co nejvýše, můžete přestat, když je 99. Pokud začnete ohraničovat nežádoucí věci, pak jste také hotovi.

  • Poté přejděte na options… nastavte minimální hodnotu zelené na 0 a maximální hodnotu zelené na 255 (tím se nastaví Ymin na 0 a Ymax na 255). Pokud začnete ohraničovat nežádoucí věci, zkuste méně široké meze, dokud neohraničujete pouze objekt, který chcete. To, co je ohraničeno, můžete obnovit kliknutím na tlačítko sendBitmap.

  • A… pak byste měli mít svých šest možností sledování… Kromě toho můžete, pokud chcete, také ručně rozšířit meze (V/červená) a (U/modrá). To pomůže učinit systém odolnějším v různých světelných podmínkách a podobně.

Chcete-li vidět, co vidí CMUcam4….

  • Zapněte režim čáry (v možnostech je zaškrtávací políčko)

  • Poté přejděte na viewing a klikněte na track color.
