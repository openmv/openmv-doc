OpenMV Shieldy¶

Přídavné desky, které se zapojují do OpenMV Cam a rozšiřují ji o síťování, řízení motorů, displeje, senzory a další. Každá stránka popisuje, co shield dělá, klíčové specifikace a API pro jeho ovládání z MicroPythonu.
