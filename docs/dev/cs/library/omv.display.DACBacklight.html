class DACBacklight – DAC podsvícení¶

Třída DACBacklight ovládá podsvícení obrazovky prostřednictvím DAC kanálu. Předejte instanci jako argument backlight libovolnému konstruktoru displeje, který přijímá ovladač podsvícení.

Konstruktory¶

class display.DACBacklight(channel: int, bits: int = 8)¶

Vytvoří ovladač podsvícení řízený pomocí DAC.

channel určuje DAC kanál (nebo GPIO pin), který se má použít.

bits určuje rozlišení DAC.

deinit() → None¶

Deinicializuje ovladač podsvícení.

backlight(value: int) → None¶

Nastaví intenzitu podsvícení v rozsahu 0-100.
