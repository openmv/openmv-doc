2. Panoramica di Python¶

La OpenMV Cam esegue MicroPython, lo stesso linguaggio del Python desktop con una libreria standard più piccola e qualche modulo aggiuntivo per dialogare con l’hardware. Tutto ciò che viene scritto per la camera – da uno script di due righe che fa lampeggiare il LED a un’applicazione completa di visione artificiale – è codice Python.
