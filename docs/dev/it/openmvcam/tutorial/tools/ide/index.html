13.1. OpenMV IDE¶

OpenMV IDE è l’applicazione desktop per lavorare con la tua OpenMV Cam. Tutto al suo interno è al servizio di un unico ciclo: modificare uno script MicroPython, eseguirlo sulla camera e osservare il frame buffer e l’output seriale tornare in tempo reale mentre viene eseguito. L’IDE è costruito sul core dell’editor Qt Creator, e l’intera applicazione è organizzata attorno a quel ciclo – uno script non richiede file di progetto, configurazioni di build o impostazioni del compilatore. Scrivi uno script e lo esegui su una camera; questo è l’intero modello.

La finestra principale ha quattro aree di lavoro. L”editor riempie il lato sinistro e contiene gli script aperti, con il terminale seriale sotto di esso che mostra tutto ciò che lo script stampa. Il visualizzatore del frame buffer si trova in alto a destra e mostra ciò che la camera ha catturato per ultimo, e l”istogramma sotto di esso traccia le statistiche dei pixel di ciò che il frame buffer mostra. Una colonna di pulsanti della barra degli strumenti lungo il bordo sinistro estremo contiene i controlli per file, modifica, connessione ed esecuzione, e una barra di stato lungo il bordo inferiore riporta ciò che sta facendo la camera connessa. Puoi comprimere qualsiasi pannello quando lo script ha bisogno di spazio.

Le scorciatoie da tastiera in questo capitolo sono scritte per Windows e Linux; su macOS sostituisci Cmd a Ctrl.
