tfp410 — Controller DVI/HDMI#

Il modulo tfp410 fornisce un driver per il serializzatore DVI/HDMI TFP410 utilizzato per pilotare un display DVI/HDMI esterno tramite un bus LCD parallelo a 24 bit.

class TFP410 – Controller DVI/HDMI#

class tfp410.TFP410(*, i2c_addr: int = 0x3F)#

Inizializza il chip controller DVI/HDMI TFP410.

i2c_addr è l’indirizzo I2C del TFP410.

isconnected() → bool#

Restituisce True se è collegato un display esterno, False altrimenti.

hotplug_callback(callback: Callable[[bool], None] | None) → None#

Registra una funzione callback che verrà chiamata ogni volta che cambia lo stato di connessione di un display esterno. Il nuovo stato di connessione (bool) viene passato come unico argomento a callback.

Passa None come callback per disabilitare il callback di hotplug.

Mentre è registrato un callback, non chiamare TFP410.isconnected().
