Historia zmian biblioteki OpenMV Arduino RPC¶

Biblioteka OpenMV Arduino RPC pozwala płytce Arduino sterować kamerą OpenMV Cam — oraz przesyłać z niej strumieniowo dane — przez CAN, I²C, SPI, UART (asynchroniczny port szeregowy) oraz USB. Jest solidna i odporna na błędy, automatycznie wracając do działania po poluzowaniu przewodów, uszkodzeniu pakietów i przekroczeniu limitów czasu. Oto co zmieniło się w każdym wydaniu — od najnowszego.
