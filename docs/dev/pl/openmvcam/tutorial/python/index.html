2. Przegląd Pythona¶

OpenMV Cam uruchamia MicroPython, ten sam język co Python na komputerze stacjonarnym, z mniejszą biblioteką standardową i kilkoma dodatkowymi modułami do komunikacji ze sprzętem. Wszystko, co napisano dla kamery – od dwuliniowego skryptu migającego diodą LED po pełną aplikację wizji maszynowej – jest kodem w Pythonie.
