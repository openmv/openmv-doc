v1.0.2-beta#

v1.0.2-beta là bản beta desktop IDE và USB tooling: thêm hộp thoại preferences với liệt kê cổng serial, giao diện gỡ lỗi OpenMV với cấu hình CDC/HID và USB gỡ lỗi riêng biệt, di chuyển file config và tập lệnh người dùng vào thư mục home, và đi kèm ảnh DFU. Không có thay đổi Python API firmware — các tập lệnh v1.0.1-beta chạy không thay đổi.

Điểm nổi bật#

  • Hộp thoại Preferences với liệt kê cổng serial.

  • Giao diện gỡ lỗi OpenMV — cấu hình CDC/HID và USB gỡ lỗi riêng biệt.

  • Ảnh DFU được đi kèm; file config/tập lệnh người dùng đã được chuyển vào thư mục home.

  • Không thay đổi firmware API — nạp lại firmware và các tập lệnh v1.0.1-beta của bạn chạy không thay đổi.

Tính năng mới#

  • Đã thêm hộp thoại preferences với liệt kê cổng serial, giao diện gỡ lỗi OpenMV (cấu hình CDC/HID và USB gỡ lỗi riêng biệt), và ảnh firmware DFU. IDE QT đã bị loại bỏ.

Các thay đổi và cải tiến khác#

  • Đã chuyển file config và tập lệnh người dùng vào thư mục home (quản lý qua appdirs); trả về cổng COM dạng chuỗi trên Windows; đổi tên bin → firmware; mở cổng serial sau openmv.init; sửa cảnh báo thiết bị bị ngắt kết nối và serial đóng.

Thay đổi API không tương thích ngược#

v1.0.2-beta không có thay đổi Python API không tương thích ngược (các thay đổi chỉ liên quan đến desktop IDE / cấu hình USB / đóng gói). Các tập lệnh chạy trên v1.0.1-beta chạy không thay đổi — chỉ cần nạp lại firmware.
