2. Tổng quan về Python#

OpenMV Cam chạy MicroPython, cùng ngôn ngữ với Python trên máy tính để bàn nhưng với thư viện chuẩn nhỏ hơn và một vài mô-đun bổ sung để giao tiếp với phần cứng. Mọi thứ được viết cho camera -- từ tập lệnh hai dòng nhấp nháy LED đến ứng dụng thị giác máy đầy đủ -- đều là mã Python.
