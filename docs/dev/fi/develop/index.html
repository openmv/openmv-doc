OpenMV MicroPythonin sisäinen toiminta¶

Tämä luku tarjoaa katsauksen MicroPythoniin kehittäjän, MicroPythoniin osallistuvan, näkökulmasta. Se toimii kattavana resurssina MicroPythonin toteutusyksityiskohdista sekä aloitteleville että kokeneille osallistujille.

MicroPythonin ympärillä tapahtuva kehitys sisältää tavallisesti ydinajonaikaisen ympäristön muokkaamista, uuden kirjaston porttausta tai ylläpitoa. Tämä opas kuvaa hyvin perusteellisesti MicroPythonin toteutusyksityiskohdat, mukaan lukien aloitusoppaan, kääntäjän sisäisen toiminnan, MicroPythonin porttaamisen uudelle alustalle ja MicroPythonin ydinkirjaston toteuttamisen.
