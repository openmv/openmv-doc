2. Pythonin yleiskatsaus#

OpenMV Cam suorittaa MicroPythonia, samaa kieltä kuin työpöydän Python, mutta pienemmällä standardikirjastolla ja muutamalla ylimääräisellä laitteiston kanssa kommunikointiin tarkoitetulla moduulilla. Kaikki kameralle kirjoitettu – kaksirivisestä LED:iä vilkuttavasta skriptistä täysimittaiseen konenäkösovellukseen – on Python-koodia.
