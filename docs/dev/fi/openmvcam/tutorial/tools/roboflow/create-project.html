13.7.1. Projektin luominen¶

Roboflow-työtilasta New Project avaa luontisivun. Projekti niputtaa yhteen aineiston, sillä koulutetut mallit ja niiden käyttöönottokohteet.

Tässä kaksi valintaa on tärkeää:

  • Project Name. Käytä mitä tahansa mieleenpainuvaa – tässä esimerkissä käytetään Finding raccoons. Nimestä tulee osa mallin tunnistetta, jonka lataat käyttöönottovaiheessa.

  • Project Type. Tämä määrittää, mitä malli ennustaa, joten sovita se siihen, mitä haluat kameran tekevän. Object Detection – kohteiden ja niiden sijaintien löytäminen rajauslaatikoilla – on oikea valinta asioiden paikantamiseen kehyksessä, ja se on se, mitä kameran YOLO-jälkikäsittelijät dekoodaavat. Muut tyypit – luokittelu, instanssisegmentointi, avainpisteiden tunnistus – kouluttavat erilaisia mallipäitä; tässä esimerkissä käytetään kohteentunnistusta.

Näkyvyys- ja lisenssivaihtoehdot vaikuttavat vain siihen, jaetaanko aineisto julkisesti Roboflow Universessa – niillä ei ole mitään merkitystä kameralle päätyvän mallin kannalta. Aseta ne haluamallasi tavalla ja valitse sitten Create Project.
