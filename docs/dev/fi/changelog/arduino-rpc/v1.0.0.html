v1.0.0¶

v1.0.0 on OpenMV Arduino RPC -kirjaston ensimmäinen julkaisu — Arduino-kirjasto OpenMV Cam -kameran (tai toisen Arduinon tai tietokoneen) ohjaamiseen ja datan suoratoistamiseen siltä väylien CAN, I²C, SPI, UART (asynkroninen sarjaliikenne) ja USB kautta. Ensimmäisenä julkaisuna siitä ei ole mitään, mistä siirtyä; tämä merkintä tiivistää, mitä siinä toimitettiin.

Kohokohdat¶

  • Etäproseduurikutsut minkä tahansa väylän yli — kutsu funktioita OpenMV Cam -kamerassa Arduinosta väylien CAN, I²C, SPI, UART tai USB kautta ja suoratoista tulokset takaisin.

  • Ohjain- ja oheislaiteroolit — Arduino voi ohjata kameraa ohjainlaitteena tai toimia oheislaitteena, johon kamera tai isäntä kutsuu.

  • Vikasietoinen — palautuu automaattisesti löysistä johdoista, pakettien vioittumisesta ja aikakatkaisuista.

  • Datan suoratoisto — suoratoistotila suurten hyötykuormien, kuten JPEG-kehysten, siirtämiseen.

Uudet ominaisuudet¶

  • RPC-rajapinta toteutettiin väylillä CAN, I²C, SPI, UART (asynkroninen sarjaliikenne) ja USB, yhteisellä ohjain- / oheislaiterajapinnalla openmv-nimiavaruuden alla.

  • Suoratoistotila suuren läpäisykyvyn siirtoihin, kuten kuvan siirtoon.

  • Esimerkkiluonnokset — suositut ominaisuudet ohjainlaitteena, JPEG-kuvan siirto ja kuvan siirtäminen.

  • Linux- / Raspberry Pi -tuki.

Rikkovat API-muutokset¶

Ei mitään. v1.0.0 on ensimmäinen julkaisu — aiempaa versiota, jonka yhteensopivuuden se voisi rikkoa, ei ole.
