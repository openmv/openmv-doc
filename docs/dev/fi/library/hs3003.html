hs3003 — HS3003-kosteus-/lämpötilasensori¶

Ajuri Renesas HS3003 -tarkkuuden suhteellisen kosteuden ja lämpötilan sensorille I2C:n kautta. Yksittäinen I2C-transaktio laukaisee uuden mittauksen ja lukee takaisin neljä tavua, jotka sisältävät sekä kosteus- että lämpötilanäytteet; ajuri skaalaa raakalukemat tekniikan yksiköiksi.

Esimerkki:

import time
from machine import Pin, I2C
from hs3003 import HS3003

bus = I2C(1, scl=Pin("P15"), sda=Pin("P14"))
hts = HS3003(bus)

while True:
    print("rH: %.2f%% T: %.2fC" % (hts.humidity(), hts.temperature()))
    time.sleep_ms(100)

Luokat¶

class hs3003.HS3003(bus: machine.I2C, address: int = 0x44)¶

Muodostaa HS3003 -ilmentymän.

bus

Määritetty machine.I2C -väylä, johon sensori on kytketty.

address

Laitteen 7-bittinen I2C-osoite. Oletuksena 0x44, kaikkien HS3003-osien käyttämä tehdasosoite.

humidity() → float¶

Laukaisee mittauksen ja palauttaa suhteellisen kosteuden prosentteina (%RH), laskettuna kaavalla (raw & 0x3FFF) / 16383 * 100.

temperature() → float¶

Laukaisee mittauksen ja palauttaa lämpötilan Celsius-asteina, laskettuna kaavalla (raw >> 2) / 16383 * 165 - 40.
