luokka SSD1351 – näytönohjain¶

SSD1351-luokkaa käytetään SSD1351 OLED -näytönohjaimen alustamiseen. Anna tämän luokan ilmentymä controller-argumenttina SPIDisplay()-luokan rakentajalle.

Rakentajat¶

class display.SSD1351¶

Luo SSD1351-näytönohjainolion.

init(display_controller: display.SPIDisplay) → None¶

Alustaa näytönohjaimen. display_controller-olion on tarjottava display.SPIDisplay.bus_write()-metodi. Tämän kutsuu pääasiallinen näyttöajuri.

ram_write(display_controller: display.SPIDisplay) → None¶

Antaa kirjoita-RAM-muistiin -komennon (SSD1351.WRITE_RAM) näytönohjaimen väylällä. display_controller-olion on tarjottava display.SPIDisplay.bus_write()-metodi.

display_on(display_controller: display.SPIDisplay) → None¶

Antaa näyttö päälle -komennon (SSD1351.DISPLAY_ON) näytönohjaimen väylällä. display_controller-olion on tarjottava display.SPIDisplay.bus_write()-metodi.

display_off(display_controller: display.SPIDisplay) → None¶

Antaa näyttö pois päältä -komennon (SSD1351.DISPLAY_OFF) näytönohjaimen väylällä. display_controller-olion on tarjottava display.SPIDisplay.bus_write()-metodi.

WRITE_RAM: int¶

Kirjoita näytön RAM-muistiin -komento (0x5C).

SET_REMAP: int¶

Aseta uudelleenkartoitus / kaksois-COM-rivitila -komento (0xA0).

DISPLAY_OFFSET: int¶

Aseta näytön siirtymä -komento (0xA2).

DISPLAY_OFF: int¶

Aseta lepotila päälle / näyttö pois päältä -komento (0xAE).

DISPLAY_ON: int¶

Aseta lepotila pois päältä / näyttö päälle -komento (0xAF).

PRECHARGE: int¶

Aseta vaiheen pituus / esilataus -komento (0xB1).

DISPLAY_ENHANCEMENT: int¶

Näytön parannus -komento (0xB2).

CLOCK_DIV: int¶

Aseta etukellon jakaja / oskillaattorin taajuus -komento (0xB3).

PRECHARGE2: int¶

Aseta toinen esilatausjakso -komento (0xB6).

PRECHARGE_LEVEL: int¶

Aseta esilatausjännitteen taso -komento (0xBB).

CONTRAST_ABC: int¶

Aseta kontrasti väreille A, B, C -komento (0xC1).

CONTRAST_MASTER: int¶

Pääkontrastin virranohjaus -komento (0xC7).

MUX_RATIO: int¶

Aseta MUX-suhde -komento (0xCA).

COMMAND_LOCK: int¶

Aseta komentolukitus -komento (0xFD).
