CMUcam4:n graafinen käyttöliittymä#

Mikä CMUcam4GUI on#

CMUcam4GUI on hyödyllinen työkalu, joka helpottaa CMUcam4:n kanssa työskentelyä. CMUcam4GUI:n avulla voit ottaa kuvia CMUcam4:lläsi ja katsella niitä tietokoneellasi sekä käyttää kuvia laskeaksesi väriseurannan rajat kuvassa oleville värillisille kohteille, joita haluat CMUcam4:n seuraavan. CMUcam4GUI:n avulla voit nopeasti määrittää kohteen väriseurannan rajat yhdessä valaistustilanteessa ja laajentaa väriseurannan rajoja tukeaksesi kohteen seurantaa muuttuvissa valaistustilanteissa. CMUcam4GUI:n avulla voit myös nähdä reaaliajassa ilman televisiota, mitä CMUcam4 seuraa, näyttämällä graafisesti CMUcam4:n lähettämien seuranta-, keskiarvo-, histogrammi- ja kehyspakettien sisällön. Lisäksi CMUcam4GUI:n avulla voit leikkiä reaaliajassa kaikilla CMUcam4:n asetuksilla, kuten panoroinnin ja kallistuksen servojen ohjaamisella, väriseuranta-asetuksilla ja matalan tason kameran säädöillä. Lopuksi CMUcam4GUI:n avulla voit hallita CMUcam4:ään liitettyä secure digital -korttia etänä CMUcam4:n kautta.

Windows-asennus#

Määritä CMUcam4GUI Windowsille noudattamalla alla olevia vaiheita:

  1. Lataa CMUcam4GUI.zip -tiedosto

  2. Pura se ja ota sisältö talteen

  3. Aja CMUcam4GUI.exe-tiedosto

Linux-asennus#

Määritä CMUcam4GUI Linuxille noudattamalla alla olevia vaiheita:

  1. Lisää käyttäjänimesi dialout-ryhmään

sudo adduser username dialout

  1. Varmista nyt kirjoittamalla

groups username

  1. Kirjaudu nyt ulos ja sitten takaisin sisään

  2. Lataa CMUcam4GUI.zip -tiedosto

  3. Pura se ja ota sisältö talteen

  4. Aja CMUcam4GUI.sh-tiedosto

Macintosh-asennus#

Määritä CMUcam4GUI Macintoshille noudattamalla alla olevia vaiheita:

  1. Asenna Qt 1.2.1 SDK (Desktop 4.8.1) Macille

  2. Lataa CMUcam4GUI-lähdekoodi

  3. Avaa CMUcam4GUI.pro-tiedosto QtCreatorilla ja vaihda debug-tilasta release-tilaan

  4. Avaa CMUcom4.h-tiedosto ja muuta #define CMUCOM4_FAST_BAUD_RATE arvosta 230400 arvoon 115200 (Macit eivät pysty käsittelemään 230400 BPS:n siirtonopeutta)

  5. Napsauta run

Kysymys: Miksi teillä ei ole valmiiksi rakennettua MAC-sovellusta?

Vastaus: Kwabenalla, CMUcam4:n kehittäjällä, ei ole MACia…

CMUcam4GUI-käyttöopas#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

Valikkorivin ja työkalurivin komennot

Työkalurivin komennot

  • Begin/Reset - Yhdistä CMUcam4:ään

  • End - Katkaise yhteys CMUcam4:ään

  • Sleep Lightly - Aseta CMUcam4 hyvin matalan virrankulutuksen tilaan

  • Sleep Deeply - Aseta CMUcam4 erittäin matalan virrankulutuksen tilaan

  • Send Frame - Ota värikuva ja näytä se katseluikkunassa

  • Send Bitmap - Ota binäärikuva ja näytä se katseluikkunassa

    • Tämä on 80x60-binäärikuva siitä, mitä CMUcam4 seuraa - CMUcam4:n täytyy seurata jotakin, jotta tästä olisi hyötyä

  • Dump Frame - Ota värikuva ja tallenna se SD-kortille

  • Dump Bitmap - Ota binäärikuva ja tallenna se SD-kortille

    • Tämä on 80x60-binäärikuva siitä, mitä CMUcam4 seuraa - CMUcam4:n täytyy seurata jotakin, jotta tästä olisi hyötyä

Valikkorivin komennot

  • Save Frame - Tallenna katseluikkunan värikuva tiedostoon

  • Save Bitmap - Tallenna katseluikkunan binäärikuva tiedostoon

    • Tämä on 80x60-binäärikuva siitä, mitä CMUcam4 seuraa - CMUcam4:n täytyy seurata jotakin, jotta tästä olisi hyötyä

  • Clear Frame - Tyhjentää värikuvan katseluikkunassa

  • Clear Bitmap - Tyhjentää binäärikuvan katseluikkunassa

    • Tämä on 80x60-binäärikuva siitä, mitä CMUcam4 seuraa - CMUcam4:n täytyy seurata jotakin, jotta tästä olisi hyötyä

  • Open - Avaa värikuva käytettäväksi seurantavalintaominaisuuden kanssa

  • Exit - Poistu sovelluksesta

Viewing-välilehti

  • Send Frame Control - Valitse vaaka- ja pystyresoluutio työkalurivin send frame -painikkeelle

    • Et voi liikuttaa sitä, mistä otat kuvaa, sillä aikaa kun send frame -komento on käynnissä

    • Jos kuvanlaatu näyttää huonolta, kytke automaattinen vahvistus ja automaattinen valkotasapaino pois päältä ja varmista, että CMUcam4 katsoo valaistua kohdetta

    • Käytä pienempää resoluutiota kaapataksesi kuvan nopeammin - kuten 160x120

  • Dump Frame Control - Valitse vaaka- ja pystyresoluutio työkalurivin dump frame -painikkeelle

    • Et voi liikuttaa sitä, mistä otat kuvaa, sillä aikaa kun dump frame -komento on käynnissä

    • Jos kuvanlaatu näyttää huonolta, kytke automaattinen vahvistus ja automaattinen valkotasapaino pois päältä ja varmista, että CMUcam4 katsoo valaistua kohdetta

    • Käytä pienempää resoluutiota kaapataksesi kuvan nopeammin - kuten 160x120

  • Opacity Control - Valitse lähetetyn ruudun ja lähetetyn bittikartan peittävyys - välillä 1.00 (100 %) ja 0.00 (0 %)

  • Tracking Control - Valitse katseluikkunasta seurattava väripläntti

    • Valintasuorakulmion tulisi sulkea sisäänsä vain ne pikselit, jotka ovat siinä värialueessa, jota haluat seurata

    • Valittuasi värilaikun napsauta track color -painiketta laskeaksesi värirajat kyseisen värilaikun seuraamiseksi

      • Algoritmi toimii laskemalla punaisen, vihreän ja sinisen värikanavan keskiarvon ja keskihajonnan valintasuorakulmion sisällä

      • Luottamusvälin tason kerroin valitsee, kuinka paljon keskihajonnasta lisätään +/- punaisen, vihreän ja sinisen värikanavan keskiarvoihin - välillä 0, joka on +/- (stdev * 1), ja 99, joka on +/- (stdev * 10)

Tracking-välilehti

Ota Line-tila käyttöön asetuspaneelissa nähdäksesi siniset seuratut pikselit.

Options-välilehti

Disk-välilehti

  • Disk Information - Hae tietoa levyn geometriasta

  • Disk Space - Hae tietoa levyn tilasta

  • Disk Control - Irrota/alusta levy

    • Huomaa: Sinun tulisi irrottaa levy minkä tahansa SD-korttiin liittyvän komennon käytön jälkeen ennen graafisen käyttöliittymän sulkemista

    • Huomaa: format disk -painikkeen napsauttaminen alustaa levyn välittömästi

  • New File - Luo uusi tiedosto tekstillä tai lisää tekstiä tiedostoon

  • New Folder - Luo uusi kansio

  • File Explorer - Käsittele SD-kortin tiedostoja/kansioita tietokoneeltasi

    • Napsauta refresh listataksesi nykyisen hakemiston sisällön

    • Muuta tiedoston tai kansion määritteitä napsauttamalla määritteiden valintaruutuja

      • Siirry kansioon kaksoisnapsauttamalla sitä

      • Avaa tiedosto kaksoisnapsauttamalla sitä

        • Älä yritä avata noin 10 kt:a suurempia tiedostoja (ellet halua odottaa ikuisesti), koska tiedostonsiirto on erittäin hidas, noin 4 kt/s

        • Voit siirtää tiedostoja ja kansioita nimeämällä ne uudelleen uudella polulla, esim. nimeä txt.txt uudelleen muotoon ../txt.txt

Miten värejä rajataan#

  • Siirry asetuksiin

  • Kytke YUV päälle

  • Odota 5 sekuntia

  • Kytke automaattinen vahvistus ja automaattinen valkotasapaino pois päältä

  • Valitse send frame -vaakaresoluutio ja aseta se arvoon 160

  • Valitse send frame -pystyresoluutio ja aseta se arvoon 120

  • Suuntaa CMUcam4 seurattavaan kohteeseen, ja sitten CMUcam4:ää tai kohdetta liikuttamatta ja kummankin ollessa liikkumatta napsauta send frame

  • Odota, kunnes kehys on lähetetty GUI:lle, älä liikuta kohdetta

  • Valitse kohteen värialue (eikä mitään sen alueen ulkopuolelta)

  • Napsauta track selection

  • Napsauta okay oletusarvolla 50

  • Jos vain kohteessa on sinistä piirrettynä, napsauta track selection uudelleen ja muuta 50 arvoon 60

  • Toista edellä mainittu, kunnes työnnät track selection -arvon mahdollisimman korkeaksi; voit lopettaa, kun se on 99. Jos alat rajata ei-toivottuja asioita, olet myös valmis.

  • Siirry sitten asetuksiin… aseta vihreän minimiarvoksi 0 ja vihreän maksimiarvoksi 255 (tämä asettaa Yminin arvoon 0 ja Ymaxin arvoon 255). Jos alat rajata ei-toivottuja asioita, kokeile vähemmän leveitä rajoja, kunnes rajaat vain haluamaasi kohdetta. Voit päivittää rajatun napsauttamalla sendBitmap-painiketta.

  • Ja… sen jälkeen sinulla pitäisi olla kuusi seurantavaihtoehtoa… Lisäksi voit myös halutessasi leventää (V/punainen)- ja (U/sininen)-rajoja manuaalisesti. Tämä auttaa tekemään järjestelmästä kestävämmän erilaisissa valaistusolosuhteissa ja vastaavissa.

Nähdäksesi, mitä CMUcam4 näkee….

  • Kytke line mode päälle (asetuksissa on valintaruutu)

  • Siirry sitten katseluun ja napsauta track color.
