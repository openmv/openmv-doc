OpenMV MicroPython -kieli ja -toteutus¶

MicroPython pyrkii toteuttamaan Python 3.4 -standardin (sekä valittuja ominaisuuksia myöhemmistä versioista) kielen syntaksin osalta, ja useimmat MicroPythonin ominaisuuksista ovat identtisiä ”Language Reference” -dokumentaatiossa osoitteessa docs.python.org kuvattujen kanssa.

MicroPythonin standardikirjasto kuvataan vastaavassa luvussa. Luku MicroPython differences from CPython kuvaa erot MicroPythonin ja CPythonin välillä (jotka koskevat enimmäkseen standardikirjastoa ja tyyppejä, mutta myös joitakin kielitason ominaisuuksia).

Tämä luku kuvaa MicroPythonin toteutuksen ominaisuuksia ja erityispiirteitä sekä parhaita käytäntöjä niiden käyttöön.
