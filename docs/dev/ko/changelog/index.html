OpenMV 변경 이력¶

각 OpenMV 제품은 자체 릴리스 이력을 관리합니다. 관심 있는 항목을 선택하세요 — 모든 항목은 무엇이 새로워졌는지, 무엇이 바뀌었는지, 그리고 (중요한 경우) 정확히 어떻게 마이그레이션하는지를 설명합니다.

OpenMV Firmware

OpenMV Cam에서 실행되는 MicroPython 머신 비전 펌웨어. 새로운 카메라, 이미지 처리, 보드 지원 기능과 스크립트의 API 변경 사항 및 마이그레이션 단계.

릴리스 보기 →

OpenMV IDE

OpenMV Cam에서 스크립트를 작성, 실행, 디버깅하는 데스크톱 개발 환경. 새로운 편집기, 카메라, 도구 기능, 버그 수정, 플랫폼 지원.

릴리스 보기 →

OpenMV Python Package

Camera Protocol V2를 통해 컴퓨터에서 OpenMV Cam과 통신하는 호스트 측 Python 라이브러리 및 openmv 커맨드라인 도구 — 스트리밍, 스크립팅, 벤치마킹.

릴리스 보기 →

OpenMV Arduino RPC Library

CAN, I²C, SPI, UART, USB를 통해 OpenMV Cam을 제어하는 Arduino 라이브러리. 새로운 보드 지원, 버스 인터페이스, 안정성 수정.
