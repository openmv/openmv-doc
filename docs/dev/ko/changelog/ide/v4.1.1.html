v4.1.1¶

v4.1.1은 Examples 메뉴에 초점을 맞춥니다. 항목이 이제 연결된 보드와 센서에 맞게 필터링되며, 중복된 폴더는 접히고 아무것도 연결되지 않았을 때는 명확한 자리표시자가 표시됩니다. 또한 프레임 버퍼 뷰어에 실시간 포커스 메트릭을 추가하고, 번들된 예제를 갱신하며, 보드 펌웨어 4.5.1을 함께 제공합니다. Qt Creator 10.0.1 기반으로 빌드되었으며, 이번 릴리스에는 Qt Creator 리베이스가 없습니다. 예제 필터링은 기본 Examples 메뉴 동작을 변경합니다 – Breaking changes를 참조하세요.

주요 사항¶

  • 예제 필터링은 examples/index.csv 설정에 따라 연결된 보드와 센서에 관련된 예제만 표시하며, Tools 메뉴 토글로 비활성화할 수 있습니다.

  • 포커스 메트릭이 이제 프레임 버퍼 뷰어와 터미널 상태 표시줄에 실시간으로 표시되어 렌즈 초점을 맞추는 데 도움을 줍니다.

  • 더 깔끔한 Examples 메뉴: 중복된 단일 하위 보드 폴더가 접히고, 빈 메뉴 대신 자리표시자가 표시됩니다.

  • 소프트 재부팅 줄 제거로 시리얼 터미널에서 실행할 때마다 콘솔이 어지럽혀지지 않습니다.

  • 번들된 보드 펌웨어가 지원되는 OpenMV 및 Arduino 보드 전반에 걸쳐 4.5.1로 업데이트되었습니다.

새 기능¶

  • 예제 필터링. Examples 메뉴는 이제 항목을 필터링하여 연결된 보드와 센서에 관련된 예제만 표시합니다. 초기 보드/센서 필터링은 filters.txt 설정에 의해 구동되었으며 (ef7190282), 이후 보드별 및 센서별 정규식과 플래튼 컬럼을 포함하는 examples/index.csv로 이동했고 (795cc20e9), 보드 폴더와 센서 유형을 대소문자 구분 없이 매칭하게 되었습니다 (3e976cb9a). 새로운 Tools > Filter examples by board and sensor type 토글(기본 체크됨, 상태 유지)로 필터링을 비활성화할 수 있습니다 (fd3defe00).

  • 포커스 메트릭. 프레임 버퍼 뷰어와 터미널 상태 표시줄이 이제 JPEG로 압축된 프레임에서 계산된 실시간 Focus 값을 표시하여 렌즈 초점을 맞추는 데 도움을 줍니다 (5834e3a41).

기타 변경 사항 및 개선 사항¶

  • 연결되면 Examples 메뉴는 이제 중복된 단일 하위 보드 폴더를 접고, 빈 메뉴 대신 명확한 “No examples found for your board” 또는 “Connect to your board first” 자리표시자를 표시합니다 (795cc20e9).

  • MicroPython MPY: soft reboot 줄이 이제 시리얼 터미널 출력에서 제거되어 실행할 때마다 콘솔을 어지럽히지 않습니다 (b3cf63299).

  • 번들된 예제가 보드 이름 폴더(예: 50-Board-Control)로 재구성되고 MIT 라이선스 헤더로 업데이트되었으며, Image Reader/Writer 예제는 원시 image.ImageIO 비디오 스트림을 사용하도록 단순화되었습니다. 메모리 내 및 디스크 상의 “그림자 제거” 프레임 차분 예제는 제거되었습니다 (1358eeb84, 7b32f2814, 9d89f1c62).

버그 수정¶

  • 특정 부트로더 펌웨어 파일을 강제할 때 항상 보드의 기본 firmware.bin으로 되돌아가는 대신 제공된 펌웨어 경로를 올바르게 사용하도록 수정했습니다 (a362c1c6b).

  • 지나치게 엄격한 RawInputDeviceList 개수 검사로 인해 실패할 수 있었던 Windows에서의 i.MX (RT1060) 장치 검색을 수정했습니다 (49571962d).

  • 누적 벡터를 64비트로 확장하여 크거나 밝은 이미지에서 발생하던 히스토그램 오버플로를 수정했습니다 (5aaf923a3).

  • 번호가 없는 예제 항목이 일관되게 정렬되도록 예제 폴더 정렬을 수정했습니다 (3cb994433).

플랫폼 및 도구 지원¶

  • Qt Creator 기반: 10.0.1.

  • 번들된 보드 펌웨어: 지원되는 OpenMV 및 Arduino 보드 전반에 걸쳐 4.5.0 -> 4.5.1로 업데이트되었습니다 – OPENMV2/3/4/4P, OPENMV PT (Pure Thermal), OPENMV RT1060, ARDUINO GIGA, NICLA VISION, PORTENTA H7, NANO RP2040 CONNECT, NANO 33 BLE SENSE를 포함하며 – UVC 및 부트로더 이미지를 포함합니다 (f6a5b12ba).

  • 새 스크립트 생성 시 이제 HM01B0, HM0360, MT9V0X2, MT9V0X4에 대해 그레이스케일 픽셀 포맷을 대입합니다(기존 MT9V034 항목은 MT9V0X2/MT9V0X4로 대체됨) (ef7190282).

Breaking changes¶

Examples 메뉴는 이제 필터링이 활성화된 경우(새 기본값) 연결된 보드를 필요로 합니다. 연결하기 전까지 메뉴에는 “Connect to your board first to see examples”가 표시되며, 연결되면 연결된 보드와 센서에 맞는 예제만 표시됩니다. 새로운 Tools > Filter examples by board and sensor type 토글을 통해 이를 비활성화할 수 있습니다.
