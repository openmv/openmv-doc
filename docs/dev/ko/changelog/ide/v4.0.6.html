v4.0.6#

v4.0.6은 Qt Creator 10.0.1을 기반으로 빌드된 집중 릴리스로, 명령줄 등록 키 대화 상자, 잠긴 공장 빌드를 위한 일련의 컴파일 타임 FORCE_* 매크로, 그리고 자동 재연결을 표준 사용자 연결 경로를 통해 라우팅하는 수정을 추가합니다. 호환성 깨짐은 없습니다.

주요 내용#

  • 등록 키 대화 상자를 이제 새로운 -form_key_dialog 명령줄 옵션으로 시작 시 요청할 수 있습니다.

  • 공장 빌드 매크로는 일련의 컴파일 타임 FORCE_* 재정의를 추가하여, 잠긴 빌드가 명령줄 인수 없이 실행되도록 합니다.

  • 자동 재연결이 이제 사용자가 시작한 연결과 동일한 경로를 따릅니다.

새로운 기능#

  • 등록 키 대화 상자. 시작 시 사용자에게 OpenMV Cam 등록 키(form key)를 입력하도록 요청하는 새로운 -form_key_dialog 명령줄 옵션을 추가했습니다. 마지막으로 입력한 키는 새로운 LastFormKey 설정을 통해 기억되며, 대화 상자가 취소되거나 비워진 상태로 남으면 애플리케이션이 종료됩니다.(416ca632b)

  • 공장 프로그래밍 빌드 매크로. 공장 프로그래밍 동작을 하드코딩하는 일련의 컴파일 타임 FORCE_* 빌드 매크로를 추가했습니다. 이는 강제 list-ports, 시리얼 번호 필터, 자동 연결, 자동 업데이트, 자동 지우기, 자동 실행, 정지 비활성화, 전체 화면, 리소스 업데이트, form key, form-key 대화 상자, 읽기/스톨 타임아웃 재정의를 다룹니다. 이를 통해 잠긴 공장 빌드가 명령줄 인수 없이 실행될 수 있습니다.(416ca632b)

버그 수정#

  • 자동 재연결이 connectClicked()를 직접 호출하는 대신 연결 작업 신호를 트리거하도록 수정하여, 재연결이 사용자가 시작한 연결과 동일한 경로를 따르도록 했습니다.(416ca632b)

플랫폼 및 도구 지원#

  • Qt Creator 베이스: 10.0.1.

호환성 깨짐#

없음. 이 릴리스는 사용자에게 보이는 워크플로나 동작을 변경하지 않으며, 새로운 옵션과 매크로는 추가적입니다.
