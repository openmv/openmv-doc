다음은 동일한 장면의 두 이미지입니다. 오른쪽 이미지는 RGB 모드로 촬영되었습니다. 왼쪽 이미지는 YUV 모드로 촬영되었습니다.

다음은 “TM”(테스트 모드)에서 촬영한 두 이미지입니다. 오른쪽 이미지는 RGB 모드로 촬영되었습니다. 왼쪽 이미지는 YUV 모드로 촬영되었습니다.

다음은 “BW”(흑백 모드)와 “NG”(네거티브 모드)로 촬영한 선반 사진입니다.

이것들은 매우 흥미로운 사진입니다. 그렇지 않나요?

다음은 다양한 해상도로 촬영한 Kwabena의 사진입니다. 원본을 보려면 각 이미지를 클릭하십시오!

각 사진을 찍기 위해 완전히 완벽하게 가만히 앉아 있어야 했기 때문에 웃고 있지 않습니다. 하지만 문제 없습니다!

다양한 해상도의 “DF”(Dump Frame) 예제 이미지 파일

여기에 YUV 모드에서 동일한 방을 다양한 해상도로 찍은 여러 사진이 있습니다 - 총 16장의 사진.

여기에 RGB 모드에서 동일한 방을 다양한 해상도로 찍은 여러 사진이 있습니다 - 총 4장의 사진.
