CMUcam4 그래픽 사용자 인터페이스#

CMUcam4GUI란 무엇인가#

CMUcam4GUI는 CMUcam4로 작업하는 것을 더 쉽게 만들어주는 유용한 도구입니다. CMUcam4GUI를 사용하면 CMUcam4로 사진을 찍어 컴퓨터에서 보고, CMUcam4가 트래킹하기를 원하는 사진 속 색깔 있는 물체의 컬러 트래킹 경계를 계산하는 데 그 사진을 사용할 수 있습니다. CMUcam4GUI를 사용하면 한 조명 장면에서 물체의 컬러 트래킹 경계를 빠르게 결정하고 변화하는 조명 장면에서 물체 트래킹을 지원하도록 컬러 트래킹 경계를 넓힐 수 있습니다. CMUcam4GUI를 사용하면 CMUcam4가 보내는 트래킹, 평균, 히스토그램 및 프레임 패킷의 내용을 그래픽으로 표시하여 TV 없이도 CMUcam4가 실시간으로 무엇을 트래킹하는지 볼 수 있습니다. 또한 CMUcam4GUI를 사용하면 팬 및 틸트 서보 제어, 컬러 트래킹 옵션, 저수준 카메라 제어 등 모든 CMUcam4 옵션을 실시간으로 다룰 수 있습니다. 마지막으로, CMUcam4GUI를 사용하면 CMUcam4를 통해 원격으로 CMUcam4에 연결된 보안 디지털 카드를 관리할 수 있습니다.

Windows 설정#

Windows용 CMUcam4GUI를 설정하려면 아래 단계를 따르십시오:

  1. CMUcam4GUI.zip 파일을 다운로드하십시오

  2. 압축을 풀고 내용을 추출하십시오

  3. CMUcam4GUI.exe 파일을 실행하십시오

Linux 설정#

Linux용 CMUcam4GUI를 설정하려면 아래 단계를 따르십시오:

  1. dialout 그룹에 사용자 이름을 추가하십시오

sudo adduser username dialout

  1. 이제 다음을 입력하여 확인하십시오

groups username

  1. 이제 로그아웃한 다음 다시 로그인하십시오

  2. CMUcam4GUI.zip 파일을 다운로드하십시오

  3. 압축을 풀고 내용을 추출하십시오

  4. CMUcam4GUI.sh 파일을 실행하십시오

Macintosh 설정#

Macintosh용 CMUcam4GUI를 설정하려면 아래 단계를 따르십시오:

  1. Mac용 Qt 1.2.1 SDK(Desktop 4.8.1)를 설치하십시오

  2. CMUcam4GUI 소스를 다운로드하십시오

  3. QtCreator를 사용하여 CMUcam4GUI.pro 파일을 열고 디버그 모드에서 릴리스 모드로 전환하십시오

  4. CMUcom4.h 파일을 열고 #define CMUCOM4_FAST_BAUD_RATE를 230400에서 115200으로 변경하십시오(Mac은 230400 BPS 보드 레이트를 처리할 수 없습니다)

  5. run을 클릭하십시오

질문: 왜 미리 빌드된 MAC 애플리케이션이 없습니까?

답변: CMUcam4 개발자인 Kwabena는 MAC이 없습니다…

CMUcam4GUI 매뉴얼#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

메뉴 모음 및 도구 모음 명령

도구 모음 명령

  • Begin/Reset - CMUcam4에 연결

  • End - CMUcam4에서 연결 해제

  • Sleep Lightly - CMUcam4를 매우 낮은 전력 상태로 전환

  • Sleep Deeply - CMUcam4를 초저전력 상태로 전환

  • Send Frame - 컬러 이미지를 촬영하여 보기 창에 표시

  • Send Bitmap - 이진 이미지를 촬영하여 보기 창에 표시

    • 이것은 CMUcam4가 트래킹하는 무엇이든의 80x60 이진 이미지입니다 - 이것이 유용하려면 CMUcam4가 무언가를 트래킹하고 있어야 합니다

  • Dump Frame - 컬러 이미지를 촬영하여 SD 카드에 저장

  • Dump Bitmap - 이진 이미지를 촬영하여 SD 카드에 저장

    • 이것은 CMUcam4가 트래킹하는 무엇이든의 80x60 이진 이미지입니다 - 이것이 유용하려면 CMUcam4가 무언가를 트래킹하고 있어야 합니다

메뉴 모음 명령

  • Save Frame - 보기 창의 컬러 이미지를 파일로 저장

  • Save Bitmap - 보기 창의 이진 이미지를 파일로 저장

    • 이것은 CMUcam4가 트래킹하는 무엇이든의 80x60 이진 이미지입니다 - 이것이 유용하려면 CMUcam4가 무언가를 트래킹하고 있어야 합니다

  • Clear Frame - 보기 창의 컬러 이미지 지우기

  • Clear Bitmap - 보기 창의 이진 이미지 지우기

    • 이것은 CMUcam4가 트래킹하는 무엇이든의 80x60 이진 이미지입니다 - 이것이 유용하려면 CMUcam4가 무언가를 트래킹하고 있어야 합니다

  • Open - 트랙 선택 기능과 함께 사용할 컬러 이미지 열기

  • Exit - 애플리케이션 종료

Viewing 탭

  • Send Frame Control - 도구 모음의 전송 프레임 버튼에 대한 가로 및 세로 해상도 선택

    • send frame 명령이 실행되는 동안 사진을 찍고 있는 대상을 움직일 수 없습니다

    • 이미지 품질이 나빠 보이면 자동 게인과 자동 화이트 밸런스를 끄고 CMUcam4가 조명이 비춰진 물체를 보고 있는지 확인하십시오

    • 이미지를 더 빨리 캡처하려면 더 작은 해상도를 사용하십시오 - 160x120 같은

  • Dump Frame Control - 도구 모음의 덤프 프레임 버튼에 대한 가로 및 세로 해상도 선택

    • dump frame 명령이 실행되는 동안 사진을 찍고 있는 대상을 움직일 수 없습니다

    • 이미지 품질이 나빠 보이면 자동 게인과 자동 화이트 밸런스를 끄고 CMUcam4가 조명이 비춰진 물체를 보고 있는지 확인하십시오

    • 이미지를 더 빨리 캡처하려면 더 작은 해상도를 사용하십시오 - 160x120 같은

  • Opacity Control - 전송된 프레임과 전송된 비트맵의 불투명도 선택 - 100%는 1.00, 0%는 0.00 사이

  • Tracking Control - 트래킹할 컬러 블롭을 보기 창에서 선택

    • 선택 사각형은 트래킹하려는 컬러 블롭 내의 픽셀만 둘러싸야 합니다

    • 컬러 블롭을 선택한 후 track color 버튼을 클릭하여 해당 컬러 블롭을 트래킹할 색상 경계를 계산하십시오

      • 이 알고리즘은 선택 사각형 내부의 red, green, blue 색상 채널의 평균과 표준편차를 계산하여 작동합니다

      • 신뢰 구간 수준 계수는 red, green, blue 색상 채널의 평균에 +/- 표준편차의 얼마를 더할지 선택합니다 - +/- (stdev * 1)의 0에서 +/- (stdev * 10)의 99 사이

Tracking 탭

파란색으로 트래킹된 픽셀을 보려면 옵션 패널에서 Line 모드를 활성화하십시오.

Options 탭

Disk 탭

  • Disk Information - 디스크의 지오메트리 정보 가져오기

  • Disk Space - 디스크의 공간 정보 가져오기

  • Disk Control - 디스크 마운트 해제/포맷

    • 참고: 그래픽 사용자 인터페이스를 닫기 전에 SD 카드 관련 명령을 사용한 후에는 디스크를 마운트 해제해야 합니다

    • 참고: format disk 버튼을 클릭하면 즉시 디스크가 포맷됩니다

  • New File - 텍스트가 포함된 새 파일을 만들거나 파일에 텍스트 추가

  • New Folder - 새 폴더 만들기

  • File Explorer - 컴퓨터에서 SD 카드의 파일/폴더와 상호 작용

    • 현재 디렉터리의 내용을 나열하려면 refresh를 클릭하십시오

    • 속성 체크박스를 클릭하여 파일이나 폴더의 속성을 변경합니다

      • 폴더에 들어가려면 더블 클릭하십시오

      • 파일을 열려면 더블 클릭하십시오

        • 파일 전송이 약 4 KB/s로 매우 느리기 때문에 (영원히 기다릴 생각이 아니라면) 약 10 KB보다 큰 파일을 열려고 하지 마십시오

        • 새 경로로 이름을 변경하여 파일과 폴더를 이동할 수 있습니다. 예: txt.txt를 ../txt.txt로 이름 변경

색상 경계를 정하는 방법#

  • 옵션으로 이동

  • YUV 켜기

  • 5초 기다리십시오

  • 자동 게인 및 자동 화이트 밸런스 끄기

  • send frame 가로 해상도를 선택하고 160으로 설정하십시오

  • send frame 세로 해상도를 선택하고 120으로 설정하십시오

  • 트래킹하려는 물체에 CMUcam4를 향하게 한 다음, CMUcam4나 물체를 움직이지 않고 둘 다 움직이지 않는 상태에서 send frame을 클릭하십시오

  • 프레임이 GUI로 전송될 때까지 기다리고, 물체를 움직이지 마십시오

  • 물체의 색상 영역을 선택하십시오(그 영역 밖의 것은 선택하지 마십시오)

  • track selection을 클릭하십시오

  • 기본값 50으로 okay를 클릭하십시오

  • 물체에만 파란색이 그려져 있으면 track selection을 다시 클릭하고 50을 60으로 만드십시오

  • track selection 값을 가능한 한 높게 올릴 때까지 위 과정을 반복하십시오. 99가 되면 멈출 수 있습니다. 원치 않는 것을 경계하기 시작하면 그것도 완료된 것입니다.

  • 그런 다음 options로 가서… green min 값을 0으로, green max 값을 255로 만드십시오(이것은 Ymin을 0으로, Ymax를 255로 설정합니다). 원치 않는 것을 경계하기 시작하면 원하는 물체만 경계할 때까지 덜 넓은 경계를 시도하십시오. sendBitmap 버튼을 클릭하여 경계된 것을 새로 고칠 수 있습니다.

  • 그리고… 그러면 여섯 가지 트래킹 옵션을 갖게 됩니다… 또한 원한다면 (V/빨강) 및 (U/파랑) 경계를 수동으로 넓힐 수도 있습니다. 이는 다양한 조명 조건 등에서 시스템을 더 견고하게 만드는 데 도움이 됩니다.

CMUcam4가 보는 것을 보려면….

  • line 모드 켜기(옵션에 체크박스가 있음)

  • 그런 다음 viewing으로 가서 track color를 클릭하십시오.

CMUcam4GUI 소스#

CMUcam4GUI의 소스 코드를 다운로드하려면 여기를 클릭하십시오.

코드를 컴파일하려면 Qt 1.2.1 SDK(Desktop 4.8.1)가 필요합니다.
