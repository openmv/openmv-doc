Security-Cam#

Security-cam은 이전 이미지와 현재 이미지를 지속적으로 비교합니다. 이미지가 미리 설정된 임계값보다 더 많이 변하면, 이미지가 jpeg로 MMC 카드에 저장됩니다. 이것은 흥미로운 놀라움을 발견하기를 바라며 어딘가에 오랫동안 남겨두기에 좋은 프로젝트입니다.

security-cam의 릴리스 버전에는 그 이후 수정된 버그가 있었습니다. 아래에 수정 사항이 포함된 업데이트된 펌웨어 이미지와 교체용 main.c 파일이 첨부되어 있습니다. 이 수정 사항은 r503에서 subversion에 커밋되었습니다.

보안 카메라 설정

  • 다음이 필요합니다:

    • 전원과 시리얼 케이블이 있는 CMUcam3

    • MMC 또는 SD 카드

    • PC의 MMC/SD 카드 리더

  • FAT 파일 시스템을 사용하여 메모리 카드를 포맷하십시오

    • Windows 기본값인 FAT32를 사용하지 마십시오

  • security-cam.hex 펌웨어 파일을 CMUcam에 플래시하십시오

  • CMUcam3를 끄고 모니터링하려는 영역으로 가져가십시오

  • MMC 카드를 삽입하고 CMUcam3를 켜십시오

    • 카메라 앞에 움직임이 있으면 빨간색 LED가 깜박이는지 확인하십시오

rev 503 버그 수정이 포함된 업데이트된 security-cam_lpc2106-cmucam3.hex 펌웨어 이미지와 업데이트된 main.c 소스 파일이 이 페이지에 첨부되었지만 아카이브에 보존되지 않았습니다.
