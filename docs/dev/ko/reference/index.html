OpenMV MicroPython 언어 및 구현¶

MicroPython은 언어 구문과 관련하여 Python 3.4 표준(이후 버전의 일부 선택된 기능 포함)을 구현하는 것을 목표로 하며, MicroPython의 대부분 기능은 docs.python.org 의 “Language Reference” 문서에 설명된 것과 동일합니다.

MicroPython 표준 라이브러리는 해당 장 에 설명되어 있습니다. MicroPython differences from CPython 장에서는 MicroPython과 CPython 간의 차이점(대부분 표준 라이브러리와 타입에 관한 것이지만 일부 언어 수준 기능도 포함)을 설명합니다.

이 장에서는 MicroPython 구현의 기능과 특성, 그리고 이를 사용하는 모범 사례를 설명합니다.
