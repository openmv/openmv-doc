13.2.7. 마무리#

그것이 바로 OpenMV Viewer입니다: 카메라를 연결하고 제품의 스크립트를 실행하여 프레임 버퍼, 시리얼 출력, 히스토그램을 실시간으로 지켜보십시오 – 카메라가 무엇을 하고 있는지 보여주는 OpenMV IDE의 부분들에서 개발 도구를 제거한 것입니다. OpenMV 제품을 구축하기보다 완성된 제품을 발표 하는 것이 목적일 때 선택할 애플리케이션입니다.

애플리케이션을 변경해야 할 때 – 스크립트 편집, 임계값 조정, 모델 재학습, 설정 조정 – 는 전체 OpenMV IDE 를 사용하십시오. 두 프로그램은 나란히 설치되므로 개발 컴퓨터에 둘 다 둘 수 있습니다: IDE에서 구축하고 Viewer에서 시연하십시오.
