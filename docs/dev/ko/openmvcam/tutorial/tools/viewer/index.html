13.2. OpenMV Viewer#

OpenMV Viewer는 OpenMV IDE 의 읽기 전용 동반 도구입니다. IDE가 OpenMV Cam에서 애플리케이션을 구축하기 위한 것이라면, Viewer는 완성된 애플리케이션을 선보이기 위한 것입니다: 카메라에 연결하여 스크립트를 실행하고 실시간 프레임 버퍼, 시리얼 출력, 히스토그램을 스트리밍합니다 – 다만 개발 도구는 없습니다. 고객, 동료 또는 부스 방문객은 전체 에디터, 머신 비전 튜닝 도구, 또는 구축 중에만 중요한 설정 없이도 제품이 무엇을 하는지 볼 수 있습니다.

Viewer는 OpenMV IDE와 별도로 독립적으로 설치되고 실행되며, 두 프로그램을 나란히 설치할 수 있습니다.

이 창은 카메라가 하고 있는 일을 보여주는 OpenMV IDE 의 부분들을 유지합니다. frame buffer viewer 는 카메라가 마지막으로 캡처한 것을 보여주고, 그 아래의 serial terminal 은 스크립트가 출력하는 모든 것을 보여주며, histogram 은 프레임의 픽셀 통계를 그립니다. 연결 및 실행 컨트롤은 툴바에 있고, 상태 표시줄은 연결된 카메라가 하고 있는 일을 보고합니다. 빠져 있는 것은 애플리케이션을 변경 하는 데 사용할 모든 것입니다: 스크립트 작성 기능, 머신 비전 튜닝 도구, AI 및 모델 도구, 그리고 IDE 자체의 개발자 설정이 모두 제거되었습니다.

편집 도구 하나는 그대로 남아 있는데, 이는 제품을 만든 사람이 아니라 제품을 uses 하는 사람을 위한 것이기 때문입니다. Viewer는 연결된 카메라에 저장된 설정 구성을 열어 폼 형태로 표시할 수 있으므로, 운영자는 IDE 없이도 제품의 설정 – 임계값, 모드, Wi-Fi 비밀번호 – 을 변경할 수 있습니다. 이는 기존 구성을 열기만 하며, 구성을 작성하는 것은 IDE의 작업입니다. 구성이 담을 수 있는 내용은 설정 편집기 를 참조하세요.

이 장의 키보드 단축키는 Windows와 Linux를 기준으로 작성되었습니다. macOS에서는 Ctrl 대신 Cmd 를 사용하십시오.
