13.1. OpenMV IDE¶

OpenMV IDE는 OpenMV Cam으로 작업하기 위한 데스크톱 애플리케이션입니다. IDE 안의 모든 것은 하나의 루프에 기여합니다: MicroPython 스크립트를 편집하고, 카메라에서 실행한 뒤, 실행되는 동안 프레임 버퍼와 직렬 출력이 실시간으로 돌아오는 것을 지켜보는 것입니다. IDE는 Qt Creator 편집기 코어를 기반으로 하며, 애플리케이션 전체가 그 루프를 중심으로 설계되어 있습니다 – 스크립트에는 프로젝트 파일, 빌드 구성, 컴파일러 설정이 필요 없습니다. 스크립트를 작성하고 카메라에서 실행합니다. 그것이 전체 모델입니다.

메인 창에는 네 개의 작업 영역이 있습니다. 편집기는 왼쪽을 채우며 열린 스크립트를 담고, 그 아래의 직렬 터미널은 스크립트가 출력하는 모든 것을 보여줍니다. 프레임 버퍼 뷰어는 오른쪽 위에 자리 잡아 카메라가 마지막으로 캡처한 것을 보여주고, 그 아래의 히스토그램은 프레임 버퍼가 보여주는 것의 픽셀 통계를 그립니다. 맨 왼쪽 가장자리를 따라 늘어선 도구 모음 버튼 열에는 파일, 편집, 연결, 실행 컨트롤이 있고, 아래쪽 가장자리를 따라 있는 상태 표시줄은 연결된 카메라가 무엇을 하고 있는지 보고합니다. 스크립트에 공간이 필요할 때는 어떤 창이든 접을 수 있습니다.

이 장의 키보드 단축키는 Windows와 Linux 기준으로 작성되었습니다. macOS에서는 Ctrl 대신 Cmd 를 사용하세요.
