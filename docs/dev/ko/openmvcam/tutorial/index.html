OpenMV Cam 튜토리얼¶

OpenMV Cam은 임베디드 머신 비전을 위해 설계된 작고 프로그래밍 가능한 MicroPython 기반 카메라입니다. 이 튜토리얼은 카메라로 무엇을 할 수 있는지 처음부터 끝까지 다룹니다. 카메라의 포장을 풀고 OpenMV IDE에서 첫 스크립트를 작성하는 것부터, 모든 스크립트가 의존하는 Python 및 하드웨어의 기초, 카메라의 이미징 스택과 그 위에 구축된 이미지 처리 및 머신러닝 도구, 애플리케이션이 성장하면서 필요로 하는 동시성 및 네트워킹 요소를 거쳐, 마지막으로 동작하는 프로토타입을 출하 가능한 제품으로 만드는 실무까지 다룹니다.

각 장은 새로운 사용자가 순서대로 읽을 수 있도록 작성되었지만, 각 장은 또한 그 자체로 완결되어 있습니다. 이미 Python을 아는 독자는 Python 개요를 건너뛸 수 있고, 이전에 제품을 출하해 본 독자는 곧장 Production으로 넘어갈 수 있습니다. 한 장이 앞에서 다룬 내용을 기반으로 할 때는 해당 부분으로 다시 링크합니다.
