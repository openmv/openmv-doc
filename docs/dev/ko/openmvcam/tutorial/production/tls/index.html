14.4. TLS 인증서 다루기¶

OpenMV Cam의 ssl 모듈(mbedTLS 기반)을 사용하면 스크립트가 암호화되고 인증된 네트워크 연결을 맺을 수 있습니다. 단순 암호화를 넘어서는 작업을 하려면 인증서가 필요합니다. 이 섹션의 페이지들은 인증서가 무엇인지, 카메라가 받아들이는 키 유형과 파일 형식, 개발용 자체 서명 인증서를 만드는 방법과 운영용 CA 서명 인증서를 얻는 방법, 그것들을 카메라에 넣고 원격 서버를 검증하는 방법, 개인 키를 보호하는 방법, 그리고 인증서 만료와 교체가 배포된 장치에 어떤 영향을 미치는지를 다룹니다.
