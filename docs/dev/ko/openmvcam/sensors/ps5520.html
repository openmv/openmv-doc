PS5520 5MP HDR 카메라#

PS5520은 PixArt의 FinePixel 및 HDR 파이프라인을 갖춘 5MP HDR 센서로, 까다로운 다이나믹 레인지의 장면을 위해 설계되었습니다.

전체 데이터시트, 사진, 주문 정보는 PS5520 5MP HDR 카메라 제품 페이지를 참조하세요.

참고

OpenMV N6에서만 지원됩니다.

주요 특징#

  • HDR 모드에서 5MP(2592x1944), 30 FPS

  • 85 dB 다이나믹 레인지, 39 dB SNR의 싱글샷 HDR

사용법#

N6에서는 프레임 속도가 MJPEG VENC 인코더에 의해 설정됩니다.

RGB565로 1280x720(HD)에서 30 FPS HDR 컬러 비디오를 스트리밍합니다:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.HD)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

RGB565로 1920x1080(FHD)에서 15 FPS HDR 컬러 비디오를 스트리밍합니다:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.FHD)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

RGB565로 전체 5MP — 2592x1944(WQXGA2) — 에서 5 FPS HDR 컬러 비디오를 스트리밍합니다:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.RGB565)
csi0.framesize(csi.WQXGA2)
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())
