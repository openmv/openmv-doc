CMUcam4 Grafische Benutzeroberfläche#

Was die CMUcam4GUI ist#

Die CMUcam4GUI ist ein nützliches Werkzeug, das die Arbeit mit der CMUcam4 erleichtert. Die CMUcam4GUI ermöglicht es Ihnen, mit Ihrer CMUcam4 Bilder aufzunehmen, sie auf Ihrem Computer anzuzeigen und die Bilder zu verwenden, um die Farbverfolgungsgrenzen farbiger Objekte im Bild zu berechnen, die die CMUcam4 verfolgen soll. Mit der CMUcam4GUI können Sie schnell die Farbverfolgungsgrenzen für ein Objekt in einer Beleuchtungsszene bestimmen und die Farbverfolgungsgrenzen erweitern, um das Verfolgen des Objekts in wechselnden Beleuchtungsszenen zu unterstützen. Die CMUcam4GUI ermöglicht es Ihnen auch, ohne Fernseher in Echtzeit zu sehen, was die CMUcam4 verfolgt, indem sie den Inhalt von Tracking-, Mittelwert-, Histogramm- und Frame-Paketen, die von der CMUcam4 gesendet werden, grafisch anzeigt. Zusätzlich ermöglicht Ihnen die CMUcam4GUI, in Echtzeit mit allen CMUcam4-Optionen zu spielen, wie etwa der Steuerung der Schwenk- und Neigeservos, den Farbverfolgungsoptionen und den Low-Level-Kamerasteuerungen. Schließlich ermöglicht Ihnen die CMUcam4GUI, die an die CMUcam4 angeschlossene Secure-Digital-Karte über die CMUcam4 aus der Ferne zu verwalten.

Windows-Einrichtung#

Befolgen Sie die folgenden Schritte, um die CMUcam4GUI für Windows einzurichten:

  1. Laden Sie die Datei CMUcam4GUI.zip herunter

  2. Entpacken Sie es und extrahieren Sie den Inhalt

  3. Führen Sie die Datei CMUcam4GUI.exe aus

Linux-Einrichtung#

Befolgen Sie die folgenden Schritte, um die CMUcam4GUI für Linux einzurichten:

  1. Fügen Sie Ihren Benutzernamen zur Gruppe dialout hinzu

sudo adduser username dialout

  1. Überprüfen Sie nun durch Eingabe von

groups username

  1. Melden Sie sich nun ab und dann wieder an

  2. Laden Sie die Datei CMUcam4GUI.zip herunter

  3. Entpacken Sie es und extrahieren Sie den Inhalt

  4. Führen Sie die Datei CMUcam4GUI.sh aus

Macintosh-Einrichtung#

Befolgen Sie die folgenden Schritte, um die CMUcam4GUI für Macintosh einzurichten:

  1. Installieren Sie das Qt 1.2.1 SDK (Desktop 4.8.1) für Mac

  2. Laden Sie den CMUcam4GUI-Quellcode herunter

  3. Öffnen Sie die Datei CMUcam4GUI.pro mit QtCreator und wechseln Sie vom Debug-Modus in den Release-Modus

  4. Öffnen Sie die Datei CMUcom4.h und ändern Sie #define CMUCOM4_FAST_BAUD_RATE von 230400 auf 115200 (Macs können die Baudrate von 230400 BPS nicht verarbeiten)

  5. Klicken Sie auf Run

Frage: Warum haben Sie keine vorgefertigte MAC-Anwendung?

Antwort: Kwabena, der CMUcam4-Entwickler, hat keinen MAC…

CMUcam4GUI-Handbuch#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

Menüleisten- und Werkzeugleistenbefehle

Werkzeugleistenbefehle

  • Begin/Reset - Mit der CMUcam4 verbinden

  • End - Verbindung zur CMUcam4 trennen

  • Sleep Lightly - Die CMUcam4 in einen sehr niedrigen Energiezustand versetzen

  • Sleep Deeply - Die CMUcam4 in einen Ultra-Niedrigenergiezustand versetzen

  • Send Frame - Ein Farbbild aufnehmen und im Ansichtsfenster anzeigen

  • Send Bitmap - Ein Binärbild aufnehmen und im Ansichtsfenster anzeigen

    • Dies ist ein 80x60-Binärbild dessen, was die CMUcam4 verfolgt – die CMUcam4 muss etwas verfolgen, damit dies nützlich ist

  • Dump Frame - Ein Farbbild aufnehmen und auf der SD-Karte speichern

  • Dump Bitmap - Ein Binärbild aufnehmen und auf der SD-Karte speichern

    • Dies ist ein 80x60-Binärbild dessen, was die CMUcam4 verfolgt – die CMUcam4 muss etwas verfolgen, damit dies nützlich ist

Menüleistenbefehle

  • Save Frame - Das Farbbild im Ansichtsfenster in eine Datei speichern

  • Save Bitmap - Das Binärbild im Ansichtsfenster in eine Datei speichern

    • Dies ist ein 80x60-Binärbild dessen, was die CMUcam4 verfolgt – die CMUcam4 muss etwas verfolgen, damit dies nützlich ist

  • Clear Frame - Das Farbbild im Ansichtsfenster löschen

  • Clear Bitmap - Das Binärbild im Ansichtsfenster löschen

    • Dies ist ein 80x60-Binärbild dessen, was die CMUcam4 verfolgt – die CMUcam4 muss etwas verfolgen, damit dies nützlich ist

  • Open - Ein Farbbild öffnen, um es mit der Track-Selection-Funktion zu verwenden

  • Exit - Die Anwendung beenden

Viewing-Tab

  • Send Frame Control - Die horizontale und vertikale Auflösung für die Send-Frame-Schaltfläche in der Werkzeugleiste auswählen

    • Sie können nicht bewegen, wovon Sie ein Bild aufnehmen, während der Send-Frame-Befehl läuft

    • Wenn die Bildqualität schlecht aussieht, schalten Sie die automatische Verstärkung und den automatischen Weißabgleich aus und stellen Sie sicher, dass die CMUcam4 auf ein beleuchtetes Objekt gerichtet ist

    • Verwenden Sie eine kleinere Auflösung, um das Bild schneller zu erfassen – wie 160x120

  • Dump Frame Control - Die horizontale und vertikale Auflösung für die Dump-Frame-Schaltfläche in der Werkzeugleiste auswählen

    • Sie können nicht bewegen, wovon Sie ein Bild aufnehmen, während der Dump-Frame-Befehl läuft

    • Wenn die Bildqualität schlecht aussieht, schalten Sie die automatische Verstärkung und den automatischen Weißabgleich aus und stellen Sie sicher, dass die CMUcam4 auf ein beleuchtetes Objekt gerichtet ist

    • Verwenden Sie eine kleinere Auflösung, um das Bild schneller zu erfassen – wie 160x120

  • Opacity Control - Die Deckkraft des gesendeten Frames und der gesendeten Bitmap auswählen - zwischen 1.00 für 100% und 0.00 für 0%

  • Tracking Control - Einen Farbfleck im Ansichtsfenster zum Verfolgen auswählen

    • Das Auswahlrechteck sollte nur die Pixel innerhalb des Farbklecks umschließen, den Sie verfolgen möchten

    • Klicken Sie nach der Auswahl des Farbflecks auf die Schaltfläche „Track Color“, um die Farbgrenzen für die Verfolgung dieses Farbflecks zu berechnen

      • Der Algorithmus funktioniert, indem er den Mittelwert und die Standardabweichung der roten, grünen und blauen Farbkanäle innerhalb des Auswahlrechtecks berechnet

      • Der Konfidenzintervall-Niveau-Koeffizient wählt aus, wie viel der Standardabweichung +/- zu den Mittelwerten der roten, grünen und blauen Farbkanäle addiert wird - zwischen 0 für +/- (stdev * 1) bis 99 für +/- (stdev * 10)

Tracking-Tab

Aktivieren Sie den Line-Modus im Optionsbereich, um die blau verfolgten Pixel zu sehen.

Options-Tab

Disk-Tab

  • Disk Information - Informationen über die Geometrie des Datenträgers abrufen

  • Disk Space - Informationen über den Speicherplatz auf dem Datenträger abrufen

  • Disk Control - Datenträger aushängen/formatieren

    • Hinweis: Sie sollten den Datenträger nach der Verwendung eines SD-Karten-bezogenen Befehls aushängen, bevor Sie die grafische Benutzeroberfläche schließen

    • Hinweis: Durch Klicken der Schaltfläche zum Formatieren des Datenträgers wird der Datenträger sofort formatiert

  • New File - Eine neue Datei mit etwas Text erstellen oder Text an eine Datei anhängen

  • New Folder - Einen neuen Ordner erstellen

  • File Explorer - Mit den Dateien/Ordnern auf der SD-Karte von Ihrem Computer aus interagieren

    • Klicken Sie auf Refresh, um den Inhalt des aktuellen Verzeichnisses aufzulisten

    • Ändern Sie die Attribute einer Datei oder eines Ordners, indem Sie auf die Attribut-Kontrollkästchen klicken

      • Doppelklicken Sie auf einen Ordner, um ihn zu betreten

      • Doppelklicken Sie auf eine Datei, um sie zu öffnen

        • Versuchen Sie nicht, Dateien zu öffnen, die größer als etwa 10 KB sind (es sei denn, Sie möchten ewig warten), da die Dateiübertragung mit etwa 4 KB/s sehr langsam ist

        • Sie können Dateien und Ordner verschieben, indem Sie sie mit einem neuen Pfad umbenennen, z.B. txt.txt in ../txt.txt umbenennen

Wie man Farben eingrenzt#

  • Gehen Sie zu den Optionen

  • YUV einschalten

  • Warten Sie 5 Sekunden

  • Autogain und Autoweißabgleich ausschalten

  • Wählen Sie die horizontale Auflösung für send frame und setzen Sie diese auf 160

  • Wählen Sie die vertikale Auflösung für send frame und setzen Sie diese auf 120

  • Richten Sie die CMUcam4 auf das Objekt, das Sie verfolgen möchten, und klicken Sie dann, ohne die CMUcam4 oder das Objekt zu bewegen und ohne dass sich beide bewegen dürfen, auf send frame

  • Warten Sie, bis der Frame an die GUI gesendet wird, bewegen Sie das Objekt nicht

  • Wählen Sie den Farbbereich des Objekts aus (und nichts außerhalb dieses Bereichs)

  • Klicken Sie auf die Track-Auswahl

  • Klicken Sie mit 50 als Standardwert auf OK

  • Wenn nur das Objekt blau eingefärbt ist, klicken Sie erneut auf Track Selection und ändern Sie die 50 auf 60

  • Wiederholen Sie das oben Genannte, bis Sie den Track-Auswahlwert so hoch wie möglich schieben, Sie können aufhören, wenn er 99 erreicht. Wenn Sie beginnen, unerwünschte Dinge einzurahmen, sind Sie ebenfalls fertig.

  • Gehen Sie dann zu den Optionen… machen Sie den Green-Min-Wert 0 und den Green-Max-Wert 255 (dies setzt Ymin auf 0 und Ymax auf 255). Wenn Sie anfangen, unerwünschte Dinge zu umranden, versuchen Sie es mit weniger weiten Grenzen, bis Sie nur noch das gewünschte Objekt umranden. Sie können das Umrandete aktualisieren, indem Sie auf die Schaltfläche sendBitmap klicken.

  • Und… dann sollten Sie Ihre sechs Verfolgungsoptionen haben… Zusätzlich können Sie die (V/rot)- und (U/blau)-Grenzen bei Bedarf auch manuell erweitern. Dies trägt dazu bei, das System unter verschiedenen Lichtbedingungen und dergleichen robuster zu machen.

Um zu sehen, was die CMUcam4 sieht….

  • Linienmodus einschalten (in den Optionen gibt es ein Kontrollkästchen)

  • Gehen Sie dann zu viewing und klicken Sie auf track color.
