OpenMV Changelog¶

Jedes OpenMV-Produkt führt seine eigene Versionshistorie. Wählen Sie diejenige, die Sie interessiert — jeder Eintrag beschreibt, was neu ist, was sich geändert hat und (wo es darauf ankommt) genau, wie zu migrieren ist.

OpenMV Firmware

Die MicroPython-Maschinensehen-Firmware für die OpenMV Cam. Neue Kamera-, Bildverarbeitungs- und Board-Support-Funktionen sowie Breaking-API-Änderungen und Migrationsschritte für Skripte.

Versionen anzeigen →

OpenMV IDE

Die Desktop-Entwicklungsumgebung zum Schreiben, Ausführen und Debuggen von Skripten auf der OpenMV Cam. Neue Editor-, Kamera- und Tool-Funktionen, Fehlerbehebungen und Plattformunterstützung.

Versionen anzeigen →

OpenMV Python Package

Die hostseitige Python-Bibliothek und das openmv-Befehlszeilen- tool zur Kommunikation mit einer OpenMV Cam vom Computer über das Kamera- Protokoll V2 — Streaming, Scripting und Benchmarking.

Versionen anzeigen →

OpenMV Arduino RPC Library

Die Arduino-Bibliothek zur Steuerung einer OpenMV Cam über CAN, I²C, SPI, UART und USB. Neue Board-Unterstützung, Bus-Schnittstellen und Zuverlässigkeitskorrekturen.
