14. Produktion¶

Ein funktionierendes Skript auf dem Entwicklungsrechner ist der Anfang, nicht das Ende. Eine Kamera, die in einem Produkt ausgeliefert wird, muss ihre Aufgabe unbeaufsichtigt erfüllen, in einem Netzwerk, das das Skript nicht kontrolliert, ohne dass der Quellcode für denjenigen lesbar ist, der das Gehäuse zerlegt. Die Seiten in diesem Abschnitt behandeln die praktische Arbeit zwischen die Demo läuft und das Gerät ist ausgeliefert.
