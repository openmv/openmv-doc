OpenMV Cam Tutorial#

Die OpenMV Cam ist eine kleine, programmierbare, von MicroPython gesteuerte Kamera, die für eingebettetes maschinelles Sehen konzipiert ist. Dieses Tutorial behandelt von Anfang bis Ende, was man damit machen kann – vom Auspacken einer Kamera und dem Schreiben des ersten Skripts in der OpenMV IDE über die Python- und Hardware-Grundlagen, auf denen jedes Skript aufbaut, bis hin zum Imaging-Stack der Kamera und den darauf aufbauenden Werkzeugen für Bildverarbeitung und maschinelles Lernen, weiter über die Nebenläufigkeits- und Netzwerk-Bausteine, auf die eine Anwendung beim Heranreifen zurückgreift, und schließlich bis zur praktischen Arbeit, einen funktionierenden Prototyp in ein ausgeliefertes Produkt zu überführen.

Die Kapitel sind so geschrieben, dass ein neuer Benutzer sie der Reihe nach lesen kann, doch jedes ist auch in sich abgeschlossen – ein Leser, der Python bereits kennt, kann den Python-Überblick überspringen, und ein Leser, der schon einmal ein Produkt ausgeliefert hat, kann direkt zu Production springen. Wo ein Kapitel auf etwas zuvor Behandeltem aufbaut, wird darauf zurückverwiesen.
