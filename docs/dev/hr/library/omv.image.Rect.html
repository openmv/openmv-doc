class Rect – objekt Rectangle¶

Objekt rect je attrtuple koji vraća Image.find_rects(). Svaki rect opisuje četverostrani četverokut koji je pronašao detektor pravokutnika, zajedno s njegovim graničnim okvirom poravnatim s osima, četiri otkrivena kuta i veličinom jakosti rubova zbrojenom duž oboda pravokutnika.

Poljima se pristupa nazivom atributa (rect.x) ili indeksom (rect[0]). Objekt nema javni konstruktor.

class image.rect¶

Pozovite Image.find_rects() za stvaranje ovog objekta.

x¶

X koordinata gornjeg lijevog kuta graničnog okvira, u pikselima. Cijeli broj. Indeks [0].

y¶

Y koordinata gornjeg lijevog kuta graničnog okvira, u pikselima. Cijeli broj. Indeks [1].

w¶

Širina graničnog okvira, u pikselima. Cijeli broj. Indeks [2].

h¶

Visina graničnog okvira, u pikselima. Cijeli broj. Indeks [3].

magnitude¶

Zbroj jakosti gradijenta rubova praćenog duž oboda pravokutnika. Veće vrijednosti znače jače rubove oko pravokutnika – tretirajte ovo kao ocjenu pouzdanosti. Cijeli broj. Indeks [4].

corners¶

4-torka (x, y) cjelobrojnih torki za četiri kuta otkrivenog četverokuta, poredana u smjeru kazaljke na satu počevši od gornjeg lijevog kuta. Možda nije poravnata s osima – koristite ove kutove za crtanje stvarnog obrisa detekcije. Indeks [5].
