class DACBacklight – DAC-achtergrondverlichting¶

De DACBacklight-klasse bestuurt de achtergrondverlichting van een scherm via een DAC-kanaal. Geef een instantie door als het backlight-argument aan elke display-constructor die een achtergrondverlichtingscontroller accepteert.

Constructors¶

class display.DACBacklight(channel: int, bits: int = 8)¶

Maakt een DAC-aangestuurde achtergrondverlichtingscontroller.

channel geeft het te gebruiken DAC-kanaal (of GPIO-pin) op.

bits geeft de DAC-resolutie op.

deinit() → None¶

Deïnitialiseert de achtergrondverlichtingscontroller.

backlight(value: int) → None¶

Stelt de sterkte van de achtergrondverlichting in van 0-100.
