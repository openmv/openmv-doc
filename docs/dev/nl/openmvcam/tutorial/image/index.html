5. Beeldverwerking¶

Elke pagina tot nu toe heeft pixelgegevens alleen lang genoeg behandeld om ze te verkrijgen: een frame vastleggen, een sensor configureren, bytes van de camera naar RAM verplaatsen. Beeldverwerking draait de richting om – de pixels zijn de invoer, en het doel is wat de applicatie er ook van nodig heeft.

De image-module is de gereedschapskist. Elke bewerking neemt een of twee Image-objecten en geeft ofwel een getransformeerde Image terug – pixels gefilterd, op drempelwaarde gebracht, vervormd, of anders samengesteld – of een lijst van resultaatobjecten: blobs, lijnen, codes, matches, statistieken. De twee varianten laten zich combineren. Een typische pijplijn schoont het vastgelegde frame op, extraheert er een set resultaatobjecten uit, zet die om in metingen, en tekent annotaties terug op het frame voor weergave.
