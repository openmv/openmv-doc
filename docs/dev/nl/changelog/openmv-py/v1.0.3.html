v1.0.3¶

v1.0.3 is de eerste release van het OpenMV Python-pakket — een host-side Python-bibliotheek en de opdrachtregeltool openmv om vanaf je computer met een OpenMV Cam te communiceren via Camera Protocol V2. Als eerste release valt er nog niets van te migreren; deze vermelding vat samen waarmee het werd geleverd.

Hoogtepunten¶

  • De ``openmv`` CLI — verbind met een camera via zijn USB seriële poort en stream zijn livevideo naar je desktop.

  • Scripts uitvoeren op de camera — stuur een .py-script naar de cam en voer het uit vanaf de opdrachtregel.

  • Camera Protocol V2-bibliotheek — een Python-implementatie van het protocol voor gescript, host-side besturing van een camera.

  • pip-installeerbaar — pip install openmv.

Nieuwe functies¶

De opdrachtregeltool openmv:

  • Stream livevideo van een verbonden camera (--port), met een aanpasbare voorbeeldschaal (--scale).

  • Een script uitvoeren op de camera (--script).

  • De protocoldoorvoer benchmarken (--bench).

  • Firmware-ELF-symbolen laden zodat de profiler functienamen toont (--firmware).

  • Stille modus (--quiet) en verbose-debugmodus (--debug).

Brekende API-wijzigingen¶

Geen. v1.0.3 is de eerste release — er is geen eerdere versie om de compatibiliteit mee te breken.
