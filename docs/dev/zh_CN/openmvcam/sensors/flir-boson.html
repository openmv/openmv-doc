FLIR Boson 适配器¶

FLIR Boson 适配器模块将 FLIR Boson 和 Boson+ 热成像机芯连接到 OpenMV Cam（单独销售）。使用它可获得比 Lepton 系列更高分辨率的热成像。

完整的数据手册、照片和订购信息请参阅 FLIR Boson 适配器产品页面。

备注

在 OpenMV N6、RT1062 和 H7 Plus 上受支持。

亮点¶

  • 兼容 FLIR Boson 和 Boson+ 热成像机芯（单独销售）

  • 分辨率高于 Lepton 系列，可呈现更精细的热成像细节

用法¶

通过 csi.CSI 驱动 Boson。设置 framesize 以匹配你所拥有的模块：320 模块使用 csi.QVGA（320x256），640 模块使用 csi.VGA（640x512）:

import csi
import time

csi0 = csi.CSI()
csi0.reset()
csi0.pixformat(csi.GRAYSCALE)
csi0.framesize(csi.QVGA)  # csi.VGA for the 640 module
clock = time.clock()

while True:
    clock.tick()
    img = csi0.snapshot()
    print(clock.fps())

若要直接从传感器获取彩色帧，请将 pixformat 切换为 csi.RGB565，并将 csi.CSI.color_palette 设置为 image.PALETTE_IRONBOW——驱动程序会输出应用了调色板的 RGB565 帧，因此 snapshot() 会直接返回 ironbow 彩色帧:

csi0.pixformat(csi.RGB565)
csi0.color_palette(image.PALETTE_IRONBOW)

备注

FLIR Boson 目前不支持温度测量。
