5. 图像处理¶

到目前为止的每一页处理像素数据都只是为了把它弄到手：捕获一帧、配置传感器、把字节从摄像头搬进 RAM。图像处理则把方向反了过来——像素是输入，而目标则是应用程序需要从中得到的任何东西。

image 模块就是这个工具箱。每个操作接受一个或两个 Image 对象，并返回一个经过变换的 Image——像素经过滤波、阈值化、扭曲，或以不同方式合成——或者返回一个结果对象列表：色块、直线、码、匹配项、统计数据。这两种风格可以组合。一条典型的流水线会先清理捕获的帧，从中提取出一组结果对象，把它们转化为测量值，然后再把标注绘制回帧上以供显示。
