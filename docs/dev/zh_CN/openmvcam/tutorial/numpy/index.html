6. NumPy¶

Image 类涵盖了摄像头本就懂得如何完成的逐像素工作——阈值处理、色块查找、边缘检测以及图像库的其余部分。numpy 模块则涵盖其他所有内容：ADC 读数及其他普通数字的缓冲区、一次性跨整个缓冲区运行的数学运算，以及图像库尚未内置的逐像素变换。

于是有了 ndarray，这是一个单一的类，用于保存一块紧密排列、类型相同的数字。numpy 中的其他所有内容都是作用于 ndarray 的数学函数。将两个数组相加、对其中一个求和、对每个元素取正弦——每一个都是单次库调用，一次性处理整个缓冲区，比等效的 Python for 循环快得多。
