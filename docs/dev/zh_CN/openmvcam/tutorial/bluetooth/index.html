11. 蓝牙¶

网络把摄像头连接到任何可以通过接入点和路由器到达的目标。蓝牙则是相反的取舍。摄像头放弃了覆盖范围，换来的是短距离、低功耗以及无需基础设施——与同一房间内的手机或传感器之间几米的直接无线链路，无需加入 Wi-Fi 网络，无需经路由器转发，功耗低到一颗纽扣电池就能让摄像头存活数月。

其 Python 一侧是 aioble 模块——一个对 asyncio 友好的封装，它把 BLE 无线电的角色、广播、发现以及读/写/通知原语转化为协程。
