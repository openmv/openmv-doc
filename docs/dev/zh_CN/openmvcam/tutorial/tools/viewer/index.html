13.2. OpenMV Viewer#

OpenMV Viewer 是 OpenMV IDE 的只读配套工具。IDE 用于在 OpenMV Cam 上构建应用程序，而 Viewer 则用于*展示*一个已完成的应用：它连接到摄像头，运行其脚本，并以流的形式呈现实时帧缓冲区、串行输出和直方图——但开发工具已被移除。客户、同事或展台访客无需完整的编辑器、机器视觉调优工具，或那些仅在构建时才有意义的设置，即可观看产品的功能。

Viewer 独立于 OpenMV IDE 单独安装和运行，两者可以并存安装。

该窗口保留了 OpenMV IDE 中用于展示摄像头当前状态的部分。frame buffer viewer 显示摄像头最后一次捕获的画面，下方的 serial terminal 显示脚本打印的所有内容，而 histogram 则绘制该帧的像素统计。连接和运行控件位于工具栏上，状态栏则报告已连接摄像头正在做什么。缺失的是所有你用来 更改 应用的功能：脚本编写功能、机器视觉调优工具、AI 与模型工具，以及 IDE 自身的开发者设置都被移除了。

有一个编辑工具确实保留了下来，因为它面向的是 uses 某个产品的人，而不是构建它的人。Viewer 可以打开存储在所连接相机上的设置配置，并将其呈现为一个表单，因此操作员无需 IDE 即可更改产品的设置 -- 某个阈值、某个模式、某个 Wi-Fi 密码 -- 。它仅打开现有配置；创作配置是 IDE 的工作。请参阅 设置编辑器 ，了解配置可以包含哪些内容。

本章中的键盘快捷键是针对 Windows 和 Linux 编写的；在 macOS 上请将 Ctrl 替换为 Cmd。
