13.2.2. 运行脚本#

Viewer 运行的是产品的脚本，而非你正在编辑的脚本。启动按钮——即连接按钮下方的绿色箭头，或按 Ctrl+R——用于运行脚本；脚本运行时，启动按钮会变成停止按钮，用以中止它。脚本可通过命令行交给 Viewer，使其在摄像头一连接时便开始运行，这也是让 Viewer 指向一个已完成应用的通常方式。

脚本运行时，其 print() 输出会流入 serial terminal，它捕获的每一帧都会出现在 frame buffer viewer 中。这正是 Viewer 的全部意义所在：连接摄像头，运行产品的脚本，并观看它运行。

相关的 Tools → Stop Script on Connect/Disconnect 选项（默认开启）会在 Viewer 连接或断开时中止正在运行的脚本，从而使摄像头始终处于已知状态。只要通电，摄像头还会持续运行以 main.py 形式存储在其上的任何脚本，无论 Viewer 是否连接——连接只是让你能够观看它运行而已。
