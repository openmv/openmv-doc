class Circle -- Circle 对象#

圆形对象是由 Image.find_circles() 返回的 attrtuple。每个圆由其圆心、半径以及一个霍夫空间幅值来描述，该幅值反映了图像中的边缘像素为它投票的强度。

字段可通过属性名（circle.x）或索引（circle[0]）访问。该对象没有公开的构造函数。

class image.circle#

请调用 Image.find_circles() 来创建此对象。

x#

圆心 x 坐标，以像素为单位。整数。索引 [0]。

y#

圆心 y 坐标，以像素为单位。整数。索引 [1]。

r#

半径，以像素为单位。整数。索引 [2]。

magnitude#

圆形在霍夫空间中的幅值。值越高表示为此圆投票的边缘像素越多——可将其视为置信度评分。整数。索引 [3]。
