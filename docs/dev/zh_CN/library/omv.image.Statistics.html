class Statistics -- Statistics 对象¶

statistics 对象是由 histogram.get_statistics()（及其别名 histogram.get_stats() / histogram.statistics()）返回，或直接由 Image.get_statistics() 返回的 attrtuple。它用每个通道八种归约值来总结底层 Histogram 的 bin 分布：均值、中位数、众数、标准差、最小值、最大值、下四分位数和上四分位数。

对于灰度数据，只有无前缀的字段（mean、median、mode、stdev、min、max、lq、uq）才携带有用的值；对于 RGB565 数据，它们被复用为 LAB L 通道的归约值 —— 因此 l_* 字段镜像无前缀字段，提供它们是为了对称。a_* 和 b_* 仅对 RGB565 直方图携带有用的值。

字段可通过属性名（statistics.mean）或索引（statistics[0]）访问。该对象没有公开的构造函数。

class image.statistics¶

请调用 histogram.get_statistics() 或 Image.get_statistics() 来创建该对象。

灰度 / LAB L（也作为下面的 l_* 公开）¶

mean¶

直方图的灰度均值。整数 0 -- 255。索引 [0]。

median¶

直方图的灰度中位数。整数 0 -- 255。索引 [1]。

mode¶

直方图的灰度众数。整数 0 -- 255。索引 [2]。

stdev¶

直方图的灰度标准差。整数 0 -- 255。索引 [3]。

min¶

直方图的灰度最小值。整数 0 -- 255。索引 [4]。

max¶

直方图的灰度最大值。整数 0 -- 255。索引 [5]。

lq¶

直方图的灰度下四分位数（第 25 百分位）。整数 0 -- 255。索引 [6]。

uq¶

直方图的灰度上四分位数（第 75 百分位）。整数 0 -- 255。索引 [7]。
