OpenMV MicroPython 内部原理#

本章从为 MicroPython 做贡献的开发者的角度，对 MicroPython 进行了一次概览。它为新手和专家级贡献者提供了关于 MicroPython 实现细节的全面资源。

围绕 MicroPython 的开发通常涉及修改核心运行时、移植或维护一个新库。本指南深入描述了 MicroPython 的实现细节，包括入门指南、编译器内部原理、将 MicroPython 移植到新平台以及实现一个 MicroPython 核心库。
