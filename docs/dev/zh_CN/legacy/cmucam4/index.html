CMUcam4#

CMUcam4 是一款完全可编程的嵌入式计算机视觉传感器。主处理器是 Parallax P8X32A（Propeller 芯片），连接到一个 OmniVision 9665 CMOS 相机传感器模块。有关更多信息，请参阅 wiki。

特性#

  • 完全开源，可使用 Propeller Tool 重新编程

  • 兼容 Arduino 扩展板

    • 带用于 Arduino 和 BASIC Stamp 的支持接口库和演示应用程序

  • VGA 分辨率 (640x480) RGB565/YUV655 彩色传感器

    • 每秒 30 帧的图像处理速率

    • 通过串行或到闪存卡的原始图像转储

      • (640:320:160:80)x(480:240:120:60) 图像分辨率

      • RGB565/YUV655 颜色空间

  • 板载图像处理 (QQVGA 160x120)

    • 在 RGB/YUV 颜色空间中跟踪用户定义的颜色斑块

    • 均值、中位数、众数和标准差数据收集 – 从 40x120 分辨率采样

    • 用于跟踪可视化的分割（阈值化）图像捕获（通过串行或到闪存卡）

      • 80x60 图像分辨率

      • 单色颜色空间

    • 直方图生成（最多 128 个区间）– 从 40x120 分辨率采样

    • 任意图像裁剪（窗口化）

  • µSD/µSDHC 闪存卡插槽，支持 FAT16/32 完整文件系统驱动

    • 带目录和文件操作

  • I/O 接口

    • 双端口舵机控制器（水平和垂直，1us 分辨率，50 Hz 刷新率）

      • 水平和/或垂直舵机通道可配置为 GPIO

    • 指示用户可控 LED（红色）和电源 LED（绿色）

    • TTL UART（最高 250,000 波特 – 默认 19,200 波特）

  • 160x120 分辨率的单色基带模拟视频输出 (NTSC/PAL)，用于跟踪可视化（以 30 FPS 叠加了颜色质心和边界框的分割（阈值化）图像）

  • 用于在 PC 上查看图像的 CMUcam4 GUI

以 PDF 形式 在此处 下载功能列表。

致谢#

本项目得到了 Parallax、Lextronic、SparkFun 和 Semiconductor Research Corporation 的慷慨支持。它来自卡内基梅隆大学的 电气与计算机工程系 和 本科生研究办公室。
