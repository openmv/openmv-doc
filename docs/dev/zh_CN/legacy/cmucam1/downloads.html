下载#

用户手册#

适用于固件 v1.12 的 CMUcam1 用户手册 v2.00 以 PDF 形式提供。它包含零件列表、电路板原理图和软件协议说明。旧版本固件的手册可以在这里找到：v1.11、v1.1、v1.0。

如果您决定尝试更改相机寄存器设置，您可能也有兴趣从 OmniVision 下载 OV6620 相机手册。

构建电路板#

如果您从头开始自己构建它，您将需要下载这个 HEX 代码文件（包含最新固件 v1.12）来对 SX28 芯片进行编程。这是实现手册中描述的功能和串行协议的代码。您可以从 Parallax Inc. 获取 SX-Key 下载器和应用程序。固件的旧 版本可以在这里找到：v1.11、v1.1、v1.0。

Java 接口程序#

我们有一个开源 java 程序，您可以下载它来与您的 CMUcam1 连接并进行测试。此接口程序允许您跟踪物体、分析数据并抓取完整的 CMUcam1 图像。这里是此接口的 Java 程序屏幕截图。

这是压缩的 java 项目，为 Linux 编写。

这是能够自动检测您操作系统的 GUI 版本，它可以在将串行 IO 视为文件（如 Linux）的 UNIX 系统以及 Windows 95/98/2000/NT/XP 下运行。我们建议至少使用运行 Java 1.2.2 或更高版本的 Pentium 2 400 MHz 处理器。

要查看 Cerebellum 微处理器和 BotBall Handyboard 与 CMUcam1 通信的 C 代码示例，请转到 此网页。

这是当前 CMUcam1 原理图（电路板版本 1.23）的高分辨率 JPEG。
