CMUcam1#

CMUcam1 是一款用于移动机器人的低成本、低功耗传感器。您可以使用 CMUcam1 视觉系统执行许多不同类型的板载实时视觉处理任务。由于 CMUcam1 使用串行端口，它可以直接与其他低功耗处理器（如 PIC 芯片）连接。有关更多信息，请参阅 wiki。

在每秒 17 帧的速度下，CMUcam1 可以执行以下操作：

  • 跟踪彩色或明亮物体的位置和大小

  • 测量图像区域的 RGB 或 YUV 统计数据

  • 自动获取并跟踪它看到的第一个物体

  • 使用直接连接的舵机进行物理跟踪

  • 通过串行端口转储一幅完整的图像

  • 转储一个显示被跟踪物体形状的位图

使用 CMUcam1，很容易制作一个可以旋转以跟踪物体的机器人头部。您还可以构建一个到处追球的轮式机器人，甚至到处追着您跑。在 图库 中，您可以看到我们和其他人用 CMUcam1 构建的一些机器人的图片和视频。

我如何使用它？#

您的第一步（在查看 图库 之后）是转到 downloads，您可以在其中获取 CMUcam1 用户手册。它包含完整的零件列表和原理图，以及串行协议和功能的说明。

确保使用我们的 java 接口程序测试您的 CMUcam1。然后您就可以开始构建自己的视觉机器人了！
