CMUcam3 Windows 快速入门指南#

这是为高级用户准备的。有关更详细的说明，请参阅 文档。您可以从 下载 页面获取所有这些软件的捆绑版本。

  • 下载并安装 cygwin

    • http://www.cygwin.com

    • 确保安装 "Make"、"CVS"、"openssh"

  • Go to the Arm GNU Toolchain download site

    • 获取适用于 Windows 的 "EABI" 或 "Bare Metal" 目标

    • 安装

  • Go to Flash Magic

    • 下载并安装 LPC ISP

  • 下载 cc3 源代码

    • 从我们的 下载 页面获取最新稳定快照的 zip

    • 或者 通过键入以下命令从 svn 检出一个全新的副本：

      svn co https://cc3.svn.sourceforge.net/svnroot/cc3/trunk cc3

  • 打开 Cygwin Shell

    • 转到 cc3

      • cd c:进入 c 盘，然后 cd 进入 cc3目录

      • 键入 ls将列出所有文件

    • 键入 make

      • 键入 make clean将删除所有编译后的文件。

      • 在编译单个项目之前，需要在根目录中运行 make

  • 在编程模式下启动

    • 在打开电源时按住按钮，以在编程模式下启动

  • 打开 LPC ISP 并加载编译好的 hex 文件

    • 浏览到 cc3/projects/cmucam2/cmucam2_lpc2106-cmucam3.hex

    • 刷写电路板

    • 关闭 ISP 程序

  • 打开一个终端程序

    • 对电路板进行电源循环

    • 一个绿色电源 LED 应亮起

    • 如果代码运行正确，红色 LED 将熄灭，蓝色和黄色 LED 将微微亮起

    • 确保它打印出启动消息

  • 使用图形用户界面查看图像并对焦相机

    • 选项 1：CMUcam3 帧抓取实用程序

      • 下载 CMUcam3 帧抓取实用程序并安装 .NET（两者都在 下载 页面上）

      • 双击 "CMUcam3 Frame Grabber.exe"

      • 选择正确的 COM 端口并按 "Connect"

      • 单击 "Send" 按钮以检查 "ACK" 消息，从而查看端口是否已连接

      • 单击 "Grab Frame" 以抓取示例图像

    • 选项 2：CMUcam2 GUI

      • 请参阅 CMUcam2 GUI 文档

      • 检查 GUI 是否检测到相机并能够发送帧

      • GUI 无法正确设置串行端口，因此必须在从 ISP 下载和使用 CMUcam2 GUI 之间运行一个终端程序
