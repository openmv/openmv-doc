OpenMV 更新日志¶

每个 OpenMV 产品都保留各自的发布历史。选择你感兴趣的那个——每条记录都描述了有哪些新内容、有哪些变更，以及（在重要的地方）确切的迁移方式。

OpenMV Firmware

运行在 OpenMV Cam 上的 MicroPython 机器视觉固件。新增摄像头、 图像处理和板级支持功能，以及破坏性 API 变更和针对你脚本的 迁移步骤。

查看版本 →

OpenMV IDE

用于在 OpenMV Cam 上编写、运行和调试脚本的桌面开发环境。 新增编辑器、摄像头和工具功能、错误修复以及平台支持。

查看版本 →

OpenMV Viewer

OpenMV IDE 的只读配套工具，用于展示成品。它以流的形式呈现 所连接摄像头的帧缓冲区、终端和直方图并运行脚本，同时移除了 开发工具。

查看版本 →

OpenMV Python Package

主机端 Python 库和 openmv 命令行工具，用于从你的 计算机通过 Camera Protocol V2 与 OpenMV Cam 通信——流传输、 脚本编写和基准测试。

查看版本 →

OpenMV Arduino RPC Library

用于通过 CAN、I²C、SPI、UART 和 USB 控制 OpenMV Cam 的 Arduino 库。新增板级支持、总线接口和可靠性修复。

查看版本 →
