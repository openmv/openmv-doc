v5.0.0¶

v5.0.0 是 OpenMV Viewer 的首个发布版本。OpenMV Viewer 是 OpenMV IDE 的一个独立的只读构建，用于展示基于 OpenMV Cam 打造的成品：它以流的形式呈现摄像头的实时输出并运行脚本，但移除了开发工具，因此客户或最终用户无需完整的开发环境即可看到产品的功能。它独立于 OpenMV IDE 安装和运行。

亮点¶

  • 只读遥测查看器——帧缓冲区、串行终端和直方图的工作方式与 OpenMV IDE 中完全一致，因此你可以实时观察摄像头捕获的画面及其脚本打印的内容。

  • 运行产品的脚本——运行和停止功能可用，能够从磁盘运行脚本，并且在查看器连接或断开时可停止正在运行的脚本，从而使设备始终处于已知状态。

  • 移除了创作工具——编辑器的开发功能、机器视觉创作工具、AI 与模型工具以及开发者设置全部被剥离，只留下一个专注于查看的应用程序。

  • 独立安装——该查看器以 OpenMV Viewer 的品牌进行标识和安装，拥有自己的应用程序名称、设置和身份，因此可与完整的 OpenMV IDE 安装并存。

包含的内容¶

  • 实时摄像头输出——帧缓冲区查看器、串行终端和直方图，具有与 OpenMV IDE 相同的控件。

  • 运行/停止以及连接时停止——从磁盘运行产品的脚本，并在查看器连接或断开时将其停止。

  • 文件和编辑菜单以及首选项——予以保留，以使查看器保持可用（打开文件、调整界面），同时隐藏仅供开发者使用的类别。

  • 固件更新与恢复——固件管理器、引导加载程序恢复以及向摄像头加载 ROM 文件系统的功能都予以保留，因此查看器仍可在现场恢复或重新配置设备。

  • 基准标记生成器——AprilTag、DataMatrix、QR 和条形码生成器都予以保留，因为它们在开发之外也很有用。

移除的内容¶

  • 脚本创作编辑器功能、机器视觉调优工具（阈值编辑器、关键点编辑器等）、AI 与模型训练工具、模型库和数据集编辑器，以及开发者设置类别。固件升级和引导入门提示被抑制，因此演示过程绝不会因更改设备而被打断。

平台与工具支持¶

  • Qt Creator 基础版本： 14.0.2（与 OpenMV IDE 共享）。

  • 支持平台： Windows、macOS 和 Linux，与 OpenMV IDE 分开安装。

破坏性变更¶

无——这是首个发布版本。
