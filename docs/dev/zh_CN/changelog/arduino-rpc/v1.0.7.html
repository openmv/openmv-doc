v1.0.7¶

v1.0.7 新增了开发板和总线支持。

新功能¶

  • 新增了对基于 Arm Mbed 的 Arduino 开发板的支持。

  • 新增了对多个 I²C 控制器接口的支持。

错误修复¶

  • 在 Wire.begin() 之后设置 I²C 的 onRequest / onReceive 处理程序，使外设模式回调能正确注册。

破坏性 API 变更¶

无。
