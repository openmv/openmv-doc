GPIO del puerto de expansión#

El conector de E/S de propósito general permite acceder al segundo UART, a varios pines de control de alimentación y a los pines SPI.

Power Enable – Cuando se pone en nivel bajo, el regulador principal de la CMUcam3 se deshabilita, haciendo que la placa consuma menos de 0.01uA. Todos los dispositivos se apagan y pierden toda la información de estado activo. Cuando la línea se libera o se pone en nivel alto, la placa se reiniciará. De forma predeterminada, el pin se mantiene internamente en nivel alto.

AUX Power – Este pin puede configurarse para alimentar externamente la placa o para alimentar una placa de expansión. De forma predeterminada, el pin está conectado a la fuente interna de 3.3 voltios. Al quitar la resistencia R11 y añadir una resistencia de puente en lugar de R6, el pin se conecta a la alimentación principal antes del regulador de 5 voltios.

  • POWER ENABLE – Poner este pin en nivel bajo cortará la alimentación de toda la placa de la cámara. Esto deshabilitará el procesador, la cámara y la FIFO, haciendo que la placa consuma menos de 1uW de potencia. Este pin se mantiene en nivel alto de forma predeterminada mediante una resistencia pull-up.

  • CAM RESET – Este pin puede utilizarse como E/S externa si no se requiere el estado de la cámara. Normalmente este pin restablece el módulo de cámara y no debe utilizarse.

  • TX2 – El pin de transmisión del UART2 no tiene nivel desplazado y, por tanto, no puede conectarse directamente a un PC ni a un dispositivo externo que no sea TTL. Este pin también puede utilizarse como GPIO.

  • RX2 – El pin de recepción del UART2 no tiene nivel desplazado y, por tanto, no puede conectarse directamente a un PC ni a un dispositivo externo que no sea TTL. Este pin también puede utilizarse como GPIO.

  • CS – Al reiniciar, si este pin se mantiene en nivel bajo, el LPC2106 entrará en modo bootstrap. El reinicio puede inducirse externamente pulsando el pin de habilitación de alimentación. Normalmente este pin será controlado por el controlador MMC. Este pin también puede utilizarse como GPIO o como el chip select del SPI cuando no hay una tarjeta MMC insertada.

  • MOSI – Normalmente este pin es controlado por el controlador MMC. Este pin también puede utilizarse como GPIO o como salida SPI cuando no hay una tarjeta MMC insertada.

  • MISO – Normalmente este pin es controlado por el controlador MMC. Este pin también puede utilizarse como GPIO o como entrada SPI cuando no hay una tarjeta MMC insertada.

  • SCK – Normalmente este pin es controlado por el controlador MMC. Este pin también puede utilizarse como GPIO o como el pin de reloj SPI cuando no hay una tarjeta MMC insertada.
