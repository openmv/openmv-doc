5. Procesamiento de imágenes¶

Hasta ahora, cada página ha manejado datos de píxeles solo el tiempo justo para obtenerlos: capturar un fotograma, configurar un sensor, mover bytes de la cámara a la RAM. El procesamiento de imágenes invierte la dirección: los píxeles son la entrada, y el objetivo es lo que sea que la aplicación necesite de ellos.

El módulo image es el conjunto de herramientas. Cada operación toma uno o dos objetos Image y retorna o bien una Image transformada – píxeles filtrados, umbralizados, deformados o compuestos de manera distinta – o bien una lista de objetos de resultado: manchas (blobs), líneas, códigos, coincidencias, estadísticas. Los dos tipos se componen entre sí. Una tubería típica limpia el fotograma capturado, le extrae un conjunto de objetos de resultado, los convierte en mediciones y dibuja anotaciones de vuelta sobre el fotograma para mostrarlas.
