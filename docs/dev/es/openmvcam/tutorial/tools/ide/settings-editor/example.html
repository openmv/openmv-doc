13.1.13.4.4.1. Leerlo de vuelta#

Tú creaste este diseño, así que conoces la ruta a cada valor. Con el asistente find de la descripción general, nombra la pestaña y el control – find recorre la barra de pestañas y cualquier grupo sin nombre que haya en medio:

enabled = find(config, "Camera", "manual_exposure")["value"]                   # the group's checkbox
exposure = find(config, "Camera", "manual_exposure", "exposure_us")["value"]   # 10000
resolution = find(config, "Camera", "resolution")["value"]                     # "qvga"

El grupo Manual Exposure tiene un name, por lo que es un paso en la ruta – y find(config, "Camera", "manual_exposure") lee su casilla de verificación. El grupo Format no tiene name, por lo que es transparente: se llega a resolution directamente desde la pestaña Camera.

Observa cómo el combobox con un arreglo values devuelve el valor almacenado ("qvga"), no la etiqueta que eligió el usuario ("QVGA") – exactamente lo que un script quiere para tomar decisiones.
