v5.0.0#

v5.0.0 es el primer lanzamiento de OpenMV Viewer. OpenMV Viewer es una compilación independiente y de solo lectura de OpenMV IDE para mostrar un producto terminado creado sobre una OpenMV Cam: transmite la salida en vivo de la cámara y ejecuta un script, pero se han eliminado las herramientas de desarrollo, de modo que un cliente o un usuario final puede ver lo que hace un producto sin el entorno de desarrollo completo. Se instala y se ejecuta de forma independiente de OpenMV IDE.

Aspectos destacados#

  • Un visor de telemetría de solo lectura — el búfer de fotogramas, la terminal serie y el histograma funcionan exactamente igual que en OpenMV IDE, de modo que puedes ver en vivo lo que captura la cámara y lo que imprime su script.

  • Ejecuta el script de un producto — Run y Stop funcionan y ejecutan un script desde el disco, y el script en ejecución se puede detener cuando el visor se conecta o se desconecta, de modo que un dispositivo siempre queda en un estado conocido.

  • Herramientas de creación eliminadas — se han retirado las funciones de desarrollo del editor, las herramientas de creación de visión artificial, las herramientas de IA y modelos y los ajustes de desarrollador, dejando una aplicación de visualización enfocada.

  • Se instala por sí solo — el visor se marca e instala como OpenMV Viewer, con su propio nombre de aplicación, ajustes e identidad, de modo que coexiste con una instalación completa de OpenMV IDE.

Qué se incluye#

  • Salida de cámara en vivo — el visor del búfer de fotogramas, la terminal serie y el histograma, con los mismos controles que OpenMV IDE.

  • Run / Stop y detener al conectar — ejecuta el script del producto desde el disco y detenlo cuando el visor se conecta o se desconecta.

  • Menús File y Edit y Preferences — se conservan para que el visor siga siendo utilizable (abrir un archivo, ajustar la interfaz), mientras que las categorías exclusivas para desarrolladores quedan ocultas.

  • Actualización y recuperación de firmware — se conservan el gestor de firmware, la recuperación del bootloader y la carga de un sistema de archivos ROM en la cámara, de modo que el visor todavía puede recuperar o reaprovisionar un dispositivo sobre el terreno.

  • Generadores de marcadores fiduciales — se conservan los generadores de AprilTag, DataMatrix, QR y códigos de barras, ya que también son útiles fuera del desarrollo.

Qué se elimina#

  • Las funciones del editor para crear scripts, las herramientas de ajuste de visión artificial (editor de umbrales, editor de puntos clave y similares), las herramientas de IA y entrenamiento de modelos, el model zoo y el editor de conjuntos de datos, y las categorías de ajustes de desarrollador. Se suprimen los avisos de actualización de firmware y de introducción para que una demostración nunca se interrumpa para modificar el dispositivo.

Compatibilidad de plataformas y herramientas#

  • Base de Qt Creator: 14.0.2 (compartida con OpenMV IDE).

  • Disponibilidad: Windows, macOS y Linux, instalado por separado de OpenMV IDE.

Cambios incompatibles#

Ninguno — este es el primer lanzamiento.
