התחלה מהירה#

תחילת העבודה#

אנא עקבו אחר השלבים שלהלן:

  1. תצטרך את שני הפריטים הבאים כדי לבדוק את ה-CMUcam4

    • ספק כוח חיצוני 4V עד 9V DC המסוגל לספק לפחות 250 mA

      • זה יכול להיות מ-Arduino, FTDI Breakout Board, FTDI Cable, או Wall Wart

    • טלוויזיית NTSC (או צג טלוויזיה תואם) ו-כבל RCA

  2. הגדרה לבדיקת ה-CMUcam4

    • חבר את טלוויזיית NTSC ל-שקע הקואקסיאלי RCA של ה-CMUcam4 באמצעות כבל RCA

    • חבר את ספק הכוח החיצוני ל-שקע ה-DC Barrel של ה-CMUcam4 (או ליציאות חשמל אחרות)

נורית ה-LED הירוקה של ההספק אמורה להאיר ברגע שתחברו את ספק הכוח החיצוני ל-CMUcam4. לאחר כ-2 שניות, נורית ה-LED העזר האדומה אמורה להאיר כדי לציין שה-CMUcam4 מוכן לפעולה. אם נורית ה-LED הירוקה של ההספק אינה מאירה אנא בדקו שוב את חיבור ספק הכוח החיצוני.

הליך בדיקה#

אנא עקבו אחר השלבים שלהלן:

  1. לחצו והחזיקו את כפתור האיפוס שעל ה-CMUcam4

  2. לחצו והחזיקו את כפתור המשתמש שעל ה-CMUcam4

  3. שחררו את כפתור האיפוס (אל תשחררו את כפתור המשתמש)

  4. המתן עד שנורית העזר האדומה תידלק (2 שניות)

  5. המתן עד שנורית העזר האדומה תתחיל להבהב ב-10 Hz ואז שחרר את ה-user button

    • הטלוויזיה אמורה להידלק (אמורים לראות מסך פתיחה מוצג על הטלוויזיה)

  6. ה-CMUcam4 יסתגל כעת לתנאי התאורה למשך 5 השניות הבאות

    • אל תניח את העצם שברצונך לעקוב אחריו מול ה-CMUcam4 במשך 5 השניות הבאות

  7. המתן עד שנורית העזר האדומה תפסיק להבהב ב-10 Hz

    • ה-CMUcam4 סיים כעת להסתגל לתנאי התאורה

    • פיני סרוו ה-pan וה-tilt אמורים להוציא פולסים של 1500 μs ב-50 Hz

  8. הניחו את העצם שברצונכם לעקוב אחריו מול ה-CMUcam4 ולחצו על כפתור המשתמש

    • אם נורית ה-LED העזר האדומה מתחילה להבהב ב-10 Hz בדוק את חיבור מודול המצלמה OV9665

      • מודול המצלמה OV9665 עשוי להיות פגום וסביר להניח שיש להחליפו

  9. כעת אתה אמור לראות את העצם הנעקב (או דומה) מוצג בטלוויזיה.

    • סרוו-מנועי ה-pan וה-tilt, אם מחוברים, ינסו גם להניע את המצלמה לעבר העצם הנעקב

  10. אנא נסו הליך זה עם עצמים שונים בסביבות שונות כדי לראות מה עובד הכי טוב

הורד את מדריך הבדיקות בפורמט PDF כאן.

כלי תקשורת#

תצטרך אחד מ-ממירי ה-USB לטורי הבאים (או דומה) כדי לתקשר עם ה-CMUcam4:

  • לוח פריצה FTDI 5V

    • תצטרך גם ספק כוח חיצוני המסוגל להזין את ה-CMUcam4

    • אנא חברו את לוח הפריצה FTDI 5V למחבר ה-6 פינים שעל ה-CMUcam4

  • לוח פריצה FTDI 3.3V

    • תצטרך גם ספק כוח חיצוני המסוגל להזין את ה-CMUcam4

    • אנא חברו את לוח הפריצה FTDI 3.3V למחבר ה-6 פינים שעל ה-CMUcam4

  • Prop Clip

    • תצטרך גם ספק כוח חיצוני המסוגל להזין את ה-CMUcam4

    • אנא חברו את Prop Clip למחבר ה-4 פינים שעל ה-CMUcam4

  • Prop Plug

    • תצטרך גם ספק כוח חיצוני המסוגל להזין את ה-CMUcam4

    • אנא חברו את Prop Plug למחבר ה-4 פינים שעל ה-CMUcam4

  • כבל FTDI 5V עם I/O של 5V

    • מומלץ למערכות הסובלות 5V

    • אנא חברו את כבל FTDI 5V עם קלט/פלט 5V למחבר ה-6 פינים שעל ה-CMUcam4

  • כבל FTDI 5V עם I/O של 3.3V

    • מומלץ למערכות הסובלות 3.3V

    • אנא חברו את כבל FTDI 5V עם קלט/פלט 3.3V למחבר ה-6 פינים שעל ה-CMUcam4

תוכנות מסוף טורי מומלצות#

ה-Parallax Serial Terminal הוא כלי שימושי לתקשורת עם מיקרו-בקרים מבוססי טורי כמו שבב ה-Parallax Propeller. זהו מסוף הטורי המומלץ לשימוש לתקשורת עם לוח המצלמה. ה-Parallax Serial Terminal (PST) הוא יישום עצמאי בגודל של פחות מ-1 MB ואינו דורש התקנה לשימוש. PST זמין להורדה מ-Parallax Inc. כאן. עקבו אחר השלבים שלהלן כדי להגדיר את PST:

  1. הריצו PST

  2. עבור אל Echo On וודא שהוא מסומן

  3. עבור אל Com Port ובחר את יציאת ה-COM שאליה מחוברת ה-CMUcam4 מהרשימה הנפתחת

  4. עבור אל Baud Rate ובחר 19200 מהרשימה הנפתחת

    • לחץ Enable אם יש צורך

עבור משתמשים שאינם ב-Windows מומלץ Brad’s SPIN Tool (BST). ניתן להוריד את BST כאן. BST הוא יישום עצמאי מסוג סביבת פיתוח משולבת (IDE) עם ממשק משתמש גרפי (GUI) בגודל של פחות מ-10 MB המיועד לשבב Parallax Propeller ואינו דורש התקנה לשימוש. BST כולל מסוף טורי מובנה וקל לשימוש. עקוב אחר השלבים למטה כדי להגדיר את המסוף הטורי המובנה והקל לשימוש של BST:

  1. הריצו BST

  2. עבור אל View ובחר Serial Terminal מהרשימה הנפתחת

    • מסוף bst אמור לקפוץ – אנא לחצו עליו

  3. עבור אל Baud ובחר 19200 מהרשימה הנפתחת

  4. עבור אל Format ובחר 8 Bits ו-Parity None מהרשימה הנפתחת

  5. עבור אל Port ובחר את יציאת ה-COM שאליה מחוברת ה-CMUcam4 מהרשימה הנפתחת

    • חפש יציאות בשם /dev/tty/USB###

  6. עבור אל Communicate ובחר Connect, Terminal Echo, ו-Reset Propeller מהתפריט הנפתח

    • עבור אל Communicate ובחר Display ASCII לפלט ASCII או Display Hex לפלט hex

הערה: לא ניתן להשתמש בכבלי ה-FTDI 5V עם BST!

הסיבה לכך היא ש BST מושכת את חוט ה-RTS הירוק למצב נמוך ב FTDI 5V Cable כאשר אתה לוחץ Connect. הדבר עוצר את CMUcam4 ללא הגבלת זמן. ניתן להסיר את חוט ה-RTS הירוק מהמחבר כדי לתקן בעיה זו.

השלבים הבאים#

הצעדים הבאים שלך (לאחר שהסתכלת על ה-גלריה) הם לעבור לדף האינטרנט של התיעוד. בנוסף, הקפד לעבור לדפי האינטרנט של הסבר מעקב הצבע, טיפים וטריקים, שאלות נפוצות, ו-פתרון בעיות שלנו למידע נוסף. לבסוף, עבור לדף האינטרנט של ספריית הממשק ל-Arduino והסתכל! ואז אתה מוכן לבנות את הרובוטים החזותיים שלך!
