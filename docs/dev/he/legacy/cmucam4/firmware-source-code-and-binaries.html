קוד מקור של הקושחה#

תוכל להוריד את קוד המקור של הקושחה כאן.

הערה

ארכיון קוד המקור של הקושחה !CMUcam4_Firmware_Code_102.zip לא נשמר בארכיון.

אנא עיינו ב-מדריך התכנות למידע נוסף.

הערה: ה-CMUcam4 אינה מתוכנתת ב-C. היא מתוכנתת בשפה ”דמוית C“ בשם ”SPIN“, lo siento. למידע נוסף אנא ראה את הסעיף P8X32A Propeller Chip Information למטה.

בנוסף, למידע נוסף על ה-Full File System Driver, הכלול בקוד המקור, אנא לחץ כאן.

קובצי קושחה בינאריים#

תוכל להוריד את הקבצים הבינאריים של קוד מקור הקושחה המהודר כאן.

אנא עיינו ב-מדריך התכנות למידע נוסף.

מידע על שבב Propeller P8X32A#

ה-CMUcam4 מונע על ידי ה-P8X32A (שבב Propeller) של Parallax.

(תמונה סופקה על ידי Parallax)

למידע נוסף על שבב ה-Propeller אנא ראה את Propeller P8X32A Q&A.

בנוסף, אנא עיין באתרים הבאים להורדות שימושיות של שבב ה-Propeller:

  • Propeller Downloads - Parallax.

    • תוכל למצוא את ה-Propeller Tool (Windows GUI IDE) או BST (Windows/Macintosh/Linux GUI IDE) כאן.

  • Propeller Object Exchange - Parallax.

    • תוכל למצוא תוכנת קוד פתוח חינמית כאן.

(תמונה סופקה על ידי Parallax)

לעיון עתידי, הנה מדריך P8X32A, גיליון נתונים P8X32A, ו-עיון מהיר P8X32A.

(מסמכים סופקו על ידי Parallax)

הערה: ה-CMUCam4 מגבירה את תדר שבב ה-Propeller ל-96 Mhz (מעל ברירת המחדל של 80 Mhz) כדי להשיג ביצועים טובים יותר (24 MIPS לכל ליבה x 8 = 192 MIPS לעומת 20 MIPS לכל ליבה x 8 = 160 MIPS).

(תמונה סופקה על ידי Parallax)

מידע על מודול המצלמה OV9665#

עקב הסכמי אי-גילוי של OmniVision איננו יכולים לספק את גיליון הנתונים של OV9665 להורדה.

  • אם ברצונך לבקש את גיליון הנתונים מ-OmniVision אנא לחץ כאן.

  • אם ברצונך לבקש תמיכה טכנית מ-OmniVision אנא לחץ כאן.

עם זאת, חילצנו את כל המידע השימושי מגיליון הנתונים של OV9665 והצבנו אותו כאן.
