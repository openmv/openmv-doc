אמולציית CMUcam2#

ל-CMUcam2 יש את היכולות הבאות:

  • עקוב אחר גושי צבע המוגדרים על ידי המשתמש בקצב של עד 50 פריימים לשנייה (קצב הפריימים תלוי בהגדרות הרזולוציה וגודל החלון)

  • עקוב אחר תנועה באמצעות הפרשי פריימים ב-26 פריימים לשנייה

  • מצא את המרכז (centroid) של כל נתוני מעקב

  • אסוף נתוני צבע ממוצע ושונות

  • אסוף היסטוגרמה של 28 תאים עבור כל ערוץ צבע

  • עיבוד תמונות מסוננות בקצה אופקי (לא נתמך)

  • העבר ביטמאפ בינארי בזמן אמת של הפיקסלים הנעקבים בתמונה

  • חלונות תמונה שרירותיים

  • דגימת חסר של תמונה

  • כוונן את מאפייני התמונה של המצלמה

  • פרוק תמונה גולמית (ערוץ יחיד או ערוצים מרובים)

  • עד רזולוציה של 176 x 255

  • תומך בקצבי שידור של: 115,200 57,600 38,400 19,200 9,600 4,800 2,400 1,200 BPS

  • שליטה ב-5 יציאות סרוו

  • מצב עיבוד תמונה מקבילי כעבד (slave) מתוך אפיק מצלמה יחיד (לא נתמך)

  • להשתמש אוטומטית בסרוואים לביצוע מעקב צבע בשני צירים

  • פלט וידאו אנלוגי בשחור-לבן (PAL או NTSC, בהתאם למודול המצלמה שבשימוש)

  • התאמה אישית גמישה של חבילת פלט

  • מצב כיבוי הספק

  • עיבוד תמונה מרובה-מעברים על תמונה מאוחסנת בבאפר

תמונות אמולציית CMUcam2 מהודרות מראש#

Compiled Firmware Image

יש מספר הבדלים בין אמולציית ה-CMUcam2 לבין ה-CMUcam2 האמיתי. להלן רשימה של כל פונקציונליות ה-CMUcam2 עם תכונות שאינן נתמכות מסומנות במחיקה. תכונות רבות שסומנו במחיקה יהיו נתמכות בעתיד הקרוב.

GB קריאת לחצן

SD שינה עמוקה

הבדלי אמולציית CMUcam2#

  • טווח כניסת סרוו

    • הסרוו-מנועים הם כעת ערכי 8 סיביות בטווח 0-255

  • מגשרים

    • עקב היעדר מגשרים ב-CMUcam3, שינויים בקצב הבאוד ובכיוון הסרוו צריכים להתבצע בזמן ההידור - אלה נשלטים על ידי #defines בראש קוד המקור cmucam.c

  • פקודת Set Input (SI pin )

    • מגדיר את ה-pin להיות קלט

  • פקודת Software Color Space (CS value )

    • CS 0 מגדיר את מרחב הצבע לברירת המחדל של החומרה (RGB או YCrCb)

    • CS 1 מבצע המרה תוכנתית למרחב הצבע HSV

      • ההמרה מניחה שהחומרה מוגדרת ל-RGB לקבלת תוצאות נכונות

      • מרחב HSV הוא מרחב שלמים שבו כל הערכים מוקנים בסקאלה בין 16 ל-240

    • הערה: זמין רק ב-r504 ומעלה

  • Set Output יכולה לשלוט בנוריות ה-LED הכחולה והכתומה

    • סרוו-מנועים 2 ו-3 יפסיקו לעבוד

    • SO 2 1 -> כחול דולק

    • SO 2 0 -> כחול כבוי

    • SO 3 1 -> כתום דולק

    • SO 3 0 -> כתום כבוי

  • אין מצב Slave

    • זה אפשרי, אך כרגע איננו תומכים בכך

  • מסכת פלט (OM)

    • תומך רק במנות T ו-S (סוג 0 ו-1)

  • מצב שורה (LM)

    • תומך רק במצב מעקב צבע (LM 0 1)

    • סטטיסטיקת מעקב צבע נוספה ב-r523 (LM 0 2)

  • שליחת JPEG (SJ)

    • שולח מסגרת דחוסה ב-jpeg, כותרת והכל

  • מצב הדגמה

    • מכיוון שהחזקת כפתור התכנות מכניסה את ה-CMUcam3 למצב תכנות, עליכם ללחוץ על הכפתור חצי שנייה לאחר ההפעלה כדי להיכנס למצב הדגמה

  • הפרש פריימים ברזולוציה גבוהה

    • זה עובד כעת באותו אופן כמו הפרשי פריימים ברזולוציה נמוכה, רק ברזולוציה גבוהה יותר
