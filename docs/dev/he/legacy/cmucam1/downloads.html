הורדות#

מדריך למשתמש#

מדריך המשתמש של CMUcam1 v2.00 עבור קושחה v1.12 זמין בפורמט PDF. הוא מכיל רשימת חלקים, תרשים חשמלי של הלוח ותיאור פרוטוקול תוכנה. מדריכים לגרסאות ישנות יותר של הקושחה ניתן למצוא כאן: v1.11, v1.1, v1.0.

אם תחליט להתנסות בשינוי הגדרות אוגרי המצלמה ייתכן שתתעניין גם בהורדת מדריך מצלמת OV6620 מ-OmniVision.

בניית הלוח#

אם אתה בונה אותה בעצמך מאפס תצטרך להוריד את קובץ קוד ה-HEX הזה (המכיל את הקושחה האחרונה v1.12) כדי לתכנת את שבב ה-SX28. זהו הקוד המיישם את הפונקציונליות ואת הפרוטוקול הטורי המתוארים במדריך. ניתן להשיג את מוריד ה-SX-Key ואת היישום מ-Parallax Inc. גרסאות ישנות יותר של הקושחה ניתן למצוא כאן: v1.11, v1.1, v1.0.

תוכנית ממשק Java#

יש לנו תוכנית java בקוד פתוח שתוכל להוריד כדי לתקשר עם ה-CMUcam1 שלך ולבדוק אותה. תוכנית ממשק זו מאפשרת לך לעקוב אחר עצמים, לנתח נתונים וללכוד תמונות CMUcam1 שלמות. הנה צילומי מסך של ממשק זה.

זהו פרויקט ה-java הדחוס שנכתב עבור Linux.

זוהי גרסת ה-GUI המזהה אוטומטית את מערכת ההפעלה שלך. היא פועלת תחת מערכות UNIX התומכות ב-IO טורי כאילו היה קובץ (כגון Linux), וכן תחת Windows 95/98/2000/NT/XP. אנו ממליצים להשתמש לכל הפחות במעבד Pentium 2 400 MHz המריץ Java 1.2.2 ומעלה.

כדי לראות דוגמאות של קוד C עבור המיקרו-מעבד Cerebellum וה-BotBall Handyboard המתקשרים עם ה-CMUcam1, עבור אל דף אינטרנט זה.

תרשים חשמלי#

הנה קובץ JPEG ברזולוציה גבוהה של ה-סכמת CMUcam1 הנוכחית (גרסת לוח 1.23).

לפני שתשאל על קוד המקור, ייתכן שתתעניין בהורדת מדריך המעבד SX28 מ-Ubicom.
