v2.2.0¶

גרסה v2.2.0 מציגה תמיכה ב-AprilTags (image.find_apriltags()), מפעילה CAN2 ב-OpenMV 2 / 3, מוסיפה מודולים נוספים של MicroPython, משחזרת את פונקציות ה-callback של אמולציית Pixy, ומיישרת את הטיפול ב-Ctrl-D/C/A ב-REPL עם לוחות MicroPython סטנדרטיים. היא אינה מציגה שינויי API שוברים — סקריפטים של v2.1.0 רצים ללא שינוי.

עיקרים¶

  • AprilTags — זיהוי סמני fiducial באמצעות image.find_apriltags().

  • CAN2 — הופעל ב-OpenMV 2 וב-OpenMV 3.

  • מודולים נוספים של MicroPython הופעלו.

  • REPL — Ctrl-D / Ctrl-C / Ctrl-A מתנהגים כעת כמו בלוחות MicroPython סטנדרטיים.

  • אין שינויי API שוברים — בצעו reflash והסקריפטים שלכם מ-v2.1.0 ירוצו ללא שינוי.

תכונות חדשות¶

  • AprilTags — נוסף זיהוי סמני fiducial מסוג AprilTag (image.find_apriltags()), עם תיקוני שחרור ובאגים.

  • CAN2 — הופעל CAN2 ב-OpenMV 2 / 3 (עם init0/de-init של CAN ב-main).

  • הופעלו מודולים נוספים של MicroPython ושוחזרו פונקציות ה-callback של אמולציית Pixy.

שינויים ושיפורים נוספים¶

  • ה-REPL מטפל כעת ב-Ctrl-D / Ctrl-C / Ctrl-A באותו אופן כמו לוחות MicroPython סטנדרטיים; עודכנו סקריפטי הדוגמה לזיהוי פנים ולמעקב אחר פנים.

תיקוני באגים¶

  • תוקנו באגים בפענוח QR-code, שלב ה-Haar cascade, ונורית ה-LED של ה-bootloader.

תמיכת חומרה ולוחות¶

  • OpenMV 2 / 3 — CAN2 הופעל; נוסף דגל למתכנת SWD.

שינויי API שוברים¶

גרסה v2.2.0 אינה מציגה שינויי API שוברים ב-Python. סקריפטים שרצו ב-v2.1.0 רצים ללא שינוי ב-v2.2.0 — פשוט בצעו reflash לקושחה.
