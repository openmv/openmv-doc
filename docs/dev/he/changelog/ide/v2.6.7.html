v2.6.7¶

v2.6.7 מעביר את זרימות העבודה של צריבת קושחת DFU והמרת וידאו אל תיבות דו-שיח של מסוף משובץ ב-IDE, מוסיף תכנות מעבד-עזר WiFi מסוג CYW4343 עבור Arduino Portenta H7 ומשפר את הטיפול בחיישני גווני אפור ואת הנחיית מצב ה-DFU. ה-IDE נשאר על Qt Creator 4.0.2, קושחת OpenMV המצורפת מתקדמת ל-3.6.9 ואין שינויים שוברים.

עיקרים¶

  • מסוף צריבת DFU משובץ מריץ את dfu-util בתוך תיבת דו-שיח של IDE הניתנת לשינוי גודל עם פלט חי (שגיאות מודגשות באדום) ולחצן Cancel, במקום חלון מסוף חיצוני של מערכת ההפעלה.

  • קושחת WiFi מסוג CYW4343 מתוכנתת כעת לצד הקושחה הראשית בעת צריבת Arduino Portenta H7 דרך DFU.

  • מסוף המרת וידאו משובץ מריץ את כלי ה-Convert Video של FFmpeg באותה תיבת דו-שיח של IDE הניתנת לשינוי גודל עם פלט חי ולחצן Cancel.

  • הנחיית מצב DFU מעוצבת מחדש משתמשת ברשימה נפתחת של גרסת הקושחה ובתיבת סימון אופציונלית Erase internal file system הזוכרת את בחירתך האחרונה.

  • תמיכה בחיישני גווני אפור מרחיבה את שכתוב RGB565 האוטומטי ל-GRAYSCALE גם ל-MT9V034 בנוסף ל-HM01B0.

תכונות חדשות¶

  • מסוף צריבת DFU משובץ. dfu-util מופעל כעת ישירות אל מסוף QDialog המזרים stdout חי כטקסט רגיל ומדגיש את stderr באדום, עם לחצן Cancel וגיאומטריית חלון שמורה, ומחליף את חלון המסוף החיצוני של מערכת ההפעלה. (c2875d871)

  • תכנות מעבד-עזר WiFi מסוג CYW4343. טעינת קושחה דרך DFU ב-Arduino Portenta H7 מריצה כעת שלב תכנות נוסף הצורב את קושחת ה-WiFi CYW4343 שעל הלוח (CYW4343/cyw4343.bin בכתובת 0x90F00000), המונע על-ידי extraProgramCommands ב-dfu.txt. (62e0ba771, ba31998ce)

  • מסוף המרת וידאו משובץ. כלי ה-Convert Video המבוסס על FFmpeg מריץ כעת את ffmpeg בתיבת דו-שיח של מסוף IDE הניתנת לשינוי גודל עם פלט חי ולחצן Cancel, במקום להפעיל מסוף חיצוני. (5987bb152)

  • אפשרות אספקה במפעל. אפשרות שורת פקודה מוסתרת -form_key, כאשר היא מסופקת, רושמת אוטומטית את הלוח/המזהה של OpenMV Cam מחוברת בשרת של OpenMV במקום להציג את הנחיית הרישום הידנית. (d5740f71e)

שינויים ושיפורים נוספים¶

  • הנחיית מצב DFU מעוצבת מחדש. ההנחיה board detected in DFU mode היא כעת תיבת דו-שיח עם רשימה נפתחת המציגה את גרסת קושחת ההפצה האחרונה המדויקת בתוספת תיבת סימון אופציונלית Erase internal file system, והיא זוכרת את הפעולה האחרונה שנבחרה ואת מצב המחיקה. (0a4d6258c)

  • תיאורי כלי לתיבת סימון המחיקה. לתיבות הסימון של מחיקת מערכת הקבצים במאתחל/DFU יש כעת תיאורי כלי המבהירים שמחיקת כונן הפלאש הפנימי אינה מוחקת כרטיס SD נשלף. (0a4d6258c)

  • שכתוב גווני אפור במקום עבור דוגמאות פתוחות. כאשר דוגמת helloworld שכבר פתוחה מורצת על חיישן HM01B0 או MT9V034, ה-IDE משכתב אותה במקום כדי להשתמש ב-GRAYSCALE כך שתפעל על חיישנים אלה התומכים בגווני אפור בלבד. (4448edb38)

  • הודעה ברורה יותר לאחר צריבה. לאחר עדכון DFU/קושחה, הודעת האישור מסבירה כעת את המקרה שבו main.py שוכתב מבלי למחוק את הכונן, כך שהמצלמה פשוט מריצה את main.py ומעלה את הכונן שלה. (62e0ba771)

  • סקריפטי דוגמה מעודכנים. ulab.py משתמש כעת ב-ulab.numerical עבור mean/std, deep_sleep.py מסדר מחדש את שינת החיישן הרכה לפני עקיפת מסדיר ה-OV7725, וקובץ 19-Low-Power/extint_wakeup.py חדש מדגים יקיצה ממצב עצירה דרך פסיקה חיצונית. (c30fcb7c3)

  • ליטוש תיבת דו-שיח של ההתחברות. טקסט ההתקדמות Connecting... עוצב מחדש עם מעברי שורה לקריאוּת. (0d3619245)

תיקוני באגים¶

  • תוקנו כל תיבות הדו-שיח של Save-As (שמירת תמונה, תבנית, מתאר (descriptor), נקודות מפתח, מודל רשת, ייצוא מערך נתונים, יומן מסוף, שמירת וידאו, פלט המרת וידאו) כך שידחו שמות קבצים ללא סיומת ויבקשו מהמשתמש להוסיף סיומת במקום לשמור קובץ ללא סיומת. (3a3ba3c5e)

  • תוקן רינדור ממשק ספציפי ל-macOS: היפוך צבע טקסט/לחצנים של התפריט (שהתבסס על בהירות רקע התפריט) ועיצוב פס הגלילה הכהה המותאם אישית אינם מוחלים עוד ב-macOS. (39179b6cb, 8b11fbcab)

תמיכת פלטפורמות וכלים¶

  • בסיס Qt Creator: 4.0.2.

  • קושחת OpenMV המצורפת: 3.6.8 ל-3.6.9.

  • קושחת WiFi מסוג CYW4343: בלוב CYW4343/cyw4343.bin חדש מצורף עבור Arduino Portenta H7.

  • סקריפטי חיישן גווני אפור: סקריפטים של New File, דוגמאות ומערכי נתונים מחליפים כעת RGB565 ל-GRAYSCALE ב-MT9V034 בנוסף ל-HM01B0 (VGA ל-QVGA רק עבור ה-HM01B0). (ef0c22efb, d5bdd8890)

שינויים שוברים¶

אין. גרסה זו תואמת לאחור; פרויקטים וסקריפטים קיימים אינם דורשים שינויים.
