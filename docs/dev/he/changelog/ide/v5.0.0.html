v5.0.0¶

v5.0.0 היא מהדורה מרכזית. היא מציגה את OpenMV Viewer — build נפרד לקריאה בלבד להצגת מוצר מוגמר — OpenMV Cam Settings Editor חדש, את פרוטוקול הניפוי v2 של המצלמה עבור מצלמות v5.0.0, ומערך גדול של שיפורים בזרימת העבודה של העורך, הטרמינל, הקושחה וראיית המכונה. היא בנויה על Qt Creator 14.0.2.

קפיצת הגרסה ל-5.0.0 מסמנת את הגעת דור מצלמות v5.0.0 והכלים התומכים בו; פרויקטים ותהליכי עבודה קיימים של v4 ממשיכים לעבוד ללא שינוי.

עיקרים¶

  • OpenMV Viewer — וריאנט build חדש המותקן בנפרד שמריץ את ה-IDE כמציג טלמטריה נעול לקריאה בלבד. הוא מציג את מאגר הפריימים של המצלמה, את הטרמינל הטורי וההיסטוגרמה, ויכול להריץ סקריפט קבוע, אך מסתיר את כלי הכתיבה, ה-AI/מודלים וההגדרות, כך שלקוח יכול להדגים מוצר הבנוי על OpenMV ללא סביבת הפיתוח המלאה.

  • OpenMV Cam Settings Editor — עורך גרפי לקובץ תצורת ההגדרות המובנה של המצלמה, כך שתוכלו לצפות ולשנות את הגדרות הלוח מתוך טופס במקום לערוך את הקובץ ידנית.

  • פרוטוקול ניפוי v2 של המצלמה — מצלמות v5.0.0 משתמשות בפרוטוקול ניפוי חדש ועמיד יותר עם אימות CRC של הכותרת, התאוששות מפערי רצף, ותצוגת קצב-פריימים מפוצלת המדווחת בנפרד על ה-FPS של המצלמה ועל ה-FPS של תצוגת ה-IDE.

  • טעינת קושחה מותאמת מקובץ .zip — מתקין הקושחה מקבל כעת ישירות חבילת .zip של פלט ה-build של הלוח, מה שמקל על צריבת תמונת קושחה מותאמת או שנבנתה מקומית.

  • רישום ניפוי של הפרוטוקול בטרמינל הטורי — מתג בטרמינל הטורי רושם את פרוטוקול הניפוי של המצלמה בשורות קריאות עם תגיות אמוג’י לאבחון בעיות חיבור.

  • תפריט עזרה ותיעוד בתוך העורך שעוצבו מחדש — תת-תפריט OpenMV Docs מחליף את הקישורים השטוחים הישנים, F1 פותח את התיעוד עבור סמל ה-Python שמתחת לסמן, ריחוף מעל שם API מציג את התיעוד שלו בתוך השורה, ותיבת הדו-שיח About עברה מודרניזציה.

תכונות חדשות¶

  • וריאנט build של OpenMV Viewer. build במצב-מציג כפוי, ממותג ומותקן בשם OpenMV Viewer, עם מזהה הרצה, מאגר הגדרות ומזהה חבילה משלו כך שהוא מתקיים לצד ה-IDE המלא. הוא מוצג כמציג טלמטריה לקריאה בלבד: Run/Stop ועצירה-בחיבור עובדים ומריצים סקריפט מהדיסק (באמצעות -auto_run), תפריטי File/Edit וה-Preferences נשארים, והתאוששות קושחה/ROMFS נשמרת, בעוד שכלי הכתיבה, כלי ראיית המכונה למפתחים, קטגוריות ההגדרות למפתחים ומשאבי AI/מודלים/תיעוד מוסרים מה-build ומהממשק. ROMFS הופך לטעינה-בלבד ומקודם לתפריט Tools, ותיבות הדו-שיח של השלמת עדכון הקושחה הופכות לגנריות (07722e78a8b, 5cd12ee260e, b97bf5447e4, 952b86de391, 1103f7988ed, f0f60261485, c743d5d67b9, b4bcdc2d71a, a37c032b6ff, 480e5668d29, 11c064cac93).

  • OpenMV Cam Settings Editor — עורך GUI לקובץ תצורת ההגדרות של המצלמה, הנפתח מתפריט Tools (issue #205) (f55a0d7fa58, 2d7f80b03cf).

  • פרוטוקול ניפוי v2 של המצלמה עבור מצלמות v5.0.0, עם אימות CRC של הכותרת לפני שסומכים על שדה כלשהו בכותרת, התאוששות מפערי רצף שמתגלגלת קדימה מעל חבילות שאבדו במקום להיתקע, וקריאת קצב-פריימים המפוצלת לקצב של המצלמה ולקצב של תצוגת ה-IDE (c377bdbaea8, a3f0488ddcb, 29108dd40f7, c15e6c02789).

  • טעינת קושחה מותאמת מקבלת חבילת .zip — בחרו את קובץ ה-zip של פלט ה-build של הלוח וה-IDE צורב את הקושחה (וה-ROMFS) שהוא מכיל (55e4eb2d900).

  • רישום ניפוי בטרמינל הטורי — כפתור מפעיל ומכבה רישום ניפוי מדורג של פרוטוקול המצלמה, כאשר כל שורה מתויגת באמוג’י ומוצגת בגופן האמוג’י לקריאות (09990c62810, 42e23c3c11e, 997705548c7, d2208de77d4).

  • Copy/Convert Current Script — פעולת Tools שמעתיקה או ממירה את הסקריפט הנוכחי, עם הגנה מפני שמירת main.py/boot.py מהודר (81a954f33cd).

  • עזרת תיעוד ב-F1 על סמל ה-Python שמתחת לסמן, המשתמשת באותו פותר סמלים כמו חלוניות הריחוף (c341330bc16).

  • חלוניות תיעוד בתוך העורך — ריחוף מעל שם API של OpenMV מציג את התיעוד שלו בתוך השורה, נפתר מתוך מקטעי ה-.pyi המצורפים ומהתיעוד של המודול עצמו, כך שתוכלו לקרוא מה עושים פונקציה, מתודה, קבוע או מודול מבלי לעזוב את הסקריפט (7ebde92d2a6, f225824ba13, 59ad01a364a, 02c1e61a2a6).

  • מתן שם למצלמה — כינוי מקומי ל-IDE שאפשר לתת למצלמה מחוברת, המוצג במקום שם היציאה הטורית כך שאפשר להבחין בין מצלמות זהות (c600158ca34).

  • עמוד קיצורי המקלדת — עמוד הקיצורים ב-Preferences מוצג כעת ומסונן לפעולות שאליהן אפשר באמת להגיע, עם מקור ופריסה ברורים יותר (4f5eae52e8e, 2d472fe5b40).

שיפורי עורך, טרמינל וממשק¶

  • תפריט העזרה עוצב מחדש לתת-תפריט OpenMV Docs (בהשמטת קישורי ה-pinout השטוחים הישנים), עם הוספת קישורים להערות המהדורה לנתיבי גרסאות המהדורה הרשמית (e1337ff67c4, 9eacc0994e6).

  • תיבת הדו-שיח About עודכנה — שנת זכויות יוצרים אוטומטית, טבלת לוגואים של שותפים, וקישור הערות-מהדורה בשורה אחת (d89fb74138d).

  • חלונית Frame Buffer מציגה כעת את קריאת הרזולוציה ותווית מצב של מאגר הפריימים לצד כפתור ה-JPG הקבוע; תווית המצב מוצגת באפור כך שהיא לא נראית ניתנת ללחיצה (cb735f82337, 376b17d4e77, 39c6ffc3b26).

  • חלונית ההיסטוגרמה — הגרפים מוחלקים ב-spline, החלונית יכולה להתכווץ (השמטת תוויות וגרפים ניתנים לכיווץ), ותווית הערוץ יושבת לצד הסטטיסטיקות (327324ef7e9, 9f8c7035483, d204e93d5e4).

  • מחיצות פאנל מצומדות — גרירה של מחיצה עם Ctrl/Cmd מזיזה יחד את מפצלי העורך/טרמינל ומאגר-הפריימים/היסטוגרמה, וידיות המפצל הקטן מקבלות הילה בריחוף עם אזור אחיזה רחב יותר (379dee250d6, 96b83984342).

  • הטרמינל הטורי — הגלילה האוטומטית אינה מוצמדת מחדש בכל מקטע פלט, כך שגלילה מעלה במהלך פלט מהיר נשארת במקומה; הטרמינל יכול להריץ קובץ מהדיסק כשאין עורך (af67a076e95, bc132aef536).

  • פקודות התיעוד מקבלות שם לפי עמוד במקום לפי אינדקס (a3201e9a4b4).

  • קנה מידה High-DPI / שברי — סמלים, שכבות-על ומסך הפתיחה מוצגים כראוי בגורמי קנה מידה שבריים (125%, 150% וכדומה), והחלון הראשי זוכר את הגיאומטריה והמסך שלו בין הפעלות מחדש בהגדרות ריבוי-מסכים (c60711e4950, 60901b6e13f, f241d2ea8ce, 54a1bc31fbd).

קושחה, לוחות וכלים¶

  • לוח יכול לקבע את גרסת הקושחה האחרונה שנשלחה עבורו, כך שלוח שאינו מקבל עוד קושחה חדשה יותר אינו מסומן כמיושן (bd987d32f42).

  • טקסט הסטטוס של קושחה מיושנת צבוע כעת באדום כדי לבלוט (9b18d96bac5).

  • רמז להתאוששות bootloader — אי-התאמה של bootloader.bin מרמזת כעת על כניסה למצב התאוששות bootloader (d46b998c76e).

  • הפעלת וידאו עמידה יותר: נתיב ה-FFmpeg מוצג כאשר הכלים אינם מצליחים להפעיל אותו, ההודעה ברורה יותר כאשר ארכיטקטורה אינה תומכת בהפעלה, פורמטי הקלט נשאלים באמצעות ffmpeg -demuxers, ונוספו חלופות טרמינל ל-Linux (0cc0e74a438, a75ec4898d0, 28e1ef10326, 51651780723).

  • Dataset Capture Script עודכן למודול csi (41a781651e4).

  • מתקין Windows — מראה האשף שופר (250599d9d9e).

תיקוני באגים¶

  • התקנת קושחת-פיתוח עבור לוחות .lst (AE3) מעמידה כעת את הקבצים הבינאריים שהרשימה מפנה אליהם בתיקיית הצריבה, והתקנת הפיתוח/המותאמת מעמידה את תמונת ה-ROMFS של החבילה תוך ניקוי תחילה של כל romfs*.img ישן, כך שה-ROMFS שהורד נצרב במקום שאריות dump של העורך (3cf56e98668, 43bedc8c591).

  • אזהרות libjpeg עבור פריימים קטועים/פגומים מסוננות מהיומן כך שזרם אובד לא מציף את הטרמינל (6e2a8bed6cf, a5166abbdc1).

  • רשימת הדוגמאות אינה מציגה עוד את ה-index.json הפנימי (14ac8ca48a7).

  • חלונות קופצים של שדרוג קושחה וקליטה מודחקים במצב מציג כך שהדגמת מוצר לעולם אינה מבקשת לדרוס את הקושחה של הלקוח (7b8528c3ad6).

  • קיצורי מקלדת תלויים וכאלה המיועדים לכתיבה בלבד שנותרו מתפריטים שהוסרו — הושבתו (be3bee8bb58, 27d1afc8705).

לוקליזציה¶

  • ארבע שפות חדשות — אינדונזית, רומנית, תאית וויאטנמית — המביאות את ה-IDE ל-37 שפות מתורגמות, וכל קטלוג עודכן, תורגם ונוקה בכל השפות (fcf04090843, c830e6c4de3, 5e22679923f).

שורת הפקודה¶

  • כינויי אפשרויות ``–word`` בסגנון GNU מתקבלים כעת לצד הצורות עם מקף יחיד, ופלט -help מציין את השקילות (66414213dd9, 768c5d0e159).

תמיכה בפלטפורמות וכלים¶

  • בסיס Qt Creator: 14.0.2.

  • גרסת OpenMV IDE הועלתה ל-5.0.0.

  • חדש: וריאנט ה-build OpenMV Viewer המותקן בנפרד.

  • תמיכה ב-Python 3.14 — מנתח ה-parso המצורף עודכן ל-0.8.7, מה ששומר על השלמת קוד ושרת השפה עובדים עם תחביר ה-Python העדכני ביותר (578a4e39d97).

שינויים שוברים¶

אין עבור פרויקטים קיימים. פרוטוקול הניפוי v2 חל על מצלמות v5.0.0; מצלמות מדור v4 ממשיכות להשתמש בפרוטוקול הקיים, וסקריפטים ותהליכי עבודה קיימים אינם מושפעים.
