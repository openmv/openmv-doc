v2.2.0¶

v2.2.0 מרחיבה את הדוגמאות והכלים המצורפים, מוסיפה שמירת יומנים ושינוי קנה מידה של וידאו ל-IDE, מחזקת את נתיב הצריבה דרך DFU, ומרעננת את קושחת המצלמה המצורפת ל-3.3.1. היא נשארת מבוססת על Qt Creator 4.0.2, ואין שינויים שוברים לפרויקטים קיימים.

עיקרי דברים¶

  • כפתורי Save שנוספו למסוף הטורי ולחלוניות פלט ההודעות/יומן מייצאים את תוכנם לקובץ .txt.

  • שינוי קנה מידה של וידאו מאפשר לך להגדיר רוחב חדש (תוך שמירה על יחס הממדים) בעת שמירה או המרה של וידאו מוקלט.

  • עורך הסף מקבל כפתור Reset Sliders לשחזור ספי grayscale או LAB לטווחי ברירת המחדל המלאים שלהם.

  • דוגמאות חדשות עבור מגני IMU, Distance, TV ו-Global Shutter, רדיומטריית Lepton, שינוי תדר ה-CPU, ומודל למידת מכונה של chars74k.

  • קושחת מצלמה מצורפת עודכנה ל-3.3.1 ונתיב הצריבה דרך DFU חוזק לשדרוגים אמינים יותר.

תכונות חדשות¶

  • נוסף כפתור Save למסוף הטורי של OpenMV ולחלוניות פלט ההודעות/יומן של ה-Core, המייצא את תוכן היומן לקובץ .txt. (fba4cd119)

  • בעת שמירה או המרה של וידאו מוקלט ה-IDE שואל כעת האם לשנות את קנה המידה שלו ומאפשר לך להזין רוחב חדש (תוך שמירה על יחס הממדים), המועבר ל-ffmpeg דרך -vf scale. (db13b4f2f)

  • כלי עורך הסף למעקב צבע קיבל כפתור Reset Sliders המשחזר את ספי ה-grayscale (0-255) או LAB (L 0-100, A/B -128..127) לטווחי ברירת המחדל המלאים שלהם. (a16d7a613)

  • נוספו תיקיות דוגמה חדשות של IMU Shield, Distance Shield ו-TV Shield (imu_read.py, distance_read.py, tv.py) בנוסף לדוגמת MAVLink optical-flow מעודכנת. (f4468637b)

  • הורחב מערך הדוגמאות המצורף עם שינוי תדר ה-CPU, טמפרטורת אובייקט ברדיומטריית Lepton, Global Shutter (קצב פריימים גבוה/מופעל בטריגר), דוגמאות ציור חדשות (אליפסה/keypoints) ומסנני תמונה (gamma/negative), בנוסף לעוזרי 100-Libraries (MQTT, BNO055, VL53L1X). (03a5696ae)

  • נוספה דוגמת זיהוי תווים Machine Learning של chars74k ומודל רשת נוירונים חדש של chars74k, ומודל ה-smile המצורף רוענן. (9d5198204)

  • צורפה חבילת אינטגרציה של ROS (Robot Operating System) (openmv_cam) להפצת ה-IDE, וחבילת ממשק LabVIEW OpenMV המצורפת עודכנה. (b43921a48, a209fdc0c)

שינויים ושיפורים נוספים¶

  • לאחר ההרצה הראשונה של דוגמת helloworld, תיבת דו-שיח חד-פעמית More Examples מפנה כעת משתמשים אל File -> Examples (עם תיבת סימון ”Don’t show this again“), והודעת לאחר שדרוג הקושחה נוסחה מחדש כדי לציין שנורית ה-RGB LED מהבהבת בכחול עם השלמת הבדיקה העצמית. (06e4f7436)

  • נתיב הצריבה דרך DFU חוזק: pydfu מנסה כעת שוב ושוב לנקות סטטוס עד DFU_IDLE ומעלה את פסק הזמן להעברת בקרה ב-USB ל-5000ms, וכללי ה-udev של Linux נכתבו מחדש כדי להתאים באופן אמין להתקני ה-DFU (0483:df11) והטורי (1209:abd1) של OpenMV וליצור קישורי symlink של /dev/openmvdfu ו-/dev/openmvcam. (49e40e8f9)

  • הודעות שגיאת זמן ריצה של MicroPython המדווחות ב-IDE מועברות כעת דרך tr() כדי שניתן יהיה לתרגמן. (ff8bcf094)

  • תיעוד ההפניות המצורף הלא-מקוון בפורמט HTML רוענן, בתוספת עמודי מודול חדשים omv.cpufreq ו-omv.tv ועדכון העמודים omv.fir, omv.image ו-omv.sensor, בנוסף לעמודי סקירת מדריך וארכיטקטורת מערכת חדשים. (33deb09a5)

תמיכת פלטפורמה וכלים¶

  • בסיס Qt Creator: 4.0.2.

  • קושחת מצלמת OpenMV מצורפת: עודכנה מ-3.2.0 ל-3.3.1 עבור OPENMV2/OPENMV3/OPENMV4 (בינאריי firmware, bootloader, openmv ו-UVC). (8a3230df4)

  • מנהל התקן DFU ל-Windows: המתקין מזהה כעת את Windows 10 ומתקין מנהל התקן STTube חתום ייעודי ל-Win10, וצורפה ערכת כלי/מנהל התקן נפרדת של STM32L DfuSe. (89f57e10a, e4ef66f12)

  • ffmpeg/ffprobe: רועננו הבינאריים המצורפים של ffmpeg/ffprobe המשמשים להמרת וידאו עבור יעדי Linux (linux-arm/Raspberry Pi, linux-x86, linux-x86_64) ו-macOS, תוך הסרת וריאנטי ffserver ו-ffmpeg ב-10 ביט שאינם בשימוש ביעדי Linux. (6ea2f68a4)

  • מנהל התקן USB-טורי FTDI ל-Windows: חבילת מנהל ההתקן המצורפת עודכנה. (919bd1d8d)
