v2.9.2#

v2.9.2 הופך את הטרמינל הטורי לאינטראקטיבי כך שאפשר להקליד ישירות אל ה-REPL של MicroPython שבמצלמה, מוסיף התקנה בלחיצה אחת של קושחת הפיתוח האחרונה, ומגדיר את ה-RTC של המצלמה לפי שעון המארח בעת ההתחברות (דורש קושחת מצלמה 4.3.2 או חדשה יותר; הקושחה המצורפת כאן היא 4.3.1). ה-IDE נשאר על בסיס Qt Creator 4.0.2 ואין שינויים שוברים לפרויקטים קיימים.

עיקרי הדברים#

  • טרמינל טורי אינטראקטיבי: הקלידו ישירות בטרמינל של ה-IDE כדי לשלוט ב-REPL של MicroPython שבמצלמה, עם מקשי החצים, Home, End ו-Delete הממופים לקודי הבקרה המתאימים של ה-REPL.

  • התקנת גרסת הפיתוח האחרונה: פריט תפריט Tools חדש מוריד את קושחת חוד החנית מ-GitHub וצורב אותה אל המצלמה המחוברת.

  • סנכרון זמן עם המארח: התחברות למצלמה מגדירה כעת את ה-RTC שלה לפי התאריך והשעה הנוכחיים של מערכת המארח (דורש קושחת מצלמה 4.3.2 או חדשה יותר).

  • עדכוני מתקין במקום: מתקין Windows/Linux מסיר אוטומטית התקנה קודמת של OpenMV IDE בתיקיית היעד לצורך עדכונים נקיים.

  • צריבת DFU של Arduino Nicla Vision תוקנה, עם סוג לוח ייעודי ומנהל התקן Windows מצורף.

תכונות חדשות#

  • טרמינל טורי אינטראקטיבי. כעת אפשר להקליד ישירות בטרמינל הטורי של ה-IDE כדי לתקשר עם ה-REPL של MicroPython שבמצלמה. מקשי החצים ממופים ל-CTRL+B/CTRL+F/CTRL+P/CTRL+N, Home ו-End ממופים ל-CTRL+A ול-CTRL+E, ו-Delete ממופה ל-CTRL+D (c50b8c716).

  • התקנת גרסת הפיתוח האחרונה. פריט תפריט Tools חדש מוריד את קושחת חוד החנית האחרונה מ-GitHub וצורב אותה אל המצלמה המחוברת, עם אפשרות למחוק את מערכת הקבצים הפנימית (87a69e700).

  • סנכרון זמן עם המארח בעת ההתחברות. בעת ההתחברות למצלמה, ה-IDE שולח כעת את התאריך והשעה הנוכחיים של מערכת המארח כדי להגדיר את ה-RTC של המצלמה. הדבר דורש קושחת מצלמה 4.3.2 או חדשה יותר; קושחת 4.3.1 המצורפת לגרסה זו אינה תומכת עדיין בפקודת קלט הזמן (d3ef9d5e0, ce82350eb).

שינויים ושיפורים נוספים#

  • מתקין Windows/Linux מציג כעת עמוד תיקיית התקנה מותאם אישית ומסיר אוטומטית כל התקנה קודמת של OpenMV IDE בתיקיית היעד לפני ההתקנה, מה שמאפשר עדכונים נקיים במקום (374808dc3, dd7e5c76e).

תיקוני באגים#

  • עצירת סקריפט או ניתוק מעבירים כעת את המצלמה למצב סרק ומרוקנים תחילה במלואם את חוצצי הטקסט והתמונה הממתינים, מה שמונע פריימים ופלט מיושנים ומרוצי פקודות (af8d18d65).

  • חלון ההתקדמות של צריבת הקושחה עבור DFU, bossac ו-picotool מעדכן כעת את שורות התקדמות המחיקה וההורדה במקום, במקום להציף את החלון באלפי שורות (23345dce4).

  • מצלמה הנמצאת במצב bootloader (מספר טורי 000000000011) אינה מוצעת עוד כיציאה רגילה הניתנת להתחברות, ומשמשת כעת רק לשחזור bootloader (9d070b669).

  • הפקודה Open OpenMV Cam Folder פותחת כעת את תיקיית .openmv_disk הנכונה ב-macOS וב-Linux במקום את נתיב היציאה הגולמי (adacf333d).

  • סגירת כרטיסיית עורך דרך סרגל הכרטיסיות סוגרת כעת את העורך כראוי מבלי להפעיל את לוגיקת הסגירה פעמיים (18408e6e5, 09a8e299b).

  • תיבות שגיאה של DFU (”No DFU settings for the selected board type!“ ו-”No DFU settings for the selected device!“) מצרפות כעת את ה-VID וה-PID הבעייתיים כדי לסייע בפתרון בעיות (640f25df3).

תמיכה בפלטפורמות וכלים#

  • בסיס Qt Creator: 4.0.2.

  • צריבת DFU של Arduino Nicla Vision תוקנה על ידי הוספת ה-VID/PID של ה-bootloader שלו למפת ה-DFU, מתן סוג לוח ייעודי NICLAV, וצירוף מנהל ההתקן שלו ל-Windows (90d1ac0ac).

  • נוספה תמיכה בחיישן התרמי FLIR Lepton 1.6 בטבלת מזהי החיישנים; רשומת Lepton 1.0 הוסרה (874e80d86).

  • קושחת המצלמה המצורפת עודכנה ל-v4.3.0 ולאחר מכן ל-v4.3.1 עבור כל לוחות OpenMV ו-Arduino, עם סקריפטים לדוגמה חדשים של Time-of-Flight, טמפרטורת אובייקט גבוהה של Lepton, ו-BLE aioble של Nano-RP2040 (3034469f7, ce82350eb).

שינויים שוברים#
