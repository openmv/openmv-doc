MicroPython differences from CPython¶

MicroPython מיישם את Python 3.4 ותכונות נבחרות של Python 3.5 ומעלה. הסעיפים הבאים מתארים את המצב הנוכחי של תכונות אלה.

  • Python 3.5
  • Python 3.6
  • Python 3.7
  • Python 3.8
  • Python 3.9
  • Python 3.10

עבור תכונות Python המיושמות על ידי MicroPython, לעיתים קיימים הבדלים בהתנהגותן בהשוואה ל-Python הסטנדרטי. הפעולות המפורטות בסעיפים הבאים מניבות תוצאות שונות ב-MicroPython בהשוואה ל-Python הסטנדרטי.
