class Statistics – אובייקט Statistics¶

אובייקט הסטטיסטיקה הוא attrtuple המוחזר על ידי histogram.get_statistics() (וכינוייו histogram.get_stats() / histogram.statistics()) או ישירות על ידי Image.get_statistics(). הוא מסכם את התפלגות התאים של ה-Histogram הבסיסי בעזרת שמונה צמצומים לכל ערוץ: ממוצע, חציון, שכיח, סטיית תקן, מינימום, מקסימום, רבעון תחתון, ורבעון עליון.

עבור נתוני גווני אפור, רק השדות ללא קידומת (mean, median, mode, stdev, min, max, lq, uq) נושאים ערכים שימושיים והם משמשים שוב כצמצומי ערוץ ה-LAB L עבור נתוני RGB565 – שדות ה-l_* משקפים אם כן את השדות ללא קידומת ומסופקים לשם סימטריה. a_* ו-b_* נושאים ערכים שימושיים רק עבור היסטוגרמות RGB565.

השדות נגישים לפי שם תכונה (statistics.mean) או לפי אינדקס (statistics[0]). לאובייקט אין בנאי ציבורי.

class image.statistics¶

אנא קרא ל-histogram.get_statistics() או Image.get_statistics() כדי ליצור אובייקט זה.

גווני אפור / LAB L (חשוף גם כ-l_* למטה)¶

סטיית תקן גווני אפור של ההיסטוגרמה. מספר שלם 0 – 255. אינדקס [3].

min¶

מינימום גווני אפור של ההיסטוגרמה. מספר שלם 0 – 255. אינדקס [4].

max¶

מקסימום גווני אפור של ההיסטוגרמה. מספר שלם 0 – 255. אינדקס [5].

lq¶

רבעון תחתון (האחוזון ה-25) של גווני אפור של ההיסטוגרמה. מספר שלם 0 – 255. אינדקס [6].
