3. בקרת חומרה¶

כל ערך Python במדריך המבוא חי בזיכרון: מספר בשם x, מחרוזת שהוחזרה מפונקציה, רשימה שעובדה בלולאה. בקרת חומרה מחליפה את העבודה בזיכרון באפקטים פיזיים – מתח על פין, רכבת פולסים על חוט, בית המוקלט אל שבב אחר דרך אפיק.

מודול machine הוא הגשר. המחלקות שלו (Pin, ADC, PWM, UART, SPI, I2C, CAN, WDT) ממפות כל אחת חלק מהסיליקון של המצלמה אל Python: מגדירים מופע, קוראים למתודות שלו, ורואים אפקטים אמיתיים על הפינים של הלוח.
