13.1.13.4.4.1. קריאה חוזרת#

אתם יצרתם את הפריסה הזו, ולכן אתם יודעים את הנתיב לכל ערך. בעזרת מסייע ה־ find מתוך overview , ציינו את שם הכרטיסייה ואת הפקד – find עובר דרך רצועת הכרטיסיות וכל קבוצה ללא שם שביניהן:

enabled = find(config, "Camera", "manual_exposure")["value"]                   # the group's checkbox
exposure = find(config, "Camera", "manual_exposure", "exposure_us")["value"]   # 10000
resolution = find(config, "Camera", "resolution")["value"]                     # "qvga"

לקבוצת Manual Exposure יש name, ולכן היא שלב בנתיב – ו-find(config, "Camera", "manual_exposure") קורא את תיבת הסימון שלה. לקבוצת Format אין name, ולכן היא שקופה: אל resolution מגיעים ישירות מלשונית Camera.

שימו לב כיצד ה-combobox עם מערך values מחזיר את הערך המאוחסן ("qvga"), ולא את התווית שהמשתמש בחר ("QVGA") – בדיוק מה שסקריפט רוצה להסתעף לפיו.
