5. עיבוד תמונה¶

כל עמוד עד כה טיפל בנתוני פיקסלים רק כדי להשיגם: לקלוט פריים, להגדיר חיישן, להעביר בייטים מהמצלמה אל ה-RAM. עיבוד תמונה הופך את הכיוון – הפיקסלים הם הקלט, והמטרה היא כל מה שהיישום צריך מהם.

המודול image הוא ארגז הכלים. כל פעולה מקבלת אובייקט Image אחד או שניים ומחזירה או Image שעבר טרנספורמציה – פיקסלים מסוננים, מסופפים, מעוותים או מורכבים אחרת – או רשימה של אובייקטי תוצאה: רכיבים, קווים, קודים, התאמות, סטטיסטיקות. שני הסוגים מתאחדים. צינור עיבוד טיפוסי מנקה את הפריים שנקלט, מחלץ ממנו אוסף של אובייקטי תוצאה, הופך אותם למדידות, ומצייר הערות בחזרה על הפריים לצורך תצוגה.
