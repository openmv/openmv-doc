4. חיישני ראייה¶

כל עמוד עד כה התייחס ל-OpenMV Cam כמיקרו-בקר קטן שבמקרה יש לו מצלמה מחוברת. חיישני ראייה מחליפים את המסגור: המצלמה היא המוצר, וכל השאר על הלוח קיים כדי להזרים ממנה פיקסלים אל תוך Python.

המודול csi הוא הגשר. מחלקה אחת (CSI) עוטפת את השרשרת הארוכה של פיזיקה וסיליקון שבין פוטונים לחוצץ. עדשה ממקדת את הסצנה על רשת של פוטודיודות ההופכות פוטונים למטען; בקר חשיפה / הגבר / קריאה מחליט באיזו בהירות כל פיקסל מדווח; מעבד אות תמונה (ISP) מתקן, מבצע debayer, מבצע דירוג-צבע, ואורז את הפיקסלים לפורמט נבחר; והתקן היקפי של ה-MCU תופס את הזרם המתקבל אל תוך RAM, מוכן ל-snapshot() שיחזירו כ-image.Image.
