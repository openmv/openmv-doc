2. סקירת Python¶

ה-OpenMV Cam מריצה MicroPython, אותה שפה כמו Python שולחני עם ספרייה סטנדרטית קטנה יותר וכמה מודולים נוספים לתקשורת עם חומרה. כל מה שנכתב עבור המצלמה – מסקריפט בן שתי שורות המהבהב את ה-LED ועד יישום ראייה ממוחשבת מלא – הוא קוד Python.
