v1.0.3-beta¶

v1.0.3-beta este o versiune beta minoră de întreținere a instrumentelor/compilării: trece compilarea la fișiere DfuSe, curăță descriptorii de configurare USB și corectează calea de împachetare a IDE-ului desktop. Nu există modificări ale API-ului Python al firmware-ului — scripturile v1.0.2-beta rulează neschimbate.

Repere¶

  • Compilare DfuSe — Makefile-urile produc acum fișiere de firmware DfuSe.

  • Curățare USB — s-a eliminat o interfață goală din a doua configurație USB.

  • Fără modificări ale API-ului firmware-ului — reflashați și scripturile dvs. v1.0.2-beta rulează neschimbate.

Alte modificări și îmbunătățiri¶

  • S-au actualizat Makefile-urile pentru a folosi fișiere DfuSe și s-au reîmprospătat binarele precompilate; s-a eliminat o interfață goală din a doua configurație USB; s-a corectat calea IDE-ului în împachetarea py2exe și s-a prins excepția la eliberarea dispozitivului.

Modificări incompatibile ale API-ului¶

v1.0.3-beta nu introduce nicio modificare incompatibilă a API-ului Python (modificările vizează doar compilarea/USB/instrumentele desktop). Scripturile care rulau pe v1.0.2-beta rulează neschimbate — doar reflashați firmware-ul.
