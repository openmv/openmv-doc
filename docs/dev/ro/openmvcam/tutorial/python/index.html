2. Prezentare generală Python¶

OpenMV Cam rulează MicroPython, același limbaj ca Python pentru desktop, cu o bibliotecă standard mai mică și câteva module suplimentare pentru comunicarea cu hardware-ul. Tot ce este scris pentru cameră – de la un script de două linii care face LED-ul să clipească până la o aplicație completă de viziune artificială – este cod Python.
