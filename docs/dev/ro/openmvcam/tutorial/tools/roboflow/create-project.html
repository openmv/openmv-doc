13.7.1. Crearea unui proiect¶

Din spațiul de lucru Roboflow, New Project deschide pagina de creare. Un proiect grupează un set de date, modelele antrenate pe acesta și țintele lor de implementare.

Aici contează două alegeri:

  • Project Name. Folosiți orice nume ușor de reținut – acest exemplu folosește Finding raccoons. Numele devine parte a identificatorului modelului, pe care îl descărcați la pasul de implementare.

  • Project Type. Acesta stabilește ce prezice modelul, așa că potriviți-l cu ceea ce doriți să facă camera. Object Detection – găsirea obiectelor și a pozițiilor lor cu casete de încadrare – este cel potrivit pentru localizarea lucrurilor în cadru și este cel pe care îl decodează post-procesoarele YOLO ale camerei. Celelalte tipuri – clasificare, segmentare de instanțe, detectare de puncte-cheie – antrenează capete de model diferite; acest exemplu folosește detectarea de obiecte.

Opțiunile de vizibilitate și licență afectează doar dacă setul de date este partajat public pe Roboflow Universe – nu au nicio influență asupra modelului care ajunge pe cameră. Setați-le după preferințe, apoi Create Project.
