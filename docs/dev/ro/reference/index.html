Limbajul și implementarea OpenMV MicroPython¶

MicroPython urmărește să implementeze standardul Python 3.4 (cu funcționalități selectate din versiuni ulterioare) în ceea ce privește sintaxa limbajului, iar majoritatea caracteristicilor MicroPython sunt identice cu cele descrise în documentația „Language Reference” de la docs.python.org.

Biblioteca standard MicroPython este descrisă în capitolul corespunzător. Capitolul MicroPython differences from CPython descrie diferențele dintre MicroPython și CPython (care privesc în mare parte biblioteca standard și tipurile, dar și unele caracteristici la nivel de limbaj).

Acest capitol descrie caracteristicile și particularitățile implementării MicroPython și cele mai bune practici de utilizare a acestora.
