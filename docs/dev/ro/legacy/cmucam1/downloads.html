Descărcări#

Manualul utilizatorului#

Manualul utilizatorului CMUcam1 v2.00 pentru firmware v1.12 este disponibil în format PDF. Conține o listă de piese, schema plăcii și descrierea protocolului software. Manuale pentru versiuni mai vechi ale firmware-ului pot fi găsite aici: v1.11, v1.1, v1.0.

Dacă decideți să experimentați cu modificarea setărilor registrului camerei, ați putea fi, de asemenea, interesat să descărcați Manualul camerei OV6620 de la OmniVision.

Construirea plăcii#

Dacă îl construiți singur de la zero, va trebui să descărcați acest fișier de cod HEX (conținând cel mai recent firmware v1.12) pentru a programa cipul SX28. Acesta este codul care implementează funcționalitatea și protocolul serial descrise în manual. Puteți obține descărcătorul și aplicația SX-Key de la Parallax Inc. Versiunile mai vechi ale firmware-ului pot fi găsite aici: v1.11, v1.1, v1.0.

Program de interfață Java#

Avem un program java open source pe care îl puteți descărca pentru a face interfața cu și a testa CMUcam1. Acest program de interfață vă permite să urmăriți obiecte, să analizați date și să capturați imagini CMUcam1 complete. Iată Capturi de ecran ale programului Java ale acestei interfețe.

Acesta este proiectul java arhivat scris pentru Linux.

Aceasta este versiunea interfeței grafice care detectează automat sistemul dvs. de operare. Rulează pe sisteme UNIX care suportă IO serial ca și cum ar fi un fișier (cum ar fi Linux), precum și pe Windows 95/98/2000/NT/XP. Recomandăm utilizarea a cel puțin unui procesor Pentium 2 de 400 MHz care rulează Java 1.2.2 sau o versiune superioară.

Pentru a vedea exemple de cod C pentru microprocesorul Cerebellum și pentru Handyboard-ul BotBall care comunică cu CMUcam1, accesați această pagină web.

Schemă#

Iată un JPEG de rezoluție înaltă al schemei CMUcam1 curente (versiunea de placă 1.23).

Înainte de a întreba despre codul sursă, ați putea fi interesat să descărcați Manualul procesorului SX28 de la Ubicom.
