Вихідний код прошивки#

Ви можете завантажити вихідний код прошивки тут.

Примітка

Архів вихідного коду прошивки !CMUcam4_Firmware_Code_102.zip не збережено в архіві.

Для отримання докладнішої інформації зверніться до Посібника з програмування.

ПРИМІТКА: CMUcam4 програмується не на C. Вона програмується мовою, «схожою на C», яка називається «SPIN», lo siento. Для отримання докладнішої інформації дивіться розділ Інформація про мікросхему P8X32A Propeller нижче.

Крім того, для отримання докладнішої інформації про Full File System Driver, включений до вихідного коду, будь ласка, натисніть тут.

Інформація про мікросхему P8X32A Propeller#

CMUcam4 живиться P8X32A (мікросхема Propeller) від Parallax.

(Зображення надано Parallax)

Для отримання докладнішої інформації про мікросхему Propeller дивіться Propeller P8X32A Q&A.

Крім того, перегляньте наступні вебсайти для корисних завантажень для мікросхеми Propeller:

  • Завантаження Propeller - Parallax.

    • Ви можете знайти Propeller Tool (Windows GUI IDE) або BST (Windows/Macintosh/Linux GUI IDE) тут.

  • Propeller Object Exchange - Parallax.

    • Ви можете знайти безкоштовне програмне забезпечення з відкритим вихідним кодом тут.

(Зображення надано Parallax)

Для подальшого використання, ось Посібник P8X32A, Технічний паспорт P8X32A та Стислий довідник P8X32A.

(Документи надані Parallax)

ПРИМІТКА: CMUCam4 розганяє мікросхему Propeller до 96 Mhz (понад типові 80 Mhz), щоб досягти кращої продуктивності (24 MIPS на ядро x 8 = 192 MIPS проти 20 MIPS на ядро x 8 = 160 MIPS).

(Зображення надано Parallax)

Інформація про модуль камери OV9665#

Через угоди про нерозголошення OmniVision ми не можемо надати технічний паспорт OV9665 для завантаження.

  • Якщо ви хочете запросити технічний паспорт від OmniVision, будь ласка, натисніть тут.

  • Якщо ви хочете запросити технічну підтримку від OmniVision, будь ласка, натисніть тут.

Однак ми виокремили всю корисну інформацію з технічного паспорта OV9665 і розмістили її тут.
