Початок роботи з CamScripter#

Огляд#

Що це таке?#

CamScripter — це інструмент, який допомагає вам писати та налагоджувати власні скрипти для CMUcam3. Він дає змогу писати скрипти, використовуючи мову програмування Lua. Використовуючи цю мову та надане Lua API для CMUcam3, ви можете налаштувати камеру майже для будь-якого використання, яке ви можете уявити. У цьому посібнику ми проведемо вас через процес встановлення, а потім розробимо зразковий скрипт для запуску на камері.

Перш ніж почати#

Цей посібник передбачає:

  • У вас вже є CMUcam3

  • Ви вже змусили камеру працювати, використовуючи або Короткий посібник із запуску, або посібник зі встановлення SDK

  • У вас є все необхідне для підключення камери до вашого комп’ютера

Встановлення, частина 1: Lua, зв’язок через послідовний порт#

Lua#

CamScripter дозволяє вам програмувати CMUcam3, використовуючи мову програмування Lua. Перш ніж ми зможемо це зробити, нам потрібно встановити цю мову на вашому комп’ютері.

  1. Завантажте відповідний двійковий файл ZIP. (Для Windows: Lua-Windows (не збережено в архіві) або для інших користувачів http://luabinaries.luaforge.net/download.html)

  2. Розпакуйте двійковий файл Lua у C:\lua

  3. Відкрийте командний рядок (у Windows виберіть «Run…» з меню Пуск і введіть „cmd“).

  4. cd до місця, куди ви розпакували файли Lua. Для мене я вводжу „cd C:lua“

  5. Введіть „luax.x.exe -v“, де x — версія Lua

Якщо все у вашому встановленні пройшло гладко, ви повинні побачити номер версії Lua, надрукований на вашому екрані, як на знімку екрана нижче (не збережено в архіві).

Якщо можливо, вам слід встановити lua з вебсайту CMUcam3. Якщо ви хочете перекомпілювати його самостійно, вам потрібно спершу пропатчити звичайний вихідний код lua, щоб він не використовував рухому кому (lua-5.1-no-fp.patch).

У windows під cygwin використовуйте наступні рядки для компіляції:

cd src
make LUAC_T=luac.exe MYCFLAGS=-mno-cygwin MYLDFLAGS=-mno-cygwin luac.exe
...
make LUA_T=lua.exe MYCFLAGS=-mno-cygwin MYLDFLAGS=-mno-cygwin lua.exe

Java Runtime Environment (JRE)#

Для CamScripter потрібне середовище виконання Java (версії 6 чи новішої). Якщо у вас ще не встановлено JRE, ви можете завантажити та встановити його тут.

RXTX#

CamScripter використовує бібліотеку послідовного зв’язку RXTX для зв’язку з CMUcam3 через послідовний порт вашого комп’ютера. Цю бібліотеку доведеться встановити на вашому комп’ютері. Спершу завантажте RXTX (принаймні версії 2.1). Версія 2.1.7 тут: ftp://ftp.qbang.org/pub/rxtx/rxtx-2.1-7-bins-r2.zip.

Щоб встановити (лише Windows — для інших ОС дотримуйтесь інструкцій у файлі rxtx INSTALL):

  1. Розпакуйте завантажений RXTX та перейдіть до отриманої папки

  2. Двічі клацніть на Windows

  3. Двічі клацніть на i368-mingw32

  4. Тепер вам доведеться з’ясувати, де розташоване ваше середовище виконання Java. Ймовірно, воно в C:\Program Files\Java\jre<version number>. Моє, наприклад, розташоване в C:\Program Files\Java\jre1.6.0_01. Скопіюйте файли rxtxSerial.dll і rxtxParallel.dll з теки RXTX i368-mingw32 до теки jre/bin. Отже, у моєму прикладі я поклав ці два файли до теки C:\Program Files\Java\jre1.6.0_01\bin.

Встановлення, частина 2: Eclipse#

Гаразд, ми майже готові! Залишилося зробити лише кілька речей.

Якщо у вас його ще немає, встановіть Eclipse (версії 3.2 чи новішої). Ви можете встановити його з http://eclipse.org/downloads. Eclipse Classic — це надійний вибір, хоча будь-який з інших перелічених пакетів також повинен працювати. Щоб встановити, просто розпакуйте обраний вами пакет у C:\Program Files\eclipse.

Тепер, коли Eclipse встановлено, нам потрібно встановити CamScripter як плагін Eclipse:

  1. Відкрийте Eclipse, двічі клацнувши eclipse.exe у C:\Program Files\eclipse\

  2. Натисніть Enter, щоб прийняти робочий простір за замовчуванням

  3. Відкрийте майстер встановлення/оновлення, вибравши Help -> Software Updates -> Find and Install

  4. Виберіть перемикач «Search for new features to install» і натисніть Next.

  5. Натисніть кнопку «New Remote Site…»

  6. Для імені введіть Camscripter, а для URL введіть http://cmucam.org/~juanpablo/. Натисніть OK.

  7. Переконайтеся, що прапорець поруч із Camscripter позначено, і натисніть кнопку «Finish».

  8. Виберіть «Camscripter» як функцію для встановлення та натисніть Next.

  9. Прийміть ліцензійну угоду та натисніть Далі

  10. Натисніть Finish - Eclipse завантажить плагін для вас

  11. Натисніть «Install», щоб встановити плагін Camscripter.

  12. Натисніть Yes, щоб перезапустити платформу Eclipse

Потім Eclipse перезапуститься зі встановленим для вас плагіном Camscripter. Щоб переконатися, що його встановлено, виберіть Window -> Show View -> Other…

Натисніть знак плюс поруч із CMUcam3, виберіть «CamScripter» і натисніть OK. Ви повинні побачити подання CamScripter у нижній частині Eclipse (область, обведена червоним на зображенні нижче, не збережено в архіві). Ви можете розгорнути нижню область, якщо вона занадто мала.

  1. Подання CamScripter

  2. Панель консолі, де відображаються текстові повідомлення

  3. Панель зворотного зв’язку, де відображаються зображення та графічні повідомлення

  4. Виконує код Lua. Ви можете виконати або весь файл, або лише вибрані рядки коду.

  5. Встановлює код Lua на картку SD/MMC CMUcam3

Налаштування#

Ми все встановили! Ще кілька кроків налаштування, і ми програмуватимемо CamScripter.

Завантажте проєкт camscripter на камеру#

Тепер, коли все встановлено, нам потрібно завантажити на камеру наш проєкт CamScripter. Запустіть make на проєкті camscripter, розташованому в projects/camscripter (у дереві вихідного коду, яке ви отримали в Посібнику зі швидкого старту), а потім завантажте hex-файл на камеру. Вимкніть камеру, щойно файл завантажиться.

Встановіть налаштування CamScripter#

Тепер поверніться до Eclipse і відкрийте екран налаштувань CamScripter: Window –> Preferences –> CamScripter Setup.

  • Встановіть розташування Luac, щоб воно вказувало на ваш файл luacx.x.exe (де x — це версія Lua) (зверніть увагу на „c“) там, де ви встановили Lua у розділі Встановлення, частина 1: Lua, зв’язок через послідовний порт.

    • Для тих, кому цікаво: luac просто компілює код lua, тоді як luax.x.exe фактично інтерпретує та запускає код. Ми компілюємо код у Eclipse, потім запускаємо його на камері – звідси й luac.

  • Вкажіть ім’я послідовного порту, до якого ви підключаєте камеру. Ви можете натиснути кнопку «See Serial Ports», щоб побачити список доступних послідовних портів.

Знімок екрана налаштувань CamScripter не збережено в архіві.

Hello, World!#

Тепер, коли все встановлено та правильно налаштовано, візьмімося за обов’язкову програму Hello, World. Спершу вам потрібно буде створити новий проєкт. Виберіть File –> New –> Project… Ви отримаєте вікно, що дозволяє вибрати тип проєкту, який ви хочете створити (див. нижче). Двічі клацніть «General», виберіть «Project» і натисніть кнопку Next.

Назвіть свій проєкт «Hello» і натисніть кнопку Finish. Потім створіть новий файл, клацнувши File –> New –> File (у Eclipse 3.3 це File –> New –> Other –> General –> File). У полі імені файлу введіть «hello_world.lua», з Hello як батьківською текою. Натисніть кнопку Finish.

Ще раз перевірте, що вашу камеру підключено до послідовного порту вашого комп’ютера. Перш ніж увімкнути камеру, важливо зрозуміти два режими, які підтримує камера:

  • Автономний: камера взагалі не очікує жодної взаємодії з ПК. За замовчуванням камера запускається в цьому режимі.

  • Інтерактивний: камера отримує скомпільований код Lua від CamScripter в Eclipse і повертає зворотний зв’язок. Щоб перевести камеру в інтерактивний режим, увімкніть її, і коли ви побачите, що світиться помаранчевий світлодіод, натисніть кнопку камери (у вас є 3 секунди, щоб натиснути кнопку). У цей момент помаранчевий світлодіод блимне кілька разів, повідомляючи вам, що ми входимо в інтерактивний режим. Якщо все пройшло добре, ви побачите повідомлення Welcome to CamScripter Interactive mode. Click the execute button to run a script.

Запустімо камеру в інтерактивному режимі, щоб ми могли виконати нашу першу програму Lua.

У вікні редактора введіть рядок коду:

print("Hello, World!");

Збережіть файл і натисніть кнопку Execute у поданні CamScripter. І вуаля!! Ваш код пройде дротом до камери, яка виконає його та надішле назад повідомлення, яке ви надрукували.

Тепер спробуймо щось трохи цікавіше. Додайте три додаткові рядки коду, щоб ваш файл hello_world.lua містив:

print("Hello, World!");
take_picture("pic_name");
print_picture("pic_name");
print("It's a picture!");

Тепер знову натисніть кнопку Execute. Цього разу виконання займе більше часу, оскільки надсилання зображень через послідовний порт досить тривале. Однак, щойно воно завершиться, ви повинні побачити ваші два повідомлення в області консолі CamScripter, а зображення, яке ви зробили камерою, в області зворотного зв’язку (ви ж зняли кришку об’єктива, чи не так?)

Ви помітите, що це зображення досить низької роздільної здатності. За замовчуванням камера налаштована робити знімки низької роздільної здатності в CamScripter, оскільки надсилання зображень високої роздільної здатності через послідовний порт займає значно більше часу. Однак ви можете змінити камеру на високу роздільну здатність за допомогою функції camera_set_resolution.

Знімки екрана проєкту Eclipse, файлу, увімкненої камери та Hello-World для цього розділу не збережено в архіві.

Корисні посилання#

Тепер CamScripter повністю встановлено та налаштовано, і ви побачили невеликий приклад, що демонструє його функціональність. Якщо ви хочете побачити більші приклади використання Camscripter, погляньте на зразкові програми, включені до каталогу camscripter/examples. Кілька інших приміток і посилань, які можуть вам допомогти:

CamScripter Lua API описує функції CMUcam3, які ви можете використовувати у своїх програмах Lua для керування діями CMUcam3.

Ось кілька посилань, які допоможуть вам почати роботу з Lua. Зверніть увагу, що зовнішні бібліотеки, як-от бібліотеки string і math, не включено до версії Lua, встановленої на CMUcam3. Їх довелося вилучити, щоб інтерпретатор Lua помістився в обмежений простір на CMUcam3.

  • Посібник Lua Посібник, який допоможе вам розпочати роботу.

  • Довідковий посібник Lua: Повна документація Lua.

  • Lua Wiki Корисне місце для пошуку відповідей. Зразки коду, поширені запитання та посилання на інші ресурси.

  • LuaEclipse — це корисний плагін Eclipse, який може стати вам у пригоді. Він забезпечує підсвічування синтаксису та інші засоби, що полегшують написання коду на Lua.
