Шілди OpenMV¶

Додаткові плати, що підключаються до OpenMV Cam та розширюють її можливостями мережі, керування двигунами, дисплеями, датчиками тощо. Кожна сторінка описує призначення шілда, ключові характеристики та API для керування ним із MicroPython.

Сучасні шілди¶

Gigabit PoE Shield

Gigabit Ethernet з PoE для потокової передачі з вищою пропускною здатністю.

Explore →

Servo Shield

Керування до 4 сервоприводів зі споживанням до 5A при живленні камери, вхід 6–36В.

Explore →

Battery Shield

Живлення від батареї 1.8–5.5В через роз'єм DC barrel jack, плюс широкий вхід 6–36В.

Explore →

Touch LCD Shield

SPI LCD 2.3" 320×240 із ємнісним мультитач-екраном і Qwiic.

Explore →

PoE Shield

10/100 Ethernet із живленням через Ethernet (PoE).

Explore →

PIR Shield

Тригер руху зі споживанням 6 мкА у режимі очікування та підсвіткою білим і ІЧ 850 нм.

Explore →

CAN/RS232 Shield

CAN-FD 8 Мб/с та RS-232 1 Мб/с для транспортних засобів і застарілого послідовного зв'язку.

Explore →

RS422/RS485 Shield

Диференційний послідовний зв'язок 10 Мб/с для промислових шин на великих відстанях.

Explore →

Driver Shield

Два драйвери двигунів по 3A або чотири лінійних драйвери по 1.5A, 6–36В.

Explore →

Relay Shield

Два реле потужністю 60 Вт для комутації навантаження AC/DC, вхід 6–36В.

Explore →

AE3 OLED Breakout

RGB OLED 128×128, 5-позиційний джойстик, Qwiic та роз'єм ARM SWD на 10 виводів.

AE3 Battery Shield

Живлення від 5–36 В, 1.8–5.5 В або Li‑Po 3.7 В для AE3 та роз'єм SPI microSD.
