5. Обробка зображень¶

Кожна попередня сторінка обробляла піксельні дані лише для їх отримання: захопити кадр, налаштувати датчик, перемістити байти з камери в RAM. Обробка зображень змінює напрям — пікселі є вхідними даними, а ціль — це те, що застосунку потрібно від них.

Модуль image — це набір інструментів. Кожна операція приймає один або два об’єкти Image і повертає або перетворений Image — пікселі відфільтровані, порогово оброблені, деформовані або скомпозовані інакше — або список об’єктів результатів: плями, лінії, коди, збіги, статистику. Обидва різновиди можна комбінувати. Типовий конвеєр очищає захоплений кадр, витягує набір об’єктів результатів із нього, перетворює їх на вимірювання та малює анотації назад на кадр для відображення.
