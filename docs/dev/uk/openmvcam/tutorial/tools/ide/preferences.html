13.1.5. Налаштування та зовнішній вигляд#

Edit → Preferences (меню програми в macOS) відкриває діалог налаштувань. Майже все в ньому можна залишити без змін – значення за замовчуванням відповідають конфігурації, описаній у решті цього розділу – але кілька категорій варто знати.

13.1.5.1. Інтерфейс і теми#

Сторінка Environment → Interface вибирає тему. IDE постачається з темною (за замовчуванням) і світлою темами, а синтаксичні кольори редактора автоматично слідують за темою. На тій самій сторінці знаходяться мова інтерфейсу та параметри масштабування відображення.

Повсякденне розташування вікон налаштовується поза діалогом. Кожна нижня панель – серійний термінал, результати пошуку – має кнопку перемикання на рядку стану; бічні панелі згортаються при перетяганні розділювачів і знову відкриваються зі стрілок у краях вікна; Full Screen у меню Window (Ctrl+Shift+F11) розгортає демонстрацію або настінний монітор на весь екран.

13.1.5.2. Налаштування текстового редактора#

Категорія Text Editor керує поверхнею редагування: Font – для гарнітури та розміру шрифту; Display – для номерів рядків, візуалізації пробілів і перенесення рядків; Behavior – для відступів (ширина табуляції, пробіли чи табуляції) і автоматичного очищення пробілів. Також можна змінити розмір шрифту на льоту за допомогою Ctrl++ / Ctrl+- / Ctrl+0 або колесом миші при утриманні Ctrl.

Сторінка Environment → Keyboard перелічує кожну команду та її комбінацію клавіш, відфільтровану до дій, які ви дійсно можете виконати, тож ви можете шукати команду й бачити (або знаходити) її прив’язку клавіш. Комбінації також показуються поруч із пунктами меню, і цей розділ наводить важливі з них там, де з’являється відповідна можливість.

13.1.5.3. Інші категорії#

Категорії Python і Copilot налаштовують мовну підтримку редактора – вбудований мовний сервер Python і необов’язковий вхід до GitHub Copilot. Решта категорій належать ядру Qt Creator і рідко потребують уваги.
