v3.0.3#

v3.0.3 додає повноцінну підтримку плати NXP i.MX RT1060 (EVKB-MIMXRT1060), включаючи процедуру відновлення та перепрошивання через послідовний завантажувач (SBL), разом із перемикачем автоматичного повторного підключення, програвачем потоків RTSP та покращеннями помічника імпорту та розбору рядків документації для автодоповнення коду. Реліз побудовано на Qt Creator 4.0.2, оновлює вбудовану мікропрограму OpenMV до v4.4.2 та не містить критичних змін для користувацьких проектів.

Основні можливості#

  • Підтримка NXP i.MX RT1060. Плата EVKB-MIMXRT1060 тепер розпізнається і може бути прошита безпосередньо з IDE.

  • Процедура відновлення завантажувача SBL. Робочий завантажувач послідовного завантаження (blhost) може відновити та перепрошити камери на базі i.MX RT1060, з можливістю стерти файлову систему флеш-пам’яті.

  • Автоматичне повторне підключення до OpenMV Cam. Новий перемикач у меню Tools автоматично підключається до камери при її виявленні без зупинки запущеного скрипту.

  • Відтворення потоку RTSP. Новий пункт Video Tools відтворює живий потік RTSP через ffplay.

  • Помічник імпорту для вкладених модулів. Модулі в нерівних ієрархіях папок тепер правильно імпортуються та копіюються на OpenMV Cam.

Нові функції#

  • Підтримка NXP i.MX RT1060 додає спеціальний завантажувач мікропрограми, який використовує вбудовані інструменти sdphost/blhost для програмування EVKB-MIMXRT1060 через USB, з новим конфігом imx.txt та записом у boards.txt OMVRT60_IMXRT60_32768_SDRAM (6a040cda3, e72ff6f52).

  • Процедура завантажувача SBL дозволяє користувачам відновлювати та перепрошивати камери на базі i.MX RT1060 через вторинний завантажувач i.MX, з можливістю стерти файлову систему флеш-пам’яті (керується існуючою дією стирання) при перепрошиванні (0bb648c94, da591b7c9, c08022fab).

  • Автоматичне повторне підключення до OpenMV Cam — перемикач у меню Tools, який автоматично підключається до камери при виявленні без зупинки запущеного скрипту; стан зберігається між сесіями та вимикає ручні дії Connect/Disconnect під час активності (0546d806e).

  • Відтворення потоку RTSP — новий пункт Video Tools (не для Linux), який запитує IP/домен, порт (за замовчуванням 554) та транспорт TCP/UDP, а потім відтворює живий потік rtsp:// через ffplay (f6a39055d).

  • Додано приклади MicroSpeech для розпізнавання ключових слів Yes/No для плат Nicla Vision та Portenta H7 (8cecde656).

Інші зміни та покращення#

  • Помічник імпорту тепер правильно імпортує та копіює модулі у вкладених (нерівних) ієрархіях папок, обчислюючи назви модулів з крапками за відносними шляхами та рекурсивно копіюючи папки; модулі з папки Documents перевизначають модулі Examples (e5696f6f8, 33a703157, 1e4a27769).

  • Розбір рядків документації для автодоповнення коду тепер правильно витягує сигнатури аргументів функцій, включаючи видалення розмітки [...] необов’язкових аргументів з вбудованої HTML документації (75cb80a09, 05125bec8, 16fde1c1e).

  • Надійність підключення/відключення покращена: цикли очищення буферів тексту та зображень тепер завжди виконуються, лише scriptStop залежить від стану автоматичного повторного підключення, тому буфери очищаються коректно незалежно від того, увімкнено автоматичне повторне підключення чи ні (6673302e1).

Виправлення помилок#

  • Виправлено кілька помилок прошивання i.MX RT: шляхи до файлів передавалися як літеральні рядки замість розв’язаних шляхів, команда переходу sdphost мала помилку у назві jump-adress, а ім’я бінарного файлу завантажувача було виправлено (0bb648c94, c08022fab).

  • Виправлено друкарську помилку в тексті діалогового вікна доступного оновлення: reources виправлено на resources (798f55c31).

Підтримка платформ та інструментів#

  • База Qt Creator: 4.0.2.

  • Нова плата: NXP i.MX RT1060 (EVKB-MIMXRT1060), запис плати OMVRT60_IMXRT60_32768_SDRAM (папка мікропрограми EVKBMIMXRT1060) (6a040cda3, e72ff6f52).

  • Вбудована мікропрограма OpenMV: оновлено до v4.4.2 (через проміжне оновлення 4.3.3), з оновленими бінарними файлами мікропрограми/uvc/завантажувача для OPENMV2/3/4/4P/4_PRO/PT, NANO33, NICLAV, PORTENTA та NANO_RP2040 (8cecde656, 292950b60).

  • Вбудовані інструменти NXP: blhost та sdphost тепер позначені як виконувані в Linux та macOS, щоб процедура завантажувача i.MX могла їх запускати (3c464ca5a).

Критичні зміни#

Відсутні. Цей реліз не вносить критичних змін для користувацьких проектів; база Qt Creator залишається Qt Creator 4.0.2.
