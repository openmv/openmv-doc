OpenMV IDE Changelog¶

Кожен реліз OpenMV IDE приносить нові можливості та покращення. Ось що нового у кожному з них — нові функції редактора, камери та інструментів, виправлення помилок, підтримка платформ, а також зміни в робочому процесі та їх вплив на вас. Релізи згруповані за мажорною версією, від найновіших. Натисніть на версію, щоб розгорнути список релізів.

OpenMV IDE побудовано на основі фреймворку застосунків Qt Creator; у кожному релізі зазначено базову версію Qt Creator, на якій він зібраний. Тут задокументовано лише власні зміни OpenMV — тисячі непов’язаних комітів upstream Qt Creator не включено.
