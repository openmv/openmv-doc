v4.4.8¶

v4.4.8 — технічний реліз, що оновлює вбудовану мікропрограму та інструменти. Він продовжує базуватися на Qt Creator 14.0.2 без перебазування в цьому діапазоні. Вбудована мікропрограма OpenMV оновлена до v4.6.0, вперше включено мікропрограму STM32N6 (OpenMV Cam N6), розпізнаються нові теплові датчики FLIR Boson, а також оновлено Alif Security Toolkit. Ніяких несумісних змін для користувачів немає, існуючі проекти не зачеплено.

Основні зміни¶

  • Вбудована мікропрограма v4.6.0: Вбудована мікропрограма OpenMV і приклади оновлені з v4.5.9 до v4.6.0 (66bfd486f).

  • Мікропрограма OpenMV Cam N6 у комплекті: Мікропрограма STM32N6 (OpenMV Cam N6) вперше включена до комплекту постачання (66bfd486f).

  • Теплові датчики FLIR Boson: Теплові датчики BOSON, BOSON-320/BOSON-640 та BOSON-320+/BOSON-640+ (Boson Plus) тепер розпізнаються IDE (8b795053c, d5068b286).

  • Alif Security Toolkit v1.101.0: Вбудований інструментарій для Windows оновлено до v1.101.0 (7d51b00b1).

  • Виправлено прошивку OpenMV Cam AE3 SBL: Плата AE3 тепер запускає правильний захищений завантажувач під час прошивання (ed7c95357).

Нові функції¶

  • Приклади FLIR Boson: Додано вбудовані приклади скриптів для потокового передавання в відтінках сірого та в палітрі ironbow з FLIR Boson, а також відстеження кольору в каталозі 01-Camera/05-Thermal-Cameras/02-FLIR-Boson, та приклад відстеження LED у відтінках сірого для GenX320 у каталозі 01-Camera/03-Event-Cameras/02-Genx320 (66bfd486f).

Інші зміни та покращення¶

  • Реорганізовано дерево вбудованих прикладів: приклади FLIR Lepton перенесено до нової категорії 01-Camera/05-Thermal-Cameras/01-FLIR-Lepton поруч із новими прикладами Boson, а приклади подієвих камер розділено на підпапки 01-Camera/03-Event-Cameras/01-Frogeye2020 та 02-Genx320 (66bfd486f).

  • Видалено невикористовувану, осиротілу папку мікропрограми OPENMV4_PRO з комплекту; жоден запис плати не посилався на неї, тому жодна плата для прошивання не постраждала (66bfd486f).

  • Оновлено вбудовану мікропрограму OpenMV AE3 SBL, завантажувач і підписані образи, що використовуються для прошивання (b8c06a8ab).

  • Оновлено вбудований Alif Security Toolkit (Windows) до v1.101.0, включаючи оновлений посібник користувача (AUGD0005) та підписані образи SP (7d51b00b1).

Виправлення помилок¶

  • Виправлено прошивання OpenMV Cam AE3 SBL, щоб запускався правильний захищений завантажувач — шляхом встановлення рядка архітектури плати OPENMV AE3 SBL (ed7c95357).

  • Збільшено таймаут стирання флеш-пам’яті blhost з 60 с до 120 с під час програмування плат i.MX RT, що запобігає перевищенню часу очікування під час стирання та прошивання великих ділянок (b48e139e4).

Підтримка платформ і інструментів¶

  • База Qt Creator: 14.0.2.

  • Вбудована мікропрограма OpenMV: оновлено 4.5.9 -> 4.6.0 (66bfd486f).

  • OpenMV Cam N6: Мікропрограма STM32N6 (firmware.bin, network.bin, openmv.bin) тепер включена до комплекту в firmware/OPENMV_N6 (66bfd486f).

  • OpenMV AE3 SBL: оновлено бінарні файли мікропрограми та завантажувача (b8c06a8ab).

  • Alif Security Toolkit (Windows): оновлено v1.100.0 -> v1.101.0 (7d51b00b1).

  • Нові розпізнані датчики: FLIR BOSON, BOSON-320, BOSON-640 (8b795053c), FLIR BOSON-320+ та BOSON-640+ (d5068b286), а також PixArt PS5520 (66bfd486f).

Несумісні зміни¶

Відсутні. Змін API скриптів немає, існуючі проекти користувачів не зачеплено. Видалена папка мікропрограми OPENMV4_PRO не використовувалася і не була прив’язана до жодної плати, тому її видалення не порушує жодного процесу прошивання.
