Внутрішня будова OpenMV MicroPython¶

Цей розділ охоплює огляд MicroPython з точки зору розробника, що робить внесок до MicroPython. Він слугує вичерпним ресурсом щодо деталей реалізації MicroPython як для початківців, так і для досвідчених учасників.

Розробка навколо MicroPython зазвичай передбачає модифікацію основного середовища виконання, перенесення або підтримку нової бібліотеки. Цей посібник детально описує деталі реалізації MicroPython, включаючи посібник для початку роботи, внутрішню будову компілятора, перенесення MicroPython на нову платформу та реалізацію основної бібліотеки MicroPython.
