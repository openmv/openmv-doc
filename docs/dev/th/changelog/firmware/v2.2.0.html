v2.2.0¶

v2.2.0 เพิ่มการรองรับ AprilTags (image.find_apriltags()), เปิดใช้งาน CAN2 บน OpenMV 2 / 3, เพิ่มโมดูล MicroPython เพิ่มเติม, คืนค่า callback การจำลอง Pixy และปรับการจัดการ REPL Ctrl-D/C/A ให้ตรงกับบอร์ด MicroPython มาตรฐาน ไม่มีการเปลี่ยนแปลง API แบบ breaking — สคริปต์ v2.1.0 ทำงานได้โดยไม่ต้องเปลี่ยนแปลง

จุดเด่น¶

  • AprilTags — การตรวจจับ fiducial marker ด้วย image.find_apriltags()

  • CAN2 — เปิดใช้งานบน OpenMV 2 และ OpenMV 3

  • เปิดใช้งาน โมดูล MicroPython เพิ่มเติม

  • REPL — Ctrl-D / Ctrl-C / Ctrl-A ทำงานเหมือนบอร์ด MicroPython มาตรฐาน

  • ไม่มีการเปลี่ยนแปลง API แบบ breaking — flash ใหม่และสคริปต์ v2.1.0 ทำงานได้โดยไม่ต้องเปลี่ยนแปลง

ฟีเจอร์ใหม่¶

  • AprilTags — เพิ่มการตรวจจับ AprilTag fiducial marker (image.find_apriltags()) พร้อมการแก้ไขบั๊ก

  • CAN2 — เปิดใช้งาน CAN2 บน OpenMV 2 / 3 (พร้อม CAN init0/de-init ใน main)

  • เปิดใช้งานโมดูล MicroPython เพิ่มเติมและคืนค่า callback การจำลอง Pixy

การเปลี่ยนแปลงและการปรับปรุงอื่น ๆ¶

  • REPL ตอนนี้จัดการ Ctrl-D / Ctrl-C / Ctrl-A เหมือนบอร์ด MicroPython มาตรฐาน; อัปเดตสคริปต์ตัวอย่างการตรวจจับและติดตามใบหน้า

การแก้ไขบั๊ก¶

  • แก้ไขบั๊กการถอดรหัส QR code, ขั้นตอน Haar cascade และ LED ของ bootloader

ฮาร์ดแวร์และการรองรับบอร์ด¶

  • OpenMV 2 / 3 — เปิดใช้งาน CAN2; เพิ่มแฟล็ก SWD-programmer

การเปลี่ยนแปลง API แบบ Breaking¶

v2.2.0 ไม่มีการเปลี่ยนแปลง Python API แบบ breaking สคริปต์ที่ทำงานบน v2.1.0 ทำงานได้โดยไม่เปลี่ยนแปลงบน v2.2.0 — เพียง flash เฟิร์มแวร์ใหม่
