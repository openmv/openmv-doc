v5.0.0¶

v5.0.0 คือรุ่นแรกของ OpenMV Viewer OpenMV Viewer เป็นบิลด์แยกแบบอ่านอย่างเดียวของ OpenMV IDE สำหรับการแสดงผลิตภัณฑ์ที่เสร็จสมบูรณ์ซึ่งสร้างบน OpenMV Cam: มันสตรีมเอาต์พุตสดของกล้องและรันสคริปต์ แต่เครื่องมือพัฒนาถูกนำออก ลูกค้าหรือผู้ใช้ปลายทางจึงสามารถเห็นสิ่งที่ผลิตภัณฑ์ทำได้โดยไม่ต้องมีสภาพแวดล้อมการพัฒนาแบบเต็ม มันติดตั้งและทำงานแยกอิสระจาก OpenMV IDE

จุดเด่น¶

  • โปรแกรมดูข้อมูลเทเลเมทรีแบบอ่านอย่างเดียว — เฟรมบัฟเฟอร์ เทอร์มินัลอนุกรม และฮิสโทแกรม ทำงานเหมือนกับใน OpenMV IDE ทุกประการ คุณจึงสามารถดูสิ่งที่กล้องจับภาพและสิ่งที่สคริปต์พิมพ์ออกมาแบบสดได้

  • รันสคริปต์ของผลิตภัณฑ์ — ปุ่ม Run และ Stop ทำงานได้และรันสคริปต์จากดิสก์ และสคริปต์ที่กำลังรันสามารถหยุดได้เมื่อ Viewer เชื่อมต่อหรือยกเลิกการเชื่อมต่อ อุปกรณ์จึงอยู่ในสถานะที่ทราบเสมอ

  • เครื่องมือสร้างงานถูกนำออก — ฟีเจอร์การพัฒนาของโปรแกรมแก้ไข เครื่องมือสร้างงานด้านแมชชีนวิชัน เครื่องมือ AI และโมเดล และการตั้งค่าสำหรับนักพัฒนา ถูกนำออกทั้งหมด เหลือไว้เพียงแอปพลิเคชันสำหรับดูที่เจาะจง

  • ติดตั้งได้ด้วยตัวเอง — Viewer ถูกจัดแบรนด์และติดตั้งในชื่อ OpenMV Viewer พร้อมชื่อแอปพลิเคชัน การตั้งค่า และอัตลักษณ์ของตนเอง จึงอยู่ร่วมกับการติดตั้ง OpenMV IDE แบบเต็มได้

สิ่งที่รวมอยู่¶

  • เอาต์พุตจากกล้องแบบสด — โปรแกรมดูเฟรมบัฟเฟอร์ เทอร์มินัลอนุกรม และฮิสโทแกรม พร้อมชุดควบคุมเดียวกับ OpenMV IDE

  • Run / Stop และหยุดเมื่อเชื่อมต่อ — รันสคริปต์ของผลิตภัณฑ์จากดิสก์ และหยุดเมื่อ Viewer เชื่อมต่อหรือถอดออก

  • เมนู File และ Edit และ Preferences — ยังคงไว้เพื่อให้ Viewer ใช้งานได้ (เปิดไฟล์ ปรับส่วนติดต่อผู้ใช้) ในขณะที่หมวดหมู่สำหรับนักพัฒนาเท่านั้นถูกซ่อนไว้

  • การอัปเดตเฟิร์มแวร์และการกู้คืน — ตัวจัดการเฟิร์มแวร์ การกู้คืน bootloader และการโหลดระบบไฟล์ ROM ลงในกล้อง ยังคงไว้ Viewer จึงยังสามารถกู้คืนหรือจัดเตรียมอุปกรณ์ใหม่ในภาคสนามได้

  • เครื่องมือสร้าง fiducial — เครื่องมือสร้าง AprilTag, DataMatrix, QR และบาร์โค้ด ยังคงไว้ เนื่องจากมีประโยชน์นอกเหนือจากการพัฒนาด้วย

สิ่งที่ถูกนำออก¶

  • ฟีเจอร์การสร้างสคริปต์ของโปรแกรมแก้ไข เครื่องมือปรับแต่งแมชชีนวิชัน (โปรแกรมแก้ไขเทรชโฮลด์ โปรแกรมแก้ไขคีย์พอยต์ และอื่น ๆ ทำนองนั้น) เครื่องมือ AI และการฝึกโมเดล คลังโมเดลและโปรแกรมแก้ไขชุดข้อมูล และหมวดหมู่การตั้งค่าสำหรับนักพัฒนา ข้อความแจ้งอัปเกรดเฟิร์มแวร์และการเริ่มต้นใช้งานถูกระงับไว้ การสาธิตจึงไม่ถูกขัดจังหวะเพื่อเปลี่ยนแปลงอุปกรณ์

การรองรับแพลตฟอร์มและเครื่องมือ¶

  • ฐาน Qt Creator: 14.0.2 (ใช้ร่วมกับ OpenMV IDE)

  • ความพร้อมใช้งาน: Windows, macOS และ Linux ติดตั้งแยกจาก OpenMV IDE

การเปลี่ยนแปลงที่ทำให้เข้ากันไม่ได้¶

ไม่มี — นี่คือรุ่นแรก
