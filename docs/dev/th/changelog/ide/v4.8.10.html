v4.8.10¶

v4.8.10 เพิ่มการรองรับบอร์ด OpenMV AE3 และรีเฟรช AI toolchains ที่รวมมา โดยอัปเดต Ethos-U Vela NPU compiler เป็น 5.0.0 และเครื่องมือ ST Edge AI เป็น 4.0.0 รีลีสนี้ยังคงใช้ Qt Creator 14.0.2 โดยไม่มีการ rebase และไม่มีการเปลี่ยนแปลงที่ทำลาย compatibility

จุดเด่น¶

  • บอร์ด OpenMV AE3 Alif Ensemble E3 รองรับแล้ว รวมถึงการเขียนโปรแกรม bootloader/DFU ด้วย alif_tools และการเร่งความเร็วด้วย NPU Ethos-U55 ต่อคอร์

  • Ethos-U Vela NPU model compiler อัปเดตจาก 4.2.0 เป็น 5.0.0 สร้างใหม่บน Python 3.13 ABI

  • เครื่องมือแปลงโมเดล ST Edge AI (stedgeai) อัปเดตจาก 3.0.0 เป็น 4.0.0 บนทุกแพลตฟอร์ม

  • Fallback การเชื่อมต่อที่เร็วขึ้น: ขณะนี้ IDE ยกเลิกการเชื่อมต่อกล้องที่ล้มเหลวหลังจากพยายามเชื่อมต่อใหม่ 2 ครั้งแทนที่จะเป็น 10 ครั้ง

ฟีเจอร์ใหม่¶

  • การรองรับบอร์ด OpenMV AE3 Alif Ensemble E3 (ชื่อรหัสภายใน "AE3 classic") เพิ่มประเภทบอร์ด AE3 / OPENMV AE3 SBL2 โดยมีชื่อแสดง IDE เป็น OpenMV AE3 SBL2 (VID:PID 37C5:16E3), การเขียนโปรแกรม bootloader/DFU ด้วย alif_tools, โครงร่าง romfs แบบ dual-core (HP/HE) และการกำหนดค่าการเร่ง NPU Ethos-U55 Vela ต่อคอร์ (High Performance Core ethos-u55-256, High Efficiency Core ethos-u55-128) บอร์ดนี้เป็นเป้าหมายแบบซ่อน/DFU-programmed ที่เข้าถึงผ่าน VID:PID mappings ของ bootloader แทนที่จะเป็นเป้าหมาย connect ที่เลือกได้ตามปกติ (adbed3825)

การเปลี่ยนแปลงและการปรับปรุงอื่นๆ¶

  • ลดจำนวนครั้งสูงสุดของการพยายามเชื่อมต่อกล้องอัตโนมัติจาก 10 เป็น 2 ทำให้ IDE ยกเลิกได้เร็วขึ้นเมื่อการเชื่อมต่อล้มเหลว (2ddecec53)

  • อัปเดต Ethos-U Vela NPU model compiler ที่รวมมาจาก 4.2.0 เป็น 5.0.0 บน Windows, Linux และ macOS arm64 สร้างใหม่บน Python 3.13 ABI พร้อม binary regor และ mlw_codec ที่คอมไพล์ใหม่และอัปเดต numpy ที่รวมมาเป็น 2.4.4 บนทุกแพลตฟอร์ม (be9d29d9d, 04c73f187, 93260af21, a58458cfb, a6e1ac7ed)

  • อัปเดตเครื่องมือแปลงโมเดล ST Edge AI (stedgeai) ที่รวมมาจาก 3.0.0 เป็น 4.0.0 บน Windows, Linux และ macOS arm64 (1a727567c, 8dcfd8472, 27aabc740)

การรองรับแพลตฟอร์มและเครื่องมือ¶

  • ฐาน Qt Creator: 14.0.2

  • บอร์ดใหม่: OpenMV AE3 SBL2 (Alif Ensemble E3) เป็นเป้าหมายแบบซ่อน/DFU-programmed ที่เขียนโปรแกรมผ่าน alif_tools พร้อมการเร่ง NPU Ethos-U55 แบบ dual-core

  • Ethos-U Vela NPU compiler: 4.2.0 -> 5.0.0 (Windows, Linux, macOS arm64) ติดตั้งใหม่บน Python 3.13 ABI

  • เครื่องมือ ST Edge AI (stedgeai): 3.0.0 -> 4.0.0 (Windows, Linux, macOS arm64)

  • numpy ที่รวมมา (สภาพแวดล้อม Vela): อัปเดตเป็น 2.4.4 บนทุกแพลตฟอร์ม

การเปลี่ยนแปลงที่ทำลาย compatibility¶

ไม่มี รีลีสนี้ไม่มีการเปลี่ยนแปลงเวิร์กโฟลว์หรือพฤติกรรมที่ผู้ใช้สังเกตเห็นได้ และโปรเจ็กต์ที่มีอยู่ยังคงทำงานได้โดยไม่มีการเปลี่ยนแปลง
