ภายใน OpenMV MicroPython¶

บทนี้ครอบคลุมการสำรวจ MicroPython จากมุมมองของนักพัฒนาที่มีส่วนร่วมกับ MicroPython โดยทำหน้าที่เป็นทรัพยากรครอบคลุมเกี่ยวกับรายละเอียดการดำเนินงานของ MicroPython สำหรับทั้งผู้มีส่วนร่วมใหม่และผู้เชี่ยวชาญ

การพัฒนาเกี่ยวกับ MicroPython มักเกี่ยวข้องกับการปรับเปลี่ยน core runtime การพอร์ตหรือดูแลไลบรารีใหม่ คู่มือนี้อธิบายรายละเอียดการดำเนินงานของ MicroPython อย่างละเอียด รวมถึงคู่มือเริ่มต้น เนื้อในของ compiler การพอร์ต MicroPython ไปยังแพลตฟอร์มใหม่ และการดำเนินการไลบรารี MicroPython หลัก
