4.10. Sensor สี¶

โฟโตไดโอดในแต่ละเซลล์ไม่สามารถแยกสีได้ มันนับโฟตอนของทุกความยาวคลื่นที่ซิลิกอนดูดซับโดยไม่แยกแดงจากเขียวจากน้ำเงิน เพื่อให้ได้ภาพสีจาก sensor ที่แยกสีไม่ได้ ผู้ผลิตจะคลุมตารางพิกเซลด้วยอาร์เรย์ตัวกรองสี (CFA): ฟิล์มบาง ๆ ของสีย้อมที่ให้แต่ละเซลล์มองเห็นเพียงหนึ่งในสามสีหลัก

4.10.1. รูปแบบ Bayer¶

เลย์เอาต์ CFA ที่โดดเด่นที่สุดคือรูปแบบ Bayer ตามชื่อผู้คิดค้นที่ Kodak ฟิล์มสลับระหว่างสองประเภทแถว: แถวคู่เป็นรูปแบบแดง-เขียวซ้ำ (แดง, เขียว, แดง, เขียว) และแถวคี่เป็นรูปแบบเขียว-น้ำเงินซ้ำ (เขียว, น้ำเงิน, เขียว, น้ำเงิน) ในไทล์ซ้ำที่เล็กที่สุด -- สองแถวคูณสองคอลัมน์ -- เซลล์หนึ่งมองเห็นแดง หนึ่งมองเห็นน้ำเงิน และสองมองเห็นเขียว

เขียวถูกสองเท่าโดยตั้งใจ การมองเห็นของมนุษย์ไวต่อสีเขียวมากกว่าสีแดงหรือสีน้ำเงินมาก และความสว่างที่รับรู้ของฉากส่วนใหญ่มาจากช่องสัญญาณสีเขียว การสุ่มตัวอย่างสีเขียวสองเท่าของความหนาแน่นของสีแดงและน้ำเงินทำให้งบประมาณความละเอียดอยู่ในจุดที่ดวงตาสังเกตเห็นมากที่สุด และซ่อนความนุ่มนวลของ chrominance ที่ตามมา

4.10.2. สิ่งที่แต่ละพิกเซลบันทึก¶

sensor สียังคงเก็บตัวเลขเพียงหนึ่งตัวต่อพิกเซล -- จำนวนโฟตอนที่ผ่านตัวกรองสีของพิกเซลนั้น เซลล์ตัวกรองแดงบันทึกค่าช่องสัญญาณแดงของมัน ค่าเขียวและน้ำเงินที่ตำแหน่งเดียวกันหายไปจากข้อมูลเพียงอย่างเดียว เช่นเดียวกันกับเซลล์เขียวและน้ำเงิน

ข้อมูลที่ออกจาก sensor ในรูปแบบ Bayer ดิบจึงเป็นช่องสัญญาณหนึ่งต่อพิกเซล จัดเรียงในรูปแบบ Bayer แทนที่จะเป็นสามช่องสัญญาณต่อพิกเซลของภาพสีสำเร็จรูป การสร้างสองช่องสัญญาณที่ขาดหายไปใหม่ในทุกตำแหน่งเซลล์เรียกว่า debayering

4.10.3. ไมโครเลนส์และมุมรังสีหลัก¶

ตัวกรองสีไม่ใช่สิ่งเดียวที่อยู่บนโฟโตไดโอด เหนือมันมีไมโครเลนส์เล็ก ๆ ที่โฟกัสกรวยแสงที่เข้ามาลงบนพื้นที่ใช้งานของโฟโตไดโอด และไมโครเลนส์ถูกออกแบบโดยสมมติว่าแสงมาในมุมใกล้ตั้งฉากกับพื้นผิว sensor เมื่อแสงมาถึงในมุมชันแทน -- มุมรังสีหลัก ที่เพิ่มขึ้นไปยังมุมของเลนส์จริงทุกตัว -- บางส่วนอาจตกบนตัวกรองของพิกเซลข้างเคียงแทน รับสีที่ผิดและเกิดการรบกวนสี พิกเซลมุมก็สูญเสียความอิ่มตัวของสีด้วย เพราะบางส่วนของกรวยพลาดโฟโตไดโอดไปทั้งหมด

sensor ชดเชยโดยเลื่อนไมโครเลนส์ของแต่ละพิกเซลออกไปตามรัศมีจากศูนย์กลาง sensor ในวงแหวนที่ขยายออกจากตรงกลางไปยังมุม การเลื่อนเป็นศูนย์ที่ศูนย์กลางและเพิ่มขึ้นเป็นไม่กี่ไมครอนที่วงแหวนนอกสุด ปรับตามโปรไฟล์ CRA ที่กำหนดไว้ในการออกแบบ sensor การจับคู่ sensor กับเลนส์ที่โปรไฟล์ CRA แตกต่างจากเป้าหมายการออกแบบอย่างมากจะทิ้งข้อผิดพลาดด้านสีและความไวที่มองเห็นในมุม ซึ่งเป็นเหตุผลที่ image sensor และเลนส์มักถูกเลือกร่วมกัน
