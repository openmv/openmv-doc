13.2. OpenMV Viewer#

OpenMV Viewer คือโปรแกรมคู่หูแบบอ่านอย่างเดียวของ OpenMV IDE ในขณะที่ IDE ใช้สำหรับสร้างแอปพลิเคชันบน OpenMV Cam นั้น Viewer ใช้สำหรับ แสดง แอปพลิเคชันที่เสร็จสมบูรณ์แล้ว: มันเชื่อมต่อกับกล้อง รันสคริปต์ของมัน และสตรีมเฟรมบัฟเฟอร์สด เอาต์พุตอนุกรม และฮิสโทแกรม -- แต่เครื่องมือพัฒนาหายไป ลูกค้า เพื่อนร่วมงาน หรือผู้เยี่ยมชมบูธสามารถดูสิ่งที่ผลิตภัณฑ์ทำได้โดยไม่ต้องมีโปรแกรมแก้ไขแบบเต็ม เครื่องมือปรับแต่งแมชชีนวิชัน หรือการตั้งค่าที่สำคัญเฉพาะขณะสร้างมันเท่านั้น

Viewer ติดตั้งและทำงานได้ด้วยตัวเอง แยกจาก OpenMV IDE และทั้งสองสามารถติดตั้งเคียงข้างกันได้

หน้าต่างนี้เก็บส่วนต่าง ๆ ของ OpenMV IDE ที่แสดงสิ่งที่กล้องกำลังทำอยู่ frame buffer viewer แสดงสิ่งที่กล้องจับภาพได้ล่าสุด serial terminal ด้านล่างแสดงทุกอย่างที่สคริปต์พิมพ์ออกมา และ histogram วาดกราฟสถิติพิกเซลของเฟรม คอนโทรลสำหรับเชื่อมต่อและรันอยู่บนแถบเครื่องมือ และแถบสถานะรายงานสิ่งที่กล้องที่เชื่อมต่ออยู่กำลังทำ สิ่งที่หายไปคือทุกอย่างที่คุณจะใช้เพื่อ เปลี่ยนแปลง แอปพลิเคชัน: ฟีเจอร์การเขียนสคริปต์ เครื่องมือปรับแต่งการมองเห็นด้วยเครื่อง เครื่องมือ AI และโมเดล รวมถึงการตั้งค่าสำหรับนักพัฒนาของ IDE เอง ถูกนำออกไปทั้งหมด

เครื่องมือแก้ไขหนึ่งอย่างยังคงอยู่ เพราะมันมุ่งไปที่ใครก็ตามที่ ใช้งาน ผลิตภัณฑ์ มากกว่าใครที่สร้างมันขึ้นมา Viewer สามารถเปิดคอนฟิกการตั้งค่าที่จัดเก็บอยู่บนกล้องที่เชื่อมต่ออยู่ และนำเสนอในรูปแบบฟอร์ม เพื่อให้ผู้ปฏิบัติงานสามารถเปลี่ยนการตั้งค่าของผลิตภัณฑ์ -- ค่าเกณฑ์ โหมด รหัสผ่าน Wi-Fi -- ได้โดยไม่ต้องใช้ IDE มันเปิดคอนฟิกที่มีอยู่แล้วเท่านั้น การสร้างคอนฟิกขึ้นมาเป็นงานของ IDE ดู settings editor สำหรับสิ่งที่คอนฟิกสามารถบรรจุได้

แป้นพิมพ์ลัดในบทนี้เขียนไว้สำหรับ Windows และ Linux; บน macOS ให้ใช้ Cmd แทน Ctrl
