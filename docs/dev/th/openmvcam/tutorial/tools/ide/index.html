13.1. OpenMV IDE¶

OpenMV IDE คือแอปพลิเคชันเดสก์ท็อปสำหรับทำงานกับ OpenMV Cam ของคุณ ทุกอย่างในนั้นรับใช้หนึ่งวงจร: แก้ไขสคริปต์ MicroPython รันบนกล้อง และดูบัฟเฟอร์เฟรมและสตรีมเอาต์พุตอนุกรมกลับมาสดขณะที่มันทำงาน IDE สร้างบนแกนตัวแก้ไข Qt Creator และแอปพลิเคชันทั้งหมดถูกออกแบบรอบวงจรนั้น -- สคริปต์ไม่ต้องการไฟล์โปรเจกต์ การกำหนดค่าบิลด์ หรือการตั้งค่าคอมไพเลอร์ คุณเขียนสคริปต์และรันบนกล้อง นั่นคือโมเดลทั้งหมด

หน้าต่างหลักมีสี่บริเวณทำงาน ตัวแก้ไข เต็มด้านซ้ายและเก็บสคริปต์ที่เปิดอยู่ พร้อม เทอร์มินัลอนุกรม ด้านล่างที่แสดงทุกสิ่งที่สคริปต์พิมพ์ โปรแกรมดูบัฟเฟอร์เฟรม อยู่ด้านขวาบนและแสดงสิ่งที่กล้องจับภาพล่าสุด และ ฮิสโตแกรม ด้านล่างวาดสถิติพิกเซลของสิ่งที่บัฟเฟอร์เฟรมแสดง คอลัมน์ปุ่มแถบเครื่องมือตามขอบซ้ายสุดมีการควบคุมไฟล์ การแก้ไข การเชื่อมต่อ และการรัน และแถบสถานะตามขอบล่างรายงานสิ่งที่กล้องที่เชื่อมต่อกำลังทำ คุณสามารถยุบบานหน้าต่างใดก็ได้เมื่อสคริปต์ต้องการพื้นที่

ทางลัดแป้นพิมพ์ในบทนี้เขียนสำหรับ Windows และ Linux บน macOS ให้แทนที่ Cmd สำหรับ Ctrl
