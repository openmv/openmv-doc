14.4. การทำงานกับ TLS certificate¶

โมดูล ssl บน OpenMV Cam (สร้างบน mbedTLS) ช่วยให้สคริปต์สามารถสร้างการเชื่อมต่อเครือข่ายที่เข้ารหัสและพิสูจน์ตัวตนได้ เพื่อทำอะไรที่มากกว่าการเข้ารหัสธรรมดา คุณต้องมี certificate หน้าต่างๆ ในส่วนนี้ครอบคลุม: certificate คืออะไร, ประเภท key และรูปแบบไฟล์ใดที่กล้องยอมรับ, วิธีสร้าง self-signed certificate สำหรับการพัฒนาและขอรับ CA-signed certificate สำหรับ production, วิธีนำ certificate ไปวางบนกล้องและตรวจสอบ server ระยะไกล, วิธีปกป้อง private key และวิธีที่การหมดอายุและการหมุนเวียน certificate ส่งผลต่ออุปกรณ์ที่ deploy ไปแล้ว

  • 14.4.1. แนวคิด: ความเชื่อถือ, key, และรูปแบบไฟล์
  • 14.4.2. ข้อกำหนดเบื้องต้น: OpenSSL และนาฬิกา
  • 14.4.3. ใบรับรอง Self-signed
  • 14.4.4. CA-signed certificate (ที่ได้รับการเชื่อถือสาธารณะ)
  • 14.4.5. การตรวจสอบเซิร์ฟเวอร์สาธารณะ (กล้องทำหน้าที่เป็น client)
  • 14.4.6. การดำเนินการ: คีย์ การหมดอายุ และการแก้ปัญหา
