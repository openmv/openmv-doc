บทเรียน OpenMV Cam#

OpenMV Cam คือกล้องขนาดเล็กที่สามารถโปรแกรมได้และขับเคลื่อนด้วย MicroPython ออกแบบมาเพื่อการมองเห็นของเครื่องแบบฝังตัว บทเรียนนี้ครอบคลุมทุกขั้นตอนตั้งแต่ต้นจนจบ -- ตั้งแต่การแกะกล่องกล้องและเขียนสคริปต์แรกใน OpenMV IDE ไปถึงพื้นฐาน Python และฮาร์ดแวร์ที่สคริปต์ทุกตัวต้องอาศัย ต่อด้วยสแต็กการสร้างภาพของกล้องและเครื่องมือประมวลผลภาพและแมชชีนเลิร์นนิงที่สร้างขึ้นเหนือมัน ผ่านส่วนการทำงานพร้อมกันและการเชื่อมต่อเครือข่ายที่แอปพลิเคชันต้องใช้เมื่อเติบโตขึ้น และสุดท้ายคืองานจริงในการนำต้นแบบที่ใช้งานได้ไปสู่ผลิตภัณฑ์ที่จัดส่ง

บทต่างๆ เขียนขึ้นให้อ่านตามลำดับสำหรับผู้ใช้ใหม่ แต่แต่ละบทก็ครบสมบูรณ์ในตัวเอง -- ผู้อ่านที่รู้จัก Python อยู่แล้วสามารถข้ามส่วนภาพรวม Python ได้ และผู้อ่านที่เคยจัดส่งผลิตภัณฑ์มาก่อนสามารถข้ามไปที่ Production ได้เลย เมื่อบทหนึ่งอ้างอิงสิ่งที่กล่าวถึงก่อนหน้า จะมีลิงก์ย้อนกลับให้
