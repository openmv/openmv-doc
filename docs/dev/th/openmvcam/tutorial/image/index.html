5. การประมวลผลภาพ¶

ทุกหน้าก่อนหน้านี้จัดการข้อมูลพิกเซลเพียงพอที่จะรับมันได้เท่านั้น: บันทึกเฟรม กำหนดค่า sensor ย้ายไบต์จากกล้องเข้าสู่ RAM การประมวลผลภาพ พลิกทิศทาง -- พิกเซลคือ input และเป้าหมายคือสิ่งที่แอปพลิเคชันต้องการจากมัน

โมดูล image คือชุดเครื่องมือ การดำเนินการทุกอย่างรับออบเจกต์ Image หนึ่งหรือสองชิ้นและส่งกลับ Image ที่แปลงแล้ว -- พิกเซลที่ผ่านการกรอง กำหนดค่าขีดแบ่ง บิดเบี้ยว หรือประกอบต่างออกไป -- หรือรายการออบเจกต์ผลลัพธ์ ได้แก่ บลอบ เส้น โค้ด การจับคู่ และสถิติ สองรูปแบบนี้ประกอบกันได้ pipeline ทั่วไปจะทำความสะอาดเฟรมที่บันทึก ดึงชุดออบเจกต์ผลลัพธ์จากมัน แปลงเป็นการวัด และวาดคำอธิบายประกอบกลับลงบนเฟรมเพื่อแสดงผล
