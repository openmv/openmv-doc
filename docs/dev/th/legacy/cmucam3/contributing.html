การมีส่วนร่วมกับ cc3#

เราสนับสนุนให้ผู้คนร่วมส่งโครงการ ไลบรารี การแก้บั๊ก และการปรับปรุงของพวกเขา

การแก้ไขบั๊ก#

วิธีที่ดีที่สุดในการร่วมส่งการแก้ไขบั๊กคือผ่านระบบทิกเก็ต ยินดีรับแพตช์

วิกิ#

เรากำลังพิจารณาให้มีหน้าส่วนตัวบนวิกิสำหรับโค้ดและไอเดียที่ร่วมส่งเข้ามา

การอภิปราย#

กระดานสนทนาเปิดให้โพสต์ ลองพิจารณาโพสต์ไอเดียหรือโค้ด!

การเข้าถึง SVN#

เรากำลังพิจารณาให้สิทธิ์เข้าถึงการคอมมิตในบางส่วนของที่เก็บควบคุมเวอร์ชัน ด้วยวิธีนี้ผู้ที่สนใจสามารถร่วมส่งโค้ดเข้าโครงการได้โดยตรง
