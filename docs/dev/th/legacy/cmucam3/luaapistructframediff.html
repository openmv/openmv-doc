Lua API: framediff#

คำอธิบาย#

โครงสร้างข้อมูล framediff เก็บข้อมูลเกี่ยวกับความแตกต่างระหว่างสองเฟรมที่แตกต่างกันที่ถ่ายด้วย CMUcam3 ภายในนั้นเฟรมเหล่านี้ถูกเก็บไว้ในเทมเพลต "current" และ "previous" การเปรียบเทียบระหว่างสองเทมเพลตนั้นทำโดยใช้ framediff_scanline_start, framediff_scanline และ framediff_scanline_finish

ตัวอย่าง#

ตัวอย่างนี้สาธิตการใช้งานโดยใช้ CMUcam3 และการทำ frame differencing มันวนซ้ำ ถ่ายภาพจนกระทั่งภาพหนึ่งแตกต่างจากภาพก่อนหน้า ณ จุดนั้นมันจะบันทึกภาพนั้นลงดิสก์โดยเรียก save_picture

-- get a new framediff, and initialize it
fdiff = framediff_new(16, 16);
fdiff:set_total_x(pixbuf_get_width());
fdiff:set_total_y(pixbuf_get_height());
fdiff:set_threshold(10);
img = image_new(pixbuf_get_width(), 1); -- need an image object for the framediff'ing

take_picture();
fdiff:set_load_frame(true);

-- take the first picture
if (framediff_scanline_start(fdiff) > 0) then
    while pixbuf_read_rows(img, 1) > 0 do
        framediff_scanline(img, fdiff);
    end
    framediff_scanline_finish(fdiff);
end

fdiff:set_load_frame(false);

-- now keep taking pictures until a picture is different than its predecessor
repeat
    take_picture();
    if (framediff_scanline_start(fdiff) > 0) then
        while pixbuf_read_rows(img, 1) > 0 do
            framediff_scanline(img, fdiff);
        end
        framediff_scanline_finish(fdiff);
    else
        print("Error!");
    end

    fdiff:swap_templates(); -- put current template in previous template spot
until fdiff:get_num_pixels() > fdiff:get_threshold()

-- we got an image that was different! Save it to disk, and send it to the Camscripter
save_picture("Diff.ppm"):
print_picture();

dispose#

fdiff:dispose()

พารามิเตอร์:

ไม่มี

ค่าที่ส่งคืน:

ไม่มี

ให้วิธีการกำจัดหน่วยความจำที่เกี่ยวข้องกับโครงสร้างข้อมูล framediff นี้อย่างชัดเจน แทนที่จะรอให้ตัวเก็บขยะของ Lua กำจัดหน่วยความจำนั้น คำเตือน: ข้อผิดพลาดมีแนวโน้มที่จะเกิดจากการใช้ framediff หลังจากเรียก dispose บนมันแล้ว ขอแนะนำอย่างยิ่งว่าคุณไม่ควรทำเช่นนั้น

get_coi#

fdiff:get_coi()

พารามิเตอร์:

ไม่มี

ค่าที่ส่งคืน:

แชนเนลที่สนใจสำหรับโครงสร้าง framediff นี้

รับช่องที่สนใจสำหรับโครงสร้างนี้ ดูค่าคงที่ของช่องที่สนใจใน Lua API: ค่าคงที่ สำหรับค่าที่อาจส่งคืนได้

get_load_frame#

fdiff:get_load_frame()

พารามิเตอร์:

ไม่มี

ค่าที่ส่งคืน:

ว่าเราต้องการโหลดเฟรมใหม่หรือไม่

รับค่าของ load_frame สำหรับออบเจกต์ framediff นี้ load_frame กำหนดว่าเราต้องโหลดเฟรมใหม่จาก CMUcam3 หรือไม่ก่อนที่จะหาความแตกต่างของเทมเพลตปัจจุบันและก่อนหน้า

get_num_pixels#

พารามิเตอร์:

ไม่มี

ค่าที่ส่งคืน:

หลังจากใช้ framediff_scanline_start, framediff_scanline และ framediff_scanline_finish แล้ว ให้เรียกเมธอดนี้เพื่อรับจำนวนพิกเซลของความแตกต่างที่พบระหว่างเทมเพลตปัจจุบันกับเทมเพลตก่อนหน้า

set_coi#

พารามิเตอร์:

channel_of_interest แชนเนลที่สนใจใหม่สำหรับการทำ frame differencing

ค่าที่ส่งคืน:

ไม่มี

ตั้งค่าแชนเนลที่สนใจสำหรับการทำ framedifferencing พารามิเตอร์ channel_of_interest ต้องเป็นหนึ่งในค่าคงที่แชนเนลที่สนใจใน Lua API: ค่าคงที่

พารามิเตอร์:

ไม่มี
