ส่วนติดต่อผู้ใช้แบบกราฟิกของ CMUcam4#

CMUcam4GUI คืออะไร#

CMUcam4GUI เป็นเครื่องมือที่มีประโยชน์ซึ่งทำให้การทำงานกับ CMUcam4 ง่ายขึ้น CMUcam4GUI ให้คุณถ่ายภาพด้วย CMUcam4 ของคุณและดูภาพเหล่านั้นบนคอมพิวเตอร์ของคุณ และใช้ภาพเพื่อคำนวณขอบเขตการติดตามสีของวัตถุสีในภาพที่คุณต้องการให้ CMUcam4 ติดตาม การใช้ CMUcam4GUI คุณสามารถกำหนดขอบเขตการติดตามสีสำหรับวัตถุในฉากแสงหนึ่งได้อย่างรวดเร็ว และขยายขอบเขตการติดตามสีเพื่อรองรับการติดตามวัตถุในฉากแสงที่เปลี่ยนแปลง CMUcam4GUI ยังให้คุณเห็นสิ่งที่ CMUcam4 กำลังติดตามแบบเรียลไทม์โดยไม่ต้องใช้ทีวี โดยแสดงเนื้อหาของแพ็กเก็ตการติดตาม ค่าเฉลี่ย ฮิสโทแกรม และเฟรมในรูปแบบกราฟิกที่ส่งมาจาก CMUcam4 นอกจากนี้ CMUcam4GUI ยังให้คุณเล่นกับตัวเลือกทั้งหมดของ CMUcam4 แบบเรียลไทม์ เช่น ควบคุมเซอร์โวแพนและก้ม ตัวเลือกการติดตามสี และการควบคุมกล้องระดับต่ำ สุดท้าย CMUcam4GUI ให้คุณจัดการการ์ด secure digital ที่ต่ออยู่กับ CMUcam4 จากระยะไกลผ่าน CMUcam4

การตั้งค่า Windows#

ทำตามขั้นตอนด้านล่างเพื่อตั้งค่า CMUcam4GUI สำหรับ Windows:

  1. ดาวน์โหลดไฟล์ CMUcam4GUI.zip

  2. แตกไฟล์ zip และดึงเนื้อหาออก

  3. รันไฟล์ CMUcam4GUI.exe

การตั้งค่า Linux#

ทำตามขั้นตอนด้านล่างเพื่อตั้งค่า CMUcam4GUI สำหรับ Linux:

  1. เพิ่มชื่อผู้ใช้ของคุณเข้าไปในกลุ่ม dialout

sudo adduser username dialout

  1. ตอนนี้ตรวจสอบโดยการพิมพ์

groups username

  1. ตอนนี้ออกจากระบบแล้วเข้าสู่ระบบใหม่

  2. ดาวน์โหลดไฟล์ CMUcam4GUI.zip

  3. แตกไฟล์ zip และดึงเนื้อหาออก

  4. รันไฟล์ CMUcam4GUI.sh

การตั้งค่า Macintosh#

ทำตามขั้นตอนด้านล่างเพื่อตั้งค่า CMUcam4GUI สำหรับ Macintosh:

  1. ติดตั้ง Qt 1.2.1 SDK (Desktop 4.8.1) สำหรับ Mac

  2. ดาวน์โหลด ซอร์ส CMUcam4GUI

  3. เปิดไฟล์ CMUcam4GUI.pro โดยใช้ QtCreator แล้วสลับจากโหมด debug เป็นโหมด release

  4. เปิดไฟล์ CMUcom4.h และเปลี่ยน #define CMUCOM4_FAST_BAUD_RATE จาก 230400 เป็น 115200 (Mac ไม่สามารถรองรับอัตราบอด 230400 BPS ได้)

  5. คลิก run

คำถาม: ทำไมคุณจึงไม่มีแอปพลิเคชัน MAC ที่สร้างไว้ล่วงหน้า?

คำตอบ: Kwabena ผู้พัฒนา CMUcam4 ไม่มีเครื่อง MAC...

คู่มือ CMUcam4GUI#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

คำสั่งแถบเมนูและแถบเครื่องมือ

คำสั่งแถบเครื่องมือ

  • Begin/Reset - เชื่อมต่อกับ CMUcam4

  • End - ตัดการเชื่อมต่อจาก CMUcam4

  • Sleep Lightly - ทำให้ CMUcam4 เข้าสู่สถานะพลังงานต่ำมาก

  • Sleep Deeply - ทำให้ CMUcam4 เข้าสู่สถานะพลังงานต่ำพิเศษ

  • Send Frame - จับภาพสีและแสดงในหน้าต่างแสดงผล

  • Send Bitmap - จับภาพไบนารีและแสดงในหน้าต่างแสดงผล

    • นี่คือภาพไบนารีขนาด 80x60 ของสิ่งที่ CMUcam4 กำลังติดตาม - CMUcam4 ต้องกำลังติดตามบางสิ่งเพื่อให้สิ่งนี้มีประโยชน์

  • Dump Frame - จับภาพสีและบันทึกลงในการ์ด SD

  • Dump Bitmap - จับภาพไบนารีและบันทึกลงในการ์ด SD

    • นี่คือภาพไบนารีขนาด 80x60 ของสิ่งที่ CMUcam4 กำลังติดตาม - CMUcam4 ต้องกำลังติดตามบางสิ่งเพื่อให้สิ่งนี้มีประโยชน์

คำสั่งแถบเมนู

  • Save Frame - บันทึกภาพสีในหน้าต่างแสดงผลลงในไฟล์

  • Save Bitmap - บันทึกภาพไบนารีในหน้าต่างแสดงผลลงในไฟล์

    • นี่คือภาพไบนารีขนาด 80x60 ของสิ่งที่ CMUcam4 กำลังติดตาม - CMUcam4 ต้องกำลังติดตามบางสิ่งเพื่อให้สิ่งนี้มีประโยชน์

  • Clear Frame - ลบภาพสีในหน้าต่างแสดงผล

  • Clear Bitmap - ลบภาพไบนารีในหน้าต่างแสดงผล

    • นี่คือภาพไบนารีขนาด 80x60 ของสิ่งที่ CMUcam4 กำลังติดตาม - CMUcam4 ต้องกำลังติดตามบางสิ่งเพื่อให้สิ่งนี้มีประโยชน์

  • Open - เปิดภาพสีเพื่อใช้กับฟีเจอร์การเลือกติดตาม

  • Exit - ออกจากแอปพลิเคชัน

แท็บ Viewing

  • Send Frame Control - เลือกความละเอียดแนวนอนและแนวตั้งสำหรับปุ่ม send frame ในแถบเครื่องมือ

    • คุณไม่สามารถขยับสิ่งที่คุณกำลังถ่ายภาพอยู่ในขณะที่คำสั่ง send frame กำลังทำงาน

    • หากคุณภาพของภาพดูแย่ ให้ปิด auto-gain และ auto-white balance และตรวจสอบให้แน่ใจว่า CMUcam4 กำลังมองที่วัตถุที่มีแสงส่อง

    • ใช้ความละเอียดที่เล็กลงเพื่อจับภาพเร็วขึ้น - เช่น 160x120

  • Dump Frame Control - เลือกความละเอียดแนวนอนและแนวตั้งสำหรับปุ่ม dump frame ในแถบเครื่องมือ

    • คุณไม่สามารถขยับสิ่งที่คุณกำลังถ่ายภาพอยู่ในขณะที่คำสั่ง dump frame กำลังทำงาน

    • หากคุณภาพของภาพดูแย่ ให้ปิด auto-gain และ auto-white balance และตรวจสอบให้แน่ใจว่า CMUcam4 กำลังมองที่วัตถุที่มีแสงส่อง

    • ใช้ความละเอียดที่เล็กลงเพื่อจับภาพเร็วขึ้น - เช่น 160x120

  • Opacity Control - เลือกความทึบของเฟรมที่ส่งและบิตแมปที่ส่ง - ระหว่าง 1.00 สำหรับ 100% และ 0.00 สำหรับ 0%

  • Tracking Control - เลือกก้อนสีในหน้าต่างแสดงผลเพื่อติดตาม

    • สี่เหลี่ยมที่เลือกควรครอบเฉพาะพิกเซลภายในหยดสี (color blob) ที่คุณต้องการติดตามเท่านั้น

    • หลังจากเลือกก้อนสีแล้ว ให้คลิกปุ่ม track color เพื่อคำนวณขอบเขตสีสำหรับติดตามก้อนสีนั้น

      • อัลกอริทึมทำงานโดยการคำนวณค่าเฉลี่ยและส่วนเบี่ยงเบนมาตรฐานของแชนเนลสีแดง เขียว และน้ำเงินภายในสี่เหลี่ยมที่เลือก

      • สัมประสิทธิ์ระดับช่วงความเชื่อมั่น (Confidence Interval Level Coefficient) เลือกว่าจะบวก/ลบ (+/-) ส่วนเบี่ยงเบนมาตรฐานเข้ากับค่าเฉลี่ยของแชนเนลสีแดง เขียว และน้ำเงินมากเพียงใด - ระหว่าง 0 สำหรับ +/- (stdev * 1) ถึง 99 สำหรับ +/- (stdev * 10)

แท็บ Tracking

เปิดใช้งานโหมด Line ในแผงตัวเลือกเพื่อดูพิกเซลที่ติดตามสีน้ำเงิน

แท็บ Options

แท็บ Disk

  • Disk Information - รับข้อมูลเกี่ยวกับเรขาคณิตของดิสก์

  • Disk Space - รับข้อมูลเกี่ยวกับพื้นที่บนดิสก์

  • Disk Control - ยกเลิกการเมาท์/ฟอร์แมตดิสก์

    • หมายเหตุ: คุณควรยกเลิกการเมาท์ดิสก์หลังจากใช้คำสั่งที่เกี่ยวข้องกับการ์ด SD ใดๆ ก่อนปิดส่วนติดต่อผู้ใช้แบบกราฟิก

    • หมายเหตุ: การคลิกปุ่มฟอร์แมตดิสก์จะฟอร์แมตดิสก์ทันที

  • New File - สร้างไฟล์ใหม่พร้อมข้อความบางส่วนหรือเพิ่มข้อความต่อท้ายไฟล์

  • New Folder - สร้างโฟลเดอร์ใหม่

  • File Explorer - โต้ตอบกับไฟล์/โฟลเดอร์บนการ์ด SD จากคอมพิวเตอร์ของคุณ

    • คลิก refresh เพื่อแสดงรายการเนื้อหาของไดเรกทอรีปัจจุบัน

    • เปลี่ยนแอตทริบิวต์ของไฟล์หรือโฟลเดอร์โดยการคลิกที่ช่องกาแอตทริบิวต์

      • ดับเบิลคลิกที่โฟลเดอร์เพื่อเข้าไป

      • ดับเบิลคลิกที่ไฟล์เพื่อเปิด

        • อย่าพยายามเปิดไฟล์ที่ใหญ่กว่าประมาณ 10 KB (เว้นแต่คุณอยากรอตลอดกาล) เพราะการถ่ายโอนไฟล์ช้ามากที่ประมาณ 4 KBs

        • คุณสามารถย้ายไฟล์และโฟลเดอร์ได้โดยเปลี่ยนชื่อด้วยพาธใหม่ เช่น เปลี่ยนชื่อ txt.txt เป็น ../txt.txt

วิธีกำหนดขอบเขตสี#

  • ไปที่ options

  • เปิด YUV

  • รอ 5 วินาที

  • ปิด autogain และ autowhite balance

  • เลือกความละเอียดแนวนอนของ send frame แล้วตั้งค่าเป็น 160

  • เลือกความละเอียดแนวตั้งของ send frame แล้วตั้งค่าเป็น 120

  • เล็ง CMUcam4 ไปที่วัตถุที่คุณต้องการติดตาม จากนั้นโดยไม่ขยับ CMUcam4 หรือวัตถุ และทั้งคู่ต้องไม่เคลื่อนไหว ให้คลิก send frame

  • รอจนกว่าเฟรมจะถูกส่งไปยัง GUI อย่าขยับวัตถุ

  • เลือกพื้นที่สีของวัตถุ (และไม่มีอะไรนอกพื้นที่นั้น)

  • คลิก track selection

  • คลิก okay โดยใช้ 50 เป็นค่าเริ่มต้น

  • หากมีเพียงวัตถุที่มีสีน้ำเงินวาดอยู่บนมัน ให้คลิก track selection อีกครั้งและทำให้ 50 เท่ากับ 60

  • ทำซ้ำข้างต้นจนกว่าคุณจะดันค่า track selection ให้สูงที่สุดเท่าที่จะเป็นไปได้ คุณสามารถหยุดได้เมื่อมันเป็น 99 หากคุณเริ่มกำหนดขอบเขตสิ่งที่ไม่ต้องการ คุณก็เสร็จเช่นกัน

  • จากนั้นไปที่ options... ทำให้ค่า green min เป็น 0 และค่า green max เป็น 255 (สิ่งนี้ตั้งค่า Ymin เป็น 0 และ Ymax เป็น 255) หากคุณเริ่มครอบสิ่งที่ไม่ต้องการ ลองใช้ขอบเขตที่แคบลงจนกว่าคุณจะครอบเฉพาะวัตถุที่คุณต้องการ คุณสามารถรีเฟรชสิ่งที่ถูกครอบได้โดยคลิกปุ่ม sendBitmap

  • และ... จากนั้นคุณควรจะมีตัวเลือกการติดตามหกแบบ... นอกจากนี้ คุณยังสามารถขยายขอบเขต (V/แดง) และ (U/น้ำเงิน) ด้วยตนเองได้ด้วยหากต้องการ สิ่งนี้จะช่วยให้ระบบทนทานมากขึ้นในสภาพแสงต่างๆ และอื่นๆ

ในการดูสิ่งที่ CMUcam4 เห็น....

  • เปิดโหมดเส้น (line mode) (มีช่องทำเครื่องหมายใน options)

  • จากนั้นไปที่ viewing และคลิก track color
