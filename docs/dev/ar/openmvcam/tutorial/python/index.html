2. نظرة عامة على Python#

تعمل OpenMV Cam بنظام MicroPython، وهي اللغة نفسها التي يستخدمها Python على سطح المكتب مع مكتبة قياسية أصغر وبعض الوحدات الإضافية للتواصل مع العتاد. كل ما يُكتب للكاميرا -- من برنامج نصي من سطرين يُومض الـ LED إلى تطبيق رؤية آلية كامل -- هو كود Python.
