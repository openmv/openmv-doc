v1.0.6¶

يجلب الإصدار v1.0.6 عدة تحسينات لواجهة سطر الأوامر والبروتوكول.

تغييرات¶

  • تم تحديث تنفيذ بروتوكول الكاميرا.

  • تمت إضافة خيار --raw إلى واجهة openmv لسطر الأوامر.

  • تم إصلاح خطأ في وسيط الكلمة المفتاحية في الـ API الخاص بالكاميرا.

  • تم إصلاح الوميض عند عرض بيانات المُحلّل في واجهة سطر الأوامر.

تغييرات كاسرة في الـ API¶

لا يوجد.
