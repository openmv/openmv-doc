v1.5.0¶

يضيف الإصدار v1.5.0 نوافذ مستقلة لـ Open Terminal عبر serial/UDP/TCP، وإكمال Python التلقائي مع تلميحات لوسائط الدوال مدفوعة بالتوثيق المرفق، وAprilTag Generator لأوراق العلامات القابلة للطباعة. ويبقى الإصدار على قاعدة Qt Creator 4.0.2 ويتضمن برنامج OpenMV الثابت بإصدار 2.2.0. لا توجد تغييرات جذرية.

أبرز النقاط¶

  • Open Terminal يفتح نوافذ طرفية مخصصة عبر serial أو UDP أو TCP، لكل منها عارض مخزن إطارات ولوح مدرج تكراري خاصان به.

  • إكمال Python التلقائي مع تلميحات لوسائط الدوال وإبراز نحوي أغنى، مصدره قاعدة بيانات كلمات مفتاحية مولّدة تلقائياً من توثيق HTML المرفق.

  • AprilTag Generator يصيّر أوراق صور علامات قابلة للطباعة لعائلات TAG16H5 وTAG25H7 وTAG25H9 وTAG36H10 وTAG36H11 وARTOOLKIT (ملصق القائمة للأخيرة يُقرأ ARKTOOLKIT).

  • إخراج تسلسلي متين: يقوم محلّل واعٍ بـ UTF-8 بتجريد ومعالجة محارف التحكم بحيث لا تعود البيانات الثنائية تفسد النص المطبوع.

  • برنامج OpenMV الثابت 2.2.0 أصبح الآن مرفقاً، مع أمثلة جديدة لـ AprilTag والوضعية ثلاثية الأبعاد.

ميزات جديدة¶

  • Open Terminal يضيف قائمة Open Terminal مستقلة تفتح نوافذ طرفية مخصصة تتصل عبر منفذ تسلسلي أو UDP أو TCP. لكل نافذة عارض مخزن إطارات ولوح مدرج تكراري خاصان بها، مع مربعات حوار اتصال serial/UDP/TCP وإدخال معدل الباود (7be9fd019، e657debea، b02e571a3، bd48a9b98).

  • إكمال Python التلقائي يضيف إكمال الكلمات المفتاحية/المعرّفات، وتلميحات أداة لوسائط الدوال، وإبرازاً نحوياً أغنى لواجهات OpenMV البرمجية. وتُولَّد قاعدة بيانات الكلمات المفتاحية والتواقيع تلقائياً عن طريق تحليل توثيق HTML المرفق إلى وحدات وفئات وبيانات ودوال وطرق ووسائط (4df150e66، ef0286403، a22869f0d، a2f4b2f25).

  • AprilTag Generator ضمن Machine Vision Tools يصيّر أوراق صور علامات PNG قابلة للطباعة لعائلات العلامات TAG16H5 وTAG25H7 وTAG25H9 وTAG36H10 وTAG36H11 وARTOOLKIT على نطاق معرّفات علامات قابل للتحديد. عنصر القائمة للعائلة الأخيرة يُقرأ ARKTOOLKIT Family (خطأ مطبعي في البناء) رغم أن العائلة نفسها هي artoolkit. كما تكتسب القائمة رابط QRCode Generator، وتعرض ملصقات قائمة العلامات الآن أعداد العلامات (ec72bf2f4، 71f78a539).

تغييرات وتحسينات أخرى¶

  • تم تحصين إخراج الطرفية التسلسلية ضد البيانات المشوّهة بمحلّل واعٍ بـ UTF-8 يجرّد ويعالج محارف التحكم (XON/XOFF، الهروب، الحذف، وغيرها) بحيث لا تعود بايتات مخزن الإطارات الثنائية وبايتات التحكم تفسد النص المطبوع (e03c7d3ef).

  • أُعيد تلوين سمة الإبراز النحوي الافتراضية Inkpot (الدوال والأنواع والأرقام والسلاسل) لتباين أفضل في المحرر (b02e571a3، 4df150e66).

  • تم تغيير روابط المنتج والمثبّت لاستخدام نطاق https://openmv.io (1793b0b5c).

إصلاحات الأخطاء¶

  • أُصلحت أخطاء طفيفة في عرض المحرر في رسم صندوق الطي وعلامة الطي، بما في ذلك تصحيح إزاحة خاص بنظام Mac، إلى جانب إصلاح ترجمة/إزاحة لنظام Linux بـ 32 بت وإصلاح خطأ ترجمة عام (c6c8dbfbb، 4dbb9e696، 66a8ce5af، cd3f23545).

دعم المنصة والأدوات¶

  • قاعدة Qt Creator: 4.0.2.

  • برنامج OpenMV الثابت المرفق: حُدّث إلى 2.2.0 لـ OpenMV Cam M4/M7 (OMV2/OMV3)، مع أمثلة جديدة لكشف AprilTag ومثال للوضعية ثلاثية الأبعاد لـ M7 (1d427612c).

  • أصبح مثبّت Windows الآن يحزم ويثبّت تلقائياً وبصمت مشغّل FTDI التسلسلي عبر USB أثناء الإعداد (1d427612c).

التغييرات الجذرية¶
