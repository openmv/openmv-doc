v1.5.0
======

v1.5.0 adds standalone serial/UDP/TCP **Open Terminal** windows, Python
autocomplete with function-argument hints driven by the bundled
documentation, and an **AprilTag Generator** for printable tag sheets. The
release stays on the Qt Creator 4.0.2 base and bundles OpenMV firmware 2.2.0.
There are no breaking changes.

.. contents:: On this page
   :backlinks: none
   :local:
   :depth: 1

Highlights
----------

- **Open Terminal** opens dedicated terminal windows over serial, UDP, or TCP,
  each with its own frame buffer viewer and histogram pane.
- **Python autocomplete** with function-argument hints and richer syntax
  highlighting, sourced from a keyword database auto-generated from the bundled
  HTML documentation.
- **AprilTag Generator** renders printable tag-image sheets for the TAG16H5,
  TAG25H7, TAG25H9, TAG36H10, TAG36H11, and ARTOOLKIT families (the menu label
  for the last reads ``ARKTOOLKIT``).
- **Robust serial output:** a UTF-8-aware parser strips and handles control
  characters so binary data no longer corrupts printed text.
- **OpenMV firmware 2.2.0** is now bundled, with new AprilTag and 3D-pose
  examples.

New features
------------

- **Open Terminal** adds a standalone ``Open Terminal`` menu that opens
  dedicated terminal windows connecting over serial port, UDP, or TCP. Each
  window has its own frame buffer viewer and histogram pane, with serial/UDP/TCP
  connect dialogs and baud-rate entry
  (`7be9fd019 <https://github.com/openmv/qt-creator/commit/7be9fd019>`__,
  `e657debea <https://github.com/openmv/qt-creator/commit/e657debea>`__,
  `b02e571a3 <https://github.com/openmv/qt-creator/commit/b02e571a3>`__,
  `bd48a9b98 <https://github.com/openmv/qt-creator/commit/bd48a9b98>`__).
- **Python autocomplete** adds keyword/identifier completion,
  function-argument hint tooltips, and richer syntax highlighting for OpenMV
  APIs. The keyword and signature database is auto-generated by parsing the
  bundled HTML documentation into modules, classes, datas, functions, methods,
  and arguments
  (`4df150e66 <https://github.com/openmv/qt-creator/commit/4df150e66>`__,
  `ef0286403 <https://github.com/openmv/qt-creator/commit/ef0286403>`__,
  `a22869f0d <https://github.com/openmv/qt-creator/commit/a22869f0d>`__,
  `a2f4b2f25 <https://github.com/openmv/qt-creator/commit/a2f4b2f25>`__).
- **AprilTag Generator** under ``Machine Vision Tools`` renders printable
  tag-image PNG sheets for the TAG16H5, TAG25H7, TAG25H9, TAG36H10, TAG36H11,
  and ARTOOLKIT tag families over a selectable tag-ID range. The menu item for
  the last family reads ``ARKTOOLKIT Family`` (a typo in the build) even though
  the family itself is ``artoolkit``. The menu also gains a
  ``QRCode Generator`` link, and tag menu labels now show tag counts
  (`ec72bf2f4 <https://github.com/openmv/qt-creator/commit/ec72bf2f4>`__,
  `71f78a539 <https://github.com/openmv/qt-creator/commit/71f78a539>`__).

Other changes and improvements
------------------------------

- Hardened the serial terminal output against malformed data with a
  UTF-8-aware parser that strips and handles control characters (XON/XOFF,
  escape, delete, and others) so binary frame-buffer and control bytes no
  longer corrupt the printed text
  (`e03c7d3ef <https://github.com/openmv/qt-creator/commit/e03c7d3ef>`__).
- Recolored the default Inkpot syntax theme (functions, types, numbers, and
  strings) for better contrast in the editor
  (`b02e571a3 <https://github.com/openmv/qt-creator/commit/b02e571a3>`__,
  `4df150e66 <https://github.com/openmv/qt-creator/commit/4df150e66>`__).
- Changed product and installer links to use the ``https://openmv.io`` domain
  (`1793b0b5c <https://github.com/openmv/qt-creator/commit/1793b0b5c>`__).

Bug fixes
---------

- Fixed minor editor rendering glitches in the folding-box and folding-marker
  drawing, including a Mac-specific offset correction, along with a 32-bit
  Linux compile/offset fix and a general compile-error fix
  (`c6c8dbfbb <https://github.com/openmv/qt-creator/commit/c6c8dbfbb>`__,
  `4dbb9e696 <https://github.com/openmv/qt-creator/commit/4dbb9e696>`__,
  `66a8ce5af <https://github.com/openmv/qt-creator/commit/66a8ce5af>`__,
  `cd3f23545 <https://github.com/openmv/qt-creator/commit/cd3f23545>`__).

Platform and tool support
-------------------------

- **Qt Creator base:** 4.0.2.
- **Bundled OpenMV firmware:** updated to 2.2.0 for OpenMV Cam M4/M7 (OMV2/OMV3),
  with new AprilTag detection examples and a 3D-pose example for the M7
  (`1d427612c <https://github.com/openmv/qt-creator/commit/1d427612c>`__).
- The Windows installer now bundles and silently auto-installs the FTDI
  USB-serial driver during setup
  (`1d427612c <https://github.com/openmv/qt-creator/commit/1d427612c>`__).

Breaking changes
----------------

None. This release is fully backward compatible; existing projects and
workflows continue to work unchanged.
