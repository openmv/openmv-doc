2. Ikhtisar Python¶

OpenMV Cam menjalankan MicroPython, bahasa yang sama dengan Python desktop dengan perpustakaan standar yang lebih kecil dan beberapa modul tambahan untuk berkomunikasi dengan perangkat keras. Semua yang ditulis untuk kamera -- dari skrip dua baris yang mengedipkan LED hingga aplikasi visi mesin lengkap -- adalah kode Python.
