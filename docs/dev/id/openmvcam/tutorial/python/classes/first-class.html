2.20. Kelas pertama¶

Kelas mengelompokkan data dan fungsi-fungsi yang beroperasi pada data tersebut di bawah satu nama. Menggunakannya bermanfaat ketika beberapa bagian status saling terkait dan sebagian besar fungsi dalam skrip mengambil kumpulan nilai yang sama sebagai argumen.

2.20.1. Kata kunci class¶

Kata kuncinya adalah class. Badan menyimpan metode __init__ yang berjalan setiap kali instans baru dibuat:

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

p = Point(3, 4)
print(p.x, p.y)

Output:

3 4

Point adalah kelasnya; p adalah instans dari kelas tersebut. Tanda kurung pada panggilan Point(3, 4) memanggil __init__, yang menetapkan argumen ke atribut pada instans baru. self adalah nama konvensional untuk instans; self.x menetapkan atribut yang akan dibaca oleh metode-metode instans lainnya.

2.20.2. Kapan menggunakan kelas¶

Gunakan kelas ketika kondisi berikut berlaku secara bersamaan:

  • Segelintir nilai terkait selalu bepergian bersama (x dan y dari titik; nilai, satuan, dan stempel waktu dari pembacaan sensor).

  • Beberapa fungsi mengambil kumpulan nilai yang sama dan beroperasi di atasnya.

  • Berguna untuk menjaga lebih dari satu kumpulan ini tetap aktif sekaligus, berbeda satu sama lain.

Jika datanya satu kali, dict atau tuple biasa biasanya cukup. Jika Anda hanya membutuhkan satu kumpulan konstanta, variabel di tingkat modul sudah baik. Kelas bukan wadah default -- kelas mendapat tempatnya ketika status dan perilaku ingin menyatu.

2.20.3. Apa yang dilakukan __init__¶

__init__ hanyalah metode biasa. Apa pun yang ditugaskan ke self di dalamnya menjadi atribut pada instans baru:

class Sensor:
    def __init__(self, name, unit):
        self.name = name
        self.unit = unit
        self.readings = []

temp = Sensor("temperature", "C")
print(temp.name, temp.unit, temp.readings)

Output:

temperature C []

readings diinisialisasi ke daftar kosong -- setiap instans baru mendapatkan daftarnya sendiri, bukan yang dibagikan. Apa pun yang tidak ditugaskan ke self bersifat lokal untuk __init__ dan hilang ketika dikembalikan.
