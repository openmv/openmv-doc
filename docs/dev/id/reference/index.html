Bahasa dan implementasi OpenMV MicroPython¶

MicroPython bertujuan untuk mengimplementasikan standar Python 3.4 (dengan fitur-fitur terpilih dari versi yang lebih baru) sehubungan dengan sintaks bahasa, dan sebagian besar fitur MicroPython identik dengan yang dijelaskan dalam dokumentasi "Language Reference" di docs.python.org.

Pustaka standar MicroPython dijelaskan dalam bab yang sesuai. Bab MicroPython differences from CPython menjelaskan perbedaan antara MicroPython dan CPython (yang sebagian besar berkaitan dengan pustaka standar dan tipe, tetapi juga beberapa fitur tingkat bahasa).

Bab ini menjelaskan fitur-fitur dan kekhasan implementasi MicroPython serta praktik terbaik untuk menggunakannya.
