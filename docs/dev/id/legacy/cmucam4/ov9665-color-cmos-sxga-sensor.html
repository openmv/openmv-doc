Antarmuka Kamera#

Kontrol Daya#

Anda perlu menyuplai 2.8V hingga 3.0V ke pin AVDD dan 1.8V hingga 3.0V ke pin DVDD. Anda harus menyalakan pin AVDD dan DVDD menggunakan catu daya rendah-noise terpisah. Selain itu, pin AGND dan DGND harus terpisah selama mungkin dalam penyiapan Anda untuk kualitas gambar yang baik.

Tarik pin RESET rendah untuk me-reset modul kamera.

Tarik pin PWDN tinggi untuk mematikan modul kamera saat dalam mode power down.

Bus Kontrol#

Pin SIO_D dan SIO_C digunakan untuk mengendalikan modul kamera. Pin SIO_D adalah pin data bus kontrol dan pin SIO_C adalah pin clock bus kontrol. Lihat bagian initialization untuk detail lebih lanjut.

Bus Data#

Pin Y0 hingga Y9 pada modul kamera membentuk bus data 10-bit. Untuk mode RGB565 dan YUV422, Y0 dan Y1 tidak digunakan. Y2 hingga Y9 digunakan untuk mengirim byte 8-bit di mana Y2 adalah LSB dan Y9 adalah MSB.

Saat pin HREF tinggi, Y2 hingga Y9 valid. Pada tepi naik pin PCLK, Y2 hingga Y9 harus disampel. Setiap piksel ditransfer dalam 2 pulsa PCLK. Dalam mode SXGA pin PCLK akan menjadi tinggi 2,560 kali saat pin HREF tinggi dan pin HREF akan menjadi tinggi 1024 kali. Dalam mode VGA pin PCLK akan menjadi tinggi 1,280 kali saat pin HREF tinggi dan pin HREF akan menjadi tinggi 480 kali. Resolusi SXGA adalah 1280x1024 piksel dan resolusi VGA adalah 640x480 piksel.

Pin VSYNC akan menjadi tinggi lalu rendah untuk menandai awal setiap frame.

Dalam mode RGB565, modul kamera akan mengirim [MSB {R4, R3, R2, R1, R0, G5, G4, G3} LSB] setiap clock ganjil dan [MSB {G2, G1, G0, B4, B3, B2, B1, B0} LSB] setiap clock genap untuk setiap piksel.

Dalam mode YUV422, modul kamera akan mengirim data YUV 8-bit di mana U dan V dikirim pada setengah bandwidth. Modul kamera mengirim Y0 pada clock pertama, diikuti oleh U pada clock kedua, diikuti Y1 pada clock ketiga, diikuti oleh V pada clock keempat dan kemudian mengulangi pola tersebut. Y0, U, dan V membentuk pixel0 dan Y1, U, dan V membentuk pixel1. Dua piksel dikirim setiap empat clock.

Inisialisasi Kamera#

Untuk mengendalikan modul kamera OV9665 Anda harus berkomunikasi dengannya menggunakan protokol SCCB -- yang persis sama dengan protokol I2C. Alamat slave perangkat adalah 0x60 untuk write dan 0x61 untuk read. Untuk menginisialisasi kamera Anda perlu menulis 0x80 ke register 0x12 untuk me-reset kamera dan kemudian menunda selama satu detik. Setelah itu Anda harus mengirim pengaturan register inisialisasi. Harap dicatat bahwa modul kamera memerlukan input clock 10 MHz hingga 24 MHz (biasanya 24 MHz) ke pin XCLK-nya untuk beroperasi.

Inisialisasi SXGA#

Untuk menginisialisasi kamera untuk mode SXGA kirim pengaturan register berikut. Untuk mengatur frame rate dalam mode SXGA tulis nilai berikut ke alamat register 0x11: 0x80 untuk 15 FPS, 0x81 untuk 7.5 FPS, 0x82 untuk 5 FPS, 0x83 untuk 3.75 FPS, 0x84 untuk 3 FPS dan seterusnya ketika XCLK digerakkan pada 24 MHz. CATATAN: Nilai FPS tidak boleh terlalu rendah atau kamera akan gagal beroperasi dengan benar.

\[\text{FPS} = \frac{15 \cdot (\text{XCLK}_\text{MHz} / 24)}{(\text{value} - \text{0x80}) + 1}\]
\[\text{PCLK Frequency} = \frac{\text{XCLK}_\text{MHz}}{(\text{value} - \text{0x80}) + 1}\]

Inisialisasi VGA#

Untuk menginisialisasi kamera untuk mode VGA kirim pengaturan register berikut. Untuk mengatur frame rate dalam mode VGA tulis nilai berikut ke alamat register 0x11: 0x80 untuk 30 FPS, 0x81 untuk 15 FPS, 0x82 untuk 10 FPS, 0x83 untuk 7.5 FPS, 0x84 untuk 6 FPS, dan seterusnya ketika XCLK digerakkan pada 24 MHz. CATATAN: Nilai FPS tidak boleh terlalu rendah atau kamera akan gagal beroperasi dengan benar.

\[\text{FPS} = \frac{30 \cdot (\text{XCLK}_\text{MHz} / 24)}{(\text{value} - \text{0x80}) + 1}\]
\[\text{PCLK Frequency} = \frac{\text{XCLK}_\text{MHz}}{(\text{value} - \text{0x80}) + 1}\]

Mode Warna#

Setelah kamera diinisialisasi ke mode SXGA atau VGA, pada laju frame tertentu, Anda kemudian harus mengatur format keluaran warna kamera. Anda dapat menggunakan RGB565 atau YUV422. Kirim pengaturan register ini untuk masuk ke mode RGB565 dan pengaturan register ini untuk masuk ke mode YUV422.
