Unduhan#

Manual Pengguna#

Manual Pengguna CMUcam1 v2.00 untuk firmware v1.12 tersedia dalam bentuk PDF. Berisi daftar komponen, skema papan dan deskripsi protokol perangkat lunak. Manual untuk versi firmware yang lebih lama dapat ditemukan di sini: v1.11, v1.1, v1.0.

Jika Anda memutuskan untuk bereksperimen dengan mengubah pengaturan register kamera, Anda mungkin juga tertarik untuk mengunduh Manual Kamera OV6620 dari OmniVision.

Membangun Papan#

Jika Anda membangunnya sendiri dari awal, Anda perlu mengunduh berkas kode HEX ini (berisi firmware terbaru v1.12) untuk memprogram chip SX28. Ini adalah kode yang mengimplementasikan fungsionalitas dan protokol serial yang dijelaskan dalam manual. Anda dapat memperoleh downloader dan aplikasi SX-Key dari Parallax Inc. Versi firmware yang lebih lama dapat ditemukan di sini: v1.11, v1.1, v1.0.

Program Antarmuka Java#

Kami memiliki program java open source yang dapat Anda unduh untuk berinteraksi dengan dan menguji CMUcam1 Anda. Program antarmuka ini memungkinkan Anda melacak objek, menganalisis data, dan mengambil gambar CMUcam1 lengkap. Berikut adalah Tangkapan Layar Program Java dari antarmuka ini.

Ini adalah proyek java yang di-zip yang ditulis untuk Linux.

Ini adalah versi GUI yang secara otomatis mendeteksi OS Anda. Ini berjalan di sistem UNIX yang mendukung IO serial seolah-olah itu adalah berkas (seperti Linux), maupun Windows 95/98/2000/NT/XP. Kami merekomendasikan menggunakan setidaknya prosesor Pentium 2 400 MHz yang menjalankan Java 1.2.2 atau lebih tinggi.

Untuk melihat contoh kode C untuk mikroprosesor Cerebellum dan BotBall Handyboard yang berkomunikasi dengan CMUcam1, pergi ke halaman web ini.

Skematik#

Berikut adalah JPEG resolusi tinggi dari Skematik CMUcam1 saat ini (versi papan 1.23).

Sebelum bertanya tentang kode sumber, Anda mungkin tertarik untuk mengunduh Manual Prosesor SX28 dari Ubicom.
