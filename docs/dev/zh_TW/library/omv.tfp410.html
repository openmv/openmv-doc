tfp410 --- DVI/HDMI 控制器¶

tfp410 模組為 TFP410 DVI/HDMI 序列化器提供驅動程式，用於透過 24 位元平行 LCD 匯流排驅動外部 DVI/HDMI 顯示器。

class TFP410 -- DVI/HDMI 控制器¶

class tfp410.TFP410(*, i2c_addr: int = 0x3F)¶

初始化 TFP410 DVI/HDMI 控制器晶片。

i2c_addr 是 TFP410 的 I2C 位址。

isconnected() → bool¶

若有外部顯示器連接則回傳 True，否則回傳 False。

hotplug_callback(callback: Callable[[bool], None] | None) → None¶

註冊一個 callback 函式，每當外部顯示器的連線狀態變更時都會被呼叫。新的連線狀態（bool）會作為唯一的引數傳遞給 callback。

將 None 作為 callback 傳入以停用熱插拔回呼函式。

在回呼函式已註冊期間，請勿呼叫 TFP410.isconnected()。
