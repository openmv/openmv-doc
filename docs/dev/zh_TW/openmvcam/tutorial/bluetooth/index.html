11. 藍牙¶

網路連線把相機連接到任何透過存取點與路由器可達的對象。藍牙則是相反的取捨。相機放棄了 觸及範圍，換來的是 短距離、低功耗與無須基礎設施——與同一房間內的手機或感測器之間幾公尺的直接無線電連結，無須加入 Wi-Fi 網路、無須透過路由器轉送，且耗電量小到一顆鈕扣電池就能讓相機運作數月。

其 Python 端是 aioble 模組——一個對 asyncio 友善的包裝層，將 BLE 無線電的角色、廣播、探索以及讀取／寫入／通知等原語轉換為協程。
