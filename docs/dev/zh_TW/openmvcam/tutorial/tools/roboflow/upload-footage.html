13.7.2. 上傳相機影片素材¶

模型在它所訓練的那類影像上表現最佳，因此資料集應該來自將要執行它的那台相機——相同的感測器、相同的鏡頭、相同的取景。快速收集大量此類影像最簡單的方式，就是用相機錄製一段短片，並讓 Roboflow 將它拆解成影格。

錄製一段目標物件的影片——移動它、改變角度與距離、變化背景與光線——並將它從相機上儲存下來。在專案的 Upload Data 頁面上，將影片檔案拖放進去。

13.7.2.1. 將影片取樣成影格¶

Roboflow 會詢問要多頻繁地將影片取樣成靜態影像。此滑桿在數量與冗餘之間取捨：太快的話，連續的影格會是近乎重複的，只會撐大資料集而不會教給模型任何新東西；太慢的話，則會錯過某些姿態。每一至兩秒取樣一個影格是個合理的起點——一段半分鐘的短片便能變成數百張多樣的影像。

挑選一個取樣率，Roboflow 便會擷取影格並將它們以未標記影像的形式加入專案。

備註

你不需要 SD 卡來錄製短片。IDE 的 Record 按鈕 會將相機的即時影片直接擷取到你電腦上的檔案——執行一個單純的擷取指令碼、錄製短片，並將該檔案上傳至 Roboflow。
