OpenMV Cam 教學¶

OpenMV Cam 是一款小巧、可程式化、由 MicroPython 驅動的相機，專為嵌入式機器視覺而設計。本教學從頭到尾涵蓋了如何使用它——從拆開相機的包裝、在 OpenMV IDE 中撰寫第一支指令碼，到每支指令碼所仰賴的 Python 與硬體基礎，再到相機的成像堆疊以及建構於其上的影像處理與機器學習工具，接著是應用程式成長到一定規模後會用到的並行處理與網路元件，最後則是將一個能運作的原型推進到出貨產品的實務工作。

本書各章節的撰寫方式讓新使用者可以依序閱讀，但每一章也都能獨立成篇——已經懂 Python 的讀者可以略過 Python 概覽，而曾經出貨過產品的讀者則可以直接跳到 Production。當某一章建立在先前介紹過的內容之上時，它會回連到對應的章節。
