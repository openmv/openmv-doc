2. Python 概觀¶

OpenMV Cam 執行 MicroPython，這與桌面版 Python 是相同的語言，只是標準函式庫較小，並多了幾個用於與硬體溝通的模組。為相機所撰寫的一切 -- 從讓 LED 閃爍的兩行指令碼，到完整的機器視覺應用程式 -- 都是 Python 程式碼。
