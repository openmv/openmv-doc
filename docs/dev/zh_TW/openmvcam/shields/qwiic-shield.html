QWIIC Shield¶

QWIIC Shield 為 OpenMV Cam 提供兩個可菊鏈串接的 Qwiic（JST-SH 4 針）連接器，並接至其 I2C 接腳，因此 SparkFun Qwiic 生態系的開發板可直接插入使用。

如需完整的資料表、照片與訂購資訊，請參閱 QWIIC Shield 產品頁面。

重點特色¶

  • 兩個 Qwiic JST-SH 連接器，可菊鏈串接 I2C 周邊裝置

  • 引出 OpenMV Cam 的 P4（SCL）與 P5（SDA）接腳

  • 相容於 SparkFun Qwiic 生態系

  • 板上未配置上拉電阻（請使用軟體上拉或外接上拉）

功能

P4

Qwiic SCL——通往兩個 Qwiic 連接器的 I²C 時脈

P5

Qwiic SDA——通往兩個 Qwiic 連接器的 I²C 資料

3.3V 電源軌

為連接的 Qwiic 裝置供電

GND 電源軌

共用接地（同時接至兩個 Qwiic 連接器）

使用方式¶

掃描 I²C 匯流排上連接的 Qwiic 裝置。此擴充板未在板上配置上拉電阻——請在軟體中啟用內部上拉電阻，或依靠所連接 Qwiic 開發板內建的上拉電阻:

from machine import Pin, SoftI2C

bus = SoftI2C(scl=Pin("P4", Pin.OPEN_DRAIN, Pin.PULL_UP),
              sda=Pin("P5", Pin.OPEN_DRAIN, Pin.PULL_UP),
              freq=100_000)

print("Qwiic devices:", [hex(addr) for addr in bus.scan()])
