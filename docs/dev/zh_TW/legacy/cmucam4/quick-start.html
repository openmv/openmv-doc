快速入門#

入門#

請依照下列步驟：

  1. 您需要以下兩項物品才能測試 CMUcam4

    • 一個 4V 至 9V DC 外部電源供應器，能夠提供至少 250 mA

      • 這可以來自 Arduino、FTDI Breakout Board、FTDI Cable 或 Wall Wart

    • 一台 NTSC 電視（或相容的電視螢幕）與一條 RCA 傳輸線

  2. 測試 CMUcam4 的設定

    • 使用 RCA 傳輸線 將 NTSC 電視 連接至 CMUcam4 的 RCA 同軸插孔

    • 將外部電源供應器連接至 CMUcam4 的 DC 圓筒插孔（或其他電源埠）

一旦您將外部電源供應器連接至 CMUcam4，綠色電源 LED 應會亮起。約 2 秒 後，紅色輔助 LED 應會亮起，以指示 CMUcam4 已準備就緒可運作。如果綠色電源 LED 未亮起，請再次確認外部電源供應器的連接。

測試程序#

請依照下列步驟：

  1. 按住 CMUcam4 上的 reset button

  2. 按住 CMUcam4 上的 user button

  3. 放開 reset button（不要放開 user button）

  4. 等到紅色輔助 LED 亮起（2 秒）

  5. 等到紅色輔助 LED 開始以 10 Hz 閃爍，然後放開 user button

    • 電視應該會開啟（您應該會看到電視上顯示一個啟動畫面）

  6. CMUcam4 現在將在接下來的 5 秒 內調整以適應照明條件

    • 在接下來的 5 秒 內，請勿將您想追蹤的物體放在 CMUcam4 前方

  7. 等到紅色輔助 LED 停止以 10 Hz 閃爍

    • CMUcam4 現在已完成對照明條件的調整

    • 平移與傾斜伺服機針腳應以 50 Hz 輸出 1500 μs 的脈衝

  8. 將您想追蹤的物體放在 CMUcam4 前方，然後按下 user button

    • 如果紅色輔助 LED 開始以 10 Hz 閃爍，請檢查 OV9665 相機模組的連接

      • OV9665 相機模組可能已損壞，而且很可能需要更換

  9. 您現在應該會看到被追蹤的物體（或類似內容）顯示在電視上。

    • 平移與傾斜伺服機（若已連接）也會嘗試驅動相機朝向被追蹤的物體

  10. 請在不同的環境中以不同的物體嘗試此程序，看看哪種效果最好

以 PDF 形式下載測試指南 這裡。

通訊工具#

您需要以下其中一款（或類似的）USB 轉序列轉換器 才能與 CMUcam4 通訊：

  • 一塊 FTDI 5V 分接板

    • 您還需要一個能夠為 CMUcam4 供電的外部電源供應器

    • 請將 FTDI 5V 分接板 連接至 CMUcam4 上的 6 針 連接器

  • 一塊 FTDI 3.3V 分接板

    • 您還需要一個能夠為 CMUcam4 供電的外部電源供應器

    • 請將 FTDI 3.3V 分接板 連接至 CMUcam4 上的 6 針 連接器

  • 一個 Prop Clip

    • 您還需要一個能夠為 CMUcam4 供電的外部電源供應器

    • 請將 Prop Clip 連接至 CMUcam4 上的 4 針 連接器

  • 一個 Prop Plug

    • 您還需要一個能夠為 CMUcam4 供電的外部電源供應器

    • 請將 Prop Plug 連接至 CMUcam4 上的 4 針 連接器

  • 一條 FTDI 5V 傳輸線（附 5V I/O）

    • 建議用於可容忍 5V 的系統

    • 請將 FTDI 5V 傳輸線（附 5V I/O） 連接至 CMUcam4 上的 6 針 連接器

  • 一條 FTDI 5V 傳輸線（附 3.3V I/O）

    • 建議用於可容忍 3.3V 的系統

    • 請將 FTDI 5V 傳輸線（附 3.3V I/O） 連接至 CMUcam4 上的 6 針 連接器

建議的序列終端機程式#

Parallax Serial Terminal 是一個方便的工具，可與諸如 Parallax Propeller 晶片這類以序列為基礎的微控制器通訊。它是建議用來與相機板通訊的序列終端機。Parallax Serial Terminal（PST）是一個大小不到 1 MB 的獨立應用程式，無須安裝即可使用。PST 可從 Parallax Inc. 此處 下載。請依照以下步驟設定 PST：

  1. 執行 PST

  2. 前往 Echo On 並確保已勾選它

  3. 前往 Com Port，並從下拉式清單中選擇 CMUcam4 所連接的 COM 埠

  4. 前往 Baud Rate，並從下拉式清單中選擇 19200

    • 必要時按一下 Enable

對於非 Windows 使用者，建議使用 Brad's SPIN Tool (BST)。BST 可從 這裡 下載。BST 是一個圖形使用者介面 (GUI) 整合開發環境 (IDE) 的獨立應用程式，大小不到 10 MB，專為 Parallax Propeller 晶片設計，且無需安裝即可使用。BST 內建一個易於使用的序列終端機。請依照下列步驟設定 BST 內建易於使用的序列終端機：

  1. 執行 BST

  2. 前往 View，並從下拉式清單中選擇 Serial Terminal

    • bst Terminal 應該會彈出——請點擊它

  3. 前往 Baud，並從下拉式清單中選擇 19200

  4. 前往 Format，並從下拉式清單中選擇 8 Bits 與 Parity None

  5. 前往 Port，並從下拉式清單中選擇 CMUcam4 所連接的 COM 埠

    • 尋找名為 /dev/tty/USB### 的連接埠

  6. 前往 Communicate，並從下拉式功能表中選擇 Connect、Terminal Echo 與 Reset Propeller

    • 前往 Communicate，並選擇 Display ASCII 以取得 ASCII 輸出，或選擇 Display Hex 以取得十六進位輸出

注意：您無法將 FTDI 5V 傳輸線與 BST 搭配使用！

這是因為 BST 在您點擊 Connect 時，將 FTDI 5V Cable 上的綠色 RTS 線拉低。 這會無限期地使 CMUcam4 停止運作。您可以從連接器移除綠色 RTS 線來修正此問題。

您接下來的步驟（在看過 圖庫 之後）是前往說明文件網頁。此外，請務必前往我們的 色彩追蹤說明、秘訣與技巧、常見問題 以及 疑難排解 網頁以取得更多資訊。最後，請前往 Arduino 介面函式庫網頁看看！接著您就準備好可以打造您自己的視覺機器人了！
