CMUcam4#

CMUcam4 是一款完全可程式化的嵌入式電腦視覺感測器。主處理器為 Parallax P8X32A（Propeller 晶片），連接至一個 OmniVision 9665 CMOS 相機感測器模組。欲了解更多資訊，請參閱 wiki。

功能#

  • 完全開放原始碼，並可使用 Propeller Tool 重新程式設計

  • 相容於 Arduino 擴充板

    • 具備供 Arduino 與 BASIC Stamp 使用的支援介面函式庫與示範應用程式

  • VGA 解析度（640x480）RGB565/YUV655 彩色感測器

    • 每秒 30 個畫格的影像處理速率

    • 透過序列埠或至快閃記憶卡的原始影像傾印

      • (640:320:160:80)x(480:240:120:60) 影像解析度

      • RGB565/YUV655 色彩空間

  • 板載影像處理（QQVGA 160x120）

    • 在 RGB/YUV 色彩空間中追蹤使用者定義的色塊

    • 平均數、中位數、眾數與標準差資料收集 – 從 40x120 解析度取樣

    • 用於追蹤視覺化的分割（閾值化）影像擷取（透過序列埠或至快閃記憶卡）

      • 80x60 影像解析度

      • 單色色彩空間

    • 直方圖產生（最多 128 個區間）– 從 40x120 解析度取樣

    • 任意影像裁剪（視窗化）

  • µSD/µSDHC 快閃記憶卡插槽，具備 FAT16/32 完整檔案系統驅動程式支援

    • 具備目錄與檔案操作功能

  • I/O 介面

    • 雙埠伺服控制器（平移與傾斜，於 50 Hz 更新率下具 1us 解析度）

      • 平移與/或俯仰伺服通道可設定為 GPIO

    • 指示用的使用者可控制 LED（紅色）與電源 LED（綠色）

    • TTL UART（最高可達 250,000 鮑 – 預設為 19,200 鮑）

  • 160x120 解析度的單色基頻類比視訊輸出（NTSC/PAL），用於追蹤視覺化（分割（閾值化）影像，以 30 FPS 疊加色彩形心與邊界框）

  • 用於在 PC 上檢視影像的 CMUcam4 GUI

以 PDF 形式下載功能清單 這裡。

致謝#

本專案得以進行，有賴於 Parallax、Lextronic、SparkFun 以及 Semiconductor Research Corporation 的慷慨支持。它出自卡內基美隆大學的 電機與電腦工程學系 以及 大學部研究辦公室。
