CMUcam4 圖形使用者介面#

CMUcam4GUI 是什麼#

CMUcam4GUI 是一個實用的工具，讓使用 CMUcam4 變得更輕鬆。CMUcam4GUI 讓您能夠用您的 CMUcam4 拍攝圖片並在電腦上檢視它們，並利用這些圖片來計算您希望 CMUcam4 追蹤的圖片中彩色物體的色彩追蹤界限。使用 CMUcam4GUI，您可以快速地在某個照明場景下確定物體的色彩追蹤界限，並拓寬色彩追蹤界限以支援在光線變化的場景下追蹤該物體。CMUcam4GUI 也讓您無須電視即可即時看到 CMUcam4 正在追蹤的內容，方法是以圖形化方式顯示 CMUcam4 所傳送的追蹤、平均值、直方圖及畫格封包的內容。此外，CMUcam4GUI 讓您能夠即時把玩所有 CMUcam4 的選項，例如控制平移與傾斜伺服機、色彩追蹤選項以及低階相機控制。最後，CMUcam4GUI 讓您能夠透過 CMUcam4 遠端管理連接於 CMUcam4 的安全數位卡。

Windows 設定#

請依照下列步驟為 Windows 設定 CMUcam4GUI：

  1. 下載 CMUcam4GUI.zip 檔案

  2. 將它解壓縮並取出內容

  3. 執行 CMUcam4GUI.exe 檔案

Linux 設定#

請依照下列步驟為 Linux 設定 CMUcam4GUI：

  1. 將您的使用者名稱加入 dialout 群組

sudo adduser username dialout

  1. 現在輸入以下內容進行驗證

groups username

  1. 現在登出，然後重新登入

  2. 下載 CMUcam4GUI.zip 檔案

  3. 將它解壓縮並取出內容

  4. 執行 CMUcam4GUI.sh 檔案

Macintosh 設定#

請依照下列步驟為 Macintosh 設定 CMUcam4GUI：

  1. 為 Mac 安裝 Qt 1.2.1 SDK（Desktop 4.8.1）

  2. 下載 CMUcam4GUI 原始碼

  3. 使用 QtCreator 開啟 CMUcam4GUI.pro 檔案，並從除錯模式切換至發行模式

  4. 開啟 CMUcom4.h 檔案，並將 #define CMUCOM4_FAST_BAUD_RATE 從 230400 變更為 115200（Mac 無法處理 230400 BPS 的鮑率）

  5. 按一下執行

CMUcam4GUI 手冊#

CMUcam4GUI 非常直覺，並實作了 CMUcam4 Arduino 介面函式庫中所有相同的功能。本手冊僅為提供 CMUcam4GUI 的概觀而存在，如果您需要更多關於 CMUcam4 命令的資訊，應閱讀 CMUcam4 命令清單使用者手冊。如果您在嘗試連接 CMUcam4 時遇到錯誤，請將您的 USB 轉序列裝置從電腦上拔除再重新插入。此外，請確保您的序列轉換器能夠處理 19200 BPS、115200 BPS 以及 230400 BPS。如果您使用的是以 FTDI 晶片為基礎的轉換器，那麼應該沒有問題。

  • 深度睡眠 - 讓 CMUcam4 進入超低功耗狀態

  • 傳送點陣圖 - 擷取二值影像並將其顯示於檢視視窗中

  • 傾印畫格 - 擷取彩色影像並將其儲存至 SD 卡

  • 傾印點陣圖 - 擷取二值影像並將其儲存至 SD 卡

    • 這是 CMUcam4 正在追蹤的任何內容的 80x60 二值影像——CMUcam4 必須正在追蹤某物，這才有用

  • 儲存畫格 - 將檢視視窗中的彩色影像儲存至檔案

  • 儲存點陣圖 - 將檢視視窗中的二值影像儲存至檔案

    • 這是 CMUcam4 正在追蹤的任何內容的 80x60 二值影像——CMUcam4 必須正在追蹤某物，這才有用

  • 清除畫格 - 清除檢視視窗中的彩色影像

  • 清除點陣圖 - 清除檢視視窗中的二值影像

    • 這是 CMUcam4 正在追蹤的任何內容的 80x60 二值影像——CMUcam4 必須正在追蹤某物，這才有用

  • 開啟 - 開啟一張彩色影像以搭配追蹤選取功能使用

  • 離開 - 離開應用程式

檢視頁籤

  • 傳送畫格控制 - 為工具列中的傳送畫格按鈕選擇水平與垂直解析度

    • 在 send frame 命令執行期間，您不能移動您正在拍攝的對象

    • 如果影像品質看起來不佳，請關閉自動增益與自動白平衡，並確保 CMUcam4 正對著一個有照明的物體

    • 使用較小的解析度以更快地擷取影像——例如 160x120

  • 傾印畫格控制 - 為工具列中的傾印畫格按鈕選擇水平與垂直解析度

    • 在 dump frame 命令執行期間，您不能移動您正在拍攝的對象

    • 如果影像品質看起來不佳，請關閉自動增益與自動白平衡，並確保 CMUcam4 正對著一個有照明的物體

    • 使用較小的解析度以更快地擷取影像——例如 160x120

  • 不透明度控制 - 選擇傳送畫格與傳送點陣圖的不透明度 - 介於 1.00（代表 100%）與 0.00（代表 0%）之間

  • 追蹤控制 - 在檢視視窗中選擇要追蹤的色彩區塊

    • 選取矩形應僅框住您想追蹤的色塊內部的像素

    • 選取色彩區塊後，按一下追蹤色彩按鈕，以計算要追蹤該色彩區塊的色彩範圍

      • 此演算法的運作方式是計算選取矩形內部紅、綠、藍色彩通道的平均值與標準差

      • 信賴區間水準係數選擇要為紅、綠、藍色彩通道的平均值加減多少標準差——從 0（代表 +/- (stdev * 1)）到 99（代表 +/- (stdev * 10)）

追蹤頁籤

在選項面板中啟用 Line 模式，以查看藍色的被追蹤像素。

磁碟頁籤

  • 磁碟資訊 - 取得磁碟幾何結構的相關資訊

  • 磁碟空間 - 取得磁碟空間的相關資訊

  • 磁碟控制 - 卸載/格式化磁碟

    • 注意：在關閉圖形使用者介面之前，您應該在使用任何 SD 卡相關指令後卸載磁碟

    • 注意：按一下格式化磁碟按鈕將立即格式化磁碟

  • 新增檔案 - 建立含有一些文字的新檔案，或將一些文字附加至檔案

  • 新增資料夾 - 建立新資料夾

  • 檔案總管 - 從您的電腦與 SD 卡上的檔案/資料夾互動

    • 按一下重新整理以列出目前目錄的內容

    • 透過點擊屬性核取方塊來變更檔案或資料夾的屬性

      • 按兩下資料夾以進入它

      • 按兩下檔案以開啟它

        • 請勿嘗試開啟大於約 10 KB 的檔案（除非您想永遠等下去），因為檔案傳輸非常慢，大約只有 4 KB/s

        • 您可以透過用新路徑重新命名檔案與資料夾來移動它們，例如將 txt.txt 重新命名為 ../txt.txt

如何界定色彩範圍#

  • 前往選項

  • 開啟 YUV

  • 等待 5 秒

  • 關閉自動增益與自動白平衡

  • 選擇傳送畫格水平解析度並將其設為 160

  • 選擇傳送畫格垂直解析度並將其設為 120

  • 將 CMUcam4 對準您想追蹤的物體，然後在不移動 CMUcam4 或物體、且兩者都不得移動的情況下，按一下傳送畫格

  • 等到畫格傳送至 GUI 為止，請勿移動物體

  • 選擇物體的色彩區域（不包含該區域以外的任何部分）

  • 按一下追蹤選取

  • 以 50 作為預設值並按一下確定

  • 如果只有該物體被畫上藍色，請再次按一下追蹤選取，並將 50 設為 60

  • 重複上述步驟，直到您將追蹤選取值推到盡可能高，當它為 99 時即可停止。如果您開始界定到不想要的東西，那麼您也完成了。

  • 接著前往 options……將 green 最小值設為 0、green 最大值設為 255（這會將 Ymin 設為 0、Ymax 設為 255）。如果您開始框住不想要的東西，就試著使用較不寬鬆的界限，直到您剛好只框住您想要的物體為止。您可以透過按一下 sendBitmap 按鈕來重新整理所框住的內容。

  • 然後……接著您應該就會有六個追蹤選項……此外，如果您願意，也可以手動放寬 (V/紅) 與 (U/藍) 的範圍。這將有助於讓系統在不同光照條件等情況下更為穩健。

若要查看 CMUcam4 所看到的內容……

  • 開啟 line mode（options 中有一個核取方塊）

  • 接著前往 viewing 並按一下 track color。
