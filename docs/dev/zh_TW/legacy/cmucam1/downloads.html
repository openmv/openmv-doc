下載#

使用者手冊#

CMUcam1 使用者手冊 v2.00（適用於韌體 v1.12）提供 PDF 格式。它包含零件清單、電路板電路圖以及軟體協定說明。較舊版本韌體的手冊可以在此處找到： v1.11、 v1.1、 v1.0。

如果您決定嘗試變更相機暫存器設定，您可能也會有興趣從 OmniVision 下載 OV6620 相機手冊。

組裝電路板#

如果您要從頭開始自行打造它，您將需要下載此 HEX 程式碼檔案（包含最新的韌體 v1.12）來為 SX28 晶片編寫程式。這是實作手冊中所述功能與序列通訊協定的程式碼。您可以從 Parallax Inc. 取得 SX-Key 下載器與應用程式。韌體的較舊 版本 可在此找到：v1.11、v1.1、v1.0。

Java 介面程式#

我們有一個開放原始碼的 java 程式，您可以下載它來與您的 CMUcam1 介接並進行測試。這個介面程式讓您能夠追蹤物體、分析資料以及擷取完整的 CMUcam1 影像。這裡是這個介面的 螢幕截圖。

這是壓縮的 java 專案 為 Linux 所撰寫。

這是 GUI 的版本，它會自動偵測您的作業系統。它可在支援將序列 IO 視為檔案的 UNIX 系統（例如 Linux）上執行，也可在 Windows 95/98/2000/NT/XP 上執行。我們建議至少使用執行 Java 1.2.2 或更高版本的 Pentium 2 400 MHz 處理器。

若要查看 Cerebellum 微處理器與 BotBall Handyboard 和 CMUcam1 通訊的 C 程式碼範例，請前往 這個網頁。
