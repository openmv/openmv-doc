OpenMV MicroPython 語言與實作¶

在語言語法方面，MicroPython 旨在實作 Python 3.4 標準（並納入較新版本中的部分精選功能），且 MicroPython 的大多數功能都與 docs.python.org 上的「Language Reference」文件所述相同。

MicroPython 標準函式庫於 對應章節 中說明。MicroPython differences from CPython 章節說明 MicroPython 與 CPython 之間的差異（這些差異大多涉及標準函式庫與型別，但也包含一些語言層級的功能）。

本章說明 MicroPython 實作的功能與特性，以及使用它們的最佳實務。
