OpenMV 變更紀錄¶

每個 OpenMV 產品都保有自己的發行歷史。挑選你感興趣的那一個——每個項目都描述了有什麼新功能、變更了什麼，以及（在重要時）確切的遷移方式。

OpenMV Firmware

在 OpenMV Cam 上運行的 MicroPython 機器視覺韌體。包含新相機、影像處理與板級支援功能，以及針對腳本的 API 重大變更與遷移步驟。

查看發行版本 →

OpenMV IDE

用於在 OpenMV Cam 上撰寫、執行和除錯腳本的桌面開發環境。包含新編輯器、相機與工具功能、錯誤修正及平台支援。

查看發行版本 →

OpenMV Python Package

主機端 Python 函式庫與 openmv 命令列工具，用於透過相機協定 V2 從電腦與 OpenMV Cam 通訊——串流、腳本執行與效能測試。

查看發行版本 →

OpenMV Arduino RPC Library

用於透過 CAN、I²C、SPI、UART 與 USB 控制 OpenMV Cam 的 Arduino 函式庫。包含新板級支援、匯流排介面與穩定性修正。

查看發行版本 →
