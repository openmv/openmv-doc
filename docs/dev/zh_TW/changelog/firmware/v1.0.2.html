v1.0.2-beta#

v1.0.2-beta 是一個桌面 IDE 與 USB 工具的 beta：它新增了具備序列埠列舉功能的偏好設定對話框、一個具有獨立 CDC/HID 與除錯 USB 設定的 OpenMV 除錯介面，將設定檔與使用者指令碼移至家目錄，並提供了一個 DFU 映像檔。韌體的 Python API 沒有任何變更 — v1.0.1-beta 的指令碼可不經修改地執行。

重點摘要#

  • 偏好設定對話框，具備序列埠列舉功能。

  • OpenMV 除錯介面 — 獨立的 CDC/HID 與除錯 USB 設定。

  • 提供 DFU 映像檔；設定／使用者指令碼已移至家目錄。

  • 韌體 API 沒有變更 — 重新燒錄後，你的 v1.0.1-beta 指令碼可不經修改地執行。

新功能#

  • 新增了具備序列埠列舉功能的偏好設定對話框、一個 OpenMV 除錯介面（獨立的 CDC/HID 與除錯 USB 設定），以及一個 DFU 韌體映像檔。已淘汰的 QT IDE 被移除。

其他變更與改進#

  • 將設定檔與使用者指令碼移至家目錄（透過 appdirs 管理）；在 Windows 上回傳字串型態的 COM 連接埠；將 bin → firmware 重新命名；在 openmv.init 之後才開啟序列埠；修正了裝置中斷連線與序列埠關閉的警告。

破壞性 API 變更#

v1.0.2-beta 沒有引入任何破壞性的 Python API 變更（這些變更僅涉及桌面 IDE ／ USB 設定／封裝）。在 v1.0.1-beta 上執行的指令碼可不經修改地執行 — 只需重新燒錄韌體即可。
