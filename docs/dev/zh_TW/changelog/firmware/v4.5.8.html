v4.5.8¶

v4.5.8 是一個小型版本：新增 uint16 模型輸出支援與一項 MicroSpeech 修正。它未引入任何破壞性 API 變更——v4.5.7 的指令碼可不變執行。

本頁內容

  • 重點摘要

  • 新功能

  • 錯誤修正

  • 破壞性 API 變更

重點摘要¶

  • uint16 模型輸出——ml.Model 現在可匹配 uint16 ndarray。

  • 無破壞性 API 變更——重新燒錄後，您的 v4.5.7 指令碼可不變執行。

新功能¶

  • ml.Model——新增 uint16 支援，使使用 uint16 的模型輸出張量能被正確匹配/處理。

錯誤修正¶

  • 修正 ml.apps 的 MicroSpeech.listen() 遺漏 return 的問題，因此偵測結果現在會確實回傳給呼叫者。

破壞性 API 變更¶

v4.5.8 未引入任何破壞性 Python API 變更。在 v4.5.7 上執行的指令碼可在 v4.5.8 上不變執行——只需重新燒錄韌體即可。
