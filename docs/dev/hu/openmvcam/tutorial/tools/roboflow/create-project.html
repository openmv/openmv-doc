13.8.1. Projekt létrehozása#

A Roboflow munkaterületről a New Project megnyitja a létrehozási oldalt. Egy projekt egy adathalmazt, a rajta betanított modelleket és azok telepítési céljait fogja össze.

Itt két választás fontos:

  • Project Name. Használj bármilyen jól megjegyezhető nevet – ez a példa a Finding raccoons nevet használja. A név a modell azonosítójának része lesz, amelyet a telepítési lépésnél töltesz le.

  • Project Type. Ez határozza meg, hogy a modell mit jósol, ezért igazítsd ahhoz, amit a kamerától el szeretnél érni. Az Object Detection – objektumok és helyzetük megtalálása határoló dobozokkal – az, amellyel dolgokat lehet a képkockában lokalizálni, és ezt dekódolják a kamera YOLO utófeldolgozói. A többi típus – osztályozás, példányszintű szegmentálás, kulcspont-észlelés – különböző modellfejeket tanít be; ez a példa objektumészlelést használ.

A láthatósági és licencbeállítások csak azt befolyásolják, hogy az adathalmaz nyilvánosan megosztásra kerül-e a Roboflow Universe-en – nincsenek hatással a kamerára kerülő modellre. Állítsd be őket tetszésed szerint, majd Create Project.
