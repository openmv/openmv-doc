OpenMV MicroPython nyelv és implementáció¶

A MicroPython célja a Python 3.4 szabvány megvalósítása (a későbbi verziók egyes funkcióival kiegészítve) a nyelvi szintaxis tekintetében, és a MicroPython funkcióinak nagy része megegyezik a docs.python.org oldalon található „Language Reference” dokumentációban leírtakkal.

A MicroPython szabványos könyvtárát a megfelelő fejezet írja le. A MicroPython differences from CPython fejezet a MicroPython és a CPython közötti különbségeket írja le (amelyek többnyire a szabványos könyvtárat és a típusokat érintik, de néhány nyelvi szintű funkciót is).

Ez a fejezet a MicroPython implementáció funkcióit és sajátosságait, valamint használatuk legjobb gyakorlatait írja le.
