Threshold osztály – Threshold objektum¶

A threshold objektum egy attrtuple, amelyet a histogram.get_threshold() ad vissza. A mögöttes Histogram minden csatornájához megadja az Otsu-féle optimális elválasztási értéket – vagyis azt a rekeszértéket, amely a csatornát a legjobban választja szét egy „háttér” és egy „előtér” felére.

Szürkeárnyalatos hisztogramokhoz használja a value mezőt (a másik három mező 0). RGB565 hisztogramokhoz az l_value / a_value / b_value adja meg az Otsu-küszöbértéket a LAB L, A és B csatornákhoz. A visszaadott értékek kiválóan alkalmasak arra, hogy közvetlenül átadja őket az Image.binary() függvénynek vagy bármely más metódusnak, amely LAB színküszöbértékeket vesz át.

A mezők elérhetők attribútumnévvel (threshold.value) vagy index szerint (threshold[0]). Az objektumnak nincs nyilvános konstruktora.

class image.threshold¶

Ennek az objektumnak a létrehozásához hívja meg a histogram.get_threshold() függvényt.

value¶

Szürkeárnyalatos Otsu-küszöbérték. Egész szám 0 – 255. Index: [0].

l_value¶

A LAB L csatorna Otsu-küszöbértéke. Egész szám 0 – 100. Index: [1].

a_value¶

A LAB A csatorna Otsu-küszöbértéke. Egész szám -128 – 127. Index: [2].

b_value¶

A LAB B csatorna Otsu-küszöbértéke. Egész szám -128 – 127. Index: [3].
