sensor — kamera érzékelő¶

Elavult a(z) 4.5 verzió óta: A sensor modul elavult. Helyette használja az új csi modult (lásd csi — kameraérzékelők). Ehhez a modulhoz nem kerülnek új funkciók, és egy jövőbeli kiadásban eltávolításra kerülhet.

A sensor modul az OpenMV Cam elsődleges kamera érzékelőjéhez tartozó örökölt, modul szintű felület. Minden hívás egy szabad függvény, amely egyetlen rejtett CSI példányon működik, ami korlátozza a több kamerával rendelkező lapok támogatását. A régebbi OpenMV szkriptekkel való visszamenőleges kompatibilitás miatt maradt meg; az új kódnak az objektumorientált csi modult kell használnia helyette.

A függvénynevek a régebbi set_pixformat / set_framesize stílust követik. Minden függvény egy az egyben megfelel a csi.CSI egy-egy metódusának; a teljes képességkészletért és az egyes argumentumok leírásáért lásd a csi modult.

Példa a használatra:

import sensor

sensor.reset()
sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.QVGA)
sensor.skip_frames(time=2000)

while True:
    img = sensor.snapshot()

Függvények¶

sensor.reset() → None¶

Inicializálja a kamera érzékelőt.

sensor.sleep(enable: bool) → None¶

Alvó állapotba helyezi a kamerát, ha az enable értéke True. Egyébként felébreszti.

sensor.shutdown(enable: bool) → None¶

Az alvásnál alacsonyabb energiafelvételű módba helyezi a kamerát. A kamerát ébredés után alaphelyzetbe kell állítani.

sensor.flush() → None¶

Átmásolja a képkocka-pufferben lévő tartalmat az IDE előnézetébe.

sensor.snapshot() → image.Image¶

Készít egy képet a kamerával, és visszaad egy image.Image objektumot.

Ha a sensor.set_auto_rotation() engedélyezve van, ez a metódus egy új, már elforgatott image.Image objektumot ad vissza.

sensor.skip_frames(n: int | None = None, time: int = 300) → None¶

Kihagy n képkockát vagy time ezredmásodpercet (amelyik meg van adva), hogy a kamera képe stabilizálódhasson a kamerabeállítások módosítása után.

Ha sem az n, sem a time nincs megadva, ez a metódus 300 ezredmásodpercig hagy ki képkockákat.

Ha mindkettő meg van adva, ez a metódus n képkockát hagy ki, de time ezredmásodperc után időtúllépéssel leáll.

sensor.width() → int¶

Visszaadja az érzékelő felbontásának szélességét.

sensor.height() → int¶

Visszaadja az érzékelő felbontásának magasságát.

sensor.get_fb() → image.Image | None¶

Visszaadja a sensor.snapshot() korábbi hívása által visszaadott kép objektumot. None értéket ad vissza, ha a sensor.snapshot() korábban nem lett meghívva.

sensor.get_id() → int¶

Visszaadja a kameramodul azonosítóját. Lásd az alábbi érzékelő-konstansokat.

sensor.get_frame_available() → bool¶

True értéket ad vissza, ha rendelkezésre áll egy képkocka, amely a sensor.snapshot() hívásával beolvasható.

sensor.alloc_extra_fb(width: int, height: int, pixformat: int) → image.Image¶

Elavult a(z) 4.5 verzió óta: Ez a függvény elavult, és OSError hibát fog kiváltani. Helyette használja az új csi modult.

sensor.dealloc_extra_fb() → None¶

Elavult a(z) 4.5 verzió óta: Ez a függvény elavult, és OSError hibát fog kiváltani. Helyette használja az új csi modult.

sensor.set_pixformat(pixformat: int) → None¶

Beállítja a kameramodul képpontformátumát. A pixformat a következők egyike:

  • sensor.BINARY

  • sensor.GRAYSCALE

  • sensor.RGB565

  • sensor.BAYER

  • sensor.YUV422

  • sensor.JPEG (csak az OV2640/OV5640 esetén)

sensor.get_pixformat() → int¶

Visszaadja a kameramodul aktuális képpontformátumát.

sensor.set_framesize(framesize: int) → None¶

Beállítja a kameramodul képkocka-méretét. Az érvényes értékekért lásd az alábbi framesize konstansokat.

sensor.get_framesize() → int¶

Visszaadja a kameramodul aktuális képkocka-méretét.

sensor.set_framerate(rate: int) → None¶

Beállítja a kameramodul képkocka-sebességét Hz-ben.

sensor.get_framerate() → int¶

Visszaadja a kameramodul képkocka-sebességét Hz-ben.

sensor.set_windowing(roi: Tuple[int, int] | Tuple[int, int, int, int] | List[int]) → None¶

Beállítja a kamera felbontását az aktuális felbontáson belüli al-felbontásra.

A roi egy téglalap tuple/lista (x, y, w, h). Megadhat (w, h) értéket is, ekkor a roi a képkockán belül középre kerül. Az argumentumok kicsomagolt pozicionális egész számokként is átadhatók.

sensor.get_windowing() → Tuple[int, int, int, int]¶

Visszaadja a sensor.set_windowing() segítségével korábban beállított roi tuple-t (x, y, w, h).

sensor.set_gainceiling(gainceiling: int) → bool¶

Beállítja a kamera képének erősítési felső határát. Az érvényes értékek: 2, 4, 8, 16, 32, 64 vagy 128. Sikeres esetén True értéket ad vissza.

sensor.set_contrast(contrast: int) → bool¶

Beállítja a kamera képének kontrasztját. Az érvényes tartomány -3 és +3 között van. Sikeres esetén True értéket ad vissza.

sensor.set_brightness(brightness: int) → bool¶

Beállítja a kamera képének fényerejét. Az érvényes tartomány -3 és +3 között van. Sikeres esetén True értéket ad vissza.

sensor.set_saturation(saturation: int) → bool¶

Beállítja a kamera képének telítettségét. Az érvényes tartomány -3 és +3 között van. Sikeres esetén True értéket ad vissza.

sensor.set_quality(quality: int) → bool¶

Beállítja a kamera képének JPEG tömörítési minőségét. Az érvényes tartomány 0 és 100 között van. Sikeres esetén True értéket ad vissza. Csak az OV2640/OV5640 kamerák esetén.

sensor.set_colorbar(enable: bool) → bool¶

Be- (True) vagy kikapcsolja (False) a színsáv tesztmódot. Sikeres esetén True értéket ad vissza.

sensor.set_auto_gain(enable: int, gain_db: float | None = None, gain_db_ceiling: float | None = None) → None¶

Az enable be- (1) vagy kikapcsolja (0) az automatikus erősítésszabályozást.

Ha az enable értéke 0, a gain_db segítségével beállíthat egy rögzített erősítést decibelben.

Ha az enable értéke nem nulla, a gain_db_ceiling segítségével beállíthatja az automatikus erősítésszabályozó algoritmus maximális erősítési felső határát decibelben.

A gain_db és a gain_db_ceiling csak kulcsszavas argumentumok.

sensor.get_gain_db() → float¶

Visszaadja a kamera aktuális erősítési értékét decibelben.

sensor.set_auto_exposure(enable: int, exposure_us: int = -1) → None¶

Az enable be- (1) vagy kikapcsolja (0) az automatikus expozíciószabályozást.

Ha az enable értéke 0, az exposure_us segítségével beállíthat egy rögzített expozíciós időt mikroszekundumban. Az exposure_us csak kulcsszavas argumentum.

sensor.get_exposure_us() → int¶

Visszaadja a kamera aktuális expozíciós értékét mikroszekundumban.

sensor.set_auto_whitebal(enable: int, rgb_gain_db: Tuple[float, float, float] | None = None) → None¶

Az enable be- (1) vagy kikapcsolja (0) az automatikus fehéregyensúlyt.

Ha az enable értéke 0, az rgb_gain_db segítségével beállíthat egy rögzített erősítést decibelben a vörös, zöld és kék csatornákra. Az rgb_gain_db csak kulcsszavas argumentum.

sensor.get_rgb_gain_db() → Tuple[float, float, float]¶

Visszaad egy tuple-t a kamera aktuális vörös, zöld és kék erősítési értékeivel decibelben.

sensor.set_auto_blc(enable: int, regs: List[int] | None = None) → None¶

Beállítja az automatikus fekete-szint kalibrálás (BLC) vezérlést a kamerán.

Az enable 1 az engedélyezéshez, vagy 0 a letiltáshoz.

A regs letiltás esetén manuálisan beállíthatja a BLC regiszterértékeket a sensor.get_blc_regs() segítségével korábban beolvasott értékek alapján. A lista hosszának meg kell egyeznie az érzékelő BLC regisztereinek számával.

sensor.get_blc_regs() → List[int]¶

Visszaadja az érzékelő BLC regisztereit egész számok listájaként. A sensor.set_auto_blc() függvénnyel való használatra.

sensor.set_hmirror(enable: bool) → None¶

Be- (True) vagy kikapcsolja (False) a vízszintes tükrözés módot. Alapértelmezetten kikapcsolva.

sensor.get_hmirror() → bool¶

True értéket ad vissza, ha a vízszintes tükrözés mód engedélyezve van.

sensor.set_vflip(enable: bool) → None¶

Be- (True) vagy kikapcsolja (False) a függőleges átfordítás módot. Alapértelmezetten kikapcsolva.

sensor.get_vflip() → bool¶

True értéket ad vissza, ha a függőleges átfordítás mód engedélyezve van.

sensor.set_transpose(enable: bool) → None¶

Be- (True) vagy kikapcsolja (False) a transzponálás módot. Alapértelmezetten kikapcsolva.

  • vflip=False, hmirror=False, transpose=False -> 0 fokos forgatás

  • vflip=True, hmirror=False, transpose=True -> 90 fokos forgatás

  • vflip=True, hmirror=True, transpose=False -> 180 fokos forgatás

  • vflip=False, hmirror=True, transpose=True -> 270 fokos forgatás

sensor.get_transpose() → bool¶

True értéket ad vissza, ha a transzponálás mód engedélyezve van.

sensor.set_auto_rotation(enable: bool) → None¶

Be- (True) vagy kikapcsolja (False) az automatikus forgatás módot. Alapértelmezetten kikapcsolva. Csak akkor működik, ha az OpenMV Cam rendelkezik beépített imu egységgel.

sensor.get_auto_rotation() → bool¶

True értéket ad vissza, ha az automatikus forgatás mód engedélyezve van.

sensor.set_framebuffers(count: int) → None¶

Beállítja a képadatok fogadásához használt képkocka-pufferek számát.

A count értéke lehet 1 (egyszeres puffer), 2 (kétszeres puffer), 3 (háromszoros puffer), vagy 4 vagy nagyobb, hogy az érzékelő-meghajtót videó FIFO módba állítsa, ahol a fogadott képkockák count puffer FIFO-jában tárolódnak.

sensor.get_framebuffers() → int¶

Visszaadja a jelenleg lefoglalt képkocka-pufferek számát.

sensor.disable_delays(disable: bool | None = None) → bool | None¶

Ha a disable értéke True, akkor letiltja az összes beállási idő késleltetést az érzékelő modulban.

Ha argumentum nélkül hívják meg, True értéket ad vissza, ha a késleltetések le vannak tiltva.

sensor.disable_full_flush(disable: bool | None = None) → bool | None¶

Ha a disable értéke True, akkor a képkocka-eldobáskor történő automatikus képkocka-puffer ürítés le van tiltva.

Ha argumentum nélkül hívják meg, True értéket ad vissza, ha az automatikus ürítés le van tiltva.

sensor.set_special_effect(sde: int) → bool¶

Beállítja a speciális digitális effektet (SDE) az érzékelőn. Az sde a sensor.NORMAL vagy sensor.NEGATIVE egyike. Sikeres esetén True értéket ad vissza.

sensor.set_lens_correction(enable: bool, radi: int, coef: int) → bool¶

Az enable True az engedélyezéshez, False a letiltáshoz. A radi a javítandó képpontok egész sugara. A coef a javítás erőssége.

Sikeres esetén True értéket ad vissza.

sensor.set_vsync_callback(cb: Callable[[int], None] | None) → None¶

Regisztrálja a cb visszahívást, hogy (megszakítási kontextusban) végrehajtódjon, valahányszor a kameramodul új képkockát generál (de mielőtt a képkockát fogadná).

A cb egy argumentumot fogad: a vsync láb aktuális állapotát a változás után.

Adjon át egy nem hívhatót (pl. None) a regisztráció megszüntetéséhez.

sensor.set_frame_callback(cb: Callable[[], None] | None) → None¶

Regisztrálja a cb visszahívást, hogy (megszakítási kontextusban) végrehajtódjon, valahányszor a kameramodul új képkockát generál, és a képkocka készen áll a sensor.snapshot() segítségével történő beolvasásra.

A cb nem fogad argumentumot.

Adjon át egy nem hívhatót (pl. None) a regisztráció megszüntetéséhez.

Végrehajt egy érzékelő-specifikus kérést. A request az IOCTL_* konstansok egyike; a fennmaradó pozicionális argumentumok és a visszatérési érték a kéréstől függenek. A támogatott kéréseket az alábbiakban érzékelő-családonként csoportosítjuk.

Ha True, utasítja a set_framesize() függvényt, hogy a látómezőre optimalizáljon a képkocka-sebesség helyett.

Visszaadja az aktuális FOV-wide állapotot bool értékként.

ioctl(IOCTL_GET_NIGHT_MODE)

Visszaadja az aktuális éjszakai-mód állapotot bool értékként.

ioctl(IOCTL_GET_RGB_STATS)

Visszaad egy 4 elemű tuple-t az érzékelőből beolvasott nyers RGB-csatorna statisztikákkal (r, gb, gr, b) (jellemzően fehéregyensúly-hangoláshoz használatos).
