v1.0.7¶

v1.0.7 adiciona suporte para novas placas e barramentos.

Novas funcionalidades¶

  • Adicionado suporte para placas Arduino baseadas em Arm Mbed.

  • Adicionado suporte para múltiplas interfaces de controlador I²C.

Correções de erros¶

  • Definidos os handlers onRequest / onReceive do I²C após Wire.begin(), para que os callbacks do modo periférico sejam registados corretamente.

Alterações de API incompatíveis¶

Nenhuma.
