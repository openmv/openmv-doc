v1.0.7¶

A v1.0.7 é uma correção de fiabilidade da camada de transporte.

Correções de erros¶

  • Corrigido um NACK recebido num pacote fragmentado na camada de transporte, melhorando a fiabilidade para transferências grandes divididas em múltiplos pacotes.

Nenhuma.
