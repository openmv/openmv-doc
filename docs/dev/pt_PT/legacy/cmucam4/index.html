CMUcam4#

A CMUcam4 é um sensor de visão por computador incorporado totalmente programável. O processador principal é o Parallax P8X32A (Propeller chip), ligado a um módulo de sensor de câmara CMOS OmniVision 9665. Para mais informações, consulte o wiki.

Funcionalidades#

  • Totalmente de código aberto e reprogramável utilizando o Propeller Tool

  • Compatível com Shield Arduino

    • c/ bibliotecas de interface de suporte e aplicações de demonstração para o Arduino e o BASIC Stamp

  • Sensor de cor VGA (640x480) RGB565/YUV655

    • Taxa de processamento de imagem de 30 fotogramas por segundo

    • Dumps de imagem raw através de série ou para cartão flash

      • (640:320:160:80)x(480:240:120:60) resolução de imagem

      • Espaço de cor RGB565/YUV655

  • Processamento de Imagem Incorporado (QQVGA 160x120)

    • Seguir blobs de cor definidos pelo utilizador no espaço de cor RGB/YUV

    • Recolha de dados de média, mediana, moda e desvio padrão – amostrado a partir de uma resolução de 40x120

    • Captura de imagem segmentada (com threshold) para visualização de seguimento (através de série ou para cartão flash)

      • resolução de imagem 80x60

      • Espaço de cor monocromático

    • Geração de histograma (até 128 Bins) – amostrado a partir de uma resolução de 40x120

    • Recorte de imagem arbitrário (janelamento)

  • ranhura para cartão flash µSD/µSDHC com suporte completo de controlador de sistema de ficheiros FAT16/32

    • c/ manipulação de diretórios e ficheiros

  • Interfaces de E/S

    • Controlador de servos de duas portas (pan e tilt com resolução de 1us a uma taxa de atualização de 50 Hz)

      • Os canais de servo de Pan e/ou Tilt podem ser configurados como GPIOs

    • LED indicador controlável pelo utilizador (vermelho) e LED de alimentação (verde)

    • TTL UART (até 250,000 baud – 19,200 baud por predefinição)

  • Saída de vídeo analógico monocromático em banda base (NTSC/PAL) de resolução 160x120 para visualização de seguimento (imagem segmentada (com threshold) com centroide de cor e sobreposição de caixa delimitadora a 30 FPS)

  • GUI da CMUcam4 para visualizar imagens no PC
