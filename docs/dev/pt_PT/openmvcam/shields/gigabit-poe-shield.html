Módulo Gigabit PoE#

O Módulo Gigabit PoE é um módulo Ethernet 10/100/1000 Mb/s com Power-over-Ethernet 802.3af para OpenMV Cams com PHY Ethernet integrado. Um único cabo para um switch PoE transporta simultaneamente a alimentação e a ligação de rede.

Para o datasheet completo, fotografias e informações de compra, consulte a página do produto Módulo Gigabit PoE.

Nota

Suportado apenas na OpenMV Cam RT1062 e N6.

Destaques#

  • Ethernet Gigabit 10/100/1000 Mb/s com PoE IEEE 802.3af

  • Até ~6 W para a câmara via VIN de 5,6 V

  • Design isolado a 1500 V — empilha com módulos de cabeçalho duplo via díodo OR integrado

Pinagem#

Referência de pinos#

A velocidade de 10/100 Mb/s utiliza apenas os pares MDI TX e MDI RX (Pares A e B). O Gigabit (1000BASE-T) é bidirecional em todos os quatro pares A/B/C/D, pelo que as linhas MDI TX ± e MDI RX ± duplicam como Par A e Par B a velocidades gigabit, e os Pares C e D transportam os pares adicionais exclusivos de gigabit.

Pino

Função

MDI LED

Linha LED de ligação/atividade do PHY

MDI TX P / DA P

Par A positivo — MDI TX+ a 10/100, BI_DA+ a gigabit

MDI TX N / DA N

Par A negativo — MDI TX− a 10/100, BI_DA− a gigabit

MDI RX P / DB P

Par B positivo — MDI RX+ a 10/100, BI_DB+ a gigabit

MDI RX N / DB N

Par B negativo — MDI RX− a 10/100, BI_DB− a gigabit

DC P

Par C positivo (BI_DC+) — apenas gigabit

DC N

Par C negativo (BI_DC−) — apenas gigabit

DD P

Par D positivo (BI_DD+) — apenas gigabit

DD N

Par D negativo (BI_DD−) — apenas gigabit

Saída VIN

5,6 V a até ~1 A a partir do regulador PoE integrado (alimenta a câmara)

Rail 3.3V

Alimenta os componentes eletrónicos integrados do módulo

Rail GND

Terra comum

Nota

Os pares DC e DD estão ligados à câmara através de resistores de 0 ohm na parte traseira do módulo. Remova-os para libertar P15–P18 (os pinos exclusivos de gigabit em câmaras como a OpenMV N6) para uso independente — o módulo passa então para Ethernet 10/100 uma vez que os pares gigabit já não estão ligados.

Utilização#

Quando o módulo está ligado a um switch PoE, o PHY gigabit da câmara aparece como uma interface network.LAN. O DHCP é executado automaticamente assim que a ligação fica ativa:

import network
import time

lan = network.LAN()
lan.active(True)
while not lan.isconnected():
    time.sleep(1)
print("Ethernet IP:", lan.ipconfig("addr4")[0])
