5. Processamento de Imagem¶

Todas as páginas até agora trataram os dados de pixel apenas o tempo suficiente para os obter: capturar um fotograma, configurar um sensor, mover bytes da câmara para a RAM. O processamento de imagem inverte a direção – os pixels são a entrada, e o objetivo é o que quer que a aplicação necessite deles.

O módulo image é o kit de ferramentas. Cada operação recebe um ou dois objetos Image e devolve ou uma Image transformada – pixels filtrados, com limiar aplicado, deformados ou compostos de forma diferente – ou uma lista de objetos de resultado: manchas, linhas, códigos, correspondências, estatísticas. Os dois tipos compõem-se. Um pipeline típico limpa o fotograma capturado, extrai um conjunto de objetos de resultado a partir dele, converte-os em medições e desenha anotações de volta no fotograma para apresentação.
