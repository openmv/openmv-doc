класс Statistics – объект Statistics¶

Объект statistics – это attrtuple, возвращаемый методом histogram.get_statistics() (и его псевдонимами histogram.get_stats() / histogram.statistics()) или непосредственно методом Image.get_statistics(). Он подводит итог распределения бинов исходной гистограммы с помощью восьми сводных показателей на канал: среднее, медиана, мода, стандартное отклонение, минимум, максимум, нижний квартиль и верхний квартиль.

Для данных в оттенках серого полезные значения несут только поля без префикса (mean, median, mode, stdev, min, max, lq, uq), и они повторно используются как сводные показатели канала LAB L для данных RGB565 – поэтому поля l_* отражают поля без префикса и предоставлены для симметрии. a_* и b_* несут полезные значения только для гистограмм RGB565.

Поля доступны по имени атрибута (statistics.mean) или по индексу (statistics[0]). У объекта нет открытого конструктора.

class image.statistics¶

Чтобы создать этот объект, вызовите histogram.get_statistics() или Image.get_statistics().

Оттенки серого / LAB L (также доступно как l_* ниже)¶

mean¶

Среднее гистограммы в оттенках серого. Целое число 0 – 255. Индекс [0].

median¶

Медиана гистограммы в оттенках серого. Целое число 0 – 255. Индекс [1].

mode¶

Мода гистограммы в оттенках серого. Целое число 0 – 255. Индекс [2].

stdev¶

Стандартное отклонение гистограммы в оттенках серого. Целое число 0 – 255. Индекс [3].

min¶

Минимум гистограммы в оттенках серого. Целое число 0 – 255. Индекс [4].

max¶

Максимум гистограммы в оттенках серого. Целое число 0 – 255. Индекс [5].

lq¶

Нижний квартиль (25-й процентиль) гистограммы в оттенках серого. Целое число 0 – 255. Индекс [6].

uq¶

Верхний квартиль (75-й процентиль) гистограммы в оттенках серого. Целое число 0 – 255. Индекс [7].
