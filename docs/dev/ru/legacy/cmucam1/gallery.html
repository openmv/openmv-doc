Следующий набор изображений CMUcam1 — это примеры того, как наша программа-интерфейс на java может использоваться для отображения вывода отслеживания цвета с CMUcam1:

Вот B2Bot — робот с CMUcam1 на серво-голове, двумя двигателями постоянного тока и специальным контроллером под названием Cerebellum:

Это MiniBot, робот CMUcam1 + Cerebellum с 2 сервоприводами, размером менее 5 дюймов в кубе:

Первое изображение ниже показывает ServoBot, CMUcam1, напрямую управляющий любительским сервоприводом. Остальные изображения показывают MiniBot3D, который использует 3 сервопривода для управления не только панорамированием, но и наклоном.

Легко запрограммировать робота на следование за яркими объектами, используя вывод CMUcam1. Фактически, если вы нажмёте здесь, вы можете загрузить код на C, который мы использовали для программирования чипа PIC, управляющего CMUcam1 на MiniBot3D.
