Журнал изменений OpenMV¶

Каждый продукт OpenMV имеет собственную историю выпусков. Выберите тот, который вас интересует — каждая запись описывает, что нового, что изменилось и (где это важно) как именно выполнить миграцию.

OpenMV Firmware

Прошивка машинного зрения на MicroPython для камеры OpenMV Cam. Новые функции камеры, обработки изображений и поддержки плат, а также критические изменения API и инструкции по миграции скриптов.

Смотреть выпуски →

OpenMV IDE

Среда разработки для написания, запуска и отладки скриптов на камере OpenMV Cam. Новые функции редактора, камеры и инструментов, исправления ошибок и поддержка платформ.

Смотреть выпуски →

OpenMV Python Package

Библиотека Python для хоста и утилита командной строки openmv для взаимодействия с камерой OpenMV Cam с компьютера по протоколу Camera Protocol V2 — потоковая передача, скриптинг и тестирование производительности.

Смотреть выпуски →

OpenMV Arduino RPC Library

Библиотека Arduino для управления камерой OpenMV Cam через CAN, I²C, SPI, UART и USB. Новая поддержка плат, шинные интерфейсы и повышение надёжности.

Смотреть выпуски →
