hs3003 — HS3003 fuktighets-/temperatursensor#

Drivrutin för Renesas HS3003 högprecisionssensor för relativ luftfuktighet och temperatur via I2C. En enda I2C-transaktion utlöser en ny mätning och läser tillbaka fyra byte som innehåller både fuktighets- och temperaturproverna; drivrutinen skalar de råa värdena till tekniska enheter.

Exempel:

import time
from machine import Pin, I2C
from hs3003 import HS3003

bus = I2C(1, scl=Pin("P15"), sda=Pin("P14"))
hts = HS3003(bus)

while True:
    print("rH: %.2f%% T: %.2fC" % (hts.humidity(), hts.temperature()))
    time.sleep_ms(100)

Klasser#

class hs3003.HS3003(bus: machine.I2C, address: int = 0x44)#

Konstruera en HS3003-instans.

bus

En konfigurerad machine.I2C-buss som sensorn är ansluten till.

address

Sensorns 7-bitars I2C-adress. Standardvärdet är 0x44, fabriksadressen som används av alla HS3003-enheter.

humidity() → float#

Utlös en mätning och returnera den relativa luftfuktigheten i procent (%RH), beräknad som (raw & 0x3FFF) / 16383 * 100.

temperature() → float#

Utlös en mätning och returnera temperaturen i grader Celsius, beräknad som (raw >> 2) / 16383 * 165 - 40.
