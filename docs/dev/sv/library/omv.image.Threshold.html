class Threshold – Threshold-objekt¶

Threshold-objektet är en attrtuple som returneras av histogram.get_threshold(). Det rapporterar det Otsu-optimala uppdelningsvärdet för varje kanal i det underliggande Histogram – dvs. det fackvärde som bäst delar upp kanalen i en ”bakgrunds”- och ”förgrunds”-halva.

För gråskalehistogram, använd value (de andra tre fälten är 0). För RGB565-histogram ger l_value / a_value / b_value Otsu-tröskelvärdet för LAB-kanalerna L, A respektive B. De returnerade värdena lämpar sig väl för att matas direkt in i Image.binary() eller någon annan metod som tar LAB-färgtröskelvärden.

Fälten är åtkomliga via attributnamn (threshold.value) eller via index (threshold[0]). Objektet har ingen publik konstruktor.

class image.threshold¶

Anropa histogram.get_threshold() för att skapa detta objekt.

value¶

Otsu-tröskelvärde för gråskala. Heltal 0 – 255. Index [0].

l_value¶

Otsu-tröskelvärde för LAB-kanalen L. Heltal 0 – 100. Index [1].

a_value¶

Otsu-tröskelvärde för LAB-kanalen A. Heltal -128 – 127. Index [2].

b_value¶

Otsu-tröskelvärde för LAB-kanalen B. Heltal -128 – 127. Index [3].
