13.7.1. Skapa ett projekt¶

Från Roboflow-arbetsytan öppnar New Project skapasidan. Ett projekt samlar en datamängd, modellerna som tränats på den och deras distributionsmål.

Två val är viktiga här:

  • Project Name. Använd något som är lätt att komma ihåg – det här exemplet använder Finding raccoons. Namnet blir en del av modellens identifierare, som du laddar ner i distributionssteget.

  • Project Type. Detta avgör vad modellen förutsäger, så matcha det mot vad du vill att kameran ska göra. Object Detection – att hitta objekt och deras positioner med begränsningsrutor – är det val som lokaliserar saker i bildrutan, och det är det som kamerans YOLO-efterbehandlare avkodar. De andra typerna – klassificering, instanssegmentering, nyckelpunktsdetektering – tränar olika modellhuvuden; det här exemplet använder objektdetektering.

Alternativen för synlighet och licens påverkar bara om datamängden delas offentligt på Roboflow Universe – de har ingen betydelse för modellen som hamnar på kameran. Ställ in dem som du vill, och tryck sedan Create Project.
