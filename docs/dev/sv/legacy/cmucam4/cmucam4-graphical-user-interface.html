CMUcam4 grafiskt användargränssnitt#

Vad CMUcam4GUI är#

CMUcam4GUI är ett användbart verktyg som gör det enklare att arbeta med CMUcam4. CMUcam4GUI gör att du kan ta bilder med din CMUcam4 och visa dem på din dator och använda bilderna för att beräkna färgspårningsgränserna för färgade objekt i bilden som du vill att CMUcam4 ska spåra. Med CMUcam4GUI kan du snabbt bestämma färgspårningsgränserna för ett objekt i en ljussättning och vidga färgspårningsgränserna för att stödja spårning av objektet i föränderliga ljussättningar. CMUcam4GUI gör det också möjligt att se vad CMUcam4 spårar i realtid utan en TV genom att grafiskt visa innehållet i spårnings-, medelvärdes-, histogram- och bildpaket som skickas av CMUcam4. Dessutom gör CMUcam4GUI att du kan leka med alla CMUcam4-alternativ i realtid, som att styra pan- och tilt-servona, färgspårningsalternativ och kamerakontroller på låg nivå. Slutligen gör CMUcam4GUI att du kan hantera det secure digital-kort som är anslutet till CMUcam4 på distans genom CMUcam4.

Windows-installation#

Följ stegen nedan för att konfigurera CMUcam4GUI för Windows:

  1. Ladda ner filen CMUcam4GUI.zip

  2. Packa upp den och extrahera innehållet

  3. Kör filen CMUcam4GUI.exe

Linux-installation#

Följ stegen nedan för att konfigurera CMUcam4GUI för Linux:

  1. Lägg till ditt användarnamn i gruppen dialout

sudo adduser username dialout

  1. Verifiera nu genom att skriva

groups username

  1. Logga nu ut och logga sedan in igen

  2. Ladda ner filen CMUcam4GUI.zip

  3. Packa upp den och extrahera innehållet

  4. Kör filen CMUcam4GUI.sh

Macintosh-installation#

Följ stegen nedan för att konfigurera CMUcam4GUI för Macintosh:

  1. Installera Qt 1.2.1 SDK (Desktop 4.8.1) för Mac

  2. Ladda ner CMUcam4GUI-källkod

  3. Öppna filen CMUcam4GUI.pro med QtCreator och växla från felsökningsläge till release-läge

  4. Öppna filen CMUcom4.h och ändra #define CMUCOM4_FAST_BAUD_RATE till 115200 från 230400 (Mac-datorer klarar inte baudhastigheten 230400 BPS)

  5. Klicka på run

Fråga: Varför har ni ingen färdigbyggd MAC-applikation?

Svar: Kwabena, CMUcam4-utvecklaren, har inte en MAC…

CMUcam4GUI-manual#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

Kommandon i menyraden och verktygsfältet

Verktygsfältskommandon

  • Begin/Reset - Anslut till CMUcam4

  • End - Koppla från CMUcam4

  • Sleep Lightly - Försätt CMUcam4 i ett mycket lågt effektläge

  • Sleep Deeply - Försätt CMUcam4 i ett ultralågt effektläge

  • Send Frame - Ta en färgbild och visa den i visningsfönstret

  • Send Bitmap - Ta en binär bild och visa den i visningsfönstret

    • Detta är en 80x60 binär bild av det som CMUcam4 spårar - CMUcam4 måste spåra något för att detta ska vara användbart

  • Dump Frame - Ta en färgbild och spara den på SD-kortet

  • Dump Bitmap - Ta en binär bild och spara den på SD-kortet

    • Detta är en 80x60 binär bild av det som CMUcam4 spårar - CMUcam4 måste spåra något för att detta ska vara användbart

Menyradskommandon

  • Save Frame - Spara färgbilden i visningsfönstret till en fil

  • Save Bitmap - Spara den binära bilden i visningsfönstret till en fil

    • Detta är en 80x60 binär bild av det som CMUcam4 spårar - CMUcam4 måste spåra något för att detta ska vara användbart

  • Clear Frame - Radera färgbilden i visningsfönstret

  • Clear Bitmap - Radera den binära bilden i visningsfönstret

    • Detta är en 80x60 binär bild av det som CMUcam4 spårar - CMUcam4 måste spåra något för att detta ska vara användbart

  • Open - Öppna en färgbild att använda med spårningsvalsfunktionen

  • Exit - Avsluta applikationen

Viewing-fliken

  • Send Frame Control - Välj horisontell och vertikal upplösning för send frame-knappen i verktygsfältet

    • Du kan inte flytta det du tar en bild av medan send frame-kommandot körs

    • Om bildkvaliteten ser dålig ut, stäng av auto-gain och automatisk vitbalans och se till att CMUcam4 tittar på ett belyst objekt

    • Använd en mindre upplösning för att fånga bilden snabbare - som 160x120

  • Dump Frame Control - Välj horisontell och vertikal upplösning för dump frame-knappen i verktygsfältet

    • Du kan inte flytta det du tar en bild av medan dump frame-kommandot körs

    • Om bildkvaliteten ser dålig ut, stäng av auto-gain och automatisk vitbalans och se till att CMUcam4 tittar på ett belyst objekt

    • Använd en mindre upplösning för att fånga bilden snabbare - som 160x120

  • Opacity Control - Välj opaciteten för skickad bild och skickad bitmapp - mellan 1.00 för 100% och 0.00 för 0%

  • Tracking Control - Välj en färgklick i visningsfönstret att spåra

    • Markeringsrektangeln ska endast omsluta pixlarna inom den färgklick du vill spåra

    • När du har valt färgklicken, klicka på knappen track color för att beräkna färggränserna för att spåra den färgklicken

      • Algoritmen fungerar genom att beräkna medelvärdet och standardavvikelsen för de röda, gröna och blå färgkanalerna inuti markeringsrektangeln

      • Konfidensintervallnivåkoefficienten väljer hur mycket av standardavvikelsen som ska adderas +/- till medelvärdena för de röda, gröna och blå färgkanalerna - mellan 0 för +/- (stdev * 1) till 99 för +/- (stdev * 10)

Tracking-fliken

Aktivera Line-läget i alternativpanelen för att se de blå spårade pixlarna.

Options-fliken

Disk-fliken

  • Disk Information - Hämta information om diskens geometri

  • Disk Space - Hämta information om utrymmet på disken

  • Disk Control - Avmontera/formatera disken

    • Obs: Du bör avmontera disken efter att ha använt något SD-kortrelaterat kommando innan du stänger det grafiska användargränssnittet

    • Obs: Att klicka på knappen formatera disk formaterar disken omedelbart

  • New File - Skapa en ny fil med lite text eller lägg till lite text i en fil

  • New Folder - Skapa en ny mapp

  • File Explorer - Interagera med filerna/mapparna på SD-kortet från din dator

    • Klicka på uppdatera för att lista innehållet i den aktuella katalogen

    • Ändra en fils eller mapps attribut genom att klicka på attributkryssrutorna

      • Dubbelklicka på en mapp för att öppna den

      • Dubbelklicka på en fil för att öppna den

        • Försök inte öppna filer större än cirka 10 KB (om du inte känner för att vänta i evighet) eftersom filöverföringen är mycket långsam på cirka 4 KB/s

        • Du kan flytta filer och mappar genom att byta namn på dem med en ny sökväg, t.ex. byt namn på txt.txt till ../txt.txt

Hur man avgränsar färger#

  • Gå till alternativ

  • Slå på YUV

  • Vänta 5 sekunder

  • Stäng av autogain och automatisk vitbalans

  • Välj horisontell upplösning för send frame och sätt den till 160

  • Välj vertikal upplösning för send frame och sätt den till 120

  • Rikta CMUcam4 mot objektet du vill spåra, och klicka sedan, utan att flytta CMUcam4 eller objektet och där båda inte får röra sig, på send frame

  • Vänta tills bilden har skickats till GUI:t, flytta inte objektet

  • Välj det färgområde på objektet (och inget utanför det området)

  • Klicka på track selection

  • Klicka på okej med 50 som standardvärde

  • Om endast objektet har blått ritat på sig, klicka på track selection igen och gör 50 till 60

  • Upprepa ovanstående tills du trycker upp track selection-värdet så högt som möjligt, du kan sluta när det är 99. Om du börjar avgränsa oönskade saker så är du också klar.

  • Gå sedan över till options… gör green min-värdet 0 och green max-värdet 255 (detta ställer in Ymin till 0 och Ymax till 255). Om du börjar avgränsa oönskade saker, prova mindre breda gränser tills du bara avgränsar det objekt du vill ha. Du kan uppdatera vad som avgränsas genom att klicka på sendBitmap-knappen.

  • Och… sedan bör du ha dina sex spårningsalternativ… Dessutom kan du även vidga gränserna (V/röd) och (U/blå) manuellt om du vill. Detta bidrar till att göra systemet mer robust i olika ljusförhållanden och liknande.

För att se vad CMUcam4 ser….

  • Slå på line-läget (det finns en kryssruta i options)

  • Gå sedan till viewing och klicka på track color.
