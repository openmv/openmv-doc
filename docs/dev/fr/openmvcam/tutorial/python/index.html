2. Aperçu de Python¶

L’OpenMV Cam exécute MicroPython, le même langage que Python de bureau avec une bibliothèque standard plus réduite et quelques modules supplémentaires pour dialoguer avec le matériel. Tout ce qui est écrit pour la caméra – d’un script de deux lignes qui fait clignoter la LED à une application complète de vision industrielle – est du code Python.
