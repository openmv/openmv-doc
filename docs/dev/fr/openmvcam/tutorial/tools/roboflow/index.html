13.7. Roboflow¶

Roboflow vous mène des images brutes à un modèle de vision par ordinateur entraîné et prêt pour la production, sans écrire une seule ligne de code d’entraînement – le tout dans le navigateur, sur leurs serveurs. OpenMV est l’une de ses cibles de déploiement : une fois l’entraînement terminé, un seul bouton télécharge des poids TFLite quantifiés en entiers, prêts pour la caméra. La documentation de Roboflow approfondit chaque étape.
