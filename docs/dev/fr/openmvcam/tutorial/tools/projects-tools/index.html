13.2. Outils OpenMV Projects¶

Les OpenMV Projects Tools forment une collection grandissante d’applications de bureau à interface graphique, chacune associée à un script MicroPython compagnon sur la caméra. La caméra capture et diffuse les données du capteur ; l’application de bureau les reçoit via USB et fournit la visualisation en temps réel, le réglage de paramètres ou le flux de calibration dont a besoin un cas d’usage précis.

Chaque outil dispose de son propre sous-répertoire dans le dépôt, contenant le script de bureau, le script compagnon côté caméra, un fichier README couvrant l’installation et l’utilisation, ainsi que des captures d’écran.

Les outils actuellement présents dans le dépôt comprennent des utilitaires de calibration pour superposer la sortie thermique ou d’une caméra événementielle sur une trame en couleur, un régleur ISP interactif pour le pipeline de couleurs de l’OpenMV N6 (debayer en direct, balance des blancs et matrice de correction des couleurs), ainsi qu’un visualiseur en temps réel pour le capteur événementiel Prophesee GenX320. De nouveaux outils arrivent dans le dépôt à mesure que de nouveaux capteurs ou flux de travail en réclament.

Chaque outil suit la même structure : une application de bureau Python construite avec DearPyGui, communiquant avec la caméra via le paquet Python openmv sur le port série USB. Python 3.12 ou plus récent est requis ; macOS et Linux offrent le meilleur débit USB.
