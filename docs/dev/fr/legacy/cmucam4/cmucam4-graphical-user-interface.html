Interface graphique de la CMUcam4#

Ce qu’est le CMUcam4GUI#

Le CMUcam4GUI est un outil utile qui facilite le travail avec la CMUcam4. Le CMUcam4GUI vous permet de prendre des photos avec votre CMUcam4 et de les visualiser sur votre ordinateur, et d’utiliser les photos pour calculer les bornes de suivi de couleur des objets colorés de la photo que vous voulez que la CMUcam4 suive. À l’aide du CMUcam4GUI, vous pouvez rapidement déterminer les bornes de suivi de couleur d’un objet dans une scène d’éclairage donnée et élargir les bornes de suivi de couleur pour permettre le suivi de l’objet dans des scènes d’éclairage changeantes. Le CMUcam4GUI vous permet également de voir ce que la CMUcam4 suit en temps réel sans téléviseur, en affichant graphiquement le contenu des paquets de suivi, de moyenne, d’histogramme et d’image envoyés par la CMUcam4. De plus, le CMUcam4GUI vous permet de jouer avec toutes les options de la CMUcam4 en temps réel, comme le contrôle des servos de panoramique et d’inclinaison, les options de suivi de couleur et les contrôles de bas niveau de la caméra. Enfin, le CMUcam4GUI vous permet de gérer à distance la carte Secure Digital connectée à la CMUcam4, par l’intermédiaire de la CMUcam4.

Configuration sous Windows#

Suivez les étapes ci-dessous pour configurer CMUcam4GUI pour Windows :

  1. Téléchargez le fichier CMUcam4GUI.zip

  2. Décompressez-le et extrayez le contenu

  3. Exécutez le fichier CMUcam4GUI.exe

Configuration Linux#

Suivez les étapes ci-dessous pour configurer CMUcam4GUI pour Linux :

  1. Ajoutez votre nom d’utilisateur au groupe dialout

sudo adduser username dialout

  1. Vérifiez maintenant en tapant

groups username

  1. Maintenant, déconnectez-vous puis reconnectez-vous

  2. Téléchargez le fichier CMUcam4GUI.zip

  3. Décompressez-le et extrayez le contenu

  4. Exécutez le fichier CMUcam4GUI.sh

Configuration Macintosh#

Suivez les étapes ci-dessous pour configurer CMUcam4GUI pour Macintosh :

  1. Installez le SDK Qt 1.2.1 (Desktop 4.8.1) pour Mac

  2. Téléchargez la source de CMUcam4GUI

  3. Ouvrez le fichier CMUcam4GUI.pro à l’aide de QtCreator et passez du mode debug au mode release

  4. Ouvrez le fichier CMUcom4.h et changez le #define CMUCOM4_FAST_BAUD_RATE de 230400 à 115200 (les Mac ne peuvent pas gérer le débit en bauds de 230400 BPS)

  5. Cliquez sur run

Question : pourquoi n’avez-vous pas d’application MAC précompilée ?

Réponse : Kwabena, le développeur de la CMUcam4, ne possède pas de MAC…

Manuel de CMUcam4GUI#

The CMUcam4GUI is very intuitive and implements all the same functionality in the CMUcam4 Arduino Interface Library. This manual only exists to give an overview of the CMUcam4GUI and you should read the CMUcam4 Command List User Manual if you need more information about CMUcam4 commands. If you encounter errors trying to connect to the CMUcam4, unplug your USB to serial device from your computer and plug it back in again. Also, make sure your serial converter can handle 19200 BPS, 115200 BPS, and 230400 BPS. If you are using a FTDI chip based converter then you should be fine.

Commandes de la barre de menus et de la barre d’outils

Commandes de la barre d’outils

  • Begin/Reset - Se connecter à la CMUcam4

  • End - Se déconnecter de la CMUcam4

  • Sleep Lightly - Mettre la CMUcam4 dans un état de très faible consommation

  • Sleep Deeply - Mettre la CMUcam4 dans un état de très basse consommation (ultra low power)

  • Send Frame - Prendre une image couleur et l’afficher dans la fenêtre de visualisation

  • Send Bitmap - Prendre une image binaire et l’afficher dans la fenêtre de visualisation

    • Ceci est une image binaire 80x60 de tout ce que la CMUcam4 suit - la CMUcam4 doit suivre quelque chose pour que cela soit utile

  • Dump Frame - Prendre une image couleur et l’enregistrer sur la carte SD

  • Dump Bitmap - Prendre une image binaire et l’enregistrer sur la carte SD

    • Ceci est une image binaire 80x60 de tout ce que la CMUcam4 suit - la CMUcam4 doit suivre quelque chose pour que cela soit utile

Commandes de la barre de menus

  • Save Frame - Enregistrer dans un fichier l’image couleur de la fenêtre de visualisation

  • Save Bitmap - Enregistrer dans un fichier l’image binaire de la fenêtre de visualisation

    • Ceci est une image binaire 80x60 de tout ce que la CMUcam4 suit - la CMUcam4 doit suivre quelque chose pour que cela soit utile

  • Clear Frame - Effacer l’image couleur dans la fenêtre de visualisation

  • Clear Bitmap - Effacer l’image binaire dans la fenêtre de visualisation

    • Ceci est une image binaire 80x60 de tout ce que la CMUcam4 suit - la CMUcam4 doit suivre quelque chose pour que cela soit utile

  • Open - Ouvrir une image couleur à utiliser avec la fonction de sélection de suivi

  • Exit - Quitter l’application

Onglet Viewing

  • Send Frame Control - Sélectionner la résolution horizontale et verticale pour le bouton send frame de la barre d’outils

    • Vous ne pouvez pas déplacer ce que vous photographiez pendant que la commande send frame est en cours d’exécution

    • Si la qualité de l’image semble mauvaise, désactivez le gain automatique et la balance des blancs automatique et assurez-vous que la CMUcam4 regarde un objet éclairé

    • Utilisez une résolution plus petite pour capturer l’image plus rapidement - comme 160x120

  • Dump Frame Control - Sélectionner la résolution horizontale et verticale pour le bouton dump frame de la barre d’outils

    • Vous ne pouvez pas déplacer ce que vous photographiez pendant que la commande dump frame est en cours d’exécution

    • Si la qualité de l’image semble mauvaise, désactivez le gain automatique et la balance des blancs automatique et assurez-vous que la CMUcam4 regarde un objet éclairé

    • Utilisez une résolution plus petite pour capturer l’image plus rapidement - comme 160x120

  • Opacity Control - Sélectionner l’opacité de l’image envoyée et du bitmap envoyé - entre 1.00 pour 100 % et 0.00 pour 0 %

  • Tracking Control - Sélectionner une tache de couleur (blob) à suivre dans la fenêtre de visualisation

    • Le rectangle de sélection ne devrait englober que les pixels situés à l’intérieur de la tache de couleur (blob) que vous souhaitez suivre

    • Après avoir sélectionné la tache de couleur, cliquez sur le bouton track color pour calculer les bornes de couleur permettant de suivre cette tache de couleur

      • L’algorithme fonctionne en calculant la moyenne et l’écart-type des canaux de couleur rouge, vert et bleu à l’intérieur du rectangle de sélection

      • Le coefficient du niveau d’intervalle de confiance sélectionne quelle proportion de l’écart-type ajouter +/- aux moyennes des canaux de couleur rouge, vert et bleu - entre 0 pour +/- (stdev * 1) et 99 pour +/- (stdev * 10)

Onglet Tracking

Activez le mode ligne (Line mode) dans le panneau d’options pour voir les pixels suivis en bleu.

Onglet Options

Onglet Disk

  • Disk Information - Obtenir des informations sur la géométrie du disque

  • Disk Space - Obtenir des informations sur l’espace du disque

  • Disk Control - Démonter/formater le disque

    • Remarque : vous devriez démonter le disque après avoir utilisé toute commande liée à la carte SD avant de fermer l’interface graphique

    • Remarque : cliquer sur le bouton de formatage du disque formatera immédiatement le disque

  • New File - Créer un nouveau fichier avec du texte ou ajouter du texte à un fichier

  • New Folder - Créer un nouveau dossier

  • File Explorer - Interagir avec les fichiers/dossiers de la carte SD depuis votre ordinateur

    • Cliquez sur refresh pour lister le contenu du répertoire actuel

    • Modifiez les attributs d’un fichier ou d’un dossier en cliquant sur les cases à cocher des attributs

      • Double-cliquez sur un dossier pour y entrer

      • Double-cliquez sur un fichier pour l’ouvrir

        • N’essayez pas d’ouvrir des fichiers de plus d’environ 10 Ko (à moins de vouloir attendre une éternité), car le transfert de fichiers est très lent, autour de 4 Ko/s

        • Vous pouvez déplacer des fichiers et des dossiers en les renommant avec un nouveau chemin, par ex. renommer txt.txt en ../txt.txt

Comment délimiter les couleurs#

  • Allez dans les options

  • Activer le YUV

  • Attendez 5 secondes

  • Désactiver le gain automatique et la balance des blancs automatique

  • Sélectionnez la résolution horizontale de send frame et réglez-la sur 160

  • Sélectionnez la résolution verticale de send frame et réglez-la sur 120

  • Pointez la CMUcam4 vers l’objet que vous souhaitez suivre, puis, sans déplacer la CMUcam4 ni l’objet, et alors que les deux ne doivent pas bouger, cliquez sur send frame

  • Attendez que l’image soit envoyée à l’interface graphique, ne bougez pas l’objet

  • Sélectionnez la zone de couleur de l’objet (et rien en dehors de cette zone)

  • Cliquez sur track selection

  • Cliquez sur OK en gardant 50 comme valeur par défaut

  • Si seul l’objet est recouvert de bleu, cliquez à nouveau sur track selection et remplacez le 50 par 60

  • Répétez ce qui précède jusqu’à ce que vous poussiez la valeur de track selection aussi haut que possible ; vous pouvez vous arrêter lorsqu’elle atteint 99. Si vous commencez à délimiter des éléments indésirables, alors vous avez également terminé.

  • Ensuite, allez dans les options… réglez la valeur min du vert sur 0 et la valeur max du vert sur 255 (cela règle Ymin à 0 et Ymax à 255). Si vous commencez à délimiter des éléments indésirables, essayez des bornes moins larges jusqu’à ne délimiter que l’objet souhaité. Vous pouvez actualiser ce qui est délimité en cliquant sur le bouton sendBitmap.

  • Et… vous devriez alors disposer de vos six options de suivi… De plus, vous pouvez aussi élargir manuellement les bornes (V/rouge) et (U/bleu) si vous le souhaitez. Cela contribuera à rendre le système plus robuste dans différentes conditions d’éclairage, etc.

Pour voir ce que voit la CMUcam4….

  • Activer le mode ligne (il y a une case à cocher dans les options)

  • Ensuite, allez dans viewing et cliquez sur track color.
