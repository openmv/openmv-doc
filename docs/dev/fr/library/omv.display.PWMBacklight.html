classe PWMBacklight – Rétroéclairage PWM#

La classe PWMBacklight contrôle le rétroéclairage d’un écran via un canal de minuteur PWM. Passez une instance comme argument backlight à n’importe quel constructeur d’affichage qui accepte un contrôleur de rétroéclairage.

Constructeurs#

class display.PWMBacklight(pin: pyb.Pin, timer: int = 3, channel: int = 3, frequency: int = 200)#

Crée un contrôleur de rétroéclairage piloté par PWM.

pin spécifie la broche (Pin) à utiliser.

timer spécifie le numéro du minuteur (Timer) à utiliser.

channel spécifie le canal du minuteur (Timer) à utiliser.

frequency spécifie la fréquence PWM en Hz.

deinit() → None#

Désinitialise le contrôleur de rétroéclairage.

backlight(value: int) → None#

Règle l’intensité du rétroéclairage de 0 à 100.
