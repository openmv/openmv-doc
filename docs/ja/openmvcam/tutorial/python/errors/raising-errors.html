2.26. гӮЁгғ©гғјгҒ®йҖҒеҮәВ¶

й–ўж•°гҒҜгҖҒдҫӢеӨ–гӮ’йҖҒеҮәгҒҷгӮӢгҒ“гҒЁгҒ§е‘јгҒіеҮәгҒ—е…ғгҒ«е•ҸйЎҢгӮ’зҹҘгӮүгҒӣгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮгӮӯгғјгғҜгғјгғүгҒҜ raise гҒ§гҒҷ:

def square_root(x):
    if x < 0:
        raise ValueError("square_root expects a non-negative number")
    return x ** 0.5

square_root(-1) гӮ’е‘јгҒіеҮәгҒҷгҒЁ raise гҒ®иЎҢгҒ§еҒңжӯўгҒ—гҖҒй–ўж•°гҒӢгӮүйЈӣгҒіеҮәгҒ—гҒҰгҖҒе‘јгҒіеҮәгҒ—е…ғгҒ§йҒ©еҗҲгҒҷгӮӢ except гӮ’жҺўгҒ—гҒҫгҒҷгҖӮгҒ©гҒ®е‘јгҒіеҮәгҒ—е…ғгӮӮжҚ•жҚүгҒ—гҒӘгҒ‘гӮҢгҒ°гҖҒгӮ№гӮҜгғӘгғ—гғҲгҒҜгғҲгғ¬гғјгӮ№гғҗгғғгӮҜгҒЁгҒЁгӮӮгҒ«зөӮдәҶгҒ—гҒҫгҒҷгҖӮ

2.26.1. з•Әе…өгӮ’иҝ”гҒҷд»ЈгӮҸгӮҠгҒ«йҖҒеҮәгҒҷгӮӢзҗҶз”ұВ¶

гҖҢдёҚжӯЈгҒӘе…ҘеҠӣгҖҚгӮ’е ұе‘ҠгҒҷгӮӢ2гҒӨгҒ®ж–№жі•:

# signal with a sentinel
def square_root_or_none(x):
    if x < 0:
        return None
    return x ** 0.5

# raise an exception
def square_root(x):
    if x < 0:
        raise ValueError("...")
    return x ** 0.5

йҖҡеёёгҒҜдҫӢеӨ–гҒ®еҪўгҒ®ж–№гҒҢе„ӘгӮҢгҒҰгҒ„гҒҫгҒҷ:

  • е‘јгҒіеҮәгҒ—е…ғгҒҜгҖҒtry гӮ’дҪҝгҒҶгҒӢгҖҒдҫӢеӨ–гӮ’дјқж’ӯгҒ•гҒӣгӮӢгҒӢгҒ—гҒҰгҖҒгӮЁгғ©гғјгҒ®е ҙеҗҲгӮ’ж„Ҹеӣізҡ„гҒ«еҮҰзҗҶгҒ—гҒӘгҒ‘гӮҢгҒ°гҒӘгӮҠгҒҫгҒӣгӮ“гҖӮз•Әе…өгҒҜеҝҳгӮҢгӮ„гҒҷгҒҸгҖҒйҖҡеёёгҒ®зөҗжһңгҒЁеҸ–гӮҠйҒ•гҒҲгӮ„гҒҷгҒ„гӮӮгҒ®гҒ§гҒҷгҖӮ

  • гӮЁгғ©гғјгғЎгғғгӮ»гғјгӮёгҒҜдҫӢеӨ–гҒЁгҒЁгӮӮгҒ«дјқгӮҸгӮҠгҒҫгҒҷгҖӮз•Әе…өгҒ®жүӢжі•гҒ§гҒҜгҖҒиЁәж–ӯжғ…е ұгӮ’гҒ©гҒ“гҒӢеҲҘгҒ®е ҙжүҖгҒ«д»ҳгҒ‘гҒӘгҒ‘гӮҢгҒ°гҒӘгӮҠгҒҫгҒӣгӮ“гҖӮ

  • жңӘеҮҰзҗҶгҒ®дҫӢеӨ–гҒ«еҜҫгҒҷгӮӢгғҮгғ•гӮ©гғ«гғҲгҒ®еӢ•дҪңгҒҜгҖҒе•ҸйЎҢгҒ®гҒӮгӮӢе‘јгҒіеҮәгҒ—гӮ’жҢҮгҒ—зӨәгҒҷгғҲгғ¬гғјгӮ№гғҗгғғгӮҜгӮ’дјҙгҒЈгҒҹжҙҫжүӢгҒӘгӮҜгғ©гғғгӮ·гғҘгҒ§гҒҷгҖӮйқҷгҒӢгҒӘ None гҒ®иҝ”еҚҙгҒҜгҖҒеҫҢгҒ§иҰӢгҒӨгҒ‘гҒ«гҒҸгҒ„гғҗгӮ°гҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

з•Әе…өгӮ’дҪҝгҒҶгҒ®гҒҜгҖҒгҖҢиҰӢгҒӨгҒӢгӮүгҒӘгҒ„гҖҚгҒ“гҒЁгҒҢдҫӢеӨ–зҡ„гҒ§гҒӘгҒ„ж—Ҙеёёзҡ„гҒӘзөҗжһңгҒ§гҒӮгӮӢе ҙеҗҲгҒ гҒ‘гҒ«гҒ—гҒҫгҒ—гӮҮгҒҶгҖӮdict.get() гҒҢгӮӯгғјгҒҢеӯҳеңЁгҒ—гҒӘгҒ„гҒЁгҒҚгҒ« None гӮ’иҝ”гҒҷгҒ®гҒҜгҖҒгҒҫгҒ•гҒ«гғ«гғғгӮҜгӮўгғғгғ—гҒҢжҷӮгҖ…гғ’гғғгғҲгҒ—гҒӘгҒ„гҒ“гҒЁгҒҢжғіе®ҡгҒ•гӮҢгҒҰгҒ„гӮӢгҒӢгӮүгҒ§гҒҷгҖӮ

2.26.2. гӮ«гӮ№гӮҝгғ дҫӢеӨ–гӮҜгғ©гӮ№В¶

е‘јгҒіеҮәгҒ—е…ғгҒҢзө„гҒҝиҫјгҒҝгӮЁгғ©гғјгҒЁеҢәеҲҘгҒ—гҒҹгҒ„гҒӢгӮӮгҒ—гӮҢгҒӘгҒ„е•ҸйЎҢгӮ’йҖҒеҮәгҒҷгӮӢгҒ«гҒҜгҖҒException гҒ®гӮөгғ–гӮҜгғ©гӮ№гӮ’е®ҡзҫ©гҒ—гҒҫгҒҷ:

class ConfigError(Exception):
    pass

def load_config(path):
    try:
        f = open(path)
    except OSError as e:
        raise ConfigError("missing config file: " + path)

try:
    load_config("settings.json")
except ConfigError as e:
    print("startup failed:", e)

class гҒ®жң¬дҪ“гҒҢз©әгҒ§гӮӮе•ҸйЎҢгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮе‘јгҒіеҮәгҒ—е…ғгҒҜгӮҜгғ©гӮ№гҒ§жҚ•жҚүгҒҷгӮӢгҒҹгӮҒгҖҒйҮҚиҰҒгҒӘгҒ®гҒҜеҗҚеүҚгҒқгҒ®гӮӮгҒ®гҒ§гҒҷгҖӮе‘јгҒіеҮәгҒ—е…ғгҒҢгғ•гӮЎгғҹгғӘгғје…ЁдҪ“гӮ’1гҒӨгҒ®гғ–гғӯгғғгӮҜгҒ§жҚ•жҚүгҒ—гҒҹгҒ„е ҙеҗҲгҒҜгҖҒй–ўйҖЈгҒҷгӮӢгӮЁгғ©гғјгӮ’е…ұйҖҡгҒ®еҹәеә•гӮҜгғ©гӮ№гҒ®дёӢгҒ«гҒҫгҒЁгӮҒгҒҫгҒҷгҖӮ

2.26.2.1. еҶҚйҖҒеҮәВ¶

except гғ–гғӯгғғгӮҜеҶ…гҒ®иЈёгҒ® raise гҒҜгҖҒзҸҫеңЁгҒ®дҫӢеӨ–гӮ’еҶҚйҖҒеҮәгҒ—гҖҒж¬ЎгҒ®гғҸгғігғүгғ©гҒёдјқж’ӯгҒ•гҒӣгҒҫгҒҷ:

try:
    do_work()
except Exception as e:
    log(e)
    raise        # let it keep going

гҒ“гӮҢгҒҜгҖҒй–ўж•°гҒҢгӮЁгғ©гғјгӮ’е®ҹйҡӣгҒ«гҒҜеҮҰзҗҶгҒӣгҒҡгҒ«иҰіжё¬гҒ—гҒҹгҒ„пјҲгғӯгӮ°гҒ«иЁҳйҢІгҒҷгӮӢгҖҒгӮ«гӮҰгғігғҲгҒҷгӮӢгҖҒйғЁеҲҶзҡ„гҒӘеӨүжӣҙгӮ’еҸ–гӮҠж¶ҲгҒҷпјүе ҙеҗҲгҒ®жӯЈгҒ—гҒ„еҪўгҒ§гҒҷгҖӮ

2.26.3. жҚ•жҚүгҒҷгҒ№гҒҚе ҙеҗҲгҒЁдјқж’ӯгҒ•гҒӣгӮӢгҒ№гҒҚе ҙеҗҲВ¶

дҫҝеҲ©гҒӘзөҢйЁ“еүҮ:

  • ж„Ҹе‘ігҒ®гҒӮгӮӢеӣһеҫ©гҒҢгҒ§гҒҚгӮӢгғ¬гғҷгғ«гҒ§дҫӢеӨ–гӮ’жҚ•жҚүгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮгғҮгғ•гӮ©гғ«гғҲгҒ§зҪ®гҒҚжҸӣгҒҲгӮӢгҖҒеҶҚи©ҰиЎҢгҒҷгӮӢгҖҒдёҚжӯЈгҒӘе…ҘеҠӣгӮ’гӮ№гӮӯгғғгғ—гҒҷгӮӢгҖҒгҒӘгҒ©гҒ§гҒҷгҖӮ

  • гӮҜгғ©гғғгӮ·гғҘд»ҘеӨ–гҒ«жңүз”ЁгҒӘгҒ“гҒЁгҒҢдҪ•гӮӮгҒӘгҒ„е ҙеҗҲгӮ„гҖҒеӣһеҫ©гҒ®ж–№жі•гӮ’зҹҘгҒЈгҒҰгҒ„гӮӢгҒ®гҒҢдёҠгҒ®еұӨгҒ§гҒӮгӮӢе ҙеҗҲгҒҜгҖҒдҫӢеӨ–гӮ’дјқж’ӯгҒ•гҒӣгҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

е‘јгҒіеҮәгҒ—гӮ№гӮҝгғғгӮҜгҒ®йҖ”дёӯгҒ«гҒӮгӮӢй–ўж•°гҒҢгҒҷгҒ№гҒҰгҒ®гӮЁгғ©гғјгӮ’йЈІгҒҝиҫјгӮ“гҒ§йқҷгҒӢгҒ«иҝ”гҒҷгҒЁгҖҒеӨұж•—гҒҢиҝҪи·ЎдёҚиғҪгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮдҫӢеӨ–гҒҜгҖҒжң¬еҪ“гҒ«еҜҫеҮҰгҒ®иЁҲз”»гӮ’жҢҒгҒӨгӮігғјгғүгҒ«еҲ°йҒ”гҒҷгӮӢгҒҫгҒ§дјқгӮҸгҒӣгӮӢж–№гӮ’е„Әе…ҲгҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ
