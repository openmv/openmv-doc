4.5. ピクセルセル¶

カメラセンサーは、ピクセルごとに1つずつ並んだ感光性セルの二次元グリッドです。各セルは、光を電圧に変換する フォトダイオード を中心に構成された小さな電気回路であり、各フレームの終わりにその電圧は単一の数値のピクセル値へとデジタル化されます。

4.5.1. 回路¶

各セルの能動素子は フォトダイオード です。これはシリコン中の小さな感光性p-n接合です。逆バイアス下で、フォトダイオードは入ってくる光子が少しずつ放出できる小さな電荷の蓄えを保持します。

4.5.2. 露光サイクル¶

各セルはフレームごとに同じ4ステップのサイクルに従います。

プリチャージ。 サイクルは、リセットスイッチ RST を閉じてフォトダイオードを電源レールに接続し、その蓄積電圧を既知の基準まで持ち上げる短いリセットパルスで始まります。その後スイッチが開き、フォトダイオードは電荷の蓄えが満たされたリセット電圧の状態で隔離されたまま残されます。

露光。 露光ウィンドウの間、フォトダイオードは光を集めるために放置されます。吸収された各光子は、フォトダイオードの蓄積電荷をわずかに消費します。光は蓄積電荷を 消失 させます。シーンが明るいほど、フォトダイオードはより速く放電し、ウィンドウの終わりまでにその電圧はより低くなります。この合計の低下量がピクセルの信号です。

サンプル。 露光ウィンドウは、転送スイッチ TX への短いパルスで終了します。TX が閉じている間に、フォトダイオードの残りの電荷がノード B に配線された小さな保持コンデンサ C に放出されます。C 上の電圧が、これでピクセルの測定値を記録します。その後 TX が再び開き、C 上の値をロックして、C が読み出しアンプの順番を待つ間にフォトダイオードを次のフレームのためにリセットできるよう解放します。

読み出し。 読み出しアンプは C 上の電圧をADCに送り、ADCがそれを整数のカウントに変換します。これは通常、ピクセルあたり生で10～12ビットの精度（ハイエンドのセンサーでは14ビットの場合もあります）です。そのカウントが 生のピクセル値 です。パイプラインが画像に対して行う他のすべて -- 補正、デベイヤー、カラーグレーディング、フォーマット変換 -- は、セルごとに1つずつあるこの数値から始まります。

4.5.3. 飽和¶

フォトダイオードには、蓄えが完全に枯渇するまでに放出できる最大の電荷量があります。その点を超えると、ピクセルは 飽和 します。追加の光は記録される電圧に影響を与えず、シーンがどれだけ明るくなってもセルは最大値を読み出します。

フォトダイオードが飽和するまでに失える最大量が、その フルウェル容量 です。物理的に大きなピクセルはより多くの蓄積電荷を保持できるため、より高いフルウェル容量を持ちます。これが、より小さく多数のピクセルを持つセンサーが、解像度の低い同等品と比べて一般にダイナミックレンジが低い理由です。
