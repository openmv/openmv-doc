13.7. Roboflow¶

Roboflow дозволяє пройти шлях від необроблених відеозаписів до навченої, готової до виробництва моделі комп’ютерного зору без написання жодного рядка навчального коду — все в браузері, на їхніх серверах. OpenMV є одним із його цільових платформ розгортання: коли навчання завершується, одна кнопка завантажує ваги TFLite з цілочисельним квантуванням, готові для камери. Власна документація Roboflow детальніше описує кожен крок.
