v1.1.0
======

v1.1.0 improves SPI reliability.

Bug fixes
---------

- Increased the SPI chip-select-to-transfer delay so slower peripheral devices
  have enough time to get ready before a transfer begins, improving reliability
  on slower hardware.

Other changes
-------------

- Documentation updates.

Breaking API changes
--------------------

None.
