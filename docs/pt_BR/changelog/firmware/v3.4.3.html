v3.4.3¶

v3.4.3 é uma pequena versão de correção: ela corrige um longo atraso quando a codificação JPEG estoura e retorna de forma limpa quando HAL_JPEG_Encode é abortado. Não introduz nenhuma mudança incompatível na API — os scripts da v3.4.2 rodam sem alterações.

Destaques¶

  • Correção de JPEG — sem mais longos atrasos quando a codificação JPEG estoura.

  • Nenhuma mudança incompatível na API — refaça o flash e seus scripts da v3.4.2 rodam sem alterações.

Correções de bugs¶

  • Corrigido um longo atraso quando a codificação JPEG estoura e agora retorna quando HAL_JPEG_Encode é abortado.

Mudanças incompatíveis na API¶

v3.4.3 não introduz nenhuma mudança incompatível na API Python. Scripts que rodavam na v3.4.2 rodam sem alterações na v3.4.3 — basta refazer o flash do firmware.
