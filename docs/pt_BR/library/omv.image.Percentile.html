classe Percentile – objeto Percentile¶

O objeto percentile é uma attrtuple retornada por histogram.get_percentile(). Ele reporta o valor do bin em uma posição da CDF fornecida pelo usuário (por exemplo, 0.05 para o 5º percentil, 0.5 para a mediana, 0.95 para o 95º percentil) para cada canal do Histogram subjacente.

Para histogramas em escala de cinza, use value (os outros três campos são 0). Para histogramas RGB565, l_value / a_value / b_value fornecem o percentil para os canais LAB L, A e B respectivamente.

Os campos são acessíveis pelo nome do atributo (percentile.value) ou pelo índice (percentile[0]). O objeto não tem construtor público.

class image.percentile¶

Chame histogram.get_percentile() para criar este objeto.

value¶

Valor do bin em escala de cinza no percentil solicitado. Inteiro de 0 a 255. Índice [0].

l_value¶

Valor do bin do canal LAB L no percentil solicitado. Inteiro de 0 a 100. Índice [1].

a_value¶

Valor do bin do canal LAB A no percentil solicitado. Inteiro de -128 a 127. Índice [2].

b_value¶

Valor do bin do canal LAB B no percentil solicitado. Inteiro de -128 a 127. Índice [3].
