Lua API
=======

The CamScripter Lua API describes the CMUcam3 functions that you can use in your
Lua programs to control the actions of the CMUcam3.

Data Structures
---------------

* :doc:`framediff <luaapistructframediff>`
* :doc:`image <luaapistructimage>`
* :doc:`pixel <luaapistructpixel>`
* :doc:`tracker <luaapistructtracker>`
