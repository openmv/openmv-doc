Downloads
=========

Here you can find the latest stable release version of the CC3 code as local
copies of any other useful software packages. For documentation please go to our
:doc:`documentation` page.

* cmucam3_install_r556.zip -- CMUcam3 Install Software (194 MB) (not preserved
  in the archive)

  * This zip contains all of the files needed for Windows and Linux and includes
    the checked out source.
  * Windows: Cygwin-Installer (with packages), arm-gcc, Philips-Flash utility,
    .NET Runtime, CMUcam3 frame grabber
  * Linux: arm-gcc compiled tar, LPC-isp Flash Utility
  * Cygwin source can be obtained through installer program (setup.exe)
  * Use this in conjunction with the SDK Install Guide
  * *Last Updated: 12/29/2008*

* cc3_r556.zip -- CC3 Source Tree (2.1MB) (not preserved in the archive)

  * This is snapshot of the latest stable CC3 source.
  * Rev 556 from 12/29/2008

* CMUcam3 Frame Grabber.exe -- CMUcam3 Frame Grabber (60 KB) (not preserved in
  the archive)

  * .NET GUI for grabbing images and testing basic CMUcam2 Emulation
    Functionality
  * For non-english Windows, try the International Frame Grabber (see
    :doc:`newtools`)

* lpc2106_isp_aug_03.zip -- LPC Flash Utility Installer (2.0 MB) (not preserved
  in the archive)

  * Windows flashing utility that allow you to download images to your CMUcam3

* cygwin-install.zip -- Local Cygwin Installer (23.7 MB) (not preserved in the
  archive)

  * Cygwin is the environment that allows you to do CMUcam3 development under
    windows

* lpc21isp_unix.zip -- Linux LPC-ISP Installer (25.9 KB) (not preserved in the
  archive)

  * Linux flashing utility

* :download:`cmucam2_lpc2106-cmucam3.hex <dl/cmucam2_lpc2106-cmucam3.hex>` --
  CMUcam2 Emulation HEX file (264.2 KB)

  * Burn this image onto your CMUcam3 to emulate the CMUcam2

* `Windows ARM-gcc Installer <http://www.codesourcery.com/gnu_toolchains/arm/portal/package2557/public/arm-none-eabi/arm-2008q1-126-arm-none-eabi.exe>`__

  * Windows Code Sourcery ARM gcc compiler installer
  * Release 323 2008 q1

* `Linux ARM-gcc Installer <http://www.codesourcery.com/gnu_toolchains/arm/portal/package2556/public/arm-none-eabi/arm-2008q1-126-arm-none-eabi.bin>`__

  * Linux Code Sourcery ARM gcc compiler executables
  * Release 323 2008 q1

* lua-win-nofpu.zip (not preserved in the archive)

  * Luac compiled for Windows without floating point support

* `CMUcam2 GUI <http://www.cs.cmu.edu/~cmucam2/downloads.html>`__

  * If you are running in CMUcam2 emulation, this java gui can be used to test
    the camera
