International CMUcam3 Frame Grabbing Tool
=========================================

The original CMUcam3 Frame Grabbing tool has trouble working on Windows machines
with non-english fonts. This is a new version of the software that has been
enhanced to run on all windows computers. In order to test this software, first
install the modified CMUcam2 emulation on your CMUcam3. Next download and un-rar
the PC side application. You should be able to execute the software by double
clicking on "cmucam location.exe".

* ``cmucam location.rar`` is the PC side application (not preserved in the
  archive)

  * Please provide feedback on the
    `forum <http://www.cmucam.org/discussion/1/202/524#524>`__
  * Source Code Coming Soon...
  * *Last Updated: 5/11/2007*

* ``cmucam3.2.rar`` is a modified CMUcam2 emulation firmware to be executed on
  the CMUcam3 (not preserved in the archive)
