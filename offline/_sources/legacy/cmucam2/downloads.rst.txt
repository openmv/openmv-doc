PC Based GUI Interface
----------------------

The GUI is written in Java and is compatible with Windows, Linux, and Mac OS:

* :download:`CMUcam2GUI <dl/CMUcam2GUI.zip>` v1.0.5

It is also mostly backward compatible with the CMUcam1.

Firmware Images
---------------

The CMUcam2 comes with the firmware pre-programmed and does not have to be
re-programmed under normal operation.

For release notes on each version please see the :doc:`faq` page.

The files below are provided for existing CMUcam2 owners who have access to an
`SX-Key <https://www.parallax.com/package/sx-key-usb-downloads/>`__ programmer and may
wish to update the firmware themselves:

* :download:`CMUcam2 v1.00 <dl/CMUcam2.v1.00.hex>`
* :download:`CMUcam2 v1.01 <dl/CMUcam2.v1.01.hex>`
